# 端口要求

 

_**上一次修改主题：** 2013-03-27_

Lync Server 要求打开防火墙上的特定端口。此外，如果在您的组织中部署了 Internet 协议安全性 (IPsec)，则必须在用于传送音频、视频和全景视频的端口范围内禁用 IPSec。
