# 删除 Lync for Windows Phone

 

_**上一次修改主题：** 2013-02-19_

若要从移动设备中删除 Lync 2013 for Windows Phone 应用程序，请执行以下步骤：

1.  在移动设备上，从开始屏幕上滑动手指以查看应用程序列表。

2.  按住 Lync 应用程序，然后选择“卸载”。
