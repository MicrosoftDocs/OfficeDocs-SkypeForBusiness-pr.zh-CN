# 其他服务器角色 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

这些 cmdlet 提供了管理未纳入服务器角色类别（如边缘服务器或通讯簿服务器）的“一次性”服务器角色（如会议服务器）的方法。

## 其他服务器角色和 Cmdlet

以下是与管理其他服务器角色和服务直接相关的 cmdlet 列表：

**其他服务器角色**

  -   
    [Set-CsConferenceServer](set-csconferenceserver.md)

  -   
    [Set-CsUserServer](set-csuserserver.md)
