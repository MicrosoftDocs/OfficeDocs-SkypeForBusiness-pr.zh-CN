# Lync Server 2013 中呼叫允许控制的基础结构要求

 

_**上一次修改主题：** 2012-08-21_

不需要其他基础结构要求，如特殊的网络路由器。部署 Lync Server 2013 时将自动安装用于强制实施呼叫允许控制 (CAC) 策略的带宽策略服务。呼叫允许控制只适用于运行 Lync 的客户端。
