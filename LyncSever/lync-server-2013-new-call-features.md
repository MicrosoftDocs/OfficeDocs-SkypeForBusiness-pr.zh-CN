# Lync Server 2013 中新的呼叫功能

 

_**上一次修改主题：** 2012-10-10_

Lync Server 2013 为呼叫转移和同时响铃提供大量的配置选项。例如，如果某个组织不想将传入呼叫向外转移到公用电话交换网 (PSTN)，则管理员可应用特殊语音策略来部署此限制。

另外，现在代理人可为向经理的传入呼叫将同时响铃设置到其移动设备。这为代理人提供了更大的灵活性，使其可代表其经理应答呼叫，而不必绑定到桌面电话。
