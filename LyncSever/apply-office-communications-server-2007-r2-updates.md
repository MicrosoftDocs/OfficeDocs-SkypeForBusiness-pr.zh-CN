# 应用 Office Communications Server 2007 R2 更新

 

_**上一次修改主题：** 2016-12-16_

在迁移到 Lync Server 2013 之前，必须对 Office Communications Server 2007 R2 环境应用更新。有关 Office Communications Server 2007 R2 的最新信息，请参阅 **Communications Server 2007 R2** 的更新，网址为 [http://go.microsoft.com/fwlink/?linkid=3052\&clcid=0x804](http://go.microsoft.com/fwlink/?linkid=3052%26clcid=0x804)。

要安装 Office Communications Server 2007 R2 的更新，建议按照 Microsoft 知识库文章 968802“Communications Server 2007 R2 的更新”中介绍的 **方法 1 服务器累积更新安装程序**过程操作，网址为 [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)。
