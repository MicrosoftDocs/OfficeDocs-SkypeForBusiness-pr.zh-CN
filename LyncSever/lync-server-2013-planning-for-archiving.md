# 在 Lync Server 2013 中规划存档

 

_**上一次修改主题：** 2012-09-28_

公司和其他组织需要服从数量不断增多的行业规定和政府法规，这些规定和法规要求保留特定类型的通信。如果您的组织具有此类要求，则可使用 Lync Server 2013 中的存档功能存档通过 Lync Server 发送的即时消息 (IM) 和会议通信，从而帮助支持部分合规性要求。
