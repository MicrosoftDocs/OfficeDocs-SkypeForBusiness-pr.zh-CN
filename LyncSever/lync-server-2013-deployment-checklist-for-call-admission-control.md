# Lync Server 2013 中呼叫允许控制的部署清单

 

_**上一次修改主题：** 2016-12-08_

若要有效规划呼叫允许控制 (CAC)，则需要考虑以下内容：

  - 部署 CAC 的先决条件。

  - CAC 所需的信息以及在部署前必须做出的配置决策。

## 呼叫允许控制的部署先决条件

部署呼叫允许控制之前，必须已部署 Lync Server 2013 内部服务器，包括 前端池或 Standard Edition Server。
