# Outlook 网页版中的 Skype for Business Online 支持

> [!IMPORTANT]
> 本文是由机器翻译的，请参阅[免责声明](305984ec-3da8-4509-bb2b-6643dcf2cb7d.md#MT_Footer)。请在 [此处](https://support.office.com/en-us/article/305984ec-3da8-4509-bb2b-6643dcf2cb7d) 中查找本文的英文版本以便参考。
  
Outlook 网页版 (Outlook Web App) 在 Office 365 中提供了从导航栏中的基本 Skype for Business web 客户端。此基本客户端仅供其管理员未配置为其 Office 365 的组织的意图 URL 的联机用户。只要用户的帐户处于联机状态，并且不具有意图的 URL，它们仍会看到体验，即使他们的组织具有一些迁移的本地用户帐户。用户帐户本地 （是否具有意图 URL 或不） 或由 Microsoft 管理用户将看到 Outlook web app 中的 Lync 体验。
  
下表总结了您可能必须不同设置和使用 web 客户端。
  
||||
|:-----|:-----|:-----|
|**已定位用户帐户** <br/> |**已配置虚 URL 或存在专用组织** <br/> |**Skype for Business 还是 Lync 体验？** <br/> |
|Online  <br/> |否  <br/> |Skype for Business Web 体验  <br/> |
|Online  <br/> |是  <br/> |Lync Web 体验  <br/> |
|混合但联机托管  <br/> |否  <br/> |Skype for Business Web 体验  <br/> |
|混合但联机托管  <br/> |是  <br/> |Lync Web 体验  <br/> |
|混合但本地托管  <br/> |否  <br/> |Lync Web 体验  <br/> |
|混合但本地托管  <br/> |是  <br/> |Lync Web 体验  <br/> |
|纯 prem 上  <br/> |否  <br/> |Lync Web 体验  <br/> |
|纯 prem 上  <br/> |是  <br/> |Lync Web 体验  <br/> |
