# <a name="user-statistics-and-organization-information"></a>用户统计信息和组织信息

业务仪表板 Skype 提供快速的服务为您的业务的设置了快照。 如果还没有配置业务的 Skype，请参阅[设置 Skype 的在线业务](set-up-skype-for-business-online.md)的分步说明。
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>如何更改 Skype 的在线业务的用户统计信息

|||
|:-----|:-----|
|**属性** <br/> |**如何更改它** <br/> |
|Skype 的在线业务用户  <br/> |要添加或阻止某些人使用 Skype 的业务应用程序，请参阅[分配或删除业务的 Office 365 的许可证](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc)或[删除用户业务的 Office 365 中的许可证](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1)。  <br/> 当用户离开您的业务操作的完整列表，请参阅[删除 Office 365 中的前员工](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1)。  <br/> |
|用户启用了拨入会议  <br/> |为了使会议拨入的用户，请参阅[设置用于商业和 Microsoft 小组 Skype 的音频会议](../audio-conferencing-in-office-365/set-up-audio-conferencing.md)。  <br/> |
|在云环境中的用户  <br/> |这是 Skype 的 Office 365 帐户业务联机用户数。  <br/> 若要将联系人添加到您的 Office 365 提供订阅 （即为其创建 Office 365 帐户），请参阅[添加用户单独或批量到 Office 365-管理帮助](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)。  <br/> |
|用户同步和在线穴  <br/> |这是为从内部部署 Active Directory 同步业务联机用户 Skype 的数目。  <br/> 若要设置内部活动目录，请参阅[设置 Office 365 的目录同步](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)。  <br/> |
   
> [!NOTE]
> 如果您看到列出不可用，这意味着没有任何信息或出现错误。
