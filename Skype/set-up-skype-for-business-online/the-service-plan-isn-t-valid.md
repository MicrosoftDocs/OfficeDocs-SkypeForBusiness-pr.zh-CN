# <a name="the-users-skype-for-business-online-service-plan-isnt-valid"></a>无效用户的 Skype 的在线业务支持服务计划

很抱歉，Skype 业务联机服务计划分配给此用户无法识别或无效。
  
请转到**Office 365 管理中心** > **活动用户**和分配有效服务计划给该用户。
