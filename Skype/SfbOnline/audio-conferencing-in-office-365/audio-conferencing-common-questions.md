# <a name="audio-conferencing-common-questions"></a>音频会议常见问题

下面是一些最重要的问题我们让我们的客户想要使用音频会议。 
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>音频会议的好处有哪些？

会议呼叫会非常有用，当人在路上，例如，不能参加会议的业务或 Microsoft 小组其便携式计算机或移动设备上的应用程序使用 Skype。 但有其他方案中的使用电话业务或 Microsoft 小组参加 Skype 会议也可能是更好的选择，比在一台计算机上使用的应用程序：
  
- Internet 连接受限。
    
- 会议只有音频。
    
- 用户尝试加入 Skype for Business 会议，但是失败。
    
- 如果他们拨入呼叫质量要好用。
    
- 用户可以使用蓝牙设备采用"免提"方式加入会议。
    
- 人们找到了更容易，更方便他们的具体情况。
    
## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>谁可以参加会议的音频会议？ 和我可以听到谁？

拨入号码和会议 ID 的任何人可以加入 Skype 业务或 Microsoft 小组的会议，除非会议组织者已锁定会议。
  
是否要在业务或 Microsoft 团队的应用程序中使用的电话或 Skype 呼叫，您将能够听到其他调用时，所有人和他们可以听到您的声音。 会议组织者有"静音"会议与会者如果他们不想听的能力。 
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>可以为我的音频会议用户添加免费电话号码？

是的免费电话号码 （服务编号） 是可用的但只在某些国家/地区。 若要查看可用的数字列表转[此处](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>目前支持多少个本地拨入号码？

没有分配给您时您购买的许可证的音频会议的当地拨入号码。 拨入号码将包含在会议邀请之中。 这些本地号码仅可供你的组织使用。 分配给您的组织的电话，该号码由启用音频会议的该组织内的用户共享。 因此，Skype 用户 A 和用户 B 另一个由计划的业务或 Microsoft 小组会议都为同一拨入号码。
  
本地拨入号码以及在某些情况下你的组织所在的国家或地区的国际拨入号码，将包含在会议邀请中。如果一名与会者使用了邀请中包含的不同号码，它将成为共享的电话号码。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-office-365-support"></a>Office 365 中的音频会议支持多少国际拨入号码？

当前国家/地区的列表，请参阅[音频电话会议的电话号码](phone-numbers-for-audio-conferencing.md)。
  
## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>可以设置本地电话号码的音频会议从全国其他城市？

如果音频电话会议的电话号码在您所在地区不可用或不能满足您的组织的需要，请向我们发送反馈在[SkypeFeedback 论坛](http://www.skypefeedback.com/forums/299910--preview/category/119971-pstn-conferencing)。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>音频会议的会议的最大长度是什么？

最长持续时间取决于与会者以及他们用于加入会议的身份验证类型。
  
|**与会者**|**会议结束时间**|
|:-----|:-----|
|有业务或 Microsoft 小组应用程序使用 Skype 已加入或已拨号加入会议的用户。  <br/> |如果 24 小时后与会者列表没有任何更改，会议结束。  <br/> |
|所有用户都是拨号加入会议，但是部分用户使用了 PIN 进入会议。  <br/> |24 小时后会议结束。  <br/> |
|所有用户将拨入会议但没有使用 PIN 输入会议的任何人。  <br/> |4 小时后会议结束。  <br/> |
   
## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>在会议中可以有多少总电话参与者？

音频会议允许最多 250 个电话与会者。
  
若要了解会议限制，请参阅 [Skype for Business Online 限制](https://technet.microsoft.com/en-us/library/skype-for-business-online-limits.aspx#bkmk_Meeting_LyncOnlineLimits)。

## <a name="can-audio-conferencing-be-used-by-the-users-who-are-part-of-an-on-premises-deployment-of-skype-for-business-server-2015"></a>音频会议可供属于内部部署的业务服务器 2015年的 Skype 的用户？

## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>可以使用 Skype 会议广播使用音频会议？

没有当前为用户启用音频会议加入 Skype 会议广播的支持。

## <a name="how-does-a-user-access-or-change-their-conference-id"></a>用户如何访问或更改他们的会议 ID？

当用户被指派**音频会议**许可证和用户在 Outlook 或在 web 上的 Outlook 中创建新 Skype 业务或 Microsoft 小组会议时，拨入电话号码并会议 Id 会自动添加到会议邀请。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>当所有与会者将使用电话拨入时，用户如何安排和启动会议？

计划将通过使用电话拨入的所有与会者加入会议并不不同于计划正则的联机会议。 但是，有两种方法启动的会议的所有参与者使用电话拨号：

- **选项 #1**： 默认情况下，如果会议组织者和所有参与者加入会议使用电话，会议组织者需要输入他 / 她的音频会议插针来启动它。 调用方被问及他们是否要拨的电话号码的联机会议时被认定为给定的会议的组织者。 保存前组织者开始将放置在大厅和将听音乐上加入会议通过拨入的所有参与者。 一旦管理器启动外接程序通过输入他 / 她的音频会议针，会议厅中的所有参与者将自动都加入会议。

- **选项 2**： 如果"允许未经身份验证的调用方将在会议中的第一人"（默认情况下禁用） 启用了对于给定的组织者，然后安排由该用户的所有会议都将能够输入管理器就不都能启动或她音频会议针。 启用此设置后，只要通过拨入电话号码的第一个参与者加入它，并且他或她不会放在大厅将启动会议。 有关其他信息，请参阅[管理音频会议的用户设置](manage-the-audio-conferencing-settings-for-a-user.md)
