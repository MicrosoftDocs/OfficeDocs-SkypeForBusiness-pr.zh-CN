# <a name="change-the-emergency-location-for-a-user"></a>更改用户的紧急位置

每个活动电话号码必须有紧急地址 （当您在 Office 365 或电话号码的转移时获得电话号码相关联） 当电话号码分配给用户。 时数与紧急通讯，还可以提供更精确的物理位置位于紧急位置。 紧急位置可以是用户所在的楼层、侧楼或办公室编号。 可以有无限的多个位置为给定的紧急地址，并可以更改紧急位置，如果用户移动到不同的办公室或建筑 — — 例如，如果用户将从地面到地板 35 34。
  
若要了解如何获取 Office 365 中的调用计划以及这些成本是多少，请参阅[附加业务和 Microsoft 小组授权的 Skype](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md)。
  
## <a name="change-the-emergency-location-for-a-user"></a>更改用户的紧急位置

1. 使用你的工作或学校帐户登录 Office 365。
    
2. Go to the **Office 365 admin center** > **Skype for Business**.
    
3. 在左侧导航转到**语音** > **语音用户**。
    
    > [!IMPORTANT]
    > 要查看业务管理中心为 Skype 在左边的导航中的**语音**选项，您必须先购买至少一个**企业 E5 许可证**、**电话系统**加载项许可证或一个**音频会议**附加许可证。
    
4. 在**语音用户**页面上，找到并选择您想要更改的紧急位置的用户。
    
5. 在操作窗格中，在" **紧急位置**"下，单击" **更改**"。
    
6. 在" **分配号码**"页面上，单击" **更改位置**"。 
    
7. 在**更改紧急地址**下的框中输入城市名称，单击**搜索**。
    
8. 从列表中选择紧急位置，然后单击**保存**。
    
    如果您想要添加一个新的紧急位置，请参阅[添加、 更改或删除您的组织的紧急位置](add-change-or-remove-an-emergency-location-for-your-organization.md)。
