# <a name="the-users-skype-for-business-online-service-plan-isnt-valid"></a>业务联机服务计划的用户的 Skype 无效

很抱歉，分配给该用户的业务联机服务计划 Skype 无法识别，或无效。
  
转到**Office 365 管理中心** > **活动用户**和分配一个有效的服务计划至该用户。
