# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>未启用此 Skype for Business Online 功能

抱歉，该用户似乎未获得此功能的许可。
  
|**可能的原因**|**建议的操作**|
|:-----|:-----|
|您有可用的许可证，但尚未启用此用户的功能。  <br/> |转到**Microsoft 365 管理中心** > **活动用户**以授权此用户。 <br/> |
|用户所在的位置无法使用此功能。  <br/> |如果可能，请转到**Microsoft 365 管理中心** > **活动用户**并将此用户分配到该功能可用的位置。 <br/> |
|尚未为你的组织启用该功能。  <br/> |转到**Microsoft 365 管理中心** > **Skype for** business，并为您的整个组织启用该功能。 <br/> |
