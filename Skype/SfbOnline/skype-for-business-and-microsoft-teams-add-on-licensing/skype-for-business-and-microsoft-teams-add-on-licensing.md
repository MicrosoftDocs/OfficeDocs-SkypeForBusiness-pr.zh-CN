# <a name="skype-for-business-and-microsoft-teams-add-on-licensing"></a>Skype 的附加业务和 Microsoft 小组授权

## <a name="what-are-add-on-licenses"></a>什么是附加许可证？

附加许可证是为特定的业务和 Microsoft 小组功能 Skype 的许可证。 某些企业希望以有竞争力的价格灵活地购买特定功能。 若要添加功能，购买每个用户都将使用它的一个附加许可证。

Skype 业务和授权的 Microsoft 小组旨在为您提供最大的灵活程度。 例如，在使用一段时间的业务功能基本 Skype 之后, 可以购买附加许可时您就可以在 Office 365 中使用更多的功能，如音频会议或电话系统。

## <a name="how-does-add-on-licensing-affect-cost"></a>加载项授权如何影响成本？

对于大多数企业，购买企业版计划中的一系列软件将降低总成本。 通常情况下，作为计划的一部分购买而不是分别的几个附加许可时，您可能会具有较高的综合成本。

成本的详细信息，请参阅[如何看到价格和购买附加许可证](skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_how)或下列项之一：
  
- [音频会议的定价](https://products.office.com/en-us/skype-for-business/audio-conferencing) 
    
- [电话系统的定价](https://products.office.com/en-us/skype-for-business/phone-system)
    
- [定价的通话方案](https://products.office.com/en-us/skype-for-business/calling-plans)
