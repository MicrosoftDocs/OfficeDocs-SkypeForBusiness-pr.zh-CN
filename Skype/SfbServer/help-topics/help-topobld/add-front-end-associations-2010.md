# <a name="add-front-end-associations-2010"></a>添加前端关联 2010

此时，通过将服务器角色与前端池关联，可以启用对需要部署其他服务器的特定功能的支持。 还可以稍后将服务器角色与前端池关联。 可以与前端池关联的服务器角色包括：

1. 存档服务器。 有关规划和实施存档支持的详细信息，请参阅规划文档中的[Defining Your Organization's Requirements for Archiving](/previous-versions/office/lync-server-2013/lync-server-2013-defining-your-requirements-for-archiving)以及部署文档中的[Define and Configure a Front End Pool or Standard Edition Server](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server)和[Deploying Archiving](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-archiving)。

2. 监控服务器。 有关实施监控支持的详细信息，请参阅规划文档中的[Planning for Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring)以及部署文档中的[Define and Configure a Front End Pool or Standard Edition Server](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server) 和[Deploying Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring)。

3. A/V 边缘服务。 有关实现 A/V 边缘服务的详细信息，请参阅 [Overview of External User Access](/previous-versions/office/lync-server-2013/lync-server-2013-overview-of-external-user-access)。
