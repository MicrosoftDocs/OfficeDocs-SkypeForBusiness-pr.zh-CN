# <a name="add-front-end-associations-2010"></a>添加前端关联 2010

你可以启用对需要部署其他服务器的特定功能的支持, 方法是将服务器角色与前端池关联。 还可以选择以后将服务器角色与前端池相关联。 可与前端池关联的服务器角色包括：

1. 存档服务器。 有关规划和实现存档支持的详细信息, 请参阅在计划文档中[定义组织的存档要求](https://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx)和[定义和配置前端池或标准版服务器](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)以及[部署](https://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx)部署文档中的存档。

2. 监视服务器。 有关实现监视支持的详细信息, 请参阅规划文档中[的监视规划](https://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)和[定义和配置前端池或标准版服务器](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)以及在部署中[部署监视](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)文档.

3. A/V 边缘服务。 有关实现 A/V 边缘服务的详细信息, 请参阅[外部用户访问概述](https://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx)。
