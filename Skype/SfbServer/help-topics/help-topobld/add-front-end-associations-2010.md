# <a name="add-front-end-associations-2010"></a>添加前端关联 2010
 
您可以启用对通过这一次与前端池关联的服务器角色需要其他服务器的部署的特定功能的支持。 还可以选择以后将服务器角色与前端池相关联。 可与前端池关联的服务器角色包括：
  
1. 存档服务器。 有关计划和实现存档支持，详见[存档定义您组织的需求](http://technet.microsoft.com/library/ce0fc0f6-7704-4b80-bf19-a1fa9818fc7a.aspx)规划文档，并[定义和配置前结束池或标准版服务器](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)和[中部署归档](http://technet.microsoft.com/library/a89edd16-12d5-4602-ad2f-194b47d1188e.aspx)部署文档中。
    
2. 监视服务器。 有关实施监视支持的详细信息，请参阅 》 中的规划文档，[定义和配置前结束池或标准版服务器](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)和部署中[部署监视](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)[监视规划](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)文档。
    
3. A / V 边缘服务。 有关实施 A / V 边缘服务，请参阅[概述外部用户访问](http://technet.microsoft.com/library/97aded6c-5fa3-4225-95a6-9ad094d61654.aspx)。
