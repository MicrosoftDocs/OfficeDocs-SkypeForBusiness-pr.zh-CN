# <a name="legacy-merge-landing"></a>旧版合并登陆
 
“指定边缘设置”页允许配置旧版边缘服务器信息。单击“添加”可指定 Office Communications Server 部署的旧版边缘服务器版本。如果旧版环境中没有任何边缘服务器，请单击“下一步”继续。
