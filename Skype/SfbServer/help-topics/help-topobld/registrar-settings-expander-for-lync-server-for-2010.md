# <a name="registrar-settings-expander-for-lync-server-for-2010"></a>Lync Server 2010 的注册设置扩展器
 
编辑**的自我恢复能力**的设置并配置以下属性：
  
- 从列表中选择**关联备份注册器池**。
    
    或者，选择**自动故障切换和回切的声音**复选框。
    
    配置**语音故障检测时间间隔 （秒）**和**语音故障回复间隔 （秒）**。 默认情况下，时间间隔是 120 秒的语音故障检测和故障回复语音 240 秒。
    
    > [!CAUTION]
    > 应仔细测试定义的故障转移和故障回复的间隔的秒数，以确保恢复正常工作。 将间隔设置为较低 （即小于 120 秒） 或故障转移和故障回复设置过于相似可能会导致实际的故障转移和故障自动恢复，未达到预期效果。 
  
 **确定** 接受更改并通过对话框提交更改。
  
 **取消** 放弃更改并关闭对话框。
  
 **帮助** 显示此帮助屏幕。
