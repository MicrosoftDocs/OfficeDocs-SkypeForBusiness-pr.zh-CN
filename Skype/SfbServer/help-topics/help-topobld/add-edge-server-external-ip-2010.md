# <a name="add-edge-server-external-ip-2010"></a>添加边缘服务器的外部 IP 2010
 
可以在此页后，可以指向的访问边缘服务、 Web 会议边缘服务和一个外部位置的边缘服务器上配置属性 / V 边缘服务。
  
- 在**外部的 IPv4 地址**，键入 IPv4 地址的外部边缘服务器在**访问边缘服务**中， **Web 会议边缘服务**，和**A / V 边缘服务**。
    
- 单击**确定**以保存并提交到拓扑文档所做的更改。
    
- 单击**取消**放弃您的更改并关闭**到边缘池中添加服务器**向导。
    
- 请单击**帮助**来阅读此帮助主题。
    
将边缘服务器添加到一个现有的拓扑结构的详细信息，请参阅[定义边拓扑](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)。 有关迁移的边缘服务器的详细信息，请参阅[阶段 5： 添加 Lync 2013 预览边缘服务器到试验池](http://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)。
