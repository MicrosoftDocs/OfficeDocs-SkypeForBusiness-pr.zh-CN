# <a name="remove-deployment-introduction"></a>删除部署简介
 
仅当要停用整个部署时才运行“删除部署向导”。该向导创建由单个前端服务器组成的空拓扑。运行该向导并发布空拓扑后，就无法还原到原始拓扑。
  
> [!IMPORTANT]
> 单击“下一步”之前，请确保删除任何非活动计算机、禁用统一通信中的所有用户和终结点并删除向导首页中指定的所有会议目录。
