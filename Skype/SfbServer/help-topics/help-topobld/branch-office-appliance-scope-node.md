# <a name="branch-office-appliance-scope-node"></a>分支机构设备范围节点
 
此节点显示了完全限定的域名 (FQDN) 、关联的池和服务器，以及有关此 Survivable 分支设备的恢复信息。 可以执行以下操作：
  
- **编辑属性**。 打开此 Survivable 分支设备的 " **编辑属性** " 页，在其中可以更改 FQDN (如果拓扑尚未发布) 、关联和复原设置。
    
- **删除**。从拓扑中删除服务器。
    
除了这些选项之外，还可以执行以下常规任务：
  
- **拓扑**。 您可以使用此选项来定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑的副本、发布拓扑、安装数据库或删除部署。
    
- **视图**。可以使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
