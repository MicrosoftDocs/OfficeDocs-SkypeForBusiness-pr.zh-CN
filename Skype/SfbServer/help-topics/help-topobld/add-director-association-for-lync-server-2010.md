# <a name="add-director-association-for-lync-server-2010"></a>添加 Lync Server 2010 的控制器关联
 
定义控制器池将是否有监控服务器与其关联。 选中复选框以监控服务器相关联的控制器或控制器池。
  
> [!TIP]
> 如果尚未创建监控服务器数据库 （和可选实例），会授予机会以后创建，如果您选中此复选框，监控服务器相关联的控制器或控制器池。
