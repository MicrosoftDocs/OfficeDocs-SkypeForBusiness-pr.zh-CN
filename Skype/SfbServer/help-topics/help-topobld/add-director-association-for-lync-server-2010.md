# <a name="add-director-association-for-lync-server-2010"></a>添加 Lync Server 2010 中的控制器关联
 
定义是否主任池将具有监视服务器与之关联。 选中复选框，若要监视的服务器相关联的主管或主管池。
  
> [!TIP]
> 如果尚未创建监视服务器数据库 （和可选的实例），您将有机会如果您选中此复选框，若要监视的服务器相关联的主管或主管稍后创建池。
