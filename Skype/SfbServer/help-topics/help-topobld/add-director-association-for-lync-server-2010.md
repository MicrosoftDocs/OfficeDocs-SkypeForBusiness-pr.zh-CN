# <a name="add-director-association-for-lync-server-2010"></a>为 Lync Server 2010 添加 Director 关联
 
定义 Director 或控制器池是否将具有与其关联的监视服务器。 选中复选框以将监视服务器与 Director 或控制器池相关联。
  
> [!TIP]
> 如果尚未创建监视服务器数据库（和可选实例），则如果选中此复选框以将监视服务器与 Director 或控制器池关联，则你将有机会在以后创建它。
