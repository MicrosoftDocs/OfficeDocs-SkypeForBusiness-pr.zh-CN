# <a name="add-director-association-for-lync-server-2010"></a>为 Lync Server 2010 添加控制器关联
 
您可以定义控制器或控制器池是否将具有与其关联的监视服务器。 选中此复选框可将监视服务器与控制器或控制器池关联。
  
> [!TIP]
> 如果尚未创建监视服务器数据库 (和可选实例) ，则在以后选中此复选框以将监控服务器与控制器或控制器池关联时，您将有机会创建该数据库。
