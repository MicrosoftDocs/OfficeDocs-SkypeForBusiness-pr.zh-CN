# <a name="add-director-association-for-lync-server-2010"></a>为 Lync Server 2010 添加控制器关联
 
定义控制器或控制器池是否将具有与之关联的监控服务器。 选中此复选框可以将监控服务器与控制器或控制器池关联。
  
> [!TIP]
> 如果尚未创建监控服务器数据库 (和可选实例) ，则以后如果选中此复选框以将监控服务器与控制器或控制器池关联，您将有机会创建它。
