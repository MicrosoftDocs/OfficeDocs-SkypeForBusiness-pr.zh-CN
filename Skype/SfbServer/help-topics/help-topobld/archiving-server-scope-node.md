# <a name="archiving-server-scope-node"></a>存档服务器范围节点
 
在拓扑生成器中，可以通过右键单击控制台树中的池并选择 " **编辑属性**" 来执行与运行存档的服务器相关的任务。 在 " **编辑属性** " 页中，您可以更改运行存档的服务器的完全限定的域名 (FQDN) 、存档 SQL server 数据库和存档文件存储。
