# <a name="archiving-server-scope-node"></a>存档服务器范围节点
 
在拓扑生成器中，通过右键单击控制台树中的池，然后选择 "**编辑属性**"，可以执行与运行存档的服务器相关的任务。 从“**编辑属性**”页面，可以更改存档服务器、存档 SQL Server 数据库和存档文件存储的完全限定域名 (FQDN)。
