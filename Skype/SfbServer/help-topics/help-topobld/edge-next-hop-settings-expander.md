# <a name="edge-next-hop-settings-expander"></a>边缘下一跃点设置扩展器

若要设置或修改边缘服务器用于与内部基础结构通信的"下一个跃点池"，请从列表中选择控制器、控制器池、前端服务器或前端服务器池。 列表中只显示已在拓扑生成器中配置的控制器或前端服务器。

有关定义和配置边缘服务设置的详细信息，请参阅[Define Your Edge Topology](/previous-versions/office/lync-server-2013/lync-server-2013-define-your-edge-topology)。
