# <a name="edge-next-hop-settings-expander"></a>边缘下一跃点设置扩展器

若要设置或修改边缘服务器将用于与内部基础结构通信的“**下一个跃点池**”，请从列表中选择控制器、控制器池、前端服务器或前端服务器池。 只有控制器或前端服务器已在拓扑生成器中配置显示在列表中。

有关定义和配置边缘服务设置的详细信息，请参阅[Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)。
