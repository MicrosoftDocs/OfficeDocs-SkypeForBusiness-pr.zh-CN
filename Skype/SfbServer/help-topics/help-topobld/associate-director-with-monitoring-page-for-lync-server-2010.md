# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>将控制器与 Lync Server 2010 监控页面关联
 
配置监控服务器并将其与控制器或控制器池关联：
  
在 **"监控服务器** "下，从列表中选择监控服务器。
  
如果没有监控服务器，并且要立即创建监控服务器，请单击"新建 **"。**
  
 **返回**：返回到之前的对话框
  
 **完成**：应用更改并完成向导
  
 **取消**：放弃更改并取消配置
  
 **帮助**：显示帮助信息，例如本主题
