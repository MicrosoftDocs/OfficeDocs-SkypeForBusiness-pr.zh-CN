# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>将控制器与 Lync Server 2010 监控页面关联
 
若要配置监视服务器并将其与 Director 或控制器池相关联, 请执行以下操作:
  
在 "**监视服务器**" 下, 从 "监视服务器" 列表中选择。
  
如果没有监视服务器, 而希望立即创建监视服务器, 请单击 "**新建**"。
  
 **后退**: 返回到上一个对话框
  
 **完成**: 提交更改并完成向导
  
 **取消**: 放弃更改并取消配置
  
 **帮助**: 显示帮助信息, 如本主题
