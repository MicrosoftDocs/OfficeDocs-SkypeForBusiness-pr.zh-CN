# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>将控制器与 Lync Server 2010 监控页面关联
 
若要配置监控服务器并将其与控制器或控制器池：
  
在**监控服务器**下从列表中选择监控服务器。
  
如果没有监控服务器，并且您希望现在创建监控服务器，请单击**新建**。
  
 **返回**： 回到上一个对话框
  
 **完成**： 应用更改并完成向导
  
 **取消**： 放弃更改并取消配置
  
 **帮助**： 显示帮助信息，如本主题
