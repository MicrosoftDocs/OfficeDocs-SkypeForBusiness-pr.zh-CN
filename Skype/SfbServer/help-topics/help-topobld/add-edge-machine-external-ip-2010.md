# <a name="add-edge-machine-external-ip-2010"></a>添加边缘计算机外部 IP 2010

您可以在此页上，指向访问边缘服务、 Web 会议边缘服务和 A 的外部位置的边缘服务器配置属性 / V 边缘服务。

- 在**外部 IPv4 地址**下的**访问边缘服务**， **Web 会议边缘服务**，键入边缘服务器的外部 IPv4 地址和**A / V 边缘服务**。

- 单击**确定**以保存并将更改应用到拓扑文档。

- 单击**取消**以放弃更改并关闭**将服务器添加到边缘池**向导。

- 单击**帮助**以阅读本帮助主题。

有关将边缘服务器添加到现有拓扑的详细信息，请参阅[Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)。 有关迁移的边缘服务器的信息，请参阅[阶段 5： 将 Lync Server 2013 Preview 边缘服务器添加到试点池](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)。
