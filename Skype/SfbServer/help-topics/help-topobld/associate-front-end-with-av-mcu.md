# <a name="associate-front-end-with-av-mcu"></a>将前端与 AV MCU 相关联
 
A / V 会议启用实时音频和视频通信，您的用户之间 （即，如果他们有适当的客户机设备，例如耳机的音频会议和视频会议的网络摄像机）。 如果您的部署支持会议并启用 web 会议和 A / V 会议可以布置 A / V 会议服务器的前端服务器)，或者您可以部署一个或多个独立的 / V 会议服务器 (A / V 会议池). 如果您选择选项来部署独立的 A / V 会议服务器，您必须定义该拓扑生成器中。
  
在站点的所有池和多个中心站点的池可以使用相同的 A / V 会议服务器，如果使用率不超过容量的 A / V 会议服务器。
