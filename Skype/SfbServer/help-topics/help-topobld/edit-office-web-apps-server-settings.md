# <a name="edit-office-web-apps-server-settings"></a>编辑 Office Web Apps Server 设置
 
编辑配置 Office Web 应用程序服务器的属性。 可编辑以下属性：
  
 **Office Web 应用程序服务器 FQDN**： 此属性定义 Office Web 应用程序服务器的完全合格的域名名称和 （如果正在使用 IPv6） 应与域名系统 (DNS) 主机 A 或 AAAA 记录。
  
 **Office Web 应用程序服务器搜索 URL**： 客户端访问 Office Web 应用程序服务器的统一资源定位器 (URL)，您可能需要编辑此地址从其默认值，如果服务器放在另一个的内部网络之外的网络区域您部署。
  
如果此服务器部署在外围网络中或部署在将外围网络、低信任度网络和互联网与你的内部部署隔离的内部防火墙外部的其他网络区域中，则选中“**在外部网络中部署 Office Web Apps Server**”复选框。
