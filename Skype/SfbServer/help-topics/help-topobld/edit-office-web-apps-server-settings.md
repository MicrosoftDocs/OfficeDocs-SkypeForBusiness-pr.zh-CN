# <a name="edit-office-web-apps-server-settings"></a>编辑 Office Web Apps Server 设置

编辑 Web Apps Server Office的属性。 可编辑以下属性：

 **Office Web Apps Server FQDN：** 此属性定义 Office Web Apps Server 的完全限定域名，如果正使用) 记录，则应该匹配域名系统 (DNS) 主机 A 或 AAAA (。

 **Office Web Apps 服务器发现 URL：** 用于客户端访问 Office Web Apps Server 的统一资源定位器 (URL) ，如果服务器位于部署的内部网络外部的其他网络区域中，您可能需要从默认地址编辑此地址。

如果此服务器部署在外围网络或将外围网络、信任度低的网络和互联网与您内部部署隔离的内部防火墙外部的其他网络区域中，则选中“Office Web Apps 服务器部署在外部网络中”复选框。
