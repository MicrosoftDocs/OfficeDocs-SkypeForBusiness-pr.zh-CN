# <a name="edge-server-add-machine"></a>边缘服务器添加计算机
 
指定要添加为此池中的边缘服务器的每台计算机的完全合格的域名称 (FQDN)。 后将计算机添加到列表中，可以更新计算机的 FQDN 或之前发布拓扑结构随时从池中，删除它。 发布拓扑之后，更改 FQDN 需要删除拓扑生成器中的服务器，然后将新服务器添加到具有新的 FQDN 的池。 关于设计边缘池的拓扑的详细信息，请参阅[定义边拓扑](http://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)边缘部署文档。
