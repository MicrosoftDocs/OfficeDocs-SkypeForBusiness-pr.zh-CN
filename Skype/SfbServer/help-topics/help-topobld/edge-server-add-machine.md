# <a name="edge-server-add-machine"></a>Edge Server 添加机器

指定要添加为此池中的边缘服务器的每台计算机的完全限定的域名（FQDN）。 将计算机添加到列表后，您可以在发布拓扑之前的任何时间更新计算机的 FQDN 或将其从池中删除。 发布拓扑后，更改 FQDN 需要先在拓扑生成器中删除服务器，然后使用新的 FQDN 将新服务器添加到池中。 有关为 Edge 池设计拓扑的详细信息，请参阅在 Edge 部署文档中[定义边缘拓扑](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)。
