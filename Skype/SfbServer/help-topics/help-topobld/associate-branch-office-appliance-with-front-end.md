# <a name="associate-branch-office-appliance-with-front-end"></a>将分支机构设备与前端关联
 
若要将 Survivable 分支装置或 Survivable 分支服务器与前端池或标准版服务器相关联, 请在 "**选择前端池**" 页面上, 从**前端**池选择前端池或标准版服务器下拉列表。 单击“**下一步**”继续。
