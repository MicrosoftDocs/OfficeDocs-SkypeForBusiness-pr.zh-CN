# <a name="file-store-scope-node"></a>文件存储范围节点
 
此节点显示此文件存储 (FQDN) 和依赖池的完全限定域名。 可以执行以下操作：
  
- **编辑属性**。打开该服务器的“编辑属性”**** 页，在其中可以更改 FQDN 和文件存储名称。
    
- **删除**。从拓扑中删除文件存储。
    
除了这些选项之外，还可以执行以下常规任务：
  
- **拓扑**。 您可以使用此选项来定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑的副本、发布拓扑、安装数据库、合并组件或删除部署。
    
- **视图**。可以使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
