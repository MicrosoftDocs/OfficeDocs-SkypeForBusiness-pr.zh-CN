# <a name="office-web-apps-server-scope-node-expander"></a>Office Web Apps 服务器范围节点扩展器
 
右键单击拓扑生成器中的 " **Office Web Apps 服务器** " 以执行以下操作：
  
- **新的 Office Web Apps server**：启动用于定义新的 Office Web apps 服务器的 " **定义新的 Office web apps 服务器** " 向导。
    
除了新的 Office Web Apps Server 选项之外，您还可以执行以下常规任务：
  
![Office WebApps Server 作用域节点](../../media/OfficeWebApps_Server_Scope_Node.jpg)
  
- **拓扑**。 您可以使用此选项来定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑的副本、发布拓扑、安装数据库、合并早期版本中的组件，或删除部署。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
