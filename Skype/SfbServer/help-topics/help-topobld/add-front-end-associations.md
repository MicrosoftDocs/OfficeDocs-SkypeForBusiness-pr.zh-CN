# <a name="add-front-end-associations"></a>添加前端关联

现在，通过将服务器角色与前端池关联，可以启用对需要部署其他服务器的特定功能的支持。 您还可以稍后将服务器角色与前端池关联。 可以与前端池关联的服务器角色包括：

- A/V 边缘服务器。 有关实现 A/V 边缘服务器的详细信息，请参阅规划文档中的 ["](https://technet.microsoft.com/library/983a272a-e1b3-4d70-8f84-836b092fe526.aspx) 规划会议"。

> [!IMPORTANT]
> 如果现在启用对这些功能的任何支持，则发布的拓扑设计将包括实现每个选定功能所需的服务器组件。 若要成功发布拓扑而不出现错误，必须将物理计算机加入域。 例如，如果现在启用存档支持，则必须部署存档服务器并配置相应的存档选项，然后才能开始为组织存档通信。
