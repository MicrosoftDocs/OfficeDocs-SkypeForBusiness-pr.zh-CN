# <a name="monitoring-servers-scope-node"></a>监视服务器的作用域节点
 
**监视服务器**节点列出了监视服务器拓扑结构中。 可以执行以下操作：
  
- **新的监视服务器**。 启动**定义新的监视服务器**向导，使用新监视服务器添加到拓扑结构中。
    
除了监视服务器的选项，您可以执行以下常规任务：
  
- **拓扑**。 您可以使用此选项来定义新的拓扑结构、 打开一个现有的拓扑结构、 下载当前拓扑、 保存一份拓扑、 发布拓扑、 安装数据库、 合并组件从以前版本的 Office 通信服务器，或删除部署。
    
- **视图**。可以使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
