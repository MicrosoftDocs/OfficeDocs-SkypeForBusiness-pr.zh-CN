# <a name="install-mirror-database-option-page"></a>安装镜像数据库选项页
 
通过以下定义配置“镜像数据库设置”：
  
- 键入 **文件共享的路径** 以定义要镜像的数据库SQL Server备份文件的位置。
    
    > [!NOTE]
    > 主SQL Server实例 (命名实例或默认实例) 必须具有此处定义的文件共享的写入权限。 镜像SQL Server实例 (命名实例或默认) 必须具有对同一文件共享的读取权限。 
  
  **确定** 接受更改并通过对话框提交更改。
  
  **取消** 放弃更改并关闭对话框。
  
  **帮助** 显示此帮助屏幕。
