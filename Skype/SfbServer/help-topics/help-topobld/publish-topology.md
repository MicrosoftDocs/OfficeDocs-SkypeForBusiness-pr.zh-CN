# <a name="publish-topology"></a>发布拓扑
 
在开始发布过程时，将显示**发布拓扑结构**页。 阅读并确认列出的任务已完成，然后再继续。 当准备好继续时，单击**下一步**。 如果您必须关闭发布过程，请单击**取消**。
