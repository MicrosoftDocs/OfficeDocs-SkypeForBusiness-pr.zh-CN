# <a name="define-a-new-trunk"></a>定义新干线
 
您定义新的会话启动协议 (SIP) 干线通过提供以下信息：
  
- **干线名称**： 表明此中继拓扑中的唯一名称
    
- **关联的 PSTN 网关**： 部署从列表中选择已部署和配置 PSTN 网关
    
- **IP/PSTN 网关的侦听端口**： IP PBX 或 PSTN 网关监听的端口。 从所有配置中部署其他干线侦听端口必须唯一
    
- **SIP 传输协议**： 从列表中选择 TCP 或 TLS
    
- **相关中介服务器**： 从列表中选择中介服务器部署并在您的部署配置
    
- **相关中介服务器端口**： 将端口值设置为等于该 SIP 中继将使用中介服务器的 TCP 或 TLS 端口值
