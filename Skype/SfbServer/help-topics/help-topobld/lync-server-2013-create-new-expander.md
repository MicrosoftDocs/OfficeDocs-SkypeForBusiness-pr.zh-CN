# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 创建新的扩展器
 
可以通过右键单击"Lync Server 2013"，选择"新建"，然后选择要创建的服务器角色，在拓扑生成器的 **Lync Server 2013** 节点中创建新的服务器角色。
