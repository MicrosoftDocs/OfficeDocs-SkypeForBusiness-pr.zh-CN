# <a name="install-database-options-page"></a>安装数据库选项页面
 
配置高级的选项的 SQL Server 上的数据库和日志文件的位置。 可用的选项有：
  
> [!IMPORTANT]
> 选择最适合您的要求和策略与在 SQL Server 的计算机上的数据和日志文件位置相关的选项。 
  
 **自动确定数据库文件的位置**: 默认选项使用一种算法，确定 SQL Server 上的可用空间，并将分发数据库和日志文件以获得最佳性能。
  
 **使用 SQL Server 实例的默认值**： 选择此选项可将数据库文件和日志文件基于 SQL Server 的实例设置。 通常，选项管理和配置与数据库管理员联系。
  
 **我们目标 SQL Server 上的这些路径**： 选择此选项可以通过键入放置数据库和日志文件的驱动器和文件夹的完整路径定义您自己的 SQL Server 数据库和日志文件的路径。
  
> [!IMPORTANT]
> 您输入的路径可根据系统中的性能优化算法进行修改。 有关详细信息，请参阅[数据库安装使用 Lync 服务器命令行管理程序](http://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx)。 
  
 **确定**： 请单击确定按钮以提交您的更改。
  
 **取消**： 单击取消放弃所有更改并返回到安装数据库屏幕。
  
 **帮助**： 单击帮助按钮来访问此帮助页。
