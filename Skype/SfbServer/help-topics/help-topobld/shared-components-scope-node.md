# <a name="shared-components-scope-node"></a>共享组件范围节点
 
右键单击**共享组件**在拓扑生成器层次结构树中创建新的共享的组件或访问级别的拓扑选项。
  
![共享组件范围节点](../../media/Shared_Components_Scope_Node.jpg)
  
选择**新建**以创建新的文件存储、 一个新的 IP/PSTN 网关、 新的 Office Web 应用程序服务器、 新的 SQL Server 存储或新干线。
  
> [!NOTE]
> 每个选项，请参阅帮助主题。 
  
![共享组件范围节点](../../media/Shared_Components_Scope_NodeB.jpg)
  
选择单击有关选项以打开一个新的拓扑文档的**拓扑结构**。 打开一个现有的拓扑文档，从中央管理存储下载当前拓扑、 当前拓扑文档发布、 安装数据库来创建当前配置的数据库的配置，当前拓扑文档的保存副本到当前部署，或删除部署合并办公通信服务器 2007 R2。
  
![共享组件范围节点 C](../../media/Shared_Components_Scope_NodeC.jpg)
