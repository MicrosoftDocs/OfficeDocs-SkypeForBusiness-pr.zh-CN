# <a name="associate-branch-office-appliance-with-edge"></a>将分支机构设备与边缘关联
 
如果已部署了边缘服务器，并且必须提供用于外部使用的媒体服务，请选择要与 Survivable 分支机构或 Survivable 分支服务器关联的边缘服务器或池，具体方法是在下拉列表中进行选择。 单击“下一步”**** 即可继续。
  
> [!NOTE]
> 如果拓扑中没有定义边缘服务器或边缘服务器池，则可跳过此步骤。 当您定义新的边缘服务器或边缘服务器池时，可以将 Survivable 分支装置或 Survivable 分支服务器与边缘服务器或边缘服务器池关联，也可以在拓扑中定义了边缘服务器或边缘服务器池之后编辑 Survivable 分支设备或 Survivable 分支服务器的属性页。
