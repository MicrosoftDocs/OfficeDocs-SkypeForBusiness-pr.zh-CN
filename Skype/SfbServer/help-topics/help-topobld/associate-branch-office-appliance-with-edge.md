# <a name="associate-branch-office-appliance-with-edge"></a>将分支机构设备与边缘关联
 
如果已部署边缘服务器并且必须提供媒体服务供外部使用，则通过在下拉列表中选择与 Survivable Branch Appliance 或 Survivable Branch Server 关联的边缘服务器或池。 单击“下一步”即可继续。
  
> [!NOTE]
> 如果拓扑中没有定义边缘服务器或边缘服务器池，则可跳过此步骤。 在定义新的边缘服务器或边缘服务器池时，或在拓扑中定义边缘服务器或边缘服务器池后编辑 Survivable Branch Appliance 或 Survivable Branch Server 的属性页时，您将能够将 Survivable Branch Appliance 或 Survivable Branch Server 与边缘服务器或边缘服务器池关联。
