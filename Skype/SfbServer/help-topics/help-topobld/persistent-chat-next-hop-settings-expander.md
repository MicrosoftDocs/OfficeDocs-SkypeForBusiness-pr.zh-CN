# <a name="persistent-chat-next-hop-settings-expander"></a>持久聊天下一跃点设置扩展器
 
通过从列表中选择一个前端服务器或前端池配置持久聊天**下一个跃点池**。
  
 **确定** 接受更改并通过对话框提交更改。
  
 **取消** 放弃更改并关闭对话框。
  
 **帮助** 显示此帮助屏幕。
