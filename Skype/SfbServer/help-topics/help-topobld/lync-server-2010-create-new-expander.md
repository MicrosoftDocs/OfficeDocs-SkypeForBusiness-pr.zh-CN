# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 创建新的扩展器
 
您可以通过右键单击**Lync Server 2010**的拓扑生成器 Lync Server 2010 节点中创建新的服务器角色，选择**新建**，然后选择要创建的服务器角色。
