# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 创建新的扩展器
 
可以通过右键单击"Lync Server 2010"，选择"新建"，然后选择要创建的服务器角色，在拓扑生成器的 Lync  **Server 2010** 节点中创建新的服务器角色。
