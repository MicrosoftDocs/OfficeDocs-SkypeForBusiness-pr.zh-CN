# <a name="legacy-merge-fqdn"></a>传统的 FQDN 合并
 
**访问边缘池内部 FQDN**用于各种方案内部用户与为联盟、 远程用户访问和公用 IM 连接的外部用户的通信在其中。 如果负载平衡边缘服务器部署在传统环境中，内部负载平衡器的输入完全合格的域名称 (FQDN)。
  
**内部 SIP 访问端口** **5061**为默认 SIP 传输控制协议 (TCP) 端口用于与客户端、 传统的前端池和服务器进行通信。 如果不使用默认值，更新**内部 SIP 访问端口：**值。
