# <a name="legacy-merge-fqdn"></a>旧版合并 FQDN
 
“访问边缘池内部 FQDN”**** 适用于内部用户与联盟、远程用户访问和公共 IM 连接的外部用户进行通信的多种情况。如果旧版环境中部署了负载平衡边缘服务器，请输入内部负载平衡器的完全限定域名 (FQDN)。
  
“内部 SIP 访问端口”**** 值“5061”**** 是与客户端、旧版前端池和服务器进行通信的默认传输控制协议 (TCP) SIP 端口。如果未使用默认值，请更新“内部 SIP 访问端口:”**** 值。
