# <a name="legacy-merge-fqdn"></a>旧版合并 FQDN
 
**访问边缘池内部 FQDN**用于各种方案，其中内部用户与外部用户进行联盟、远程用户访问和公共 IM 连接。 如果在您的旧环境中部署了负载平衡的边缘服务器，请输入内部负载平衡器的完全限定的域名（FQDN）。
  
**5061**的**内部 SIP 访问端口**值是用于与客户端、旧式前端池和服务器通信的默认传输控制协议（TCP） SIP 端口。 如果未使用默认值，请更新 "**内部 SIP 访问端口：** " 值。
