# <a name="add-trusted-application-machine"></a>添加受信任的应用程序的计算机
 
指定您想要添加为受信任的应用程序服务器该池中的每台计算机的完全限定的域名 (FQDN)。 将计算机添加到列表中后，可以更新计算机的 FQDN 或在发布拓扑前随时将其从池中删除。 发布拓扑之后，更改 FQDN 需要删除拓扑生成器中的服务器，然后将新服务器添加到具有新的 FQDN 的池。
