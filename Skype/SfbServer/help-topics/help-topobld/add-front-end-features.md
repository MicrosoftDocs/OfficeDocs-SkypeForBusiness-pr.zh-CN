# <a name="add-front-end-features"></a>添加前端功能
 
选择您要在其中启用前端池中每个功能。 至少，您必须部署即时消息 (IM)，它总是处于启用状态并且无法取消选择。 您还可以选择部署其他功能，如会议，企业语音，并调用许可控制 (CAC)。 某些功能是依赖于您的拓扑结构。 
  
- 网络会议
    
- 企业语音
    
- 调用许可控制 (CAC)
    
- 存档
    
- 监控
    
> [!NOTE]
> 如果您想要在部署中启用 CAC，您必须启用每个中心站点的一个池中的 CAC。 我们建议 CAC，如果您正在部署语音功能或 A / V 会议。 
  
有关指定企业版池的功能的详细信息，请参阅部署文档中的[定义和配置前结束池](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)。
