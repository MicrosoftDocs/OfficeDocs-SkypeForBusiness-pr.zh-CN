# <a name="install-and-create-databases"></a>安装和创建数据库
 
选择要在其中创建您的部署的数据库。 默认情况下，数据库将在定义的站点中定义的 SQL Server 上创建并将自动部署并配置基于正在将数据库放在 SQL Server 的数据库文件。
  
 **选择您想要创建的数据库**： 选择您打算部署和配置的任何数据库的复选框。 请选择您将部署的任何或所有数据库。 
  
> [!CAUTION]
> SQL Server 必须已进行了配置 （如果有） 的实例，必须打开的防火墙端口以适应您要部署到的数据库的实例。 有关详细信息，请参阅[将 SQL Server 配置 Lync 服务器 2013年预览](http://technet.microsoft.com/library/375e5cc4-e436-46dc-9b02-5063f35cdcc1.aspx)
  
 **高级**： 在 SQL Server 上单击，单击**高级**按钮以选择文件位置上 SQL Server 数据库的选项。 高级的数据库文件的位置的详细信息，请参阅[数据库安装使用 Lync 服务器命令行管理程序](http://technet.microsoft.com/library/c90a6449-4dd5-4b18-b21c-ea2c2a64dc3c.aspx)
  
 **回**： 单击此按钮将返回到前一个屏幕 （不总是可能可用，请根据您如何到达此对话框）。
  
 **下一步**： 单击此按钮提交在对话框中当前所选内容并将您带到下一对话框用于配置的其他信息
  
 **取消**： 单击此按钮将退出配置并放弃所做的更改。 某些但不是所有配置屏幕将提示您，如果您想要退出并放弃更改。 选择**是**将关闭当前配置并关闭当前的配置并使您返回到拓扑生成器。 选择**否**会使您返回到配置对话框，允许您继续配置。
  
 **帮助**： 单击**帮助**按钮显示该帮助信息，与当前配置对话框。
