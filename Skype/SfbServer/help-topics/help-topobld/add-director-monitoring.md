# <a name="add-director-monitoring"></a>添加控制器监控
 
您可以通过配置以下属性的**定义监视 SQL Server 存储**：
  
- **监控 SQL Server 存储**： 从列表中选择 SQL Server 完全限定的域名 (FQDN) （和 （可选） 的已命名的 SQL Server 实例）。
    
    单击**新建**以创建一个新的 SQL Server FQDN 定义，以及 （可选） 的监控服务器存储的实例名称。
    
- 如果您想要添加数据库镜像监控服务器，请选择**启用 SQL Server 存储镜像**复选框。
    
    从列表中选择现有**监控 SQL Server 存储镜像**。
    
    单击**新建**以创建一个新的 SQL Server FQDN 定义和镜像存储的实例名称 （可选）。
    
- 如果选中**启用 SQL Server 存储镜像**，或者选择**使用 SQL Server 镜像见证启用自动故障转移**以选择 SQL Server 镜像见证存储从列表中。
    
    单击**新建**以创建一个新的 SQL Server FQDN 定义和镜像见证存储的实例名称 （可选）。
    
单击“**上一步**”以回到上一个池定义对话框。
  
完成为此对话框，然后继续进行配置输入选项后，请单击**下一步**。
  
单击**取消**以放弃所有更改并结束该向导。
  
单击**帮助**以访问上下文相关帮助，例如此页面。
