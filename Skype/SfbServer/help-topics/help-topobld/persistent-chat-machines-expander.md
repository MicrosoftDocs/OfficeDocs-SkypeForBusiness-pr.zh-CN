# <a name="persistent-chat-machines-expander"></a>持久聊天计算机扩展器
 
激活或使用的服务器或池在**机器状态设置**中列出停用已部署持续聊天服务器或持久聊天服务器池。 在列表中选择一个服务器或池，并单击“**变为活动状态**”按钮可将服务器或池设为活动状态。
  
通过选择列表中的服务器或池设置为非活动状态已部署持续聊天服务器或持久聊天服务器池并单击**使处于非活动状态**。 必须至少将一台服务器设置为活动状态。
  
 **确定** 接受更改并通过对话框提交更改。
  
 **取消** 放弃更改并关闭对话框。
  
 **帮助** 显示此帮助屏幕。
