# <a name="add-front-end-monitoring-store-page"></a>添加前端监控存储页
 
通过配置以下属性“定义监视 SQL Server 存储”****：
  
- **监视 Sql server 存储**：选择 sql server 完全限定的域名 (并从列表中选择一个实例) （可选）。
    
    单击 " **新建** " 以创建新的 SQL Server FQDN 定义，还可以选择监视服务器存储的实例名称。
    
- 如果要为监视服务器添加数据库镜像，请选中 " **启用 SQL Server 存储镜像** " 复选框。
    
    从列表选择一个现有的“监控 SQL Server 存储镜像”****。
    
    单击 " **新建** " 创建新的 SQL Server FQDN 定义，还可以选择为镜像存储指定实例名称。
    
- 如果选择 " **启用 Sql server 存储镜像**"，则可以选择 " **使用 sql server 镜像见证启用自动故障转移** "，从列表中选择 SQL server 镜像存储。
    
    单击 " **新建** " 创建新的 SQL Server FQDN 定义，还可以选择为镜像见证存储指定实例名称。
    
单击“上一步”**** 以回到上一个池定义对话框。
  
完成为此对话框输入选项后单击“下一步”****，以继续进行配置。
  
单击“取消”**** 将放弃所有更改并关闭对话框。
  
单击“帮助”**** 以访问上下文相关帮助，例如此页面。
