# <a name="add-front-end-monitoring-store-page"></a>添加前端监控存储页
 
通过配置以下属性“定义监视 SQL Server 存储”：
  
- **监控SQL Server存储**：从SQL Server选择一个完全限定 (，也可以选择一) 实例。
    
    单击 **"** 新建"以SQL Server FQDN 定义和监控服务器存储的实例名称（可选）。
    
- 如果要为 **SQL Server** 服务器添加数据库镜像，请选中"启用存储镜像"复选框。
    
    从列表选择一个现有的“监控 SQL Server 存储镜像”。
    
    单击 **"** 新建"以SQL Server FQDN 定义和镜像存储的实例名称（可选）。
    
- 如果选择 **"启用** SQL Server存储镜像"，可选择选择"使用 **SQL Server** 镜像见证启用自动故障转移"以从SQL Server选择镜像见证存储。
    
    单击 **"** 新建"以SQL Server FQDN 定义和镜像见证存储的实例名称（可选）。
    
单击“上一步”以回到上一个池定义对话框。
  
完成为此对话框输入选项后单击“下一步”，以继续进行配置。
  
单击“取消”将放弃所有更改并关闭对话框。
  
单击“帮助”以访问上下文相关帮助，例如此页面。
