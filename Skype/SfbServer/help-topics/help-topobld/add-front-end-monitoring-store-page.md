# <a name="add-front-end-monitoring-store-page"></a>添加前端监控存储页
 
您**定义的监视 SQL Server 存储**通过配置以下属性：
  
- **监视 SQL Server 存储**： 从列表中选择一个 SQL Server 完全限定的域名称 （和可选实例）。
    
    单击**新建**以创建新的 SQL Server FQDN 定义，也可以监视服务器存储的实例名称。
    
- 如果您想要添加数据库镜像用于监视服务器，请选择**启用 SQL Server 存储镜像**复选框。
    
    从列表中选择**监视 SQL Server 存储镜像**现有。
    
    单击**新建**以创建新的 SQL Server FQDN 定义，也可以为镜像存储一个实例名称。
    
- 如果您选择**启用 SQL Server 存储镜像**，或者选择**使用 SQL Server 镜像见证启用自动故障转移**SQL Server 镜像见证商店从列表中选择。
    
    单击**新建**以创建新的 SQL Server FQDN 定义，也可以镜像见证存储的实例名称。
    
单击“**上一步**”以回到上一个池定义对话框。
  
输入此对话框以继续进行配置的选项之后，请单击**下一步**。
  
单击**取消**放弃所有更改并结束该向导。
  
单击“**帮助**”以访问上下文相关帮助，例如此页面。
