# <a name="front-end-start-page"></a>前端开始页面

使用此向导为网站定义新的前端池。 每个网站必须至少有一个企业版池或标准版服务器。 有关使用向导创建新的前端池的要求的详细信息，请参阅在部署文档中[定义和配置前端池](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)。
