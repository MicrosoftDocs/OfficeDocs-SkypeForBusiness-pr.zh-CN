# <a name="front-end-start-page"></a>前端开始页

使用此向导为站点定义一个新的前端池。 每个站点必需至少具有一个 Enterprise Edition 池或 Standard Edition 服务器。 有关使用向导创建新的前端池的要求的详细信息，请参阅部署文档中的[Define and Configure a Front End Pool](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)。
