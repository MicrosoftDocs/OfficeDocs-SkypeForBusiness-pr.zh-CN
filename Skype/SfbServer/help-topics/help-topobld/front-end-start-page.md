# <a name="front-end-start-page"></a>前端开始页

使用此向导为站点定义一个新的前端池。每个站点必需至少具有一个 Enterprise Edition 池或 Standard Edition 服务器。有关使用向导创建新的前端池的要求的详细信息，请参阅部署文档中的[Define and Configure a Front End Pool](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server)。
