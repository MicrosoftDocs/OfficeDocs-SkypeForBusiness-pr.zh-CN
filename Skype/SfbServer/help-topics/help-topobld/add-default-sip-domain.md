# <a name="add-default-sip-domain"></a>添加默认 Sip 域
 
指定您希望部署支持的主要会话启动协议 (SIP) 域。 主要的 SIP 域可以是相同，或与，Active Directory 域服务域不同。 例如，Active Directory 域可能是 fabrikam.net，并可指定在您主要的 SIP 域为 contoso.net。
