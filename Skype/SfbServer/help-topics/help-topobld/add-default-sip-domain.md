# <a name="add-default-sip-domain"></a>添加默认的 SIP 域
 
指定希望您部署支持的主会话初始协议 (SIP) 域。 主 SIP 域可以与 Active Directory 域服务域相同，也可以不同。 例如，您的 Active Directory 域是 fabrikam.net，而您可以将主 SIP 域指定为 contoso.net。
