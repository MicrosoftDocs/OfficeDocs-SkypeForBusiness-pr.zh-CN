# <a name="topology-builder-startup-dialog"></a>拓扑生成器启动对话框
 
首先，拓扑生成器配置属性和 Microsoft Lync Server 2013 部署的配置。 在启动时，拓扑生成器显示一个对话框，询问拓扑文档的来源将是什么。 您可以选择三个选项之一：
  
- **下载从现有部署的拓扑结构**如果您有运行拓扑生成器并已成功发布到中央管理服务器拓扑结构，请选择此选项。 此选项请求并且下载当前的配置，使您可以编辑和配置现有部署的本地副本。
    
- **从本地文件打开拓扑**如果配置的是部署但尚未发布配置，请选择此选项。 此选项允许您将配置工作的本地副本保存在拓扑生成器并回到它在以后的时间。
    
- **新的拓扑结构**选择此选项将开始一个新的拓扑文档配置。 此选项将打开一个新的拓扑文档，配置以满足您将配置部署的要求。
