# <a name="associate-front-end-with-archiving"></a>将前端与存档相关联
 
每个前端池和高存活力的分支装置只能具有一个与之关联的存档服务器。 当对网站启用存档时，存档服务器然后收集和记录即时消息 (IM) 消息内容和 web 会议内容涉及用户的对话从宿主关联的池和高存活力的分支装置。 
  
在站点的所有池和多个中心站点的池可以使用同一存档服务器，如果使用率不超过了存档服务器的容量。
