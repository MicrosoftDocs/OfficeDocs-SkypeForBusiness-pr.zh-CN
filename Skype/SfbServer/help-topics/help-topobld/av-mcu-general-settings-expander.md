# <a name="av-mcu-general-settings-expander"></a>AV MCU 常规设置扩展器
 
若要编辑的属性 A / V 会议服务器池，您已经定义，您可以执行以下操作：
  
## <a name="general-settings"></a>常规设置

更新一个完全限定的域名 (FQDN) / V 会议服务器池通过键入该池，因为它定义在域名系统 (DNS)。
  
## <a name="av-conferencing-service"></a>A/V 会议服务

相关联的 A / V 会议服务器池中选择前端池或池有与 A / V 会议服务器池。
  
> [!NOTE]
> 如果没有前端池显示在**选择将与此相关联的前端池 / V 会议池**窗格中，既有搭配 A / V 会议服务器与前端池或没有选择在**功能中添加会议和功能**的前端池属性的部分。
  
## 

完成后，单击**确定**以完成编辑 A / V 会议服务器池设置。 单击“**取消**”将放弃所有更改。
