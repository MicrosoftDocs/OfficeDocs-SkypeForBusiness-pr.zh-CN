# <a name="publish-topology-next-step"></a>发布拓扑下一步

完成拓扑发布后，将显示“发布向导已完成”页面。

在发布拓扑过程中完成操作步骤时，将创建包含有关执行的任务信息的日志。若要查看与给定步骤相关的日志文件，请选择需要查看的日志文件所对应的步骤，然后单击“查看日志”。

> [!NOTE]
> 并非发布拓扑过程中的所有步骤都有关联的日志文件。如果选择某个步骤时“查看日志”可用，则此步骤已创建日志文件。如果“查看日志”按钮不可用（显示为灰色），则未针对此步骤创建日志文件。

若要查看为完成拓扑所建议的后续步骤，请单击“单击此处打开待办事项列表”链接。 通过单击此链接打开的文本文件将指示必须运行本地设置的服务器，以及完成部署所需执行的任何其他步骤。 有关列出的每项任务，请参阅部署文档中相应的章节。 有关详细信息，请参阅[Deploying Lync Server 2013 Preview](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-lync-server)，并参阅待办事项列表中记录的每个服务器角色的相关章节。

单击“完成”。

有关详细信息，请参阅[Publish the Topology](/previous-versions/office/lync-server-2013/lync-server-2013-publish-the-topology)。
