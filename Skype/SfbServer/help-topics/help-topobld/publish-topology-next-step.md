# <a name="publish-topology-next-step"></a>发布拓扑下一步

完成拓扑发布后，将显示**发布向导完成**页。

在发布拓扑过程中完成步骤，如中创建日志包含有关已执行的任务的信息。 若要查看日志文件与给定的步骤，选择您需要查看日志文件，该步骤，然后单击**查看日志**。

> [!NOTE]
> 在发布拓扑过程不是所有步骤将具有都关联的日志文件。 如果您选择一个步骤，可**查看日志**，此步骤已创建的日志文件。 如果**查看日志**按钮不可用 （灰显），没有日志文件创建的此步骤。

若要查看建议的完成您的拓扑的后续步骤中，单击**单击此处打开待办事项列表**链接。 通过单击此链接打开文本文件将指出的服务器必须运行本地安装程序，以及有需要完成部署所需的其他步骤。 对于每个列出的任务，请参阅部署文档中的相应部分。 有关详细信息，请参阅[部署 Lync Server 2013 Preview](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)、 和引用每个服务器角色待办事项列表中记下的相关节。

单击**完成**

有关详细信息，请参阅[Publish the Topology](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)。
