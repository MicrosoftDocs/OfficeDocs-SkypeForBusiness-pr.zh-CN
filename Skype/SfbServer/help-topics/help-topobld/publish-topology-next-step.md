# <a name="publish-topology-next-step"></a>发布拓扑结构下一步
 
拓扑的发布完成时，会显示**完成发布向导**页。
  
如在发布拓扑过程完成步骤，创建了日志包含有关已执行的任务的信息。 若要查看与某一给定的步骤日志文件，选择需要查看的日志文件的步骤，然后单击**查看日志**。 
  
> [!NOTE]
> 在发布拓扑过程并不是所有步骤将具有都关联的日志文件。 如果您选择一个步骤并且**查看日志**不可用，此步骤创建了日志文件。 如果**查看日志**按钮不可用 （灰显），没有日志文件创建的这一步。
  
若要查看下一步行动建议来完成您的拓扑结构，请单击**单击此处可打开待办事项列表**链接。 单击此链接可打开的文本文件将显示运行本地安装程序，您必须在服务器和任何其它步骤，您需要采取以完成部署。 对于每一个列出的任务，请参阅部署文档中的相应章节。 有关详细信息，请参阅[部署 Lync 服务器 2013年预览](http://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)，并指在待办事项列表中记下每个服务器角色的相关部分。
  
单击**完成**
  
有关详细信息，请参阅[发布拓扑结构](http://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)。
