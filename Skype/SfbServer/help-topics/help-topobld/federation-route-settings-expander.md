# <a name="federation-route-settings-expander"></a>联合身份验证路由设置扩展器
 
若要设置站点联盟路由分配，首先必须在边缘服务器或池边缘服务器上启用联盟。 如果边缘服务器或池上未启用联合身份验证，则网站的联合工艺路线作业设置将不能进行修改。
  
## 

如果已配置的边缘服务器或池的联合身份验证设置，您可以配置以下选项： 
  
- **允许联合路由分配到的所有网站**此设置将影响所有站点。 请确保在此站点所配置的设置适用于所有的站点。
    
- **启用 SIP 联盟**选择此选项可启用 SIP 联合工艺路线，并选择主管或边缘池作为联合工艺路线。
    
- **启用 XMPP 联盟**选择此选项可启用 XMPP 联合路由，并选择主管或边缘池作为联合工艺路线。
