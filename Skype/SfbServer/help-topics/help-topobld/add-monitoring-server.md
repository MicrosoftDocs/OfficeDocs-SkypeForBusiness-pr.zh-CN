# <a name="add-monitoring-server"></a>添加监控服务器
 
若要将监视服务器合并到您的拓扑设计，必须指定您要在其部署监视服务器的服务器的完全合格的域名称 (FQDN)。 之前发布包括监视服务器的拓扑结构和 Skype 业务服务器安装在服务器上，您应该完成部署监视的所有先决条件。 有关这些系统必备组件的详细信息，请参阅部署文档中的[部署监视](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)。
