# <a name="publish-topology-create-mirror-database-page"></a>发布拓扑创建镜像数据库页
 
这是初步文档，可能会更改。 空白主题作为占位符包含在内。
  
拓扑生成器分析已定义的拓扑文档中的所有配置设置。 如果已配置镜像SQL Server数据库，则会显示一个屏幕，显示镜像存储、主存储、文件共享路径和数据库路径。 选中与数据库镜像存储关联的复选框可创建镜像数据库和文件共享路径。 如果不想创建数据库并部署到定义的数据库，请清除数据库镜像的SQL Server。
  
 如果此时选择不安装所有这些数据库或其中某个数据库，则清除不安装的数据库。 稍后，选择使用" **安装** 数据库"，方法是右键单击拓扑生成器中的大多数节点项。
