# <a name="publish-topology-create-mirror-database-page"></a>发布拓扑创建镜像数据库页面
 
本文档是预备文档，可能随时更改。 空白主题均以占位符的形式包含在内。
  
拓扑生成器分析你定义的拓扑文档中的所有配置设置。 如果已配置镜像 SQL Server 数据库，则会显示一个屏幕，显示镜像存储、主存储、文件共享路径和数据库路径。 选择与数据库镜像存储相关联的复选框以创建镜像数据库和文件共享路径。 如果不希望创建数据库并将其部署到已定义的 SQL Server，请清除数据库镜像的复选框。
  
 如果您选择现在不安装其中一个或所有这些数据库，您将清除不会安装的数据库。 稍后，通过右键单击拓扑生成器中的大多数节点项目，可选择使用 "**安装数据库**"。
