# <a name="publish-topology-create-mirror-database-page"></a>发布拓扑创建镜像数据库页
 
本文档是预备文档，可能随时更改。空白主题均以占位符的形式包含在内。
  
拓扑生成器分析拓扑定义的文档中的所有配置设置。 如果已配置了镜像的 SQL Server 数据库，将会出现一个屏幕，显示镜像存储、 主存储、 文件共享路径和数据库路径。 选择要创建镜像的数据库和文件共享路径的数据库镜像存储相关联的复选框。 如果您不想创建并部署到已定义的 SQL Server 数据库，请清除复选框用于数据库镜像。
  
 如果您选择选择不在此时安装的一个或所有这些数据库，您清除不安装数据库。 稍后，您选择使用多数节点项拓扑生成器中右键单击**安装的数据库**。
