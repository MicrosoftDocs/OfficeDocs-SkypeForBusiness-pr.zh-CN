# <a name="director-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010 中的控制器一般设置扩展器
 
若要配置控制器或控制器池的**常规**属性：
  
 **FQDN**： 键入总监或总监池的完全合格的域名称
  
 **关联**
  
 **文件存储**： 从列表中选择文件存储控制器或控制器池
  
单击**新建**以定义一个新的文件存储区
  
选择**关联监视服务器**复选框，然后从列表中选择一个监视服务器
  
单击**新建**以定义新的监视服务器
  
 **确定** 接受更改并通过对话框提交更改。
  
 **取消** 放弃更改并关闭对话框。
  
 **帮助** 显示此帮助屏幕。
