# <a name="legacy-merge-edge-type"></a>旧版合并 Edge 类型
 
对于**单个边缘服务器**或**负载平衡的统一边缘**服务器，单击 "**下一步**" 以指定配置信息。 对于**展开的边缘**，必须先将每台服务器的配置更改为统一边缘配置，然后才能将其与 Lync server 2013 合并。
