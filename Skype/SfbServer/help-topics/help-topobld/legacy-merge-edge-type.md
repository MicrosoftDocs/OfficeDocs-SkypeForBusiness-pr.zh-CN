# <a name="legacy-merge-edge-type"></a>旧版合并 Edge 类型
 
为**单台边缘服务器**或**负载平衡合并边缘**服务器，单击**下一步**来指定配置信息。 **扩展边缘**，必须在每台服务器的配置更改为合并的边缘配置它可与 Lync Server 2013 合并之前。
