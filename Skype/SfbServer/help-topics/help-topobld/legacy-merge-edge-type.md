# <a name="legacy-merge-edge-type"></a>传统的合并边缘类型
 
对于**单个边缘服务器**或**负载平衡合并边缘**服务器，请单击**下一步**要指定配置信息。 对于**扩展边缘**，每个服务器的配置必须到统一的边缘配置之前更改它可以合并使用 Lync Server 2013。
