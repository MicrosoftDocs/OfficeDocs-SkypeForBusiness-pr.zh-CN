# <a name="legacy-merge-edge-type"></a>旧版合并边缘类型
 
对于 **“单个边缘服务器”** 或 **“负载平衡合并边缘”** 服务器，单击 **“下一步”** 以指定配置信息。 对于" **扩展边缘**"，必须将每个服务器的配置更改为合并边缘配置，然后才能与 Lync Server 2013 合并。
