# <a name="simple-url-settings-expander"></a>简单 URL 设置扩展器

若要指定新、 删除现有或更改活动的简单 Url，请执行以下操作：

## <a name="simple-urls"></a>简单 URL

若要修改或定义简单 Url，请首先决定哪些您想要编辑或更改三种简单 Url。 可以从电话访问 URL、会议 URL 和管理访问 URL 中进行选择。

要修改电话访问 URL 或会议 URL，请选择需要更改的 URL。 单击“**编辑 URL**”。 编辑 URL，并单击**确定**以保存 URL。 单击“**取消**”将放弃所有更改。

要添加新 URL，请单击“**添加**”。 在**添加简单 URL**对话框中，指定的 URL，，然后单击**确定**以保存 URL。 如果需要将新的 URL 设置为活动 URL，请选择“**将此设为选定域的活动 URL**”。 单击“**取消**”将放弃所有更改。

要将其他 URL 为活动 URL （有绿色复选标记 URL 旁边），选择相应的 URL，然后单击**变为活动状态**。

> [!NOTE]
> 只能有一个活动 URL 为每个会话初始协议 (SIP) 域。

要删除的 URL，选择相应的 URL，然后单击**删除**。

> [!CAUTION]
> 请务必阅读简单 Url 设置对话框页上的信息。 删除会议 URL 可呈现由用户无法再访问计划的现有会议。 请考虑使新的会议 URL 处于活动状态后保留现有的 URL。 确保用户不再使用旧的会议 URL 时，您可以安全地将其删除。

要编辑或更改管理访问 URL，请编辑该条目。

在编辑属性页上的所有更改都完成后，单击**确定**以保存所做的更改。 单击“**取消**”将放弃更改。
