# <a name="simple-url-settings-expander"></a>简单的 URL 设置扩展器
 
若要指定新、 删除现有的或简单的 Url 来更改活动，请执行以下操作：
  
## <a name="simple-urls"></a>简单 URL

修改或定义简单的 Url，请首先确定哪些您想要编辑或更改三个简单的 Url。 可以从电话访问 URL、会议 URL 和管理访问 URL 中进行选择。
  
要修改电话访问 URL 或会议 URL，请选择需要更改的 URL。 单击“**编辑 URL**”。 编辑该 URL，然后单击**确定**以保存该 URL。 单击“**取消**”将放弃所有更改。
  
要添加新 URL，请单击“**添加**”。 在**添加简单的 URL**对话框中，指定的 URL，然后单击**确定**以保存该 URL。 如果需要将新的 URL 设置为活动 URL，请选择“**将此设为选定域的活动 URL**”。 单击“**取消**”将放弃所有更改。
  
要使一个不同的 URL （以绿色的复选标记的 URL 旁边的设定） 的有效 URL，选择的 URL 并单击**激活**。
  
> [!NOTE]
> 只能有一个活动会话启动协议 (SIP) 的每个域 URL。 
  
要删除一个 URL，请选择 URL，然后单击**删除**。
  
> [!CAUTION]
> 请务必阅读简单的 Url 设置对话框页面上的信息。 删除会议 URL 可以呈现现有会议由用户无法再进行访问计划。 考虑进行新的会议 URL 活动后保留现有的 URL。 当您确信用户都不再使用旧的会议 URL 时，您可以安全地删除它。 
  
要编辑或更改管理访问 URL，请编辑该条目。
  
您已完成编辑属性页上的所有更改后，单击**确定**以保存所做的更改。 单击“**取消**”将放弃更改。
