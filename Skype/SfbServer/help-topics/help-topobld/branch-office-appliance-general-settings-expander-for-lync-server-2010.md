# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010 分支机构设备常规设置扩展器
 
若要编辑属性高存活力的分支装置或自动恢复分支服务器，在**常规**下配置：
  
- **FQDN**： 键入高存活力的分支装置或自动恢复分支服务器的完全合格的域名称。
    
    **使用已配置的所有 IP 地址**使用高存活力的分支装置或自动恢复分支服务器上配置的所有目的的 IP 地址。
    
    **将服务用途限制为所选 IP 地址** 配置定义服务器的不同地址和要用于 PSTN 的 IP 地址。
    
    **主 IP 地址**：除了 PSTN 关联的功能，针对所有用途定义和配置此 IP 地址。
    
    **PSTN IP 地址**：与公用电话交换网 (PSTN) 功能关联的 IP 地址。
    
- 配置以确保其他服务器角色的配置和高存活力的分支装置或自动恢复分支服务器相关联的**关联**。
    
    **将存档服务器**从列表中选择您想要的高存活力的分支装置或自动恢复分支服务器相关联的存档服务器
    
    单击**新建**（如果还未创建要与此高存活力的分支装置或自动恢复分支服务器相关联的存档服务器
    
    **将监视服务器**从列表中选择您想要的高存活力的分支装置或自动恢复分支服务器相关联的监视服务器
    
    如果您尚未创建要与此高存活力的分支装置或自动恢复分支服务器相关联的监视服务器，请单击**新建**
    
    **将相关联的边池 （对于媒体组件）**从列表中选择您想要的高存活力的分支装置或自动恢复分支服务器相关联的边池的边缘服务器
    
 **确定** 接受更改并通过对话框提交更改。
  
 **取消** 放弃更改并关闭对话框。
  
 **帮助** 显示此帮助屏幕。
