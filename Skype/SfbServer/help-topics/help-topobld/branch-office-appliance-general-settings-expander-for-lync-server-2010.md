# <a name="branch-office-appliance-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010 分支机构设备常规设置扩展器
 
若要编辑的 Survivable Branch Appliance 或 Survivable Branch Server，在**常规**下属性配置：
  
- **FQDN**： 键入 Survivable Branch Appliance 或 Survivable Branch Server 的完全限定的域名。
    
    **使用所有已配置的 IP 地址**使用 Survivable Branch Appliance 或 Survivable Branch Server 上配置针对所有用途的 IP 地址。
    
    **将服务用途限制为所选 IP 地址** 配置定义服务器的不同地址和要用于 PSTN 的 IP 地址。
    
    **主 IP 地址**：除了 PSTN 关联的功能，针对所有用途定义和配置此 IP 地址。
    
    **PSTN IP 地址**：与公用电话交换网 (PSTN) 功能关联的 IP 地址。
    
- 配置**关联**确保其他服务器角色，配置并与 Survivable Branch Appliance 或 Survivable Branch Server 关联。
    
    **关联存档服务器**从列表中选择您想要与 Survivable Branch Appliance 或 Survivable Branch Server 关联的存档服务器
    
    单击**新建**，如果尚未创建要与此 Survivable Branch Appliance 或 Survivable Branch Server 关联的存档服务器
    
    **关联监控服务器**从列表中选择您想要与 Survivable Branch Appliance 或 Survivable Branch Server 关联的监控服务器
    
    单击**新建**，如果尚未创建要与此 Survivable Branch Appliance 或 Survivable Branch Server 关联的监控服务器
    
    **关联边缘池 （用于媒体组件）** 从列表中选择您想要与 Survivable Branch Appliance 或 Survivable Branch Server 关联的边缘池的边缘服务器
    
  **确定** 接受更改并通过对话框提交更改。
  
  **取消** 放弃更改并关闭对话框。
  
  **帮助** 显示此帮助屏幕。
