# <a name="external-application-next-hop-settings-expander"></a>外部应用程序下一个跃点设置扩展器
 
你可以通过从下拉列表中选择定义的企业版前端池或标准版前端服务器来指定受信任的应用程序服务器池的下一个跃点服务器。 Director 或控制器池不是受信任的应用程序服务器下一跃点的有效选择, 并且不会显示在列表中。 单击 **"确定"** 接受并保存所做的更改。 单击“**取消**”将放弃所做的更改并退出属性页面。
