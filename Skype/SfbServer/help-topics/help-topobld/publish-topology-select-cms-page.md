# <a name="publish-topology-select-cms-page"></a>发布拓扑选择 CMS 页面

您可以使用拓扑生成器发布已配置的拓扑。 系统将要求您从列表中进行选择，以便前端服务器或前端池担任保存中央管理存储的角色。 在任何给定时间，只有一个前端服务器或前端池可以保留此角色。

有关中央管理存储的详细信息，请参阅 Move the [Lync server 2010 Central Management Server To Lync Server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx)。
