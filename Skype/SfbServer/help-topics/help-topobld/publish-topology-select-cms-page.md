# <a name="publish-topology-select-cms-page"></a>发布拓扑选择 CMS 页面
 
发布已配置使用拓扑生成器的拓扑。 要求您从前端服务器或前端池将担任持有中央管理存储的角色的列表中进行选择。 在任何给定时间，只能有一个前端服务器或前端池可以容纳此角色。
