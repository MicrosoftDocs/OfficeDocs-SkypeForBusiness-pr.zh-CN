# <a name="publish-topology-select-cms-page"></a>发布拓扑选择 CMS 页面

你可以使用拓扑生成器发布已配置的拓扑。 系统将要求你从列表中选择前端服务器或前端池将承担拥有中央管理存储的角色。 在任何给定时间，只有一个前端服务器或前端池可以持有此角色。

有关中央管理存储的详细信息，请参阅[将 Lync server 2010 中央管理服务器移动到 Lync server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx)。
