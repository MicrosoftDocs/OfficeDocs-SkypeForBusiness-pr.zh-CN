# <a name="add-director-features"></a>添加控制器功能

如果希望主管池包含监视服务器, 请选择 "**监视" (CDR 和 QoE 度量值)**。

有关添加控制器池的更多详细信息, 请参阅 Director 和[设置 director](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)的[方案](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx)。
