# <a name="add-director-features"></a>添加控制器功能

如果您想要包括监控服务器的控制器池，选择**监视 （CDR 和 QoE 度量）**。

有关添加控制器池的详细信息，请参阅[方案 Director](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx)和[Setting Up the Director](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)。
