# <a name="add-edge-server-nat-ip-2010"></a>添加边缘服务器 NAT IP 2010
 
如果通过网络地址转换 (NAT) 转换此边缘池的公共 IP 地址，则必须指定要转换的公共 IP 地址。该地址应为 A/V 边缘服务的外部 IP 地址。
  
若要更改边缘池是否使用 NAT 转换，请右键单击边缘池的名称，然后单击“编辑属性”****。
