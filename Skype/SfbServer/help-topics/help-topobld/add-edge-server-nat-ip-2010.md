# <a name="add-edge-server-nat-ip-2010"></a>添加边缘服务器 NAT IP 2010
 
如果此边缘池的公用 IP 地址的网络地址转换 (NAT)，翻译必须指定要转换的公用 IP 地址。 这应该是外部 IP 地址的 A / V 边缘服务。
  
若要让边缘池使用 NAT 转换，击边池的名称，然后单击**编辑属性**。
