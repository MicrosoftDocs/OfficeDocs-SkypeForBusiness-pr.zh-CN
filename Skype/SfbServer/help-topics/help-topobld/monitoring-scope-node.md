# <a name="monitoring-scope-node"></a>监控范围节点
 
在 Lync Server 2013 拓扑生成器中，可以执行与运行监控的单个服务器相关的任务，方法是：在控制台树中右键单击要运行监视的服务器，单击工具栏中的 " **操作** "，或者在 "操作" 窗格中单击某个任务，然后为运行监视的服务器单击下列选项之一：
  
- **编辑属性**。 打开运行监视的服务器的 " **编辑属性** " 页，在其中可以更改运行监视或监视 SQL server 数据库的服务器的完全限定域名 (FQDN) 。
    
- **删除**。 从拓扑和监视 SQL Server 数据库中删除运行监视的服务器。
    
除了本主题之前介绍的监控服务器选项之外，“监控”节点还具有控制台树中的其他节点也具有的一些常规选项。其中包括以下选项：
  
- **拓扑**。 您可以使用此选项来定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑的副本、发布拓扑、安装数据库或删除部署。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
