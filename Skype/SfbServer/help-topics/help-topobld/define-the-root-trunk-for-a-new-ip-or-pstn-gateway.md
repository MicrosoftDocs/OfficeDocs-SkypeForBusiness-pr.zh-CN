# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>为新的 IP 或 PSTN 网关定义根中继线
 
配置下列的 ip 地址或公用交换的电话网络 (PSTN) 为定义根主干：
  
- **干线名称**： 定义与主干完全合格的域名称
    
- **IP/PSTN 网关的侦听端口**： 定义该干线将侦听的端口
    
- **SIP 传输协议**： 从列表中选择**TCP**或**TLS**，根据干线要求
    
- **相关中介服务器**： 从部署中可用的中介服务器的列表中选择
    
- **相关中介服务器端口**： 定义所选的中介服务器正在侦听的端口
