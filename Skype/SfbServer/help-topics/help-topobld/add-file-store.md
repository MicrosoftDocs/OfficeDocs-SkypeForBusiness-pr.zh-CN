# <a name="add-file-store"></a>添加文件存储
 
要定义新文件存储，请指定以下内容：
  
- 将托管新文件存储的服务器的完全限定域名 (FQDN)。 在发布拓扑之前，服务器的 FQDN 必须存在于域名系统 (DNS) 中。
    
- 主机服务器中的文件共享的名称。文件共享必须存在，然后才能发布拓扑。
