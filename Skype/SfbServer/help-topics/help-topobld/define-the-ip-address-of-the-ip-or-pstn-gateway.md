# <a name="define-the-ip-address-of-the-ip-or-pstn-gateway"></a>定义的 ip 地址或 PSTN 网关的 IP 地址
 
您通过选择**启用 IPv4**或**启用 IPv6**定义的 IP 或电话交换的电话网络 (PSTN) 网关的 IPv4 或 IPv6 地址。 您可以选择**使用所有已配置的 IP 地址**或**限制到选定的 IP 地址的服务使用**。
  
如果您选择**使用所有已配置的 IP 地址**，将使用配置网关的所有地址。 还可以选择**限制到服务使用所选 IP 地址**，这使**PSTN IP 地址**条目。 输入此网关分配的 IPv4 或 IPv6 地址。
