# <a name="file-store-settings-expander"></a>文件存储设置扩展器
 
要编辑已定义的文件存储的属性, 请执行以下操作:
  
对于文件共享名称, 你可以定义新共享的名称。 然后, 必须在 "文件服务器完全限定的域名 (FQDN)" 属性中指定的服务器上创建共享。
  
> [!NOTE]
> 不能更改 "文件服务器 FQDN" 属性中指定的服务器的 FQDN。 如果需要将文件存储文件共享放置在其他服务器上, 请创建新的文件存储项。
