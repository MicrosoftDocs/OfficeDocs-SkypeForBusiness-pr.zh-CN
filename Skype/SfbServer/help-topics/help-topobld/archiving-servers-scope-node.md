# <a name="archiving-servers-scope-node"></a>存档服务器范围节点
 
在拓扑生成器中，可以执行与单个存档服务器相关的任务，方法是右键单击控制台树中的 **存档服务器** ，单击工具栏中的 " **操作** "，或在 "操作" 窗格中单击某个任务，然后单击 "以下存档服务器" 选项。
  
- **新建存档服务器**。启动“定义新的存档服务器”**** 向导，从中指定新存档服务器的设置，包括完全限定的域名 (FQDN)、存档 SQL Server 数据库、存档文件存储和池关联。
    
> [!NOTE]
> 如果您之前已将存档服务器添加到拓扑生成器的拓扑中，则“存档服务器”节点将包含存档服务器。 但是，在设置存档服务器（包括在要用作存档服务器的服务器上安装 Skype for Business Server 并配置对存档的支持）之前，不能存档即时消息或 web 会议 (邮件) 。 
  
除了本主题中之前介绍的存档服务器选项之外，“存档服务器”节点还具有控制台树中的其他节点也具有的一些常规选项。其中包括以下选项：
  
- **拓扑**。 使用此选项可定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑的副本、发布拓扑、安装数据库、合并组件或删除部署。
    
- **视图**。使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。使用此选项访问活动节点的“帮助”。
