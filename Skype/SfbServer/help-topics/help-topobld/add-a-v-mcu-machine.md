# <a name="add-av-mcu-machine"></a>添加 A / V MCU 机
 
指定想要添加的每台计算机的完全限定的域名 (FQDN) / V 会议服务器在此 A / V 会议池。 将计算机添加到列表中后，可以更新计算机的 FQDN 或在发布拓扑前随时将其从池中删除。 发布拓扑之后，更改 FQDN 需要删除拓扑生成器中的服务器，然后将新服务器添加到具有新的 FQDN 的池。
