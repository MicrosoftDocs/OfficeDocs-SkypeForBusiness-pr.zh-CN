# <a name="legacy-merge-next-hop"></a>传统的合并下一跃点
 
**下一个跃点的 FQDN**标识传统边缘服务器将内部通信路由到下一个跃点服务器。 键入此边缘服务器将内部的通信路由到下一个跃点服务器的完全合格的域名称 (FQDN)，然后选择是否**前端池**或**控制器**。
