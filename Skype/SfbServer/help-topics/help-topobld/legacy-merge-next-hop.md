# <a name="legacy-merge-next-hop"></a>旧版合并下一跃点
 
“下一个跃点 FQDN”标识旧版边缘服务器将内部流量路由到的下一个跃点服务器。键入此边缘服务器将内部流量路由到的下一个跃点服务器的完全限定的域名 (FQDN)，然后选择该服务器是“前端池”还是“控制器”。
