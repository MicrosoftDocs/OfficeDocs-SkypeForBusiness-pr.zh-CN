# <a name="legacy-merge-add-expanded-edge"></a>旧版合并添加扩展边缘
 
Skype for Business Server 不支持合并旧版 Office Communications Server 2007 R2 扩展边缘服务器配置。 必须先将扩展边缘服务器转换成合并边缘服务器配置，然后才能合并边缘服务器信息。
