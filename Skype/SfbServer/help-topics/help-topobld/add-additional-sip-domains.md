# <a name="add-additional-sip-domains"></a>添加其他的 SIP 域
 
指定您想要支持您的部署中的每个其他域的域名称。 例如，如果您还想要为您部署中的 SIP 通信支持 fabrikam.net 作为主要的会话启动协议 (SIP) 域指定了 contoso.net，与其他受支持的域指定 fabrikam.net。
