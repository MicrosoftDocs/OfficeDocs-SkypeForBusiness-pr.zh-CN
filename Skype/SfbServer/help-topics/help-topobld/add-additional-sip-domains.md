# <a name="add-additional-sip-domains"></a>添加其他 SIP 域
 
指定要在部署中支持的每个其他域的域名。例如，如果指定了 contoso.net 作为主会话初始协议 (SIP) 域，并且要针对部署中的 SIP 流量支持 fabrikam.net，则指定 fabrikam.net 作为其他支持域。
