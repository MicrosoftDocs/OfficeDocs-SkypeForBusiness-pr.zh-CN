使用此页可以选择哪些前端池和标准版服务器将使用此边缘池 A / V 通信。 
  
每个前端池可与边缘池只能有一个相关联。 如果选择已与另一个边缘池池，将更改从以前的边缘池关联。
  
您还可以选择将此边缘服务器或池边缘与中介服务器或中介服务器池来配置和启用外部用户的企业语音相关联。
