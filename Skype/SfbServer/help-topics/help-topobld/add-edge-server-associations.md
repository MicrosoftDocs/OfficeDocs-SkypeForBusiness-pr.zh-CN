# <a name="add-edge-server-associations"></a>添加 Edge Server 关联
 
使用此页面选择哪些前端池和 Standard Edition 服务器会将此边缘池用于 A / V 流量。 
  
每个前端池可与只有一个边缘池相关联。 如果选择已与其他边缘池关联的池，将会更改离开的以前的边缘池的关联。
  
您还可以选择将此边缘服务器或边缘池与中介服务器或中介服务器池来配置和为外部用户启用企业语音。
