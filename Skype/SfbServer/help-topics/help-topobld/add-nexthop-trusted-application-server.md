# <a name="add-nexthop-trusted-application-server"></a>添加 NextHop 受信任应用程序服务器
 
要设置受信任应用程序服务器或池的下一个跃点，请执行下列操作：
  
在“下一个跃点池”下拉列表中，选择受信任应用程序池要用作下一个跃点的 Enterprise Edition 池或 Standard Edition 前端服务器。
  
单击“完成”以完成受信任应用程序池创建。
