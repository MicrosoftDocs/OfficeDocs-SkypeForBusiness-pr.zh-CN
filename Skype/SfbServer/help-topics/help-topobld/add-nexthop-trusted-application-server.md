# <a name="add-nexthop-trusted-application-server"></a>添加需要受信任的应用程序服务器
 
要设置的受信任的应用程序服务器或池的下一个跃点，执行以下操作：
  
在**下一个跃点池**下拉列表中，选择企业版池的标准版前端服务器所需的要用作下一个跃点的受信任的应用程序池。
  
单击**完成**以完成创建受信任的应用程序池。
