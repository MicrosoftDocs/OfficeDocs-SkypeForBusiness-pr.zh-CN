# <a name="add-survivable-branch-appliance-pool"></a>添加 Survivable Branch Appliance 池
 
**定义 Survivable Branch Appliance FQDN**，在**FQDN**文本框中键入 Survivable Branch Appliance 或 Survivable Branch Server 的完全限定的域名 (FQDN) （如定义在域名系统 (DNS)）。 单击“**下一步**”继续。
  
> [!NOTE]
> 除非它另有说明，否则，定义 Survivable Branch Appliance 或 Survivable Branch Server 是相同的过程。
