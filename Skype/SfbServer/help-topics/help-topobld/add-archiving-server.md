# <a name="add-archiving-server"></a>添加存档服务器
 
要在拓扑设计中部署存档服务器，则需要指定要在其上部署存档的服务器的完全限定域名 (FQDN)。 在发布包括存档和安装 Skype for Business 服务器的拓扑之前，你应该已完成部署存档的所有先决条件。
