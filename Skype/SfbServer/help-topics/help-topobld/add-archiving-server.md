# <a name="add-archiving-server"></a>添加存档服务器
 
要在拓扑设计中部署存档服务器，则需要指定要在其上部署存档的服务器的完全限定域名 (FQDN)。 之前发布的拓扑结构，其中包含存档并安装 Skype 业务服务器，则应该完成部署存档的所有先决条件。
