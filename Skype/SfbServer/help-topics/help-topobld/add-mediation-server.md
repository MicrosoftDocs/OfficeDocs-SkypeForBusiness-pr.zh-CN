# <a name="add-mediation-server"></a>添加中介服务器
 
若要定义中介服务器，请指定中介服务器池的完全限定的域名 (FQDN)。 如果要创建中介服务器池，则需要为要部署的中介服务器指定该池的名称。 您将在另一步骤中定义各台服务器的 FQDN。 如果要部署单台中介服务器，指定的 FQDN 应该是此单台中介服务器的 FQDN。 有关详细信息，请参阅[Deploy a Mediation Server in Topology Builder in Skype for Business Server 2015。](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
