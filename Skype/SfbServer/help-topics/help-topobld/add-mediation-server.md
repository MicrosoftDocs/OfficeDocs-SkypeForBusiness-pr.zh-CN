# <a name="add-mediation-server"></a>添加中介服务器
 
若要定义中介服务器，请指定中介服务器池完全合格的域名称 (FQDN)。 如果您正在创建的中介服务器池，将指定要部署中介服务器池的名称。 您将在另一步骤中定义各台服务器的 FQDN。 如果您正在部署一个中介服务器，则指定 FQDN 将单个的中介服务器的 FQDN。 有关详细信息，请参阅[部署在拓扑生成器在 Skype 业务服务器 2015年的中介服务器](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
