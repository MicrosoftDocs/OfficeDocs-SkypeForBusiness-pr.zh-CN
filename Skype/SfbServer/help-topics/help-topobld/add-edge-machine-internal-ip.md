# <a name="add-edge-machine-internal-ip"></a>添加边缘机内部 IP
 
使用此页可以指定的内部 IP 地址和用于边缘服务器的内部完全合格的域名称 (FQDN)。
  
您指定的 FQDN 必须与在服务器上配置的计算机名相同。 默认情况下，未加入域的计算机的计算机名称为短名称，而不是 FQDN。 拓扑生成器使用 FQDN，而不是短名称。 因此，您必须配置域名系统 (DNS) 后缀作为未加入到域边缘服务器进行部署的计算机的名称。 有关将 DNS 后缀添加到计算机名称的详细信息，请参阅[边缘支持配置 DNS](http://technet.microsoft.com/library/955493e6-aa29-424d-bb81-1ef87b3b15e3.aspx)
