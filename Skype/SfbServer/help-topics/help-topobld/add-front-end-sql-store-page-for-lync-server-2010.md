# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>添加 Lync Server 2010 的前端 SQL 存储页
 
为您要配置的新前端池定义 SQL Server 存储的属性。
  
> [!IMPORTANT]
> 如果要配置 Standard Edition server，将显示此对话框，但无法配置设置。 单击“下一步”**** 继续。
  
可以配置的设置：
  
- **使用之前定义的 Sql server 存储**：从前端池存储的列表中选择现有的 sql server 存储。
    
- 选择 " **定义新的 Sql server 存储** " 以创建新的 sql server 存储。
    
    在**Sql SERVER FQDN**中键入 sql server 的完全限定的域名
    
    最初选择的是“默认实例”****。如果偏好定义实例，则选择“命名实例”**** 并键入需要创建的实例的名称。
    
  **返回**：回到上一个对话框
  
  **下一步**：应用更改并移到下一步的对话框
  
  **取消**：放弃更改并取消配置
  
  **帮助**：显示帮助信息，例如本主题
