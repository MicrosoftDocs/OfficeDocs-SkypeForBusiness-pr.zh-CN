# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>为 Lync Server 2010 添加前端 SQL Store 页面
 
为要配置的新前端池定义 SQL Server 存储的属性。
  
> [!IMPORTANT]
> 如果你配置的是标准版服务器，将显示此对话框，但你将无法配置设置。 单击“**下一步**”继续。
  
可以配置的设置：
  
- **使用以前定义的 Sql server 应用商店**：从前端池存储的列表中选择现有 sql server 应用商店。
    
- 选择 "**定义新的 Sql server 应用商店**" 以创建新的 sql server 应用商店。
    
    在**Sql SERVER FQDN**中键入 sql server 的完全限定的域名
    
    默认情况下，将首先选择**默认实例**。 如果你希望定义实例，请选择 "**命名实例**"，然后键入你需要创建的实例的名称。
    
  **后退**：返回到上一个对话框
  
  **下一**步：提交更改并移动到 "下一步" 对话框
  
  **取消**：放弃更改并取消配置
  
  **帮助**：显示帮助信息，如本主题
