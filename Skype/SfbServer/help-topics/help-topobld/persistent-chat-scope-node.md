# <a name="persistent-chat-scope-node"></a>持久聊天范围节点
 
若要开始创建新的持久聊天服务器或持久聊天服务器池，在拓扑生成器**持久聊天池**上右键单击，然后选择**新的持久聊天池**
  
- **拓扑**。 您可以使用此选项来定义新的拓扑结构、 打开一个现有的拓扑结构、 下载当前拓扑、 保存一份拓扑、 发布拓扑、 安装数据库、 合并组件从以前版本的 Office 通信服务器，或删除部署。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
