# <a name="add-front-end-features-2010"></a>添加前端功能 2010

选择要为前端池启用的每个功能。 至少必须部署即时消息 (IM)，IM 始终启用且不能取消选择。 还可以选择部署其他功能，如会议、企业语音以及呼叫允许控制 (CAC)。 某些功能依赖于拓扑。

有关指定 Standard Edition Server 或 Enterprise Edition 池的功能的详细信息，请参阅部署文档中的"定义和配置前端池"或["Standard Edition Server"。](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)
