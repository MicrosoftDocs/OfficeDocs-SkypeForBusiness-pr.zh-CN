# <a name="legacy-merge-summary-page"></a>传统的合并摘要页
 
从 Microsoft Office 通信服务器 2007 R2 Lync Server 2013 到迁移使用拓扑生成器的功能合并现有的拓扑结构。 摘要屏幕指定的所有操作和办公通信服务器 2007 R2 的合并配置 Lync Server 2013 拓扑生成器配置期间所做的选择。
  
查看摘要并确定是否没有正确定义您的选择。 如果您在摘要中反映出来的选择感到满意，请继续合并而成。
