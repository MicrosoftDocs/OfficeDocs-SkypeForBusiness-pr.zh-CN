# <a name="legacy-merge-summary-page"></a>旧版合并摘要页

从 Microsoft Office Communications Server 2007 R2 迁移到 Lync Server 2013 使用拓扑生成器的功能合并现有拓扑。 摘要屏幕指定在配置将 Office Communications Server 2007 R2 合并到 Lync Server 2013 拓扑生成器配置期间执行的所有操作和选择。

查看摘要并确定是否正确定义了您的选择。如果满意摘要中所反映的选择，则继续执行合并。
