# <a name="legacy-merge-summary-page"></a>旧版合并摘要页面

从 Microsoft Office 通信服务器 2007 R2 迁移到 Lync Server 2013 使用拓扑生成器的功能合并现有拓扑。 "摘要" 屏幕指定在将 Office 通信服务器 2007 R2 合并到 Lync Server 2013 拓扑结构配置期间进行配置时所做的所有操作和选择。

查看摘要并确定所做的选择是否已正确定义。 如果对摘要中反映的选项感到满意，请继续合并。
