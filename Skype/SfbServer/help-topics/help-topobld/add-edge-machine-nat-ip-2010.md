# <a name="add-edge-machine-nat-ip-2010"></a>添加 Edge 机器 NAT IP 2010
 
如果此边缘池的公共 IP 地址由网络地址转换（NAT）转换，则必须指定要转换的公共 IP 地址。 这应该是 A/V 边缘服务的外部 IP 地址。
  
若要更改 Edge 池是否使用 NAT 转换，请右键单击 "边缘池" 的名称，然后单击 "**编辑属性**"。
