# <a name="add-edge-server-fqdn"></a>将边缘服务器 FQDN 添加
 
必须指定访问边缘服务完全合格的域名称 (FQDN)。 如果您没有选择**使用一个单一的 FQDN &amp; IP 地址**选项**选择功能**页上，您还必须指定 FQDN Web 会议边缘服务以及 A / V 边缘服务。
  
此外，如果您选择了**使用一个单一的 FQDN &amp; IP 地址**选项，必须为每个边缘服务指定一个不同的端口号 (建议的端口设置： 访问边缘的 444 服务、 Web 会议边缘服务 8057 和 443 a / V边缘服务）。 如果不选择该选项，可以为所有三个服务中使用相同的端口号 （例如 443)。
