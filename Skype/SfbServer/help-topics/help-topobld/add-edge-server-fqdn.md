# <a name="add-edge-server-fqdn"></a>添加边缘服务器 FQDN
 
必须指定访问边缘服务的完全限定域名 (FQDN)。 如果未在 "**选择功能**" 页上选择 "**使用单个 FQDN &amp; IP 地址**" 选项，则还必须为 Web 会议边缘服务和 a/V 边缘服务指定一个 fqdn。
  
此外，如果选择了 " **使用单个 FQDN &amp; IP 地址** " 选项，则必须为每个边缘服务指定不同的端口号。 (建议的端口设置：444用于访问边缘服务，8057用于 Web 会议边缘服务，以及 a/V 边缘服务) 的443。 如果未选择此选项，则可以为所有三个服务使用相同的端口号（如 443）。
