# <a name="add-simple-url"></a>添加简单 URL
 
若要向部署的定义中添加其他简单的 Url，请指定新的简单 URL 的 URL。 URL 的格式为_http://\<简单的 url 类型\>。\<域名\> _或_http://\<\>域名/简单 url 类型\>，其中简单 url 类型是下列情况之一：\<_
  
- 会议
    
- 拨入
    
> [!IMPORTANT]
> 你可以随时使用一个拨号的简单 URL 和一个 "匹配简单 URL"。 添加新的简单 URL 并使其成为活动 URL，将所有未来请求设置为使用新的简单 URL。 保留非活动的简单 URL 以处理会议和在较旧的简单 URL 处于活动状态时所做的其他请求。 
  
有关添加简单 URL 的更多详细信息，请参阅[Skype for Business Server 2015 中的简单 url 的 DNS 要求](../../plan-your-deployment/network-requirements/simple-urls.md)。
