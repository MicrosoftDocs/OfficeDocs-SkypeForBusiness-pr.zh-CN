# <a name="add-office-web-apps-server"></a>添加 Office Web Apps Server
 
**定义新 Office Web 应用程序服务器**向导部署中定义新的 Office Web 应用程序服务器。 请填写以下信息：
  
 **Office Web 应用程序服务器 FQDN**： 键入的服务器将承载 Office Web 应用程序服务器的完全限定的域名
  
 **Office Web 应用程序服务器搜索 URL**： 键入 Office Web 应用程序服务器的完整统一资源定位器 (URL)
  
> [!TIP]
> **Office Web 应用程序服务器搜索 URL**的默认行为是创建基于格式 Office Web 应用程序服务器的 FQDN 的 URL: `https://<FQDN of the Office Web Apps Server/hosting/discovery` 。 在大多数情况下，不需要更改默认格式。 您可能需要更改默认格式的 Office Web 应用程序服务器和 Office Web 应用程序服务器搜索 URL 必须不同。 例如，Office Web 应用程序服务器放置在外围网络中，将具有不同的 URL 的位置。
  
 **Office Web 应用程序服务器被部署在外部网络 （也就是说，外围/互联网）**： 如果 Office Web 应用程序服务器放置在内部防火墙，如外围网络、 外部网络或其他网络区域之外，请选中复选框这不是与您的内部网络。
