# <a name="legacy-merge-add-machine"></a>旧版合并添加计算机
 
若要定义组成边缘服务器池的计算机，请输入至少一台计算机的完全限定的域名 (FQDN) ，该计算机属于旧版负载平衡的统一边缘池。
