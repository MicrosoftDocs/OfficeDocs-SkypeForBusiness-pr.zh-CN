# <a name="legacy-merge"></a>传统的合并
 
**Web 会议外部 FQDN**允许外部用户参加内部会议。 传统的边缘服务器的 web 会议外部接口的输入完全合格的域名称 (FQDN)。
  
**外部的 Web 会议外部端口** **443**为默认传输控制协议 (TCP) 会话启动协议 (SIP) 的会议客户机配置的端口。 如果不使用默认值，更新的**外部 Web 会议外部端口**值。
  
如果您计划使用联盟此边缘服务器，请选择**此边缘池用于联盟和公用 IM 连接**复选框。 如果您有多个部署的边缘服务器，只有一个将启用联盟。 如果未选中此框，稍后再决定您想要启用联合身份验证，您必须再次运行合并拓扑生成器向导，以及发布您的拓扑结构。 有关详细信息，请参阅[阶段 4： 合并拓扑](http://technet.microsoft.com/library/81eb5bb2-1fd7-4611-a2aa-eb2393c8abc9.aspx)。
