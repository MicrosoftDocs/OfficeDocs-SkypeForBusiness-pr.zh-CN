# <a name="av-mcu-settings-expander"></a>AV MCU 设置扩展器
 
要编辑的属性的 a / V 会议服务器池具有已定义的情况下，执行以下操作：
  
## <a name="general-settings"></a>常规设置

若要更新的完全限定的域名 (FQDN) / V 会议服务器池，类型为该池的名称定义在域名系统 (DNS)。 您必须具有 DNS 主机 (A) 记录对应与 A / V 会议服务器池的名称。
  
## <a name="av-conferencing-service"></a>A / V 会议服务

要关联 A / V 会议服务器池，选择前端池或多个必须与 A / V 会议服务器池。
  
> [!NOTE]
> 如果没有前端池都显示在**选择要与此 A 关联的前端池 / V 会议池**窗格中，或者已并置的 A / V 会议服务器与前端池，或未选择要在**功能中添加会议和功能**前端池属性的部分。
  


完成后，单击**确定**以完成编辑 A / V 会议服务器池设置。 单击“**取消**”将放弃所有更改。
