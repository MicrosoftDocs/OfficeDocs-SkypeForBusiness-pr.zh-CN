# <a name="av-mcu-settings-expander"></a>AV MCU 设置扩展器
 
若要编辑的属性 A / V 会议服务器池，您已经定义，请执行以下操作：
  
## <a name="general-settings"></a>常规设置

若要更新一个完全限定的域名 (FQDN) / V 会议服务器池，池名称的类型定义在域名系统 (DNS)。 您必须具有一个对应的 DNS 主机 (A) 记录 / V 会议服务器池名称。
  
## <a name="av-conferencing-service"></a>A / V 会议服务

相关联的 A / V 会议服务器池中选择前端池或池有与 A / V 会议服务器池。
  
> [!NOTE]
> 如果没有前端池显示在**选择将与此相关联的前端池 / V 会议池**窗格中，既有搭配 A / V 会议服务器前端池，或没有选择在**功能中添加会议功能**的前端池属性的部分。
  
## 

完成后，单击**确定**以完成编辑 A / V 会议服务器池设置。 单击“**取消**”将放弃所有更改。
