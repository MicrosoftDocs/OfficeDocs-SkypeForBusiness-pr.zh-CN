# <a name="exchange-unified-messaging-planning-tool"></a>Exchange Unified Messaging (Planning Tool)

Skype for Business Server 2015 支持与 Exchange 统一消息 (UM) 集成, 以将语音消息和电子邮件合并到单个消息传递基础结构中。
