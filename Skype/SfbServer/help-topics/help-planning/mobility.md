# <a name="skype-for-business-server-mobility-planning-tool"></a>Skype for Business Server：移动规划工具
 
Skype for Business Server 2015 中的移动功能支持Skype for Business移动功能。
  
移动功能与 Lync Server 2013 中的功能相同。 有关准备环境实现移动的更多详细信息，请参阅[Planning for mobility in Lync Server 2013。](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility)
