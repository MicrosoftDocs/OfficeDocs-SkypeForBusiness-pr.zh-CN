# <a name="mobility-planning-tool"></a>移动 （规划工具）
 
在业务服务器 2015年的 Skype 的移动功能支持 Skype 在移动设备上的业务功能。
  
移动性功能是 Lync Server 2013 一样。 有关准备用于移动环境的更详细信息，请参阅[规划 Lync Server 2013 中移动](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)。
