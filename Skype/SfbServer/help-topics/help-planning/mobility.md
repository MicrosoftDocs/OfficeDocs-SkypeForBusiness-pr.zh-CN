# <a name="mobility-planning-tool"></a>Mobility (Planning Tool) 
 
Skype for Business Server 2015 中的移动功能支持移动设备上的 Skype for Business 功能。
  
移动功能与 Lync Server 2013 中的功能相同。 有关准备环境实现移动的更多详细信息，请参阅规划 Lync [Server 2013](https://technet.microsoft.com/library/hh689981%28v=ocs.15%29.aspx)中的移动功能。
