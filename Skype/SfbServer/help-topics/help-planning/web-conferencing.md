# <a name="web-conferencing-planning-tool"></a>Web 会议 （规划工具）
 
Skype 的业务服务器 2015 web 会议包括文档协作、 共享应用程序和桌面共享。
  
> [!NOTE]
> 商业服务器 2015年的 Skype 使用 Office Web 应用程序和 Office Web 应用程序服务器来处理共享的 PowerPoint 演示文稿的呈现。 在 Microsoft Lync 2013 中一样，网络会议功能作用完全相同。 有关安装和配置 Office Web 应用程序服务器的详细信息，请参阅[配置与的集成 Office Web 应用程序服务器以及 Lync Server 2013](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx)。
