# <a name="ip-version-planning-tool"></a>IP 版本 (规划工具) 
 
在 Skype for Business Server 2015 中配置 IP 地址时，有三个选项可供选择。 可以将 Skype for Business Server 2015 配置为仅支持 IP 版本 4 (IPv4) 、仅支持 IP 版本 6 (IPv6) ，或同时支持这两个 (（称为双协议栈) ）。
  
有关选择 IP 基础结构的更多详细信息，请参阅 Plan [for IPv6 in Skype for Business。](../../plan-your-deployment/network-requirements/ipv6.md)
