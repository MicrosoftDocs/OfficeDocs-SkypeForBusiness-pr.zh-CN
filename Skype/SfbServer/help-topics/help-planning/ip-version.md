# <a name="skype-for-business-server-ip-version-planning-tool"></a>Skype for Business Server IP 版本规划工具
 
在 2015 年 3 月配置 IP 地址时，Skype for Business Server三个选项。 您可以将 Skype for Business Server 2015 配置为仅支持 IP 版本 4 (IPv4) 、仅支持 IP 版本 6 (IPv6) ，或支持两者的组合 (称为双协议栈) 。
  
有关选择 IP 基础结构的更多详细信息，请参阅规划 [IPv6 Skype for Business](../../plan-your-deployment/network-requirements/ipv6.md)。
