# <a name="monitoring-planning-tool"></a>监视 （规划工具）
 
在业务服务器 2015年的 Skype，监视用于收集用法信息和体验质量 (QoE) 中涉及用户的通信会话有关的数据。
  
有关监视服务器规划的详细信息，请参阅[监视计划](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)。
