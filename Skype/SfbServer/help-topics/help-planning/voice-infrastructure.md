# <a name="voice-infrastructure-planning-tool"></a>Voice Infrastructure (Planning Tool)
 
Skype for Business Server 2015 通过使用 SIP 中继、直接 SIP 连接到 PSTN 网关或直接 SIP 连接到 PBX，提供可靠、可扩展的 PSTN 连接。 有关详细信息，请参阅[在 Skype for Business Server 2015 中规划 PSTN 连接](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)。
