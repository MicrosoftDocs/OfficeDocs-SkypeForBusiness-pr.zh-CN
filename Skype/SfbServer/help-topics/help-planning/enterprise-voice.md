# <a name="enterprise-voice-planning-tool"></a>企业语音 （规划工具）
 
企业语音业务服务器 2015年的 Skype 提供独立语音 (voip)，提出要增强或替代传统的专用分组交换机 (PBX) 系统。 企业语音用户可以调用的同事组织的 VoIP 网络或 PBX，并且它们可以调用您组织外的传统电话号码。 企业语音解决方案包括公共电话功能，如答案、 转发、 传输、 保存、 转移、 释放和停放，并增强 9-1-1 (E9-1-1) 调用 （E9-1-1 是仅在美国可用。企业语音还支持一系列当前和旧的 IP 和 USB 设备。
  
有关企业语音计划的更多详细信息，请参阅[规划业务服务器 2015年的 Skype 在企业语音](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md)。 部署独立的中介服务器的详细信息，请参阅[部署在拓扑生成器在 Skype 业务服务器 2015年的中介服务器](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
