# <a name="enterprise-voice-planning-tool"></a>企业语音 (规划工具) 
 
借助 企业语音，Skype for Business Server 2015 提供独立的 Internet 语音 (VoIP) 产品，以增强或替换传统的专用交换机 (PBX) 系统。 企业语音可以呼叫您组织的 VoIP 网络或 PBX 上的同事，并且他们可以呼叫组织外部的传统电话号码。 企业语音 解决方案包括常见呼叫功能，如应答、转接、转移、保留、转移、释放和保留，以及增强型 9-1-1 (E9-1-1) 呼叫 (E9-1-1 仅在美国可用。) 企业语音 还支持各种当前和较旧的 IP 和 USB 设备。
  
有关规划 skype for Business Server 企业语音的详细信息，请参阅[Plan for 企业语音 in Skype for Business Server 2015。](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md) 有关部署独立中介服务器的更多详细信息，请参阅 [Skype for Business Server 2015](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)中的拓扑生成器中部署中介服务器。
