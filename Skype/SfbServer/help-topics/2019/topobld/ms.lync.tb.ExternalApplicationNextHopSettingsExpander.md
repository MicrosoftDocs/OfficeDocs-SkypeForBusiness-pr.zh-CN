# <a name="external-application-next-hop-settings-expander"></a>外部应用程序下一个跃点设置扩展器
 
您可以通过从下拉列表中选择的定义的 Enterprise Edition 前端池或 Standard Edition 前端服务器指定受信任应用程序服务器池的下一个跃点服务器。 控制器或控制器池不是有效的选定内容的受信任应用程序服务器下一个跃点，并且它将不会出现在列表中。 单击**确定**接受并保存所做的更改。 单击“**取消**”将放弃所做的更改并退出属性页面。
