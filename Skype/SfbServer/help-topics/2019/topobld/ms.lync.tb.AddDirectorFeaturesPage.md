# <a name="add-director-features"></a>添加控制器功能

如果希望控制器池包含监控服务器，请选择"监控" (**CDR 和 QoE 指标)**。

有关添加控制器池的更多详细信息，请参阅 [控制器](/previous-versions/office/lync-server-2013/lync-server-2013-scenarios-for-the-director) 的方案和 [设置控制器](/previous-versions/office/lync-server-2013/lync-server-2013-setting-up-the-director)。
