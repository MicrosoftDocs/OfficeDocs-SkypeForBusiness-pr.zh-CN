# <a name="add-nexthop-trusted-application-server"></a>添加受信任应用程序服务器下一个跃点
 
若要为受信任的应用程序服务器或池设置下一跃点, 请执行下列操作:
  
在**下一个 "跃点池**" 下拉列表中, 选择希望受信任的应用程序池用作下一个跃点的企业版池或标准版前端服务器。
  
单击 "**完成**" 以完成受信任的应用程序池创建。
