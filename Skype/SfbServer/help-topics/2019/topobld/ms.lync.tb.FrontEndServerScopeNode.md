# <a name="front-end-server-scope-node"></a>前端服务器范围节点
 
此节点显示该前端池或 Standard Edition Server 的配置信息。可以执行以下操作：
  
- **新建服务器**。如果是前端池，则可以选择该选项以启动“定义新的服务器”向导，向该池中添加其他前端服务器。
    
- **编辑属性**。打开该池或服务器的“编辑属性”页，在其中可以启用和禁用功能，指定与文件存储、SQL Server 数据库及其他服务器的关联，并设置 Web 服务器和中介服务器信息。
    
- **删除**。从拓扑中删除前端池或 Standard Edition Server。
    
除了这些选项之外，还可以执行以下常规任务：
  
- **拓扑**。 可以使用此选项来定义新的拓扑、打开现有拓扑、下载当前拓扑、保存拓扑副本、发布拓扑、安装数据库、合并组件或删除部署。
    
- **视图**。可以使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
