# <a name="add-edge-server-associations"></a>添加 Edge Server 关联
 
使用此页面选择哪些前端池和标准版服务器将此 Edge 池用于 A/V 流量。 
  
每个前端池只能与一个 Edge 池相关联。 如果你选择的池已与另一个 Edge 池关联, 则它将更改与上一个边缘池关联的关联。
  
你还可以选择将此 Edge 服务器或边缘池与中介服务器或中介服务器池相关联, 以便为外部用户配置和启用企业语音。
