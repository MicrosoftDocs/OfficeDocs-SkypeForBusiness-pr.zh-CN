# <a name="associate-branch-office-appliance-with-front-end"></a>将分支机构设备与前端关联
 
若要将 Survivable Branch Appliance 或 Survivable Branch Server 与前端池或 Standard  Edition Server 关联，请在"选择前端池"页上，从前端池下拉列表中选择前端池或 Standard Edition Server。 单击“下一步”继续。
