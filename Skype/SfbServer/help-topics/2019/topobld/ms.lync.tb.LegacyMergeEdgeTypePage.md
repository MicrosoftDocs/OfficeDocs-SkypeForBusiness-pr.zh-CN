# <a name="legacy-merge-edge-type"></a>旧版合并边缘类型
 
为**单台边缘服务器**或**负载平衡合并边缘**服务器，单击**下一步**来指定配置信息。 对于**扩展边缘**，必须在每台服务器的配置更改到合并边缘配置之前可以合并。
