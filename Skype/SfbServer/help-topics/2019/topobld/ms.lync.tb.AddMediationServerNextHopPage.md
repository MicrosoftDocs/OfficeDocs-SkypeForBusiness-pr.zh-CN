# <a name="add-mediation-server-nexthop"></a>添加中介服务器下一跃点
 
要定义要定义的中介服务器的下一个跃点，请从"下一个跃点池"下拉列表中选择将充当中介服务器的下一个跃点的 Enterprise Edition 服务器或 Standard Edition 服务器。 单击“下一步”继续。
