# <a name="add-mediation-server-nexthop"></a>添加中介服务器下一跃点
 
若要为你定义的中介服务器定义下一跃点, 请从**下一个 "跃点池**" 下拉列表中选择要充当中介服务器的下一跃点的企业版服务器或标准版服务器。 单击“**下一步**”继续。
