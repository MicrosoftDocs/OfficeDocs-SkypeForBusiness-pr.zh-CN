# <a name="add-front-end-features-2010"></a>添加前端功能 2010
 
选择您要在其中启用的前端池的每个功能。 必须至少部署即时消息 (IM) 哪些总是处于启用状态，无法取消选择。 您还可以选择部署其他功能，如会议，企业语音和呼叫允许控制 (CAC)。 某些功能是取决于您的拓扑。
  
有关指定 Standard Edition server 或企业版池的功能的详细信息，请参阅部署文档中的[Define and Configure a Front End Pool or Standard Edition Server](http://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx) 。
