# <a name="add-branch-site-details"></a>添加分支机构详细信息
 
要指定站点的站点详细信息，请执行下列操作：
  
指定新分支站点的城市名称。“城市”值是可选条目。
  
指定新分支站点的省/市/自治区名称。“省/市/自治区”值是可选条目。
  
指定新分支站点的国家/地区代码名称。“国家/地区代码”值是可选条目。
  
有关其他信息，请参阅[Sites](http://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx)。
