# <a name="add-default-sip-domain"></a>添加默认 Sip 域
 
指定要部署以支持的主要会话初始协议 (SIP) 域。 主 SIP 域可以是相同，或在不同您 Active Directory 域服务的域。 例如，Active Directory 域无法是 fabrikam.net，并且您可以指定为 contoso.net 您主 SIP 域。
