# <a name="add-default-sip-domain"></a>添加默认 SIP 域
 
指定你希望部署支持的主会话初始协议（SIP）域。 主要 SIP 域可以与你的 Active Directory 域服务域相同，或不同于你的 Active Directory 域服务域。 例如，你的 Active Directory 域可能是 fabrikam.net，你可以将你的主要 SIP 域指定为 contoso.net。
