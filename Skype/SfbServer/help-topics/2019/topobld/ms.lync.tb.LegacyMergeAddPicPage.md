# <a name="legacy-merge"></a>旧版合并
 
**Web 会议的外部 FQDN**允许外部用户可以加入内部会议。 输入旧边缘服务器的 web 会议外部接口的完全限定的域名 (FQDN)。
  
**外部 Web 会议的外部端口** **443**的值是为会议客户端配置的默认传输控制协议 (TCP) 会话初始协议 (SIP) 端口。 如果不使用默认值，更新**外部 Web 会议的外部端口**值。
  
如果您打算对联盟使用此边缘服务器，请选择**此边缘池用于联盟和公共 IM 连接**复选框。 如果您有多台边缘服务器部署，其中的一个仅将启用联盟。 如果不选中此框，并且您以后决定要启用联盟，您必须再次运行拓扑生成器合并向导，以及发布拓扑。 有关详细信息，请参阅[第 4 阶段： 合并拓扑](http://technet.microsoft.com/library/81eb5bb2-1fd7-4611-a2aa-eb2393c8abc9.aspx)。
