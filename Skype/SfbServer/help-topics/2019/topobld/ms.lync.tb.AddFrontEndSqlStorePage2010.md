# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>添加 Lync Server 2010 的前端 Sql 存储页
 
定义新前端池所配置的 SQL Server 存储的属性。
  
> [!IMPORTANT]
> 如果您配置 Standard Edition server，则显示此对话框，但无法配置设置。 单击“**下一步**”继续。
  
您可以配置的设置：
  
- **使用先前定义的 SQL Server 存储**： 从前端池存储的列表中选择现有的 SQL Server 存储。
    
- 选择**定义新的 SQL Server 存储**以创建新的 SQL Server 存储。
    
    在**SQL Server FQDN**中键入 SQL Server 的完全限定的域名
    
    最初选择**默认实例**。 如果您希望定义实例，选择**命名实例**，然后键入您需要创建实例的名称。
    
 **返回**： 回到上一个对话框
  
 **下一步**： 应用更改并将移至下一步的对话框
  
 **取消**： 放弃更改并取消配置
  
 **帮助**： 显示帮助信息，如本主题
