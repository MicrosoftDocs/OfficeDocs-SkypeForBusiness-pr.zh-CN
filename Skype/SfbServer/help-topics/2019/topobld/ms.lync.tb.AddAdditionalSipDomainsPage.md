# <a name="add-additional-sip-domains"></a>添加其他 SIP 域
 
指定要在您部署中支持的每个其他域的域名。 例如，如果您还希望支持的部署中的 SIP 流量 fabrikam.net contoso.net 指定作为主要的会话初始协议 (SIP) 域，为其他支持域指定 fabrikam.net。
