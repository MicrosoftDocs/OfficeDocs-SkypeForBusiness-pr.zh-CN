# <a name="add-av-mcu-machine"></a>添加 A/V MCU 计算机
 
指定要添加为此 A/V 会议池中 A/V 会议服务器的每台计算机的完全限定域名 (FQDN)。将计算机添加到列表中后，可以更新计算机的 FQDN 或在发布拓扑前随时将其从池中删除。发布拓扑后，更改 FQDN 要求在拓扑生成器中删除相应的服务器，然后向池中添加具有新 FQDN 的新服务器。
