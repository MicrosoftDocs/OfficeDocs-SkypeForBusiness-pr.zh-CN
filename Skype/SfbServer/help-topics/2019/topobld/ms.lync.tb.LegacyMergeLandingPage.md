# <a name="legacy-merge-landing"></a>旧版合并登录
 
通过 "**指定边缘设置**" 页面，你可以配置旧版边缘服务器信息。 单击 "**添加**" 以指定你的 Office 通信服务器部署的旧版 Edge 服务器版本。 如果您的旧环境中没有任何边缘服务器，请单击 "**下一步**" 以继续。
