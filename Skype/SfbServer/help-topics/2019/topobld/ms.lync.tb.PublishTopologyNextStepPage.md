# <a name="publish-topology-next-step"></a>发布拓扑下一步

拓扑发布完成后，将显示 "**发布向导完成**" 页面。

在发布拓扑过程中完成步骤时，将创建日志，其中包含有关已执行的任务的信息。 若要查看与给定步骤相关的日志文件，请选择查看日志文件所需的步骤，然后单击 "**查看日志**"。

> [!NOTE]
> 在发布拓扑流程期间，并非所有步骤都有关联的日志文件。 如果你选择了一个步骤并**查看日志**可用，此步骤将创建一个日志文件。 如果 "**查看日志**" 按钮不可用（灰显），则不会为此步骤创建日志文件。

若要查看建议的后续步骤以完成拓扑，请单击链接 **"单击此处打开待办事项列表**"。 通过单击此链接打开的文本文件将指示必须运行本地安装的服务器以及完成部署所需执行的任何其他步骤。 对于列出的每个任务，请参阅部署文档中的相应部分。

单击 "**完成**"

有关详细信息，请参阅[发布拓扑](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)。
