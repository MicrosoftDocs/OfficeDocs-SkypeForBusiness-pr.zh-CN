# <a name="add-file-store"></a>添加文件存储
 
若要定义新的文件存储，请指定以下内容：
  
- 将托管新文件存储的服务器的完全限定的域名（FQDN）。 在发布拓扑之前，服务器的 FQDN 必须存在于域名系统（DNS）中。
    
- 主服务器上文件共享的名称。 在发布拓扑之前，必须存在文件共享。
