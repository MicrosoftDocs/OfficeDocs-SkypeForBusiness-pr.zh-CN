# <a name="add-file-store"></a>添加文件存储
 
要定义新文件存储，请指定以下内容：
  
- 完全限定的域名 (FQDN) 的服务器的将承载新的文件存储。 发布拓扑之前，对服务器的 FQDN 必须存在在域名系统 (DNS)。
    
- 主机服务器上的文件共享名称。 发布拓扑之前，必须存在的文件共享。
