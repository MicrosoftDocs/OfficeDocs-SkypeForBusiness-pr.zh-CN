# <a name="publish-topology"></a>发布拓扑
 
开始发布过程时，将显示 "**发布拓扑"** 页面。 在继续操作之前，请阅读并确认列出的任务是否完成。 准备好继续时，单击 "**下一步**"。 如果您必须关闭发布过程，请单击 "**取消**"。
