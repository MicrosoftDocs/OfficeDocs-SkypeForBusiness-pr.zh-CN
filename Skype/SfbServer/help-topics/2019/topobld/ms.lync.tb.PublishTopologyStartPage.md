# <a name="publish-topology"></a>发布拓扑
 
开始发布过程时将显示“发布拓扑”页。继续操作前，请阅读并确认列出的任务已完成。准备好继续操作时，单击“下一步”。如果必须停止发布过程，请单击“取消”。
