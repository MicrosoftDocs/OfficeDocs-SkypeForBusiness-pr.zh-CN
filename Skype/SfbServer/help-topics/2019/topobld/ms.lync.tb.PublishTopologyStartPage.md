# <a name="publish-topology"></a>发布拓扑
 
开始发布过程时，将显示**发布拓扑**页。 阅读并确认列出的任务已完成，然后您才能继续。 当您准备好继续时，单击**下一步**。 如果您必须关闭超出发布过程，请单击**取消**。
