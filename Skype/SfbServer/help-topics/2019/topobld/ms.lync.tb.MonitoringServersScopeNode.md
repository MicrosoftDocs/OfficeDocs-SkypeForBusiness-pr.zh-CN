# <a name="monitoring-servers-scope-node"></a>监控服务器范围节点
 
**监控服务器**节点列出了监控服务器拓扑中。 可以执行以下操作：
  
- **新的监控服务器**。 启动**定义新的监控服务器**向导，用于向拓扑中添加新的监控服务器。
    
除了监控服务器选项，可以执行以下常规任务：
  
- **拓扑**。 此选项可用于定义新拓扑、 打开现有拓扑、 下载当前拓扑、 保存一份拓扑、 发布拓扑，安装数据库、 合并从早期版本的 Office Communications Server 组件或删除部署。
    
- **视图**。可以使用此选项通过在控制台树视图中选择要显示和隐藏的项目来自定义拓扑生成器视图。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
