# <a name="add-mediation-machine"></a>添加中介计算机
 
指定要添加为此池中前端服务器的每台计算机的完全限定的域名 (FQDN)。 将计算机添加到列表中后，可以更新计算机的 FQDN 或在发布拓扑前随时将其从池中删除。 发布拓扑后，更改 FQDN，必须删除拓扑生成器中的服务器，然后将新服务器添加到新的 FQDN 与池。
