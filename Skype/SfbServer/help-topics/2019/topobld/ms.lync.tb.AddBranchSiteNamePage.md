# <a name="add-branch-site-name"></a>添加分支站点名称

要定义新的分支站点，请指定以下内容：

新分支站点的名称。定义的名称在部署中必须是唯一的。可使用您所选定的任何命名约定。通常建议使用城市、位置名称、分支机构名称。名称值是必需条目。

可选的说明字段提供了指定其他描述性信息的位置，这些信息不会包含在基础结构名称中，但为您提供了输入进一步描述站点的文本的位置。

有关详细信息，请参阅[站点](/previous-versions/office/lync-server-2013/lync-server-2013-sites)。
