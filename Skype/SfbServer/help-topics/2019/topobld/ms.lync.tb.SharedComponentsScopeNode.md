# <a name="shared-components-scope-node"></a>共享组件范围节点
 
右键单击**共享**组件在拓扑生成器层次结构树中，创建新的共享的组件或访问拓扑级别选项。
  
![共享组件范围节点](../../../media/Shared_Components_Scope_Node.jpg)
  
选择**新建**创建新的文件存储、 新的 IP/PSTN 网关、 新的 Office Web Apps Server、 新的 SQL Server 存储或新的中继。
  
> [!NOTE]
> 每个选项，请参阅帮助主题。 
  
![共享组件范围节点](../../../media/Shared_Components_Scope_NodeB.jpg)
  
选择**拓扑**选项可打开一个新拓扑文档上单击。 打开现有拓扑文档，从中央管理存储下载当前拓扑、 保存副本发布当前拓扑文档，安装数据库创建的当前配置的数据库配置，当前拓扑文档到当前部署，或删除部署合并 Office Communications Server 2007 R2。
  
![共享组件范围节点 C](../../../media/Shared_Components_Scope_NodeC.jpg)
