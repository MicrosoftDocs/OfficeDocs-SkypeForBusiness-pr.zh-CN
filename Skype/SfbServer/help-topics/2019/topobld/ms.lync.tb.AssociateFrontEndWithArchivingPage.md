# <a name="associate-front-end-with-archiving"></a>将前端与存档关联
 
每个前端池和 Survivable Branch Appliance 只能有一个与之关联的存档服务器。 启用对网站的存档后，存档服务器然后收集并记录即时消息 (IM) 消息内容和 web 会议内容的用户所参与对话驻留在关联的池和 Survivable Branch Appliance 上。 
  
在站点的所有池和多个中央站点的池可以使用相同的存档服务器，如果使用量不超过存档服务器的容量。
