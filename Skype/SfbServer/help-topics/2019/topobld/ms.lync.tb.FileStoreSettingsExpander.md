# <a name="file-store-settings-expander"></a>文件存储设置扩展器
 
若要编辑的定义的文件存储的属性：
  
对于文件共享名称，您可以定义新的共享的名称。 然后，您必须在名为完全限定的文件服务器域的域名 (FQDN) 属性中的服务器上创建共享。
  
> [!NOTE]
> 不能更改文件服务器 FQDN 属性中指定服务器的 FQDN。 如果您需要将文件存储文件共享放在不同的服务器上，您将创建新的文件存储区项目。
