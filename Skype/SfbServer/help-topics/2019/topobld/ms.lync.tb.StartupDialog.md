# <a name="topology-builder-startup-dialog"></a>拓扑生成器启动对话框
 
启动拓扑生成器配置的属性和部署的配置。 在启动时，拓扑生成器显示一个对话框，询问将哪些拓扑文档的来源。 您选择三个选项之一：
  
- **从现有部署下载拓扑**如果您已运行拓扑生成器并已成功发布到中央管理服务器拓扑，请选择此选项。 此选项请求和下载的当前配置，从而使您可以编辑和配置现有部署的本地副本。
    
- **从本地文件打开拓扑**如果您配置的是部署，但尚未发布配置，请选择此选项。 此选项允许您在拓扑生成器中保存的配置工作的本地副本和回到其在以后的时间。
    
- **新的拓扑**选择此选项以开始新拓扑文档的配置。 此选项可打开一个新的拓扑文档的配置以满足您将配置的部署要求。
