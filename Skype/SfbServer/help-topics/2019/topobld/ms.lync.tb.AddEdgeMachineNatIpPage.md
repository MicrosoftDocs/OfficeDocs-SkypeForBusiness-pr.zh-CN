# <a name="add-edge-machine-nat-ip"></a>添加边缘计算机 NAT IP
 
如果此边缘池的公共 IP 地址已翻译的网络地址转换 (NAT)，则必须指定要转换的公共 IP 地址。 这应该是外部 IP 地址的 A / V 边缘服务。
  
若要更改边缘池是否使用 NAT 转换，右键单击边缘池的名称，然后单击**编辑属性**。
