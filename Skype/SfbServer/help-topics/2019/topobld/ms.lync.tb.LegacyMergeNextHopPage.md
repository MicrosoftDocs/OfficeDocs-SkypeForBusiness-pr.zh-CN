# <a name="legacy-merge-next-hop"></a>旧版合并下一个跃点
 
**下一跃点 FQDN**标识了旧式服务器路由内部流量的下一跃点服务器。 键入此边缘服务器路由内部流量的下一个跃点服务器的完全限定的域名（FQDN），然后选择它是**前端池**还是**控制器**。
