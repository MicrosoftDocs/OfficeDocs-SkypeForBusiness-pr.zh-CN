# <a name="add-mediation-server"></a>添加中介服务器
 
若要定义中介服务器，请指定中介服务器池的完全限定的域名 (FQDN)。 如果您正在创建中介服务器池，您将指定要部署中介服务器池的名称。 您将在另一步骤中定义各台服务器的 FQDN。 如果您正在部署一台中介服务器，您指定的 FQDN 将是单个中介服务器的 FQDN。 有关详细信息，请参阅[Deploy Skype 业务服务器中的拓扑生成器中的中介服务器](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
