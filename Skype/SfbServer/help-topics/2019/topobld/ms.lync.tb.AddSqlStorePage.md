# <a name="add-sql-store"></a>添加 SQL 存储

若要定义新的 SQL 存储，这意味着要指定基于 SQL Server 数据库和 SQL Server 的实例 — 默认实例或命名的实例 — 指定以下。

指定将承载您要定义的数据库实例的 SQL Server 的完全限定的域名 (FQDN)。

指定将承载数据的 SQL Server 实例。 您可以指定默认的实例，或者您可以指定的命名的实例。

应该非常清楚地了解并置的特定实例中的数据库。 有关服务器并置和数据库实例并置的详细信息，请参阅[服务器并置在前端最终池部署](https://technet.microsoft.com/library/0516b18d-14c0-4237-9279-0f92e341b1bd.aspx)和[服务器并置在 Standard Edition 服务器部署中](https://technet.microsoft.com/library/0763ffab-4fd6-463a-8e62-d97876b376d3.aspx)。
