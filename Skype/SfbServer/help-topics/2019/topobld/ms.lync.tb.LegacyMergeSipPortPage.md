# <a name="legacy-merge-sip-port"></a>旧版合并 SIP 端口
 
与客户端、旧版前端池和服务器通信的默认传输控制协议 (TCP) SIP 端口为 5061。如果未使用默认值，则选择“**所有池和服务器的 SIP 端口：**”并输入新值。
  
如果旧版部署配置了自动客户端登录，请选择“**自动发现每个池的 SIP 端口配置**”，并单击“**下一步**”继续。
