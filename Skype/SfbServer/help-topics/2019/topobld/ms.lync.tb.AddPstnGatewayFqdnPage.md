# <a name="define-the-pstn-gateway-fqdn"></a>定义 PSTN 网关 FQDN

指定公用电话交换网 (PSTN) 网关的完全限定域名 (FQDN) ，该网关可将 企业语音 组件连接到 PSTN 或专用交换机 (PBX) 。

有关详细信息，请参阅[PSTN connectivity components in Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
