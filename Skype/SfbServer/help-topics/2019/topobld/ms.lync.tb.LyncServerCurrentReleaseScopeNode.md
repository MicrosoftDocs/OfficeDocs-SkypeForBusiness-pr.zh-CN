# <a name="skype-for-business-server-create-new-expander"></a>Skype 业务服务器创建新的扩展器
 
右键单击**Skype 业务服务器**，选择**新建**，然后选择要创建的服务器角色，您可以在拓扑生成器中创建新的服务器角色。
