# <a name="add-monitoring-server"></a>添加监控服务器

若要将监控服务器纳入拓扑设计中，必须指定要在其上部署监控服务器的服务器的完全限定域名 (FQDN) FQDN。 在发布包含监控服务器的拓扑并在服务器上安装 Skype for Business Server 之前，应完成部署监控的所有先决条件。 有关这些先决条件的详细信息，请参阅部署文档中的[Deploying Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring)。
