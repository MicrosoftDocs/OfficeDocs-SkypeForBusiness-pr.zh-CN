# <a name="add-monitoring-server"></a>添加监控服务器
 
若要监控的服务器合并拓扑设计，必须指定要在其部署监控服务器的服务器的完全限定的域名 (FQDN)。 发布包含监控服务器的拓扑和 Skype 业务服务器安装在服务器上之前，您应完成所有必备组件的部署监控。 有关这些先决条件的详细信息，请参阅部署文档中的[Deploying Monitoring](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx) 。
