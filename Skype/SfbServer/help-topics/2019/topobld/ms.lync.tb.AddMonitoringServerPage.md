# <a name="add-monitoring-server"></a>添加监控服务器

若要将监视服务器合并到拓扑设计中，你必须指定要在其上部署监视服务器的服务器的完全限定的域名（FQDN）。 在发布包含监视服务器并在服务器上安装 Skype for Business 服务器的拓扑之前，应完成部署监视的所有先决条件。 有关这些先决条件的详细信息，请参阅部署文档中的[Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)。
