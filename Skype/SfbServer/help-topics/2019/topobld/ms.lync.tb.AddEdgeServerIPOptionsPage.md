# <a name="add-edge-server-ip-options"></a>添加 Edge Server IP 选项
 
Skype 业务服务器可以为边缘服务器和边缘池配置为每个接口的 IPv4 和 IPv6 地址。 若要执行此操作，请执行以下操作：
  
- **内部接口上启用 IPv4**： 如果您想要将 IPv4 地址应用到边缘服务器或边缘池内部接口选中复选框
    
- **内部接口上启用 IPv6**： 如果您想要将 IPv6 地址应用到边缘服务器或边缘池内部接口选中复选框
    
- **外部接口上启用 IPv4**： 如果您想要将 IPv4 地址应用到边缘服务器或边缘池的外部接口选中复选框
    
- **外部接口上启用 IPv6**： 如果您想要将 IPv6 地址应用到边缘服务器或边缘池的外部接口选中复选框
    
您还可以配置边缘服务器或边缘池以使用网络地址转换地址的外部 IP 地址。 通过选择**此边缘池的外部 IP 地址由 NAT 转换**复选框来执行此操作。
  
NAT 支持。 使用硬件负载平衡，以便执行选择 NAT 选项，如果要部署带有硬件负载平衡的边缘服务器池时不支持网络地址转换 (NAT)。
