# <a name="persistent-chat-scope-node"></a>持久聊天范围节点
 
若要开始创建新的持久聊天服务器或持久聊天服务器池，右键单击**持久聊天池**，然后选择**新的持久聊天池**的拓扑生成器中
  
- **拓扑**。 此选项可用于定义新拓扑、 打开现有拓扑、 下载当前拓扑、 保存一份拓扑、 发布拓扑，安装数据库、 合并从早期版本的 Office Communications Server 组件或删除部署。
    
- **帮助**。可以使用此选项访问活动节点的“帮助”。
