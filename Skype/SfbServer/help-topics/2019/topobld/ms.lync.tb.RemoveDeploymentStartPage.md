# <a name="remove-deployment-introduction"></a>删除部署简介
 
只有当您想要停止整个部署时, 才运行 "**删除部署向导**"。 此向导将创建一个由单个前端服务器组成的空拓扑。 运行此向导并发布空拓扑后, 无法还原为原始拓扑。
  
> [!IMPORTANT]
> 单击 "**下一步**" 之前, 请确保删除所有非活动计算机、禁用统一通信中的所有用户和终结点, 并删除向导第一页上指定的所有会议目录。
