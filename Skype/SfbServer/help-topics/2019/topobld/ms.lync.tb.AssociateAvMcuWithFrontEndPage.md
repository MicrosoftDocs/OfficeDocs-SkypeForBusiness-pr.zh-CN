# <a name="associate-av-mcu-with-front-end"></a>将 AV MCU 与前端关联
 
所有启用了会议的企业版前端池必须具有 A/V 会议服务 collocated，或者必须使用独立的 A/V 会议池。 任何当前具有 A/V 会议服务 collocated 的现有企业版前端池都可能被选中以使用此新的独立 A/V 会议池。
