# <a name="sql-server-reporting-services-credentials"></a>SQL Server Reporting Services（凭据）
 
要指定 SQL Server Reporting Services (SSRS) 的凭据以访问监控数据库，请执行以下操作：
  
在“指定凭据”页中，指定要用于访问监控数据库（QoEMetrics 和 LcsCDR）的用户凭据。 
  
> [!NOTE]
> 有关详细信息，请参阅 位于 的"指定报告数据源的凭据和 (SSRS [https://go.microsoft.com/fwlink/p/?LinkId=263431](/sql/reporting-services/report-data/specify-credential-and-connection-information-for-report-data-sources)) "。 
  
有关为运行监控的服务器部署 Reporting Services 报告的详细信息，请参阅 Install [Monitoring Reports in Skype for Business Server](../../../deploy/deploy-monitoring/install-monitoring-reports.md)。
