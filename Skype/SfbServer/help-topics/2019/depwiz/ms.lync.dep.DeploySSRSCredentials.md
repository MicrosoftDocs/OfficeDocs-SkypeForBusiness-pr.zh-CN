要指定的 SQL Server Reporting Services (SSRS) 以访问监控数据库的凭据，请执行以下操作：
  
在**指定凭据页**中，指定您想要用于访问监控数据库 （QoEMetrics 和 LcsCDR） 的用户凭据。
