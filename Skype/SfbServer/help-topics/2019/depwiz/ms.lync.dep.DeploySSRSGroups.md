# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services（组）

在“指定只读组”中，指定要向其授予对监控报告的只读访问权限的组。 使用格式 &lt; 域 &gt; \\ &lt; 访问组 &gt; 。 例如，contoso\RTCUniversalReadOnlyAdmins。
  
为组分配只读访问权限是可选的。如果不想现在授予只读访问权限，可以以后直接在 SQL Server Reporting Services 中配置。
