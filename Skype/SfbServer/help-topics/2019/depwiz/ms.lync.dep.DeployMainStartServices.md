# <a name="start-services"></a>启动服务
 
若要部署的服务器角色上启动服务，请单击中的**运行****步骤 4： 启动服务**。 您必须以本地 Administrators 组或等效的成员身份登录。 
  
> [!NOTE]
> 如果已运行**启动服务**，您可以通过单击**再次运行**重新运行该命令。
