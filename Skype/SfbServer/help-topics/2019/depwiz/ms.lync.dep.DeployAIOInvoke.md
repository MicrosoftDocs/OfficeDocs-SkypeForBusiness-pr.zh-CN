# <a name="prepare-single-standard-edition-server-invoke"></a>准备单个 Standard Edition Server（调用）
 
在**正在执行命令**页上的 SQL Server Express 安装和配置为充当中央管理存储的任务可以查看在任务窗格中。 默认情况下创建名为 RTC 的基于 SQL Server 数据库的实例。 此外可以创建防火墙规则允许入站和出站访问服务器和客户端与数据库和实例进行通信。 在任务完成后，您可以从下拉列表中选择日志文件。 日志文件中名为**Bootstrap 本地计算机**。 选择日志文件之后，单击**查看日志**。 查看日志文件的任何错误和警告。 当您准备好继续时，单击**完成。** 如果您没有，现在应定义您的拓扑使用拓扑生成器。
  
> [!IMPORTANT]
> 安装 SQL Server Express 可能需要一些时间才能完成。 安装期间，没有任何进度屏幕或任务窗格可见。 若要监视安装，应使用 Windows 任务管理器，并为 SQL Server 查找 MSIExec 流程和安装程序文件。 这种方式，您可以查看的安装状态，并确保是继续安装。 根据此帮助主题的范围之外的因素，它可以需要长达 15 分钟或更多安装 SQL Server 实例完成。
