# <a name="pending-certificate-returned"></a>待处理的证书 （已返回）
 
**联机证书请求状态**页上返回证书的当前状态。 如果证书仍处于挂起的审批状态与证书颁发机构管理员 — 的，如果它不具有尚未批准 — 其状态将显示仍在处理的证书。
  
如果已得到批准待处理的请求并检索证书是成功，其状态将显示证书已分配给本地证书存储，以及状态窗格还将显示证书指纹。
  
如果证书颁发机构管理员拒绝证书请求，状态窗格将报告具有给定 ID （维护证书颁发机构） 的待处理证书请求被拒绝。
  
单击**完成**完成待处理的证书的处理。
