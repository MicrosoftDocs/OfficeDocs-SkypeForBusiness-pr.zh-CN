# <a name="startservices-intro"></a>启动服务（简介）
 
在要部署的 Business Server 角色服务器 Skype 上启动服务，请查看简介屏幕。 请注意，启动该服务不等待发出业务服务器启动 Skype 的命令。 这是启动服务的最佳操作，您应当使用 Windows 服务 Microsoft 管理控制台 (MMC) 监控服务的实际状态。 单击“**下一步**”开始“**启动服务**”任务。
