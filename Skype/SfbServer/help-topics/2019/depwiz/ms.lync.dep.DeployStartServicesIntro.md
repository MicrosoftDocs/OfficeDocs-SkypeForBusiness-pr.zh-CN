# <a name="startservices-intro"></a>启动服务（简介）
 
若要在要部署的 Skype for Business Server 角色服务器上启动服务，请查看简介屏幕。 请注意，为启动 Skype for Business Server 而发出的命令不会等待服务启动。 这是启动服务的最佳操作，并且您应当使用 Windows 服务 Microsoft 管理控制台 (MMC) 监控服务的实际状态。 单击“下一步”开始“启动服务”任务。
