# <a name="certificate-assignment-from-request"></a>证书分配（根据请求）
 
**证书分配**页上启动向角色服务器上的请求用法分配证书的过程。 单击**查看证书详细信息**，以查看证书。 您可以选择**常规**选项卡、**详细信息**选项卡或**证书路径**选项卡。**常规**选项卡将显示有关该证书，如颁发的证书颁发机构和私钥的状态的基本信息。 详细信息选项卡将显示证书信息，如使用者名称、 使用者替代名称和其他详细信息。 **证书路径**选项卡将根证书颁发机构，从任何颁发或中间证书颁发机构 （如果它们存在） 通过证书显示证书"的信任链"。
  
单击“**下一步**”继续。
