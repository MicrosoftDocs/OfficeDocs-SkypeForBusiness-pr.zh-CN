# <a name="certificate-assignment-from-request"></a>证书分配（根据请求）
 
" **证书分配** "页启动将证书分配给角色服务器上请求的用法的过程。 单击“查看证书详细信息”查看证书。 可以选择"常规 **"** 选项卡、" **详细信息** "选项卡或" **证书路径"** 选项卡。" **常规** "选项卡将显示有关证书的基本信息，例如证书颁发机构以及私钥的状态。 “详细信息”选项卡将显示证书的使用者名称、使用者替代名称以及其他详细信息。 " **证书路径** "选项卡将显示来自证书的"信任链"，该证书通过任何颁发或中间证书颁发机构显示 (如果它们存在) 根证书颁发机构。
  
单击“下一步”继续。
