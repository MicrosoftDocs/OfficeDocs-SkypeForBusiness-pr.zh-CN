# <a name="prepare-single-standard-edition-server-intro"></a>准备单个 Standard Edition Server（简介）
 
若要开始安装将保留中央管理存储以及选择的其他并点服务的 Skype for Business Server Standard Edition 服务器，您必须以将成为 Standard Edition 服务器的服务器上本地 管理员组 的成员身份登录。 “准备单个 Standard Edition Server”页将详细阐述初始安装要求。 计算机必须是要部署在其中的域的成员，并且必须已成功为林完成架构、林和域准备。
  
这一特定任务旨在将 Standard Edition Server 设置为基础结构中的第一台服务器。 此任务将中央管理存储（SQL Server Express）安装到Standard Edition服务器上。 如果已部署其他 Standard Edition Server 或前端池，则应单击“取消”。
  
> [!NOTE]
> 完成此任务后，如果尚未安装拓扑生成器 (拓扑生成器) 配置拓扑文档。 在提供中央管理存储（通过完成本主题中介绍的任务进行部署）之前，无法发布拓扑文档。
