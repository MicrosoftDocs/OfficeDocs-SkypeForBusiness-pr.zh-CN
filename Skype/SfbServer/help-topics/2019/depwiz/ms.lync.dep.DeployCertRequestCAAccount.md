# <a name="certificate-request-certificate-authority-account"></a>证书请求（证书颁发机构帐户）
 
若要提交请求, 您的证书颁发机构 (CA) 可能需要您当前登录的用户以外的凭据。 若要允许证书请求作为其他用户, 请选中 "**指定证书颁发机构的备用凭据**" 复选框, 然后为可以申请证书的用户键入用户名或_域_\ _用户名_。. 在 "**密码**" 文本框中, 键入您指定的用户的密码。 然后, 将用户名和密码作为证书请求过程的一部分 (而不是在实际证书请求中) 发送到联机 CA。
