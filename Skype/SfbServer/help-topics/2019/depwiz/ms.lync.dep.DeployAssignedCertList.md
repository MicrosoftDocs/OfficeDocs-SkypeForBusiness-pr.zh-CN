# <a name="view-certificate"></a>查看证书
 
在“查看证书”页上，会显示一个窗格，通过此窗格可选择本地证书存储中的任何有效证书。可以选择某个证书，然后单击“查看证书详细信息”。该证书将显示在新窗格中。在“证书”窗格中，可以查看常规证书信息。单击“详细信息”选项卡可显示证书的详细信息，如使用者名称和使用者替代名称。“证书路径”将显示证书的“信任链”路径。查看完证书信息后，单击“确定”。
