# <a name="view-certificate"></a>查看证书
 
在 "**查看证书**" 页面上，有一个窗格，可让你选择本地证书存储中的任何有效证书。 您可以选择证书，然后单击 "**查看证书详细信息**"。 证书将显示在新窗格中。 在 "**证书**" 窗格中，您可以查看一般的证书信息。 单击 "**详细信息**" 选项卡以显示有关证书的详细信息，如 "主题名称" 和 "主题备用名称"。 证书路径将显示证书的 "信任链" 的路径。 查看完证书信息后，单击 **"确定"**。
