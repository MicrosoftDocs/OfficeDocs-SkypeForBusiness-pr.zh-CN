# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>安装或删除 Skype for Business Server 组件（调用）
 
“执行命令”显示为部署服务器角色而运行的任务的运行摘要。 任务完成后（部署和激活，或者停用和卸载）将显示摘要。 此外，在下拉列表中，可以选择命令运行时创建的任何日志文件。 使用下拉列表选择要查看的日志文件，然后单击“查看日志”。
  
查看完摘要和日志文件后，单击“完成”。
