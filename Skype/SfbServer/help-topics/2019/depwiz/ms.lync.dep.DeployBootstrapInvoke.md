# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>安装或删除 Skype for Business Server 组件（调用）
 
**执行命令**显示运行运行部署服务器角色的任务的摘要。 完成任务后，部署和激活或停用和卸载一个摘要的显示。 此外，在下拉列表中，您可以选择任意命令运行时所创建的日志文件。 使用下拉列表，选择您想要查看的日志文件并单击**查看日志**。
  
后查看完摘要和日志文件中，单击**完成**。
