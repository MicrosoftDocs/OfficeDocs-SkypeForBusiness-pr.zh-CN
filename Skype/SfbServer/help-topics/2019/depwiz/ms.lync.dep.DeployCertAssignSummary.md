# <a name="certificate-assignment-summary"></a>证书分配（摘要）
 
完成证书分配前，您会看到“证书分配摘要”，其中列出了将分配的证书的详细信息。请检查列出的信息以确保使用者名称和使用者替代名称正确无误。检查完证书摘要后，单击“下一步”。
