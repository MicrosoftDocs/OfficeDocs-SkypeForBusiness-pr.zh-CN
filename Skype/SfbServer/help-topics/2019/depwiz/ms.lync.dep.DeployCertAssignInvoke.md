# <a name="certificate-assignment-invoke"></a>证书分配（调用）
 
“正在执行命令”页显示向服务器分配证书的任务所采取的操作。任务完成后，可以查看详细描述证书分配过程的日志。要查看日志文件，请单击“查看日志”。
