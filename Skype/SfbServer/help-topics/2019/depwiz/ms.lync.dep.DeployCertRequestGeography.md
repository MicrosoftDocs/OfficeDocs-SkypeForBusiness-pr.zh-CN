# <a name="certificate-request-geography"></a>证书请求（地理位置）
 
**地理信息**页面请求证书请求的其他信息。 在此页面上，选择 "**国家/地区**"，然后在文本框中键入**州/省**和市/自治区和**市/县**。 请注意，不应使用缩写词，并且应该拼写出州/省和市/自治区的完整名称。
