# <a name="remove-certificate-invoke"></a>删除证书（调用）
 
“正在执行命令”页提供有关删除证书的详细信息。检查证书删除的摘要信息。请确保删除证书日志文件显示在下拉列表中，然后单击“查看日志”检查证书删除的详细信息。检查完摘要和日志文件后，单击“完成”。
