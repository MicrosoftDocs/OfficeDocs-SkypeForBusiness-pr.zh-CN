# <a name="remove-certificate-invoke"></a>删除证书（调用）
 
**正在执行命令**页上提供了有关删除证书的详细信息。 查看证书删除的摘要信息。 请确保删除证书日志文件在下拉列表中，可见，然后单击**查看日志**，以查看证书删除操作的详细信息。 完成查看摘要和日志文件后，单击**完成**。
