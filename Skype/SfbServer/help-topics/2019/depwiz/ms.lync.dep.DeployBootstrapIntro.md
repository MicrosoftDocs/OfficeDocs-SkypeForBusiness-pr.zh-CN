# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>安装或删除 Skype for Business Server 组件（简介）
 
您正准备安装和激活或停用 Skype for Business 服务器组件。 将运行的进程是通过发布到中央管理存储的定义的当前配置定义的。 若要成功完成任务，必须以本地管理员组的成员身份登录，并且至少必须能够读取 Active Directory 域服务。
  
单击“**下一步**”继续进入“**正在执行命令**”屏幕。
