# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>安装或删除 Skype for Business Server 组件（简介）
 
您准备安装和激活或停用和卸载 Skype 业务服务器组件。 将运行的进程是通过发布到中央管理存储的定义的当前配置定义的。 若要成功完成任务，您必须登录以本地 Administrators 组的成员身份，并至少情况下，您必须能够读取 Active Directory 域服务。
  
单击“**下一步**”继续进入“**正在执行命令**”屏幕。
