# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>安装或删除 Skype for Business Server 组件（简介）
 
你正在准备安装并激活或停用和卸载Skype for Business Server组件。 将运行的进程是通过发布到中央管理存储的定义的当前配置定义的。 要成功完成这些任务，必须以本地域管理员身份登录管理员组并且您至少必须能够读取 Active Directory 域服务。
  
单击“下一步”继续进入“正在执行命令”屏幕。
