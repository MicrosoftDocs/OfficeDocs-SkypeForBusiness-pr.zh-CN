# <a name="start-services-invoke"></a>启动服务（调用）
 
在**正在执行命令**窗格的摘要窗格中显示颁发启动的服务的业务服务器角色的服务器，在要部署的 Skype 的任务的状态。 任务窗格中的摘要不代表服务启动状态的实时指示。 业务 Server 服务的 Skype 的一些可能需要较长的时间开始的初始启动过程。 任务会发出启动命令，但不等待确定服务是否成功启动。 如果必须监控服务启动和服务状态，应使用 Windows 服务 Microsoft 管理控制台 (MMC)。
  
任务窗格下方是将显示“**启动服务**”日志文件的下拉列表。要查看日志文件，请单击“**查看日志**”。单击“**完成**”完成任务。
