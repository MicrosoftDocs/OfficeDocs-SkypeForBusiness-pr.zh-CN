# <a name="prepare-current-forrest-invoke"></a>准备当前林（调用）
 
"**执行命令**" 页面显示任务运行时的摘要。 查看摘要屏幕，确保任务成功，并且没有错误。 查看警告以确保它们正常。 在 "摘要" 窗格下，应查看 "**任务状态：已完成**"。 在下拉列表中，日志文件**准备林**应可见。 如果不是，请选择日志文件，然后单击 "**查看日志**" 以查看任务的详细操作和结果。 单击 "**完成**" 以完成林的准备。
