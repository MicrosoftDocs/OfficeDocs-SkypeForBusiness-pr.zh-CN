# <a name="certificate-request-certificate-signing-request-generated"></a>证书请求（已生成证书签名请求）
 
"**证书申请文件**" 页面在您完成证书的脱机请求后显示。 若要显示证书请求并准备将其发送到脱机证书颁发机构（CA）或公共 CA，请单击 "**视图**" 按钮。 证书请求显示在 Windows 记事本中（默认情况下）。 你可以从该显示器复制和粘贴编码的证书申请，或者按照所选脱机 CA 或公共 CA 中的说明完成对证书申请的处理。 单击 "**完成**" 以继续。
