# <a name="certificate-request-certificate-signing-request-generated"></a>证书请求（已生成证书签名请求）
 
完成证书的脱机请求后，将显示“证书请求文件”页。要显示证书请求并准备将其发送到脱机证书颁发机构 (CA) 或公共 CA，请单击“查看”按钮。证书请求将显示在 Windows 记事本（默认）中。您可以从此显示中复制和粘贴编码的证书请求，或按照选定的脱机 CA 或公共 CA 的指导完成证书请求的处理。单击“完成”继续。
