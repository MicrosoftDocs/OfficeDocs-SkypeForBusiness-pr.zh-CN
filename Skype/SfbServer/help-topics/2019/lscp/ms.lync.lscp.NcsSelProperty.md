# <a name="assign-properties"></a>分配属性
 
使用 "**分配属性**" 对话框一次为多个子网指定描述和网络站点 ID。 在 "**说明**" 中，键入要用于所选子网的说明。 在 "**网络站点 id**" 中，选择要分配给所选子网的网络站点 id。 若要将这些属性分配给所选的子网，请单击 **"确定"**。
