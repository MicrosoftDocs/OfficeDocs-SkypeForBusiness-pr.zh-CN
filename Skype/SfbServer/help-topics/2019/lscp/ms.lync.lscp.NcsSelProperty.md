# <a name="assign-properties"></a>分配属性
 
使用**分配属性**对话框中一次指定为多个子网的说明和网络站点 ID。 在**说明**框中，键入您想要用于所选的子网的说明。 在**网络站点 ID**框中，选择您想要分配给所选的子网的网络站点 ID。 若要将这些属性分配给所选的子网，请单击**确定**。
