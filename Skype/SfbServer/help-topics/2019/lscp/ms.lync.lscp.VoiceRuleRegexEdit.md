# <a name="translation-rule-type-a-regular-expression"></a>转换规则：键入正则表达式
 
在**匹配此模式**字段中，指定将用于匹配要转换的号码的模式。 在**转换规则**字段中，指定转换后的号码格式的模式。 例如，如果输入 ^\+(\d{9}\d+)$ 中**匹配此模式**字段和 011$ 1**转换规则**字段中，该规则会将 + 441235551010 转换为 011441235551010。
