# <a name="translation-rule-type-a-regular-expression"></a>转换规则：键入正则表达式
 
在“匹配此模式”字段中，指定将用于匹配要转换的号码的模式。 在“转换规则”字段中，指定转换号码格式的模式。 例如，如果在"匹配此模式"字段中输入 ^\+ (\d  {9}\d+) $，在"转换规则"字段中输入 011$1，则规则将 +441235551010 转换为 011441235551010。
