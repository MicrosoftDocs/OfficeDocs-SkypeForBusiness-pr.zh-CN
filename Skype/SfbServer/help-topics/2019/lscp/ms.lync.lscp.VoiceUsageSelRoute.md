# <a name="select-routes"></a>选择路由
 
使用 "**选择路由**" 对话框查找您可以使用的现有语音路由。
  
> [!NOTE]
> 若要检索所有可用的语音路由，请不要在搜索框中键入任何内容，然后单击 "搜索" 图标。
