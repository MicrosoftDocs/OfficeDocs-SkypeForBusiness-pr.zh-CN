# <a name="uncommitted-voice-configuration-settings-review"></a>未提交的语音配置设置： 审阅
 
使用**未提交的语音配置设置**对话框中查看所有所做的更改。 滚动对话框可查看所有未提交的更改，或单击箭头以展开或折叠不同的部分。 完成后，单击**关闭**。 关闭此对话框后，您可以修改所做的更改，或提交它们。
