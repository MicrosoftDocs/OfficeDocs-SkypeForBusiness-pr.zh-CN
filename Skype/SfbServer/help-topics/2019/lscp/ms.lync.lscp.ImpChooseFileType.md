# <a name="select-file-type-extensions"></a>选择文件类型扩展名
 
使用“选择文件类型扩展名”对话框，通过选中（以阻止）或清除（以允许）“文件类型扩展名”下的扩展名旁边的复选框，可以选择要阻止或允许的文件类型扩展名。单击箭头可展开类别并查看该类别的所有文件类型扩展名。如果要选择某个类别的所有文件类型扩展名，请选择该类别名称。 
  
如果“文件类型扩展名”中未列出要选择的文件类型扩展名，请在“向列表中添加新的文件扩展名”下的文本框中键入相应的扩展名，然后单击“添加”。 
  
将需要的所有文件类型扩展名添加到“选中的文件类型扩展名”列表中后，单击“确定”。
