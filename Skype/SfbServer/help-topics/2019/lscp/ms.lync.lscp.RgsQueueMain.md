# <a name="response-groups-queue"></a>响应组队列
 
响应组组成代理组、 队列和工作流。 响应组队列保留对响应组呼叫，直到代理应答呼叫。 
  
**响应组** - **队列**页显示一个为组织定义的所有响应组队列的列表。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在**响应组**执行以下任务 - **队列**页：
  
- 创建新队列
    
- 更改现有队列
    
- 删除队列
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。
  
- **新**启动一个新的队列。
    
- **编辑**打开所选的队列以进行编辑，在列表中，选择所有队列或者删除所选的队列。
    
- **刷新**刷新队列的列表。
    
下表介绍了该页上的各个字段。
  
- **名称**标识队列的唯一名称。
    
- **服务**承载队列的**ApplicationServer**服务。
    
- **说明**队列的说明。
    
有关响应组特性和功能的详细信息，请参阅规划文档中的[计划中的业务服务器 2015 Skype 的响应组应用程序](../../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 有关使用响应组队列的详细信息，请参阅操作文档中的[Managing Response Group Queues](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) 。
