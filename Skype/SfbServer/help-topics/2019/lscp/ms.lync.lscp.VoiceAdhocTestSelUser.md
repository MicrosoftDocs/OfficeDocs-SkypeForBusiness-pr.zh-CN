# <a name="select-enterprise-voice-users"></a>选择企业语音用户
 
使用 **"企业语音用户** "对话框查找已启用此企业语音。 在搜索框中，键入要查找的用户的 SIP 地址或显示名称，然后单击“查找”。 若要选择用户，请单击搜索结果列表中的用户，然后单击“确定”。 如果找不到要查找的用户，则可能无法为该用户启用企业语音。 
  
> [!NOTE]
> 若要检索所有启用企业语音的用户，请不要在搜索框中键入任何内容。 还可以查找显示名称或 SIP 地址以特定字符串开头的所有用户。 例如，要查找 SIP 地址以 ken 开头的所有用户，请键入 sip:ken。 这将返回诸如 sip:kenmyer@fabrikam.com、sip:kenkwok.fabrikam.com 和 sip:kenmallit@fabrikam.com 的用户。
