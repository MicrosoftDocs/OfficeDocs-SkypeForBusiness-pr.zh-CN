# <a name="hybrid-manageability-wizard"></a>混合可管理性向导

你可以使用混合管理向导执行一些配置任务，这些任务是使用 Skype for Business Online 设置混合部署所必需的。 有关如何配置混合部署的详细信息，请参阅[通过混合管理向导配置你的环境](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
