# <a name="topology-server-application"></a>拓扑：服务器应用程序

“拓扑: 服务器应用程序”页将显示部署中的服务器应用程序。

## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在“拓扑: 服务器应用程序”页上执行以下任务：

- 查看部署中的服务器应用程序

- 启用或禁用服务器应用程序

- 将应用程序选择为关键或取消选择
