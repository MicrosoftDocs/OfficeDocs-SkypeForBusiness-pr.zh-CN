# <a name="select-a-region-link"></a>选择区域链接
 
使用“**区域链接**”对话框查找环境中可用的区域链接。要使用现有区域链接，请单击列表中的某个区域链接，然后单击“**确定**”。
  
> [!NOTE]
> 要检索所有可用的区域链接，请不要在搜索框中键入任何内容，然后单击搜索图标。
