# <a name="select-trunk"></a>选择中继
 
使用 "**选择主干**" 对话框查找可使用的现有主干。
  
> [!NOTE]
> 若要检索所有可用的中继, 请不要在搜索框中键入任何内容, 然后单击 "搜索" 图标。
