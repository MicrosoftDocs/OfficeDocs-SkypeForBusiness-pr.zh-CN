# <a name="select-overflow-queue"></a>选择溢出队列
 
使用“选择溢出队列”对话框可选择在响应组队列已满时转接呼叫的队列。
  
在搜索框中，键入要使用队列的完整名称或部分名称。若要返回所有可用队列，请将搜索框留空并单击搜索图标。
  
在搜索返回的队列列表中，选择希望用于在响应组队列已满时转接呼叫的队列。
