# <a name="select-overflow-queue"></a>选择溢出队列
 
使用 "**选择溢出队列**" 对话框选择当响应组队列已满时将调用转发到的队列。
  
在 "搜索" 框中, 键入要使用的队列的完整名称或部分名称。 若要返回所有可用的队列, 请将搜索框留空, 然后单击 "搜索" 图标。
  
在搜索返回的队列列表中, 选择当响应组队列已满时要用于转发呼叫的队列。
