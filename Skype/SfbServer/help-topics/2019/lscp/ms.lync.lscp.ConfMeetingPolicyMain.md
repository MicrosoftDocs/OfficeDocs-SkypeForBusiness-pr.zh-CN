# <a name="conferencing-policy"></a>会议策略
 
会议策略定义的特性和功能 （也称为会议） 会议期间用户可以使用。
  
会议策略包括全局策略和一个或多个站点和用户策略 （可选）：
  
- **全局策略：** 默认情况下创建全局策略。 可以编辑全局策略，但无法将其删除。 如果您尝试删除全局策略，则所有设置将重置为默认值。
    
- **的网站策略 （可选）：** 您可以创建一个或多个站点会议策略，其中每个适用于特定站点。 站点策略会覆盖全局策略。
    
- **用户策略 （可选）：** 您可以创建一个或多个用户会议策略，其中每个适用于特定用户或用户组。 用户策略会覆盖全局策略和站点策略。
    
**会议策略**页上显示为组织定义的所有会议策略的列表。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在“**位置策略**”页上执行以下任务：
  
- 创建新的站点会议策略或用户会议策略
    
- 更改全局策略或现有站点策略或用户策略
    
- 删除站点策略或用户策略
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。
  
- **新**开始一个新的站点会议策略或用户会议策略。
    
- **编辑**打开所选的会议策略以进行编辑，在列表中，选择所有会议策略，或者删除所选的站点策略或用户策略。
    
    > [!NOTE]
    > 对于全局策略，“**删除**”会将设置重置为默认值。
  
- **刷新**刷新会议策略的列表。
    
下表介绍了该页上的各个字段。
  
- **名称**标识会议策略。
    
- **范围**标识会议策略的范围： 全局、 站点或用户。
    
- **数据协作**如果会议策略指定在会议中允许数据协作，则检查。
    
- **应用程序共享**检查是否会议策略指定在会议中允许应用程序共享。
    
- **音频**如果会议策略指定在会议中允许音频，则检查。
    
- **视频**如果会议策略指定在会议中允许视频，则检查。
    
- **PSTN**检查是否会议策略指定允许 PSTN 电话拨入式会议。
    
- **录制**检查是否会议策略指定在会议中允许录制。
    
有关会议特性和功能的详细信息，请参阅规划文档中[概述的会议](http://technet.microsoft.com/library/5bb90e69-3d4f-4d59-a1ee-2550de84439f.aspx)。 有关使用会议策略的详细信息，请参阅操作文档中的[会议策略](http://technet.microsoft.com/library/8f92eb7c-ee66-4df6-a726-4bff93b122cb.aspx)。
