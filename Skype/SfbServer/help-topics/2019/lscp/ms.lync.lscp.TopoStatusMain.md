# <a name="topology-status"></a>拓扑：状态

" **拓扑**： **状态** "页显示 Skype for Business 拓扑中服务器的状态。

## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在“拓扑: 状态”页上执行以下任务：

- [查看运行 Skype for Business Server 或 Lync Server 的计算机列表](https://technet.microsoft.com/library/44eeec27-8b99-44f0-b0bd-622c12393d34.aspx)

- [查看有关服务的详细信息](https://technet.microsoft.com/library/bc8e8202-cd68-47e4-95b2-bb36e51cc124.aspx)

- [启动或停止服务](https://technet.microsoft.com/library/1c70b4ec-9de5-4f7a-a3c9-c0eb76710505.aspx)

- [阻止与服务的新连接](https://technet.microsoft.com/library/977dcc5c-2aac-48ef-86a1-a8d47b4d9e74.aspx)
