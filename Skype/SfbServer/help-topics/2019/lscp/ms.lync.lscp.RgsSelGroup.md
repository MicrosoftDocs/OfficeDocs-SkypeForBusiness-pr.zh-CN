# <a name="select-groups"></a>选择组
 
响应组队列保留对响应组呼叫，直到代理应答呼叫。 队列已分配的代理组。 当您创建或编辑队列时，您使用**选择组**对话框选择要分配给队列的一个或多个代理组。
  
## <a name="ui-reference"></a>用户界面参考

以下列表描述的控件和在**选择组**对话框中的字段。
  
- 搜索框中搜索的代理组的名称。 输入的全部或部分名称。 将搜索框保留为空，以显示您的组织的所有代理组。 
    
- **名称**显示返回的代理组的名称。
    
- **参与策略**显示为代理组定义的参与策略： 正式或非正式。
    
- **路由方法**显示为代理组定义的路由的类型。
    
有关使用队列的详细信息，请参阅操作文档中的[Managing Response Group Queues](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) 。
