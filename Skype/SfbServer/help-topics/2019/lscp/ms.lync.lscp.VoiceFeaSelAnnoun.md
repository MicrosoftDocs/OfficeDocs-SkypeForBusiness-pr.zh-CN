# <a name="select-an-announcement-service"></a>选择通知服务
 
使用“选择通知服务”对话框可选择将处理到未分配号码范围的传入呼叫的通知服务。
  
在搜索框中，键入要使用通知服务的完整名称或部分名称。若要返回所有可用通知服务，请将搜索框留空并单击搜索图标。
  
在搜索返回的通知服务列表中，选择希望用于处理到未分配号码范围的传入呼叫的服务。
