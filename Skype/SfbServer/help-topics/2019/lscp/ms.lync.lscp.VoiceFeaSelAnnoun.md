# <a name="select-an-announcement-service"></a>选择通知服务
 
使用 "**选择通知服务**" 对话框选择一种公告服务，该服务将处理传入呼叫到一系列未分配的号码。
  
在 "搜索" 框中，键入要使用的公告服务的全名或部分名称。 若要返回所有可用的通知服务，请将搜索框留空，然后单击 "搜索" 图标。
  
在搜索返回的公告服务列表中，选择要用于处理对未分配号码范围的传入呼叫的服务。
