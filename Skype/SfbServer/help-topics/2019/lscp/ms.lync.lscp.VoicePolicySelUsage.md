# <a name="select-pstn-usage-record"></a>选择 PSTN 用法记录
 
使用**选择 PSTN 用法记录**对话框查找可以使用现有公用电话交换网 (pstn) 用法记录。
  
> [!NOTE]
> 若要检索所有可用的 PSTN 用法记录，请执行不在搜索框中键入任何内容，然后单击搜索图标。
