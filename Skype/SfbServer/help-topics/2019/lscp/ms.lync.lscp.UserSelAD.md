# <a name="select-from-active-directory"></a>从 Active Directory 中选择
 
使用 **"从 Active Directory** 中选择"对话框查找要为此目录启用Skype for Business Server。 在搜索框中，键入显示名称用户的 SIP 地址或地址。 若要检索所有可用用户，请不要在搜索框中键入任何内容。
