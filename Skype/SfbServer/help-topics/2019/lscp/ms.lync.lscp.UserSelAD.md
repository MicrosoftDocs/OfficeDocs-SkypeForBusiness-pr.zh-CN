# <a name="select-from-active-directory"></a>从 Active Directory 中选择
 
使用**从 Active Directory 中选择**对话框查找您想要启用的 Skype 业务 Server 的用户。 在搜索框中，键入要查找的用户的 SIP 地址或显示名称。 要检索所有可用的用户，请不要在搜索框中键入任何内容。
