# <a name="save-voice-routing-test-case-information"></a>保存语音路由测试用例信息
 
可以使用"保存 **语音路由测试用例** 信息"对话框保存语音路由测试用例。 在“保存语音路由测试用例信息”的“名称”中，键入要赋予此测试用例的名称，然后单击“确定”。 
  
> [!NOTE]
> 保存临时测试用例时，不会将其保存到中央管理存储。 而会在“语音路由”页的“测试语音路由”选项卡上显示测试用例。 测试用例的状态将显示为“未提交”。 若要将测试用例保存到中央管理存储，必须通过单击"语音路由"页中任何"操作"菜单中的"全部提交"来 **提交语音** 配置。
