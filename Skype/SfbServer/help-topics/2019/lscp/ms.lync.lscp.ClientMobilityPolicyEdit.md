# <a name="edit-client-mobility-policy"></a>编辑客户端移动策略

在“编辑移动策略”对话框上，您可以编辑移动策略的详细信息。

## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的菜单、命令、字段和属性。


- **提交** 单击以保存对策略设置所做的更改。

- **取消** 单击以停止编辑策略并放弃对策略设置进行的任何更改。

- **名称** 移动策略的名称。

- **说明** 移动策略的说明。

有关移动策略的详细信息，请参阅[Mobility Policies](/previous-versions/office/lync-server-2013/lync-server-2013-mobility-policies)。
