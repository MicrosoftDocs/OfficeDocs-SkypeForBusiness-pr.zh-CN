# <a name="hybrid-wizard-sign-in-to-microsoft-365-or-office-365"></a>混合向导：登录 Microsoft 365 或 Office 365

使用 Skype for Business Server 混合可管理性向导帮助你配置环境以与 Skype for Business Online 进行混合。

## <a name="sign-in-to-microsoft-365-or-office-365"></a>登录到 Microsoft 365 或 Office 365

输入 Microsoft 365 或 Office 365 组织的用户名和密码，以登录本地部署，并与 Microsoft 365 或 Office 365 组织连接。 注册 Microsoft 365 或 Office 365 时，将需要使用为 onmicrosoft 域创建的管理员帐户，例如 admin@contoso.onmicrosoft.com。

若要详细了解如何配置混合环境，请参阅 [混合部署概述](/previous-versions/office/lync-server-2013/lync-server-2013-overview-of-hybrid-deployments)。
