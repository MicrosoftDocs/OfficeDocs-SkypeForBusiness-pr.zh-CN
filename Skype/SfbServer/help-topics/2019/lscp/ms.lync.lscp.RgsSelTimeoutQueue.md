# <a name="select-time-out-queue"></a>选择超时队列
 
使用**选择超时队列**对话框中选择队列超时后转接呼叫。
  
在搜索框中，键入您想要使用的队列的名称的全部或部分。 要返回所有可用的队列，请将搜索框保留为空，然后单击搜索图标。
  
在搜索返回的队列列表中，选择您希望用于超时呼叫的队列。
