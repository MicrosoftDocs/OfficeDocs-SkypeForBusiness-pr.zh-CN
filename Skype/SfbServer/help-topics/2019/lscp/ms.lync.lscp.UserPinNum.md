# <a name="pin-set-successfully"></a>PIN 设置成功
 
若要使用电子邮件发送给用户的 PIN，请单击**打开我的电子邮件应用程序发送给用户的新 PIN**。 若要复制到电子邮件的 PIN，选择**显示 PIN**。
