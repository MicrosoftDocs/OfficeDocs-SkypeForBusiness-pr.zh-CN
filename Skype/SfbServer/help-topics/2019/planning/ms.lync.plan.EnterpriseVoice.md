# <a name="enterprise-voice-planning-tool"></a>企业语音 (规划工具) 
 
借助 企业语音，Skype for Business Server 提供了独立的 Internet 语音 (VoIP) 产品，用于增强或替换 PBX (的传统专用交换机) 系统。 企业语音可以呼叫您组织的 VoIP 网络或 PBX 上的同事，并且他们可以呼叫组织外部的传统电话号码。 企业语音 解决方案包括常见呼叫功能，如应答、转接、转移、暂停、释放和 (增强型 9-1-1 (E9-1-1) 呼叫 (E9-1-1 仅在美国可用。) 企业语音 还支持多种当前和较旧的 IP 和 USB 设备。
  
有关规划规划的企业语音，请参阅规划企业语音[中的Skype for Business Server。](../../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md) 有关部署独立中介服务器的更多详细信息，请参阅 Skype for Business Server 中的在拓扑生成器[中部署中介Skype for Business Server。](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)
