# <a name="monitoring-planning-tool"></a>监控 （规划工具）
 
在业务服务器 Skype，监控用于收集有关您的用户所涉及的通信会话的用户体验质量 (QoE) 数据和使用情况信息。
  
有关监控服务器规划的详细信息，请参阅[Planning for Monitoring](http://technet.microsoft.com/library/26cead5a-183c-42f1-a4b0-0e8d61c6159d.aspx)。
