# <a name="skype-for-business-server-monitoring-planning-tool"></a>Skype for Business Server监控规划工具

在Skype for Business Server中，监控用于收集有关用户参与的通信会话的使用情况信息和用户体验质量 (qoE) 数据。

有关规划监控服务器的更多详细信息，请参阅 [Planning for Monitoring](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring)。
