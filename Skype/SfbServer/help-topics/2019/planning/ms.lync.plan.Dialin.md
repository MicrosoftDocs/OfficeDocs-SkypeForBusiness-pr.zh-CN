# <a name="dial-in-conferencing-planning-tool"></a>电话拨入式会议 （规划工具）
 
Skype 业务服务器 2015年电话拨入式会议，用户可以通过使用公用电话交换网 (pstn) 电话，而无需第三方音频会议提供商加入 Skype 业务服务器 2015年会议的音频部分。
  
有关为组织启用电话拨入式会议的规划详细信息，请参阅以下主题及其链接到的主题： 
  
- [规划业务服务器 2015 Skype 中的会议](../../../plan-your-deployment/conferencing/conferencing.md)
