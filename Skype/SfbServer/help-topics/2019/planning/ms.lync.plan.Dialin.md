# <a name="dial-in-conferencing-planning-tool"></a>电话拨入式会议 (规划工具) 
 
Skype for Business Server电话拨入式会议使用户可以使用公用电话交换网 (PSTN) 电话加入 Skype for Business Server 会议的音频部分，而无需第三方音频会议提供商。
  
有关为组织启用电话拨入式会议的计划详细信息，请参阅以下主题及其链接到的主题： 
  
- [规划会议Skype for Business Server](../../../plan-your-deployment/conferencing/conferencing.md)
