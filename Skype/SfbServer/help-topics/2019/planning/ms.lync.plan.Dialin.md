# <a name="dial-in-conferencing-planning-tool"></a>Dial-in Conferencing (Planning Tool)
 
Skype Business Server 电话拨入式会议，用户可以通过使用公用电话交换网 (pstn) 电话，而无需第三方音频会议提供商加入 Skype Business Server 会议的音频部分。
  
有关为组织启用电话拨入式会议的规划详细信息，请参阅以下主题及其链接到的主题： 
  
- [规划业务服务器中 Skype 的会议](../../../plan-your-deployment/conferencing/conferencing.md)
