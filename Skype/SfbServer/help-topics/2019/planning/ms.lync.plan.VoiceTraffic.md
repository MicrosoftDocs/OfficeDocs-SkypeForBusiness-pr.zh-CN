# <a name="voice-traffic-planning-tool"></a>Voice Traffic (Planning Tool)
 
规划工具使用有关用户流量的信息来确定要在部署中使用的端口数。 此数字依次确定所需中介服务器和网关的数量。 有关详细信息，请参阅[估计 Skype for Business 服务器的语音使用和流量](../../../plan-your-deployment/capacity/estimating-voice-traffic.md)。
