# <a name="mediation-collocation-planning-tool"></a>中介并置 (规划工具) 
 
默认情况下，中介服务器与前端服务器并排。 出于性能原因，还可以在独立池中部署中介服务器，或者如果部署 SIP 中继（在这种情况下，强烈建议使用独立池）。 与 Lync Server 2013 中Skype for Business Server一样，在服务器中的并置功能也一样。 有关详细信息，请参阅下列主题：
  
- [Lync Server 2013 中支持的服务器并置](/previous-versions/office/lync-server-2013/lync-server-2013-supported-server-collocation)
    
- [中介服务器组件Skype for Business Server](../../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
