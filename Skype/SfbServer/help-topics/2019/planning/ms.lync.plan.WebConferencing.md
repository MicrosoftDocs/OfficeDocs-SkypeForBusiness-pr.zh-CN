# <a name="web-conferencing-planning-tool"></a>Web 会议 (规划工具) 
 
Skype for Business Server Web 会议包括文档协作、应用程序共享和桌面共享。
  
> [!NOTE]
> Skype for Business Server使用 Office Web Apps 和 Office Web Apps Server 来处理演示文稿PowerPoint和呈现。 Web 会议功能的功能与在 Microsoft Lync 2013 中的功能完全相同。 有关安装和配置 Web Apps Server Office的详细信息，请参阅[Configuring integration with Office Web Apps Server and Lync Server 2013。](/previous-versions/office/lync-server-2013/lync-server-2013-enabling-office-web-apps-server-and-lync-server-2013)
