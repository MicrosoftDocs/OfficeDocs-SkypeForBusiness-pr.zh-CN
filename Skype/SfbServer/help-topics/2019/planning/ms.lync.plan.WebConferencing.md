# <a name="web-conferencing-planning-tool"></a>Web 会议 （规划工具）
 
Skype Business Server web 会议包括文档协作、 应用程序共享和桌面共享。
  
> [!NOTE]
> Skype 业务服务器使用 Office Web Apps 和 Office Web Apps Server 处理共享 PowerPoint 演示文稿的呈现。 在 Microsoft Lync 2013 中一样，web 会议功能正常完全相同。 有关安装和配置 Office Web Apps Server 的详细信息，请参阅[配置与 Office Web Apps Server 和 Lync Server 2013 的集成](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx)。
