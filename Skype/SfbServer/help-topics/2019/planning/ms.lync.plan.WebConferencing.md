# <a name="web-conferencing-planning-tool"></a>Web 会议（规划工具）
 
Skype for Business Server web 会议包括文档协作、应用程序共享和桌面共享。
  
> [!NOTE]
> Skype for Business Server 使用 Office Web Apps 和 Office Web Apps Server 处理 PowerPoint 演示文稿的共享和呈现。 Web 会议功能的工作方式与 Microsoft Lync 2013 中的功能完全相同。 有关安装和配置 Office Web Apps Server 的详细信息，请参阅[配置与 Office Web Apps server 和 Lync Server 2013 的集成](https://technet.microsoft.com/library/jj204792%28v=ocs.15%29.aspx)。
