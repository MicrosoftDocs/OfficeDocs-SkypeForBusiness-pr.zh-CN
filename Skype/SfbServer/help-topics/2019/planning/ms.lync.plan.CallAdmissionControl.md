# <a name="call-admission-control-planning-tool"></a>呼叫允许控制 （规划工具）
 
呼叫允许控制 (CAC) 根据可用网络带宽确定是否允许建立实时通信会话（如语音呼叫或视频呼叫）。
  
有关规划呼叫允许控制的详细信息，请参阅[Plan for Business Server 的 Skype 中的呼叫允许控制](../../../plan-your-deployment/enterprise-voice-solution/call-admission-control.md)。
