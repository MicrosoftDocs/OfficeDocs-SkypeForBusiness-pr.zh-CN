# <a name="disaster-recovery-planning-tool"></a>Disaster Recovery (Planning Tool)
 
Skype for business 服务器通过使你能够对位于两个数据中心的前端池进行配对，提供了灾难恢复措施。
  
有关为灾难恢复准备环境的详细信息，请参阅[在 Skype For Business 服务器中规划高可用性和灾难恢复](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md)。
