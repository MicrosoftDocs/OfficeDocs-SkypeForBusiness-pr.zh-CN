# <a name="mobility-planning-tool"></a>移动性（规划工具）
 
Skype for Business Server 中的移动功能支持移动设备上的 Skype for Business 功能。
  
在 Skype for Business Server 中，与 Lync Server 2013 中的移动功能相同。 有关准备环境以实现移动性的更多详细信息，请参阅[在 Lync Server 2013 中规划移动](https://technet.microsoft.com/library/hh689981%28v=ocs.15%29.aspx)功能。
