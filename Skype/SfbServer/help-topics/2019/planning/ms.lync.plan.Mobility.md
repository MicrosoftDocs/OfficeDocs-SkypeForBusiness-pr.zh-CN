# <a name="mobility-planning-tool"></a>Mobility (Planning Tool)
 
Skype for Business 服务器中的移动功能支持移动设备上的 Skype for business 功能。
  
Skype for Business 服务器中的移动功能与 Lync Server 2013 中的相同。 有关为移动性准备环境的更多详细信息，请参阅[在 Lync Server 2013 中规划移动](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)。
