# <a name="edge-network-planning-tool-in-skype-for-business"></a>Skype for Business (边缘) 规划工具 
 
大多数边缘组件都部署在外围网络中。 以下组件组成外围网络的边缘拓扑。 除非特别说明，否则这些组件是 [Skype for Business Server 中边缘服务器](../../../plan-your-deployment/edge-server-deployments/scenarios.md) 方案的一部分，并且位于外围网络中。 边缘组件包括下列各项：
  
- 边缘服务器
    
- 反向代理
    
- 防火墙
    
- 控制器 (，且逻辑上位于内部网络) 
    
- 扩展的边缘拓扑的负载平衡（DNS 负载平衡或硬件负载平衡器）
