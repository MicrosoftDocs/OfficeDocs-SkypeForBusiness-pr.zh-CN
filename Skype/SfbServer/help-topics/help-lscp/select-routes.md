# <a name="select-routes"></a>选择路由
 
使用**路由选择**对话框中查找现有的语音路由，您可以使用。
  
> [!NOTE]
> 若要检索所有可用语音路由，执行不在搜索框中键入任何内容，然后单击搜索图标。 
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
