# <a name="prefix-edit"></a>前缀编辑
 
若要定义前缀或启动数字，添加页码前缀应该处理该路由，其中包括在必要时 +。 例如，您可以匹配所有以 +123 开头的数字。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
