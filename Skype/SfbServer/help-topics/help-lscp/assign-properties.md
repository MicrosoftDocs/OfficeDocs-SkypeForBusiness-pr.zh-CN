# <a name="assign-properties"></a>指定属性
 
使用**指定属性**对话框中指定说明和网络站点标识为多个子网一次。 在**说明**框中，键入要用于所选子网的描述。 在**网络站点 ID**，选择您要分配给所选子网的网络站点 ID。 要将这些属性分配给所选的子网，请单击**确定**。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
