# <a name="assign-properties"></a>分配属性
 
使用“分配属性”对话框一次指定多个子网的说明和网络站点 ID。在“说明”中，键入要用于选定的子网的说明。在“网络站点 ID”中，选择要分配给选定的子网的网络站点 ID。要将这些属性分配给选定的子网，请单击“确定”。
  
有关可以使用控制面板执行的不同过程的详细信息，Skype for Business Server Manage [Skype for Business Server 2015](../../manage/manage.md)。
