# <a name="select-file-type-extensions"></a>选择的文件类型扩展名
 
使用**选择文件类型的扩展名**对话框中选择您想要阻止或允许通过选择 （到块） 或清除 （以允许） 的文件类型扩展名在**文件类型扩展名**的扩展旁边的复选框。 单击箭头以展开某一类别，然后查看该类别的文件类型扩展名的所有。 如果您想要选择所有类别的文件类型扩展名，选择类别名称。 
  
如果不看**文件类型扩展**中列出您想要选择的文件类型的扩展名，**向列表添加新的文件扩展名**下的文本框中键入扩展名，然后单击**添加**。 
  
添加完所有所需的文件类型扩展名与**所选文件类型的扩展名**列表后，单击**确定**。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
