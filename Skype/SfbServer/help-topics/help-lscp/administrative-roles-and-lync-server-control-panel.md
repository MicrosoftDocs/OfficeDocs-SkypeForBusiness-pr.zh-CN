# <a name="administrative-roles-and-lync-server-control-panel"></a>管理角色和 Lync Server 控制面板

为了使您能够在保持高安全标准的同时委派管理任务，Skype for Business Server 支持基于角色的访问控制 (RBAC) 。 使用 RBAC，可通过将用户分配给预定义的管理角色来授予管理权限。 Skype for Business Server 包括一组丰富的内置管理角色，还允许您创建新角色并为每个新角色指定 cmdlet 的自定义列表。 您还可以将 cmdlet 的脚本添加到所允许的预定义和自定义 RBAC 角色任务中。 有关详细信息，包括 Skype for Business Server 中提供的预定义管理角色，请参阅 [基于角色的访问控制](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control)。
