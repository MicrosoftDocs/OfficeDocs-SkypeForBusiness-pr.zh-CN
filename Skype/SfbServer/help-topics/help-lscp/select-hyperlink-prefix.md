# <a name="select-hyperlink-prefix"></a>选择超链接前缀
 
使用**作为前缀选择的超链接**对话框中选择想要阻止，发送一条警告消息，或允许通过选择 （以阻止或发送一个警告消息） 或清除 （允许） 的超链接的类型及其扩展名对应的复选框下**超链接前缀**的类别。 单击箭头可展开类别，并查看所有超链接前缀为该类别。 如果您想要选择的所有超链接前缀为类别，选择类别名称。 
  
如果不看您要选择一个超链接前缀列**超链接前缀**中，**向列表添加新超链接前缀**，则在文本框中键入扩展名，然后单击**添加**。 
  
所有所需的超链接前缀添加到**所选超链接前缀**列表后，单击**确定**。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
