# <a name="select-from-active-directory"></a>从 Active Directory 中选择
 
使用 **"从 Active Directory** 中选择"对话框查找要启用此Skype for Business Server。 在搜索框中，键入要显示名称用户的 SIP 地址或地址。 若要检索所有可用用户，请不要在搜索框中键入任何内容。
  
有关可以使用控制面板执行的不同过程的详细信息，Skype for Business Server Manage [Skype for Business Server 2015](../../manage/manage.md)。
