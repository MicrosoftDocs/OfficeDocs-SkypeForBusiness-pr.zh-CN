# <a name="select-enterprise-voice-users"></a>选择企业语音用户
 
使用**选择企业语音用户**对话框中查找用户启用了企业语音。 在搜索框中键入 SIP 地址或显示名称您想要查找的用户，然后单击**查找**。 若要选择用户，单击搜索结果列表中的用户，然后单击**确定**。 如果您无法找到要查找的用户，该用户可能没有启用为企业语音。 
  
> [!NOTE]
> 若要检索所有用户启用企业语音，不要键入任何内容的搜索框中。 您还可以查找所有用户的显示名称，或以特定字符串开头的 SIP 地址。 例如，若要查找所有用户的 SIP 地址开头 ken，键入 sip: ken。 这将返回用户，如： sip:kenmyer@fabrikam.com、 sip:kenkwok.fabrikam.com 和 sip:kenmallit@fabrikam.com。 
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
