# <a name="select-exchange-um-auto-attendant"></a>选择交换 UM 自动助理
 
使用**选择 Exchange UM 自动助理**对话框中选择 Exchange UM 自动助理将处理传入呼叫的未分配的数字范围。
  
在搜索框中，键入全部或部分交换 UM 自动助理想要使用的名称。 若要返回所有可用 Exchange UM 自动助理，将搜索框保留为空，请单击搜索图标。
  
在 Exchange UM 自动助理搜索返回的列表中，选择您要用于处理传入呼叫的未分配的数字范围自动助理。
