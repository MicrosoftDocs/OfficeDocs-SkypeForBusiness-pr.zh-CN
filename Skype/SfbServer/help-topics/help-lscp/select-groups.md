# <a name="select-groups"></a>选择组
 
直到代理应答呼叫，响应组队列包含响应组对的调用。 队列已指派代理组。 当您创建或编辑队列时，则使用**选择组**对话框中选择要分配给该队列的一个或多个代理组。
  
## <a name="ui-reference"></a>用户界面参考

下面的列表描述控件并在**选择组**对话框中的字段。
  
- 代理程序组名称的搜索框中搜索。 输入所有或部分名称。 将搜索框留空，以显示您的组织的所有代理组。 
    
- **名称**显示返回的代理组的名称。
    
- **参与策略**显示定义代理组的参与策略： 正式或非正式。
    
- **路由方法**显示路由代理组定义的类型。
    
有关如何使用队列的详细信息，请参阅[管理响应组队列](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx)操作文档中。
