# <a name="uncommitted-voice-configuration-settings-review"></a>未提交的语音配置设置： 审查
 
使用**未提交语音配置设置**对话框中查看所有您所做的更改。 滚动对话框可查看所有未提交的更改，或单击箭头以展开或折叠不同的部分。 当您完成时，单击**关闭**。 关闭此对话框后，您可以修改所做的更改或并提交。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
