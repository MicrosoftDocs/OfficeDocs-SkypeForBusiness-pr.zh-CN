# <a name="uncommitted-voice-configuration-settings-review"></a>未提交的语音配置设置：审查
 
使用 "**未提交的语音配置设置**" 对话框查看您所做的所有更改。 滚动对话框可查看所有未提交的更改，或单击箭头以展开或折叠不同的部分。 完成后，单击 "**关闭**"。 关闭此对话框后，您可以修改或提交更改。
  
有关可通过使用 Skype for Business 服务器控制面板执行的不同过程的详细信息，请参阅[管理 skype for Business server 2015](../../manage/manage.md)。
