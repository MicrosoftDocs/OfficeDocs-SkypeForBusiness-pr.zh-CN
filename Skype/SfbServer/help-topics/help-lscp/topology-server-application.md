# <a name="topology-server-application"></a>拓扑：服务器应用程序

"**拓扑**：**服务器应用程序**" 页面显示你的部署中的服务器应用程序。

## <a name="tasks-you-can-perform"></a>可执行的任务

可以在 "**拓扑**：**服务器应用程序**" 页面上执行以下任务：

- 查看部署中的服务器应用程序

- 启用或禁用服务器应用程序

- 选择或取消选择应用程序为关键
