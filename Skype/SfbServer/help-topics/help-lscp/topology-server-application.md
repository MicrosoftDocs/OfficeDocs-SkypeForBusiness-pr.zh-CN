# <a name="topology-server-application"></a>拓扑： 服务器应用程序
 
**拓扑结构**：**服务器应用程序**页显示在您的部署中的服务器应用程序。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在**拓扑结构**上执行以下任务：**服务器应用程序**页：
  
- 查看部署中的服务器应用程序
    
- 启用或禁用服务器应用程序
    
- 选择或取消选择为关键应用程序
