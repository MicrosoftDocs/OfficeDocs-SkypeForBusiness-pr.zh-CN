# <a name="select-normalization-rules"></a>选择规范化规则
 
规范化规则使用 .NET Framework 正则表达式将拨打的电话号码转换为 E.164 格式。必须将每个拨号计划分配给一个或多个规范化规则。
  
要使用现有的规范化规则，请单击列表中的某个规则，然后单击“**确定**”。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
