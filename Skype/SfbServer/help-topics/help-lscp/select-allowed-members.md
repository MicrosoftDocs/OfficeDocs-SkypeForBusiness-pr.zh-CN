# <a name="select-allowed-members"></a>选择允许的成员
 
创建和管理持续聊天房间是非常简单的类别的正确使用。 持续的聊天管理员可以为每个类别，定义**AllowedMembers**和**创建者**，还可以定义默认聊天室设置和将应用于在该类别中创建的所有聊天室的行为。 持久的聊天管理员创建，并通过使用控制面板或 Windows PowerShell cmdlet 类别进行管理。
  
标识为类别创建者的用户、组织单位 (OU) 和用户组是仅有的被允许在类别中创建聊天室的个人和组。 在创建该类别后，它们可以用户、 Ou 和用户组从列表中选择该类别的**AllowedMembers**作为研讨室经理和成员，以管理和参与空间。
  
## <a name="tasks-that-you-can-perform"></a>可执行的任务

您可以在“**选择允许的成员**”页上执行以下任务：
  
- [配置类别](http://technet.microsoft.com/library/4547f514-f0c0-404d-890f-092ddeeac852.aspx)
    
- [新的持久的聊天服务器功能](http://technet.microsoft.com/library/c3ec6f33-6261-4bf5-aa31-baa8ab2a87d8.aspx)
    
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
  
## <a name="to-configure-categories-for-chat-rooms"></a>配置聊天室的类别

在中的**成员资格**，**允许的成员**部分中，添加或删除用户，并允许被添加为成员的聊天室其他 Active Directory 域服务主体 （用户、 通讯组、 组织单位，等）属于该类别。 类别允许的主体可搜索此类别的聊天室（除非隐藏了聊天室，在这种情况下，仅聊天室的成员可在此目录中搜索聊天室）。
