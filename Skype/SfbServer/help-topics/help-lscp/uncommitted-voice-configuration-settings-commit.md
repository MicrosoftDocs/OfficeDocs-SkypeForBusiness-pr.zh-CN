# <a name="uncommitted-voice-configuration-settings-commit"></a>未提交的语音配置设置：提交
 
您可以使用"未 **提交的语音** 配置设置"对话框查看您做出的所有更改。 滚动对话框可查看所有未提交的更改，或单击箭头以展开或折叠不同的部分。 完成后，单击“提交”提交所有这些更改，或单击“取消”返回并修改所做的更改。
  
> [!IMPORTANT]
> 如果单击“提交”，则将提交“未提交的语音配置设置”对话框中列出的 **所有** 更改。
  
有关可以使用控制面板执行的不同过程的详细信息，请参阅 Manage [Skype for Business Server Skype for Business Server 2015](../../manage/manage.md)。
