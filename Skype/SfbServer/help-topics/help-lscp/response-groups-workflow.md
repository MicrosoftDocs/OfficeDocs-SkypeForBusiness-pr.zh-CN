# <a name="response-groups-workflow"></a>响应组工作流

响应组组成代理组、 队列和工作流。 响应组工作流定义响应组应用程序接收电话呼叫时执行的操作。

**响应组** - **工作流**页显示一个为组织定义的所有响应组工作流的列表。

## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在**响应组**执行以下任务 - **工作流**页：

- 创建或更改智能寻线工作流

- 创建或更改互动工作流

## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。

- **创建或编辑工作流**打开响应组配置工具创建或编辑工作流。

- **刷新**刷新工作流的列表。

下表介绍了该页上的各个字段。

- **名称**分配给工作流的唯一名称。

- **服务**承载工作流的**ApplicationServer**服务。

- **SIP 地址**将应答工作流呼叫的组的 SIP 地址。

- **电话**调用它以接通此响应组的电话号码。

- **语言**用于语音识别和文本到语音转换的语言。

- **IVR**指示工作流是否寻线还是互动工作流。

- **启用**指示是否激活工作流接收呼叫。

有关响应组特性和功能的详细信息，请参阅规划文档中的[计划中的业务服务器 2015 Skype 的响应组应用程序](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 有关使用响应组工作流的详细信息，请参阅操作文档中的[管理响应组工作流](https://technet.microsoft.com/library/42cfccdd-2844-4875-b4e3-813e1df15f08.aspx)。
