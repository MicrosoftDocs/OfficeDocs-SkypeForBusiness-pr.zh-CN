# <a name="response-groups-workflow"></a>响应组工作流
 
响应组组成代理组、 队列和工作流。 响应组的工作流定义响应组应用程序接收电话呼叫时所执行的操作。 
  
**响应组** - **工作流**页面显示为您的组织定义的所有响应组工作流的列表。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以从**响应组**中执行下列任务 - **工作流**页面：
  
- 创建或更改的查寻组工作流
    
- 创建或更改交互式的工作流
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。
  
- **创建或编辑工作流**打开响应组配置工具创建或编辑工作流。
    
- **刷新**刷新工作流的列表。
    
下表介绍了该页上的各个字段。
  
- **名称**向工作流分配唯一名称。
    
- **服务**承载工作流**应用程序服务器**服务。
    
- **SIP 地址**将应答呼叫到工作流组的 SIP 地址。
    
- **电话**调用以到达此响应组电话号码。
    
- **语言**使用语音识别和文本到语音转换的语言。
    
- **IVR**指示工作流是否查寻组或交互式的工作流。
    
- **启用**指示是否启用工作流来接听电话。
    
有关响应组的特性和功能的详细信息，请参阅规划文档中[规划业务服务器 2015年的 Skype 的响应组应用程序](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 有关使用工作流响应组的详细信息，请参阅[管理响应组工作流](http://technet.microsoft.com/library/42cfccdd-2844-4875-b4e3-813e1df15f08.aspx)操作文档中。
