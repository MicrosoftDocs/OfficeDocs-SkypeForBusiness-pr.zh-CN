# <a name="save-voice-routing-test-case-information"></a>保存语音路由测试用例信息
 
可以使用**路由选择测试用例保存语音信息**对话框中保存 atemporary 语音路由测试用例。 **保存语音路由测试用例信息**，键入想要赋予此测试用例在**名称**，然后单击**确定**的名称。 
  
> [!NOTE]
> 保存临时测试用例时，将不保存为中央管理存储。 相反，它使该测试用例可从**语音路由**页中的**测试语音路由**选项卡。 **Uncommitted**的状态将显示该测试用例。 将测试用例保存到中央管理存储，必须通过单击从任何**语音路由**页中的**操作**菜单上的**全部提交**提交语音配置。
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
