# <a name="select-regions"></a>选择区域
 
使用 **"选择** 区域"对话框查找环境中可用的区域。 要使用现有区域，请单击列表中的某个区域，然后单击“确定”。
  
> [!NOTE]
> 要检索所有可用区域，请不要在搜索框中键入任何内容，然后单击搜索图标。
