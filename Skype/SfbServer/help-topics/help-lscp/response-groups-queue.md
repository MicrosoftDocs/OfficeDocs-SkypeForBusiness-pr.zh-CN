# <a name="response-groups-queue"></a>响应组队列
 
响应组组成代理组、 队列和工作流。 直到代理应答呼叫，响应组队列包含响应组对的调用。 
  
**响应组** - **队列**页显示为您的组织定义的所有响应组队列的列表。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以从**响应组**中执行下列任务 - **队列**页：
  
- 创建新的队列
    
- 更改现有队列
    
- 删除队列
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。
  
- **新**启动一个新队列。
    
- **编辑**打开所选的队列对其进行编辑，在列表中，选择所有队列或删除所选的队列。
    
- **刷新**将刷新队列的列表。
    
下表介绍了该页上的各个字段。
  
- **名称**唯一名称，用于标识队列。
    
- **服务****应用程序服务器**服务的队列中。
    
- **说明**对队列的说明。
    
有关响应组的特性和功能的详细信息，请参阅规划文档中[规划业务服务器 2015年的 Skype 的响应组应用程序](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 有关使用响应组队列的详细信息，请参阅[管理响应组队列](http://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx)操作文档中。
