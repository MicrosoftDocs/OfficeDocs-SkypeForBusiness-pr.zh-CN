# <a name="response-groups-queue"></a>响应组队列

响应组由代理组、队列和工作流组成。 响应组队列保留对响应组的呼叫，直到代理应答呼叫。

"**响应组**  -  **队列**"页显示为组织定义的所有响应组队列的列表。

## <a name="tasks-you-can-perform"></a>可执行的任务

可以从"响应组队列"页 **执行**  -  **以下** 任务：

- 创建新队列

- 更改现有队列

- 删除队列

## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。

- **新建** 启动新队列。

- **编辑** 打开所选队列进行编辑、选择列表中的所有队列或删除所选队列。

- **刷新** 刷新队列列表。

下表介绍了该页上的各个字段。

- **名称** 标识队列的唯一名称。

- **服务** 托管 **队列的 ApplicationServer** 服务。

- **说明** 队列的说明。

有关响应组特性和功能的详细信息，请参阅规划文档中 [的"规划 Skype for Business Server 2015](../../plan-your-deployment/enterprise-voice-solution/response-group.md) 中的响应组应用程序"。 有关使用响应组队列的详细信息，请参阅操作文档中的 ["管理响应组](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) 队列"。
