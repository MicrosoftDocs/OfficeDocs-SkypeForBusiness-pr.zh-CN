# <a name="select-an-announcement-service"></a>选择通知服务
 
使用**选择通知服务**对话框中选择将处理到未分配号码范围的传入呼叫通知服务。
  
在搜索框中，键入您想要使用的公告服务的名称的全部或部分。 要返回所有可用的通知服务，请将搜索框保留为空，然后单击搜索图标。
  
在搜索返回的通知服务列表中，选择您想要用于处理到未分配号码范围的传入呼叫的服务。
