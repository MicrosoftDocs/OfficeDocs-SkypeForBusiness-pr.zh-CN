# <a name="select-time-out-queue"></a>选择超时队列
 
使用**选择超时队列**对话框中选择队列调用位置转发时他们的超时时间。
  
在搜索框中，键入要使用的队列的名称的全部或部分。 若要返回所有可用的队列，请将搜索框保留为空，请单击搜索图标。
  
在队列由搜索返回的列表中，选择要用于调用超时的队列。
