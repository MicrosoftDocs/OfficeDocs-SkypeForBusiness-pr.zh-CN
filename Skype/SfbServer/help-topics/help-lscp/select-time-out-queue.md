# <a name="select-time-out-queue"></a>选择超时队列
 
使用“选择超时队列”对话框可选择呼叫在超时后被转接到的队列。
  
在搜索框中，键入要使用的队列的完整名称或部分名称。若要返回所有可用队列，请将搜索框留空并单击搜索图标。
  
在搜索返回的队列列表中，选择希望用于超时呼叫的队列。
