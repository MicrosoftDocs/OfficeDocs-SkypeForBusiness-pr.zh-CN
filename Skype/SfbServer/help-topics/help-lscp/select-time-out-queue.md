# <a name="select-time-out-queue"></a>选择超时队列
 
使用 "**选择超时队列**" 对话框选择呼叫在拨出时转发的队列。
  
在 "搜索" 框中, 键入要使用的队列的完整名称或部分名称。 若要返回所有可用的队列, 请将搜索框留空, 然后单击 "搜索" 图标。
  
在搜索返回的队列列表中, 选择要用于超时呼叫的队列。
