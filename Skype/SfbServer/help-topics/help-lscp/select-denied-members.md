# <a name="select-denied-members"></a>选择被拒绝的成员
 
持续的聊天管理员可以创建并管理聊天室类别。 为创建和管理聊天室类别的一部分，持续的聊天管理员可以配置主体 （Active Directory 域服务组/容器/用户） 所拥有的权能的特定类别的聊天室的成员/创造者。 持续的聊天管理员还可以向类别中添加 DeniedMembers，它们将成为到允许列表中显式排除。 DeniedMembers 重写中的 AllowedMembers。
  
## <a name="tasks-that-you-can-perform"></a>可执行的任务

您可以在“**选择被拒绝的成员**”页上执行以下任务：
  
- [配置类别](http://technet.microsoft.com/library/4547f514-f0c0-404d-890f-092ddeeac852.aspx)
    
- [新的持久的聊天服务器功能](http://technet.microsoft.com/library/c3ec6f33-6261-4bf5-aa31-baa8ab2a87d8.aspx)
    
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
  
## <a name="to-configure-categories-for-chat-rooms"></a>配置聊天室的类别

在中的**成员资格**，**拒绝的成员**部分中，添加或删除用户和其他 Active Directory 主体与被拒绝从文件室的成员。
