# <a name="response-groups-group"></a>响应组： 组
 
响应组组成代理组、 队列和工作流。 代理程序组是人员的一组指定应答呼叫响应组。 
  
**响应组** - **组**页显示为您的组织定义的所有代理组的列表。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以从**响应组**中执行下列任务 - **组**页：
  
- 创建新的代理组
    
- 更改现有代理组
    
- 删除代理组
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的各个命令。
  
- **新**启动新的代理组。
    
- **编辑**打开选定的代理组对其进行编辑，在列表中，选择所有的代理组或删除选定的代理组。
    
- **刷新**刷新的代理组的列表。
    
下表介绍了该页上的各个字段。
  
- **名称**标识代理组的唯一名称。
    
- **服务****应用程序服务器**服务承载的代理组。
    
- **说明**代理程序组的描述。
    
有关响应组的特性和功能的详细信息，请参阅规划文档中[规划业务服务器 2015年的 Skype 的响应组应用程序](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 有关使用代理组的详细信息，请参阅在运营文档资料的[管理代理组](http://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx)。
