# <a name="select-agents"></a>选择代理
 
代理是被指定为应答呼叫响应组的用户。 响应组必须具有一个分配的代理组来识别可接收针对响应组的呼叫的代理。 创建代理组的一种方法就是通过选择合格用户来定义自定义组。 符合条件的用户启用为 Skype 业务服务器和企业语音。 
  
使用“**选择代理**”对话框可选择要添加到代理组的用户。
  
## <a name="ui-reference"></a>用户界面参考

下表介绍了“**选择代理**”对话框中的各个控件。
  
- **查找**搜索的 SIP 地址或显示名称用户。 输入的地址或名称的全部或部分。 将搜索框留空，以显示所有用户启用了 Skype 业务服务器和企业语音。
    
- **若要显示的最多用户**更改显示返回的结果数。 使用此计数器可以限制搜索，如果您希望结果多。
    
下表介绍了“**选择代理**”对话框中的各个字段。
  
- **代理**显示由搜索返回的用户名称。
    
- **SIP 地址**显示由搜索返回的用户 SIP 地址。
    
- **电话服务**显示**电话**字段中为用户定义的值。
    
- **启用**显示为用户定义的**Lync 服务器启用**字段的值。
    
有关使用代理组的详细信息，请参阅在运营文档资料的[管理代理组](http://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx)。
