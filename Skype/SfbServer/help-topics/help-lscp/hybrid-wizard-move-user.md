# <a name="hybrid-wizard-move-user"></a>混合向导：移动用户

使用"移动用户"向导将本地池中的用户帐户移动到 Skype for Business Online 上。 有关详细信息，请参阅使用 [混合可管理性向导](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
