# <a name="hybrid-wizard-move-user"></a>混合向导：移动用户

使用移动用户向导将驻留在本地池中的用户帐户移动到 Skype for business Online 上。 有关更多信息，请参阅[Use the Hybrid Manageability wizard](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
