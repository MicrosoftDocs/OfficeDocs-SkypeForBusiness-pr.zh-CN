# <a name="select-a-service"></a>选择一个服务
 
使用“**选择服务**”对话框查找环境中可用的服务。要使用现有服务，请单击列表中的某个服务，然后单击“**确定**”。 
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在“**选择服务**”页上执行以下任务：
  
- [配置类别](http://technet.microsoft.com/library/4547f514-f0c0-404d-890f-092ddeeac852.aspx)
    
- [为房间配置外接程序](http://technet.microsoft.com/library/4eeaf19e-8369-4f6f-af65-a283cf7daa1c.aspx)
    
- [在全局范围内或持久聊天服务器池配置持久聊天服务器选项](http://technet.microsoft.com/library/1e8d5245-cd58-4aad-9a1c-35b24189bc40.aspx)
    
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
  
## <a name="to-configure-categories-for-chat-rooms"></a>配置聊天室的类别

中**选择一个服务**，请选择对应于持久的聊天服务器池需要创建类别的服务。 该服务是持久聊天服务器池持续聊天 （客户端） 用来标识该池类别属于。 类别可以属于一个持久聊天服务器池，并且无法移到另一个，或与另一个池共享。 
  
## <a name="to-configure-add-ins-for-chat-rooms"></a>配置聊天室外接程序

中**选择一个服务**，请选择对应于您需要创建外接程序的持久的聊天服务器池的服务。 外接程序无法从一个池移到另一个池，或者在不同池之间共享。
  
## <a name="to-configure-persistent-chat-options-for-a-specific-persistent-chat-server-pool"></a>若要配置特定的持久性聊天服务器池的持久聊天选项

中**选择一个服务**，请选择与要配置持久聊天服务器池相关联的服务。
  
持久的聊天服务器的特性和功能的详细信息，请参阅规划文档中的[持久聊天服务器的概述](http://technet.microsoft.com/library/23f7c886-304d-495a-ae70-3cbb44241acd.aspx)。 有关使用持久聊天服务器配置的详细信息，请参阅[配置持久聊天服务器](http://technet.microsoft.com/library/d90a4049-b268-4e8e-9f24-0cef08c8d9ed.aspx)部署文档和[管理 Lync Server 2013、 持久聊天服务器](http://technet.microsoft.com/library/82befdc6-5d32-45f1-bfd7-aaedffed1ab8.aspx)中操作文档中。
