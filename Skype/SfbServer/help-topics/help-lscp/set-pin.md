# <a name="set-pin"></a>设置 PIN

若要生成用户的 PIN，请选择 **自动生成有效的 PIN。**

要创建自己的 PIN，请选择“手动输入特定 PIN”，然后在文本框中键入满足 PIN 策略设置中指定的 PIN 要求的 PIN。

有关可以使用控制面板执行的不同过程的详细信息，请参阅 Manage [Skype for Business Server Skype for Business Server 2015](../../manage/manage.md)。
