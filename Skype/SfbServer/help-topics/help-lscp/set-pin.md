# <a name="set-pin"></a>设置 PIN
 
要生成该用户的 PIN，请选择**自动生成有效的针**。 
  
要创建自己的 PIN，请选择“**手动输入特定 PIN**”，然后在文本框中键入满足 PIN 策略设置中指定的 PIN 要求的 PIN。 
  
您还可以通过 Skype 业务服务器控件面板的不同过程的详细信息，请参阅[管理的业务服务器 2015年的 Skype](../../manage/manage.md)。
