# <a name="set-pin"></a>设置 PIN

若要生成用户的 PIN，请选中**自动生成有效 PIN**。

要创建自己的 PIN，请选择“**手动输入特定 PIN**”，然后在文本框中键入满足 PIN 策略设置中指定的 PIN 要求的 PIN。

有关您可以使用适用于业务 Server Control Panel Skype 执行的各种过程的详细信息，请参阅[管理的 Skype 的业务服务器 2015年](../../manage/manage.md)。
