您希望执行什么操作？单击您要执行的任务所对应的链接。
  
 **使用 IP 电话的 PIN 策略**该个人标识号码 (PIN) 策略支持通过 IP 电话业务服务器连接到 Skype 的用户身份验证。
- [在 Skype 为业务服务器 2015年中创建的新 PIN 策略](../../manage/authentication/create-a-new-pin-policy.md)
    
- [修改现有的 PIN 策略在 Skype 业务服务器 2015](../../manage/authentication/modify-an-existing-pin-policy.md)
    
- [删除业务服务器 2015年在 Skype 的 PIN 策略](../../manage/authentication/delete-a-pin-policy.md)

- [创建或修改站点或组用户拨入会议 PIN 设置](http://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)
    
- [修改默认拨入会议 PIN 设置](http://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)
    
- [删除站点或组用户拨入会议 PIN 设置](http://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
