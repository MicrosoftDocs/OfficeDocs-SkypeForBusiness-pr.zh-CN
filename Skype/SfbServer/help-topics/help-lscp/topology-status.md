# <a name="topology-status"></a>拓扑：状态

“拓扑****: 状态****”页将显示 Skype for Business 拓扑中服务器的状态。

## <a name="tasks-you-can-perform"></a>可执行的任务

您可以在“拓扑****: 状态****”页上执行以下任务：

- [查看运行 Lync Server 2010 的计算机列表](https://technet.microsoft.com/library/44eeec27-8b99-44f0-b0bd-622c12393d34.aspx)

- [查看有关服务的详细信息](https://technet.microsoft.com/library/bc8e8202-cd68-47e4-95b2-bb36e51cc124.aspx)

- [启动或停止 Lync Server 2010 服务](https://technet.microsoft.com/library/1c70b4ec-9de5-4f7a-a3c9-c0eb76710505.aspx)

- [防止新连接到服务](https://technet.microsoft.com/library/977dcc5c-2aac-48ef-86a1-a8d47b4d9e74.aspx)

有关您可以使用适用于业务 Server Control Panel Skype 执行的各种过程的详细信息，请参阅[管理的 Skype 的业务服务器 2015年](../../manage/manage.md)。
