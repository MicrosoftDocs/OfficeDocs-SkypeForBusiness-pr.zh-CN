# <a name="clients-mobility-policy"></a>客户端： 移动策略
 
**客户端****移动策略**页显示可用您的部署中的行动策略。
  
## <a name="tasks-you-can-perform"></a>可执行的任务

在**移动策略**页上，可以执行以下任务：
  
- 查看现有的移动策略
    
- 创建新的移动策略
    
- 编辑移动策略
    
## <a name="ui-reference"></a>用户界面参考

下表介绍了该页上的菜单、命令、字段和属性。
  
### 

- **搜索字段**若要筛选所显示的策略的文本框中键入字符串。
    
- **新**单击**新建**，然后选择**站点**或**用户**以创建新的移动策略。
    
- **编辑**选择策略，然后单击**编辑**来编辑所选策略的属性。
    
- **刷新**刷新显示的策略列表。 当单击**刷新**时，搜索筛选器字符串被清除并且显示所有现有的策略。
    
有关配置移动策略的详细信息，请参阅[配置移动策略](http://technet.microsoft.com/library/595536e0-9bb3-49a3-8d13-1a77351ebc62.aspx)。
