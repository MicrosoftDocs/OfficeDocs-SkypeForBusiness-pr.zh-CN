# <a name="users-view-persistent-chat"></a>用户： 查看持久聊天

**CsPersistentChatAdministrator**角色 （持久聊天管理员） 的成员可以向其他用户和自己授予聊天室管理员权限和权限。

A Persistent Chat Administrator 可以执行以下操作：

- 创建聊天室类别和聊天室。

- 设置所有类别和聊天室的成员身份。

- 管理所有类别和聊天室的所有设置。

- 启用或禁用策略 for Persistent Chat Server。

- 设置和管理持久聊天服务器池上的配置设置。

有关详细信息，请参阅部署文档中的[Adding a Persistent Chat Administrator](https://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx) 。
