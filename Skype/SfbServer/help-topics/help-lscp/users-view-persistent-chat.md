# <a name="users-view-persistent-chat"></a>用户：查看持久聊天

**CsPersistentChatAdministrator** 角色的成员 (持久聊天管理员) 可以将聊天室管理员权限授予其他用户和自己。

持久聊天管理员可以执行以下操作：

- 创建聊天类别和聊天室。

- 设置所有类别和聊天室的成员身份。

- 管理所有类别和聊天室的所有设置。

- 启用或禁用持久聊天服务器的策略。

- 在持久聊天服务器池上设置和管理配置设置。

有关详细信息，请参阅部署文档中的[Adding a Persistent Chat Administrator](/previous-versions/office/lync-server-2013/lync-server-2013-adding-a-persistent-chat-administrator)。

## <a name="tasks-that-you-can-perform"></a>可执行的任务

您可以在“用户: 查看持久聊天”页上执行以下任务：

- [允许用户管理类别、聊天室和用户权限](/previous-versions/office/lync-server-2013/lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions)

- [了解持久聊天成员身份](/previous-versions/office/lync-server-2013/understanding-persistent-chat-membership)

有关可以使用 Skype for Business Server 控制面板执行的不同过程的详细信息，请参阅[管理 Skype for Business Server 2015。](../../manage/manage.md)
