# <a name="certificate-assignment-invoke"></a>证书分配（调用）
 
"**执行命令**" 页面显示任务将证书分配给服务器所执行的操作。 完成任务后，您可以查看详细介绍证书分配过程的日志。 要查看日志文件，请单击“**查看日志**”。
