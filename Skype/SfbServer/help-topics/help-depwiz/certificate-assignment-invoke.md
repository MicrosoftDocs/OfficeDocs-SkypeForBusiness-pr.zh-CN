# <a name="certificate-assignment-invoke"></a>证书分配 （调用）
 
**执行命令**页显示将证书分配给服务器的任务所采取的操作。 任务完成后，您可以查看证书分配过程的详细信息的日志。 要查看日志文件，请单击“**查看日志**”。
