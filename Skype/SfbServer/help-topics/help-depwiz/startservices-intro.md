# <a name="startservices-intro"></a>启动服务（简介）
 
在 Skype 正在部署的业务服务器 2015年角色服务器上启动服务，请查看简介屏幕。 请注意，发出启动 Skype 业务服务器的命令不等待服务能够启动。 这是启动服务的最佳操作，您应当使用 Windows 服务 Microsoft 管理控制台 (MMC) 监控服务的实际状态。 单击“**下一步**”开始“**启动服务**”任务。
