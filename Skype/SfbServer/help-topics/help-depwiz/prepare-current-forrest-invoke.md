# <a name="prepare-current-forrest-invoke"></a>准备当前廖 （调用）
 
**执行命令**页运行时显示任务的摘要。 查看摘要屏幕并确保任务均成功，并且没有任何错误。 检查以确保在预期的警告。 在摘要窗格，您应看到**任务状态： 已完成**。 在下拉列表中，日志文件**准备林**应是可见的。 如果不是，选择日志文件，然后单击**查看日志**查看详细的操作和结果的任务。 单击**完成**以完成此目录林的准备工作。
