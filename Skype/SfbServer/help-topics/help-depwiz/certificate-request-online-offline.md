# <a name="certificate-request-online-offline"></a>证书请求（联机/脱机）
 
要创建延迟请求或脱机请求，请选择“**立即准备请求，但是稍后发送（脱机证书请求）**”。 若要创建即时请求给联机证书颁发机构 （通常在贵组织的公钥基础结构内部系统），选择**发送给联机证书颁发机构立即请求**。
  
做出选择后，单击“**下一步**”继续。如果要取消而不进行更改，请单击“**取消**”。要返回上一步，请单击“**上一步**”。
