# <a name="prepare-current-forest-universal-group-location"></a>准备当前林 （通用组位置）
 
您必须指定需要在林中创建通用组。 **通用组位置**页使用您登录到计算机的成员作为默认值的域。 如果您希望在不是默认**本地域**的域中创建通用组，选择**域的 FQDN** ，并在文本框中键入的完全合格的域名称 (FQDN)。 创建提供非通用组所在域中运行林准备任务时，备用域很有帮助。 单击“**下一步**”继续。
