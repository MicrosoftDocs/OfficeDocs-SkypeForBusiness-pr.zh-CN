# <a name="certificate-request-certificate-signing-request-generated"></a>证书申请 （证书签名生成的请求）
 
已完成脱机证书请求后，会显示**证书请求文件**页。 若要显示的证书申请，并准备将其发送给您脱机的证书颁发机构 (CA) 或公共 CA，请单击**视图**按钮。 证书请求 （默认） 显示在 Windows 记事本。 您可以复制和粘贴此显示屏，从编码的证书申请或完成按照说明从您选择脱机 CA 或公共 CA 的证书请求的处理。 单击**完成**以继续。
