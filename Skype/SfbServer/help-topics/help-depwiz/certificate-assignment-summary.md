# <a name="certificate-assignment-summary"></a>证书分配（摘要）
 
在证书分配完成之前，你将看到一个**证书分配摘要**，其中列出了将分配的证书的详细信息。 查看信息，确保主题名称和主题备用名称正确。 查看完证书摘要后，单击 "**下一步**"。
