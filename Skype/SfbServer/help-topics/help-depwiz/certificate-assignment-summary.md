# <a name="certificate-assignment-summary"></a>证书分配 （摘要）
 
证书分配完毕之前，系统会显示**证书分配摘要**列出将分配证书的详细信息。 检查以确保使用者名称和主题备用名称正确无误的信息。 完成审阅证书摘要后，单击**下一步**。
