# <a name="start-services-invoke"></a>启动服务（调用）
 
**执行命令**窗格上的摘要窗格中显示为开始为要部署的业务服务器 2015年角色服务器 Skype 的服务而发出的任务的状态。 任务窗格中的摘要不代表服务启动状态的实时指示。 有些业务服务器服务 Skype 可能会开始在初始启动过程中较长的时间。 任务会发出启动命令，但不等待确定服务是否成功启动。 如果必须监控服务启动和服务状态，应使用 Windows 服务 Microsoft 管理控制台 (MMC)。
  
任务窗格下方是将显示“**启动服务**”日志文件的下拉列表。要查看日志文件，请单击“**查看日志**”。单击“**完成**”完成任务。
