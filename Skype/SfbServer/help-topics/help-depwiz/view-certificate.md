# <a name="view-certificate"></a>查看证书
 
在**查看证书**页中，没有一个窗格使您能够在本地证书存储中选择任何有效的证书。 您可以选择一个证书，，然后单击**查看证书详细信息**。 该证书将显示在一个新窗格。 在**证书**窗格中，可以查看一般的证书信息。 单击**详细信息**选项卡显示有关该证书使用者名称和主题备用名称等详细的信息。 证书路径将显示"信任链"的证书的路径。 当您完成查看证书信息，请单击**确定**。
