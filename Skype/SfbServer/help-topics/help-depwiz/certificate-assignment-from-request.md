# <a name="certificate-assignment-from-request"></a>证书分配（根据请求）
 
**证书分配**页启动过程将证书分配给角色的服务器上的请求的用法。 单击**查看证书详细信息**以查看证书。 您可以选择**常规**选项卡、**详细信息**选项卡或**证书路径**选项卡。**常规**选项卡将显示基本的证书，如证书颁发机构和一个私有密钥的状态信息。 详细信息选项卡将显示证书信息，例如使用者名称、 主题备用名称和其他详细信息。 **证书路径**选项卡将显示证书"信任链"通过任何颁发或中间证书颁发机构 （如果它们存在） 证书从根证书颁发机构。
  
单击“**下一步**”继续。
