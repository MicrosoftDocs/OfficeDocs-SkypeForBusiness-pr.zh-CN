# <a name="sql-server-reporting-services-intro"></a>SQL Server 报表服务 （简介）
 
每个前端池和高存活力的分支装置只能具有一个与之相关联的监视服务器。 当对网站启用监视时，监视服务器将提供呼叫详细记录 (CDR) 和体验质量 (QoE) 数据收集和报告。
  
站点上的所有池和多个中心站点的池可以使用同一个监视服务器，如果使用率不超过监视服务器的容量。 设计的拓扑结构，以支持监视的详细信息，请参阅部署文档中[将一个前端池在业务服务器 2015年的 Skype 的监视存储相关联](../../deploy/deploy-monitoring/associate-a-monitoring-store.md)。
