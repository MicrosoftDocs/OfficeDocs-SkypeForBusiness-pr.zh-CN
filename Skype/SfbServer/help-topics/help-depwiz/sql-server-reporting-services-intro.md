# <a name="sql-server-reporting-services-intro"></a>SQL Server Reporting Services（简介）
 
每个前端池和 Survivable Branch Appliance 只能拥有一台关联的监控服务器。如果为站点启用监控，监控服务器会提供呼叫详细信息记录 (CDR) 和用户体验质量 (QoE) 数据收集和报告。
  
站点中的所有池和多个中央站点的池可以使用相同的监控服务器，前提是使用量不超过监控服务器的容量。 有关设计支持监控的拓扑的详细信息，请参阅部署文档中的 Associate [a monitoring store with a Front End pool in Skype for Business Server 2015](../../deploy/deploy-monitoring/associate-a-monitoring-store.md)。
