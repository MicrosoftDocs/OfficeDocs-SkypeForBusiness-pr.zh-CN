# <a name="remove-certificate-invoke"></a>删除证书 （调用）
 
**执行命令**页提供有关删除证书的详细信息。 查看证书删除摘要信息。 请确保删除证书日志文件在下拉列表中，可见，然后单击**查看日志**以查看证书删除的详细信息。 检查摘要和日志文件完毕后，单击**完成**。
