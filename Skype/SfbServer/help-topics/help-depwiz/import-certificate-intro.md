# <a name="import-certificate-intro"></a>导入证书 （简介）
 
导入证书，您必须提供的证书文件的路径。 在**选择证书文件**文本框中，您可以键入完整路径和文件名，或单击**浏览**按钮，导航到的路径位置和文件名 （通常.p7b、.pfx 或.cer 文件）。
  
如果证书包含私钥，请选择**证书文件包含证书的专用密钥**。 选中此复选框后，将启用**密码**文本输入。 如果您有与它相关联的私钥的证书，密码通常放置在私钥创建证书时。 您输入的密码允许证书的专用密钥和私钥导入到证书存储区。 当您提供了信息的证书的文件路径，和私钥密码 （可选） 如果需要，请单击**下一步**。
  
> [!IMPORTANT]
> 如果您不知道的私钥的密码，则导入将失败。
