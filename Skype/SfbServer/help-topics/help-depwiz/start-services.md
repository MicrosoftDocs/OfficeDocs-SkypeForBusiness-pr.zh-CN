# <a name="start-services"></a>启动服务
 
要启动部署的服务器角色上的服务，请单击“步骤 4: 启动服务”中的“运行”。必须以本地 Administrators 组的成员或等效身份登录。 
  
> [!NOTE]
> 如果已运行“启动服务”，可以单击“再次运行”来重新运行此命令。
