# <a name="start-services"></a>启动服务
 
已部署的服务器角色上启动服务，单击**运行**中**步骤 4： 启动服务**。 您必须作为本地管理员组或与之等同的成员身份登录。 
  
> [!NOTE]
> 如果已运行**启动服务**，您可以通过**运行再次**单击重新运行命令。
