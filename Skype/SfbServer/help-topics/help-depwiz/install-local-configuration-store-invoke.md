# <a name="install-local-configuration-store-invoke"></a>安装本地配置存储（调用）
 
“执行命令”窗格中显示安装中央管理存储的本地只读副本的任务的状态。任务运行过程中，会根据运行的命令更新任务窗格。应检查任务状态并注意显示的所有错误或警告。任务完成后，将显示“任务状态: 已完成”消息，并且日志文件下拉列表将显示日志文件的名称。要查看创建的日志文件，请选择要查看的日志文件并单击“查看日志”。检查所有日志文件的成功、错误和警告消息。
  
检查日志文件后，单击“完成”以完成本地配置存储的安装。
