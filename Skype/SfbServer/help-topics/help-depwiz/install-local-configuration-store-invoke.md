# <a name="install-local-configuration-store-invoke"></a>安装本地配置存储区 （调用）
 
**执行命令**窗格显示有关安装中央管理存储的本地只读副本任务的状态。 在任务运行时，命令运行时，将更新任务窗格。 您应该查看任务状态并注意任何错误或警告，显示。 当任务完成后，消息**任务状态： 已完成**显示，以及日志文件的下拉列表中显示的日志文件的名称。 若要查看已创建的日志文件，请选择日志文件查看并单击**查看日志**。 查看所有日志文件成功，错误和警告消息。
  
您已经检查过的日志文件后，单击**完成**以完成安装的本地配置存储。
