# <a name="pending-certificate-invoke"></a>待处理的证书（调用）
 
"**执行命令**" 页面显示正在检索挂起证书的命令的状态。 任务完成后，"**获取挂起的证书**" 日志文件可用。 单击 "**查看日志**" 以查看处理和任务完成状态。 单击“**下一步**”继续。
  
> [!IMPORTANT]
> 如果证书尚未由证书颁发机构管理员批准，则检索挂起的请求日志文件将指示存在挂起的请求。 如果证书颁发机构管理员已拒绝请求，则日志文件将指示失败并报告错误，指示请求被拒绝，以及拒绝请求的人员的用户名。
