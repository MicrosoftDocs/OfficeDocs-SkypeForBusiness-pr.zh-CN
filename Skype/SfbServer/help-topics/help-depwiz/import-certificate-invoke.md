# <a name="import-certificate-invoke"></a>导入证书 （调用）
 
在**执行命令**页上查看导入的证书的摘要信息。 在下拉列表中，选择导入证书日志文件。 请确保导入证书日志文件将显示在下拉列表中。 单击要查看的证书导入详细信息**查看日志**。 查看完摘要和日志文件后，单击**完成**。
