# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services（组）

在 "**指定只读" 组**中，指定要向其授予对监视报告的只读访问权限的组。 使用 "设置&lt;域&gt;\\&lt;访问权限&gt;" 组。 例如： contoso\RTCUniversalReadOnlyAdmins。
  
分配组只读访问权限是可选的。 如果您不希望立即授予只读访问权限，则可以在以后直接在 SQL Server Reporting Services 中对其进行配置。
