# <a name="sql-server-reporting-services-groups"></a>SQL Server 服务 （组） 的报告

**指定只读组**中，指定要向其授予对监视报告的只读访问权限的组。 使用格式&lt;域&gt;\\&lt;访问组&gt;。 例如： contoso\RTCUniversalReadOnlyAdmins。
  
分配组只读访问权限的操作是可选的。 如果您不想现在授予只读访问权限，您可以配置它以后直接在 SQL Server 报告服务。
