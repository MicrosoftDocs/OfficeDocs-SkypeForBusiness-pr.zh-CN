# <a name="certificate-request-certificate-signing-request"></a>证书申请 （申请进行签名的证书）
 
若要创建脱机请求，您必须首先创建证书签名请求 (CSR)。 在**文件名**文本框中键入该 CSR 的路径和文件名称。 您可以单击**浏览**以选择一个位置来创建 CSR。 单击“**下一步**”继续。
  
单击**重新**回到前一个屏幕。 单击**取消**以退出没有更改系统。
  
> [!IMPORTANT]
> 在此步骤中创建的文件到您的内部脱机 CA，发送到公用证书颁发机构 (CA) 或提交给 CA 管理员进行处理。 请注意文件位置和名称，以便可以将它检索出来并提交给 CA 的 CSR。 默认位置是文档目录运行证书向导的人员。
