# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>安装或删除 Skype for Business Server 组件（调用）
 
"**执行" 命令**显示用于部署服务器角色的运行任务的运行摘要。 任务完成后（部署和激活，或停用和卸载-将显示摘要。 此外，在下拉列表中，你可以选择运行命令时创建的任何日志文件。 使用下拉列表选择要查看的日志文件，然后单击 "**查看日志**"。
  
查看完摘要和日志文件后，单击 "**完成**"。
