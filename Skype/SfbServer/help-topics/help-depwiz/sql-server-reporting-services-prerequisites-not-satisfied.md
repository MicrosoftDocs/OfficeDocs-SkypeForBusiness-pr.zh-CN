# <a name="sql-server-reporting-services-prerequisites-not-satisfied"></a>SQL Server Reporting Services（不满足先决条件）
 
如果未在基础结构中部署监控服务器，则会看到此页。这表示尚未满足部署监控服务器报告的最低要求。 
  
要解决此问题，请确保您具有监视服务器加入到域，定义拓扑生成器和已发布的拓扑结构。 SQL Server 报表服务也必须在 SQL Server 中，并作为一项功能安装到监视服务器上 SQL Server 数据库。 
  
有关详细信息，请参阅[在 Skype 的业务服务器 2015年安装监视报告](../../deploy/deploy-monitoring/install-monitoring-reports.md)和[部署监视](http://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)。
