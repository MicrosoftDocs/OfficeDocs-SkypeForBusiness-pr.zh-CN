# <a name="certificate-request-geography"></a>证书请求（地理位置）
 
“地理信息”页要求提供证书请求的其他信息。在此页上，选择“国家/地区”，然后在文本框中键入“省/市/自治区”和“市/县”。请注意，应避免使用缩写，并拼写出省/市/自治区和市/县的完整名称。
