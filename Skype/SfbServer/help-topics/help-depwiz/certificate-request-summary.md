# <a name="certificate-request-summary"></a>证书请求（摘要）
 
“证书请求摘要”页显示证书的所有选项和配置的详细摘要。摘要中显示证书请求将包含的内容，以及后续将如何颁发证书。您应仔细阅读摘要以确保选择的所有条目和选项都是正确的。如果选择正确，可以单击“下一步”继续生成请求的操作，并且准备将其发送给证书颁发机构 (CA)。
