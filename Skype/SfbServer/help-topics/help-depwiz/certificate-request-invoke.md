# <a name="certificate-request-invoke"></a>证书请求（调用）
 
运行生成证书请求的命令时，**“执行命令”** 页会显示任务状态窗格。 任务完成后，摘要窗格下的下拉列表中包含日志文件列表。 使用下拉列表选择要查看的日志文件，然后单击"查看 **日志"。** Internet Explorer 将启动并显示该日志文件。 如果请求失败，应检查日志文件以确定失败的原因。 可以重新运行证书请求以解决问题，并更正 日志文件。 继续浏览证书请求页面，然后再次请求证书。
