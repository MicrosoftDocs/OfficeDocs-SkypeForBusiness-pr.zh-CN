# <a name="certificate-request-invoke"></a>证书申请 （调用）
 
**执行命令**页将任务状态窗格显示为运行生成证书请求的命令。 完成任务后，摘要窗格下面的下拉列表将包含日志文件的列表。 使用下拉列表选择要查看，日志文件，然后单击**查看日志**。 Internet Explorer 将启动并显示日志文件。 对于失败的请求，您应该查看日志文件以确定失败的原因。 可以重新运行证书申请，若要解决此问题，请更正该问题以日志文件表示。 尽管继续证书请求页面，并再次请求证书。
