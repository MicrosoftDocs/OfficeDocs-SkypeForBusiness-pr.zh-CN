# <a name="install-or-update-skype-for-business-server-system"></a>安装或更新 Skype for Business Server 系统

若要开始安装或更新 Skype for Business Server 2015 的现有部署，请单击"安装"或"更新 **Skype for Business Server"。** 本节中的步骤将基于拓扑生成器文档安装特定于给定服务器角色的组件。 在开始这一系列步骤之前，必须在拓扑生成器中定义部署。 必须以本地管理员和服务器所在的域中 Domain Admins 组成员的身份登录到要部署的计算机。 在完成此步骤之前，计算机必须加入该域。

如果要更新当前部署的服务器，则必须在拓扑生成器中进行更改。
