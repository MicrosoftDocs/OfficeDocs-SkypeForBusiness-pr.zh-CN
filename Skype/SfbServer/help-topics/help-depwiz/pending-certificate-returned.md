# <a name="pending-certificate-returned"></a>挂起的证书 （返回）
 
**联机证书请求的状态**页返回的证书的当前状态。 如果证书仍处于待定审批状态的证书颁发机构管理员--，的如果没有尚未批准 — — 状态将表明证书仍在处理。
  
如果挂起的请求已获批准，而且成功检索的证书，其状态将显示已分配给本地证书存储中，证书，和状态窗格也会显示证书指纹。
  
如果证书颁发机构管理员拒绝证书申请，则状态窗格将报告具有给定 ID （由证书颁发机构维护） 的挂起证书请求被拒绝。
  
单击**完成**以完成挂起的证书处理。
