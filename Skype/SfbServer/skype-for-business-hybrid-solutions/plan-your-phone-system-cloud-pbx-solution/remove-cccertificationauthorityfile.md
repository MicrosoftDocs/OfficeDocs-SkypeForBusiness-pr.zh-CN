# <a name="remove-cccertificationauthorityfile"></a>删除 CcCertificationAuthorityFile
 
删除 CcCertificationAuthorityFile cmdlet 删除证书颁发机构服务备份文件"&lt;SiteRootDirectory&gt;\CA\SfB CCE Root.p12"业务云连接器的 Skype 网站共享目录下的 CA 文件夹中版。 
  
```
Remove-CcCertificationAuthorityFile
```

## <a name="parameters"></a>参数

无
  
## <a name="examples"></a>示例
<a name="Examples"> </a>

### <a name="example-1"></a>示例 1

下面的示例删除证书颁发机构服务备份文件"&lt;SiteRootDirectory&gt;\CA\SfB CCE Root.p12"网站共享目录下的 CA 文件夹中：
  
```
Remove-CcCertificationAuthorityFile
```

## <a name="input-types"></a>输入类型
<a name="InputTypes"> </a>

无。Remove-CcCertificationAuthorityFile cmdlet 不接受主线输入。
  
## <a name="return-types"></a>返回类型
<a name="ReturnTypes"> </a>

无
