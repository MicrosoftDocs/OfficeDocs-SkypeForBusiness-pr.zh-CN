# <a name="get-ccappliancestatus"></a>Get-CcApplianceStatus
 
Get-CcApplianceStatus cmdlet 提供设备诊断Skype for Business 云连接器版本信息。
  
此 cmdlet 适用于 Skype for Business 云连接器版本 2.1。
  
```
    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>详细说明

当Get-CcApplianceStatus连接器主机上运行时，Get-CcApplianceStatus cmdlet 会提供云连接器设备的诊断Skype for Business信息。
  
## <a name="parameters"></a>参数

无
  
## <a name="input-types"></a>输入类型

无。 Get-CcApplianceStatus cmdlet 不接受通过管道的输入。
  
## <a name="return-types"></a>返回类型

该命令返回设备的状态。
