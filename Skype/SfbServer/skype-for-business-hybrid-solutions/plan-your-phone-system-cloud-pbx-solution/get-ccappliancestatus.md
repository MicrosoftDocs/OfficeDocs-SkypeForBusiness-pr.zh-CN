# <a name="get-ccappliancestatus"></a>获得 CcApplianceStatus
 
获得 CcApplianceStatus cmdlet 提供商务云连接器版装置的 Skype 的诊断信息。
  
此 cmdlet 适用于 Skype 业务云连接器版本 2.1。
  
```

    Get-CcApplianceStatus 
```

## <a name="detailed-description"></a>详细说明

获得 CcApplianceStatus cmdlet，云连接器宿主计算机上，运行时提供 Skype 业务云接口装置的诊断的信息。
  
## <a name="parameters"></a>参数

无
  
## <a name="input-types"></a>输入类型

无。 获得 CcApplianceStatus cmdlet 不接受管道的输入。
  
## <a name="return-types"></a>返回类型

该命令返回设备的状态。
