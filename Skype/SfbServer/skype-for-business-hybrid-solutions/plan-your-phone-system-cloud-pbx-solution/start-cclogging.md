# <a name="start-cclogging"></a>Start-CcLogging
 
Start-CcLogging cmdlet 用于为 Skype for Business 云连接器版本设备生成传入和传出呼叫日志。 
  
```
Start-CcLogging
```

## <a name="parameters"></a>参数

无
  
## <a name="examples"></a>示例
<a name="Examples"> </a>

### <a name="example-1"></a>示例 1

以下示例生成传入和传出呼叫日志：
  
```
Start-CcLogging
```

## <a name="detailed-description"></a>详细说明
<a name="DetailedDescription"> </a>

开始 CcLogging cmdlet 提供了一种方法，供管理员用来开始日志记录云连接器装置的传入和传出呼叫。 默认情况下，日志记录将在四个小时后自动停止。
  
## <a name="input-types"></a>输入类型
<a name="InputTypes"> </a>

无。Start-CcLogging cmdlet 不接受主线输入。
  
## <a name="return-types"></a>返回类型
<a name="ReturnTypes"> </a>

无
