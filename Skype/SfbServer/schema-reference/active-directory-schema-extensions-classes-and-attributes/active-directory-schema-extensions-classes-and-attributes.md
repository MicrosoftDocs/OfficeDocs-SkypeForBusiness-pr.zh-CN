此引用部分包括以下信息： 
  
- 将新的或更改的业务服务器的 Skype 的活动目录架构扩展
    
    Active Directory 架构中包含可以在 Active Directory 目录林中创建的每个对象类的正式定义。 该架构还包含可存在于 Active Directory 对象的每个属性的正式定义。 Active Directory 全局编录包含的目录林的每个对象属性的子集以及所有对象的副本。 本部分介绍的类和新的或更改在 Skype 业务服务器的属性。
    
- 用于通过 Skype 业务服务器，每个说明的所有类
    
- 用于通过 Skype 业务服务器，每个说明的所有属性
    
- 可能包含用于通过 Skype 业务服务器与每个属性的类的列表
    
- 全局设置和对象，除在林准备过程中创建的通用服务和管理组
    
- 在域准备过程创建域根目录和内置容器的访问控制项 (Ace)
    
- 由 Grant_CsSetupPermission cmdlet 的 Active Directory 组织单位 (OU) 进行的更改。
    
- 活动目录 OU 上通过 Grant_CsOUPermission cmdlet 所做的更改。
