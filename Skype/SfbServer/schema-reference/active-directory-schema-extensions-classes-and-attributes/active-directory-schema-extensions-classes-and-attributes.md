# <a name="active-directory-schema-extensions-classes-and-attributes"></a>Active Directory 架构扩展名、类和属性
 
本参考部分包括以下信息： 
  
- 新增或更改的 Skype 业务服务器的 active Directory 架构扩展
    
    Active Directory 架构包含可在 Active Directory 林中创建每个对象类的正式定义。 该架构还包含每个属性的 Active Directory 对象上可能存在的正式定义。 Active Directory 全局编录包含林，以及每个对象的属性的子集的所有对象的副本。 本节介绍的类和属性的新的或更改 Skype 业务服务器中。
    
- 每个说明中使用 Business Server Skype 的所有类
    
- 每个说明中使用 Business Server Skype 的所有属性
    
- 与每个属性中使用 Business Server Skype 的类列表以及可能包含
    
- 全局设置和对象，以及在林准备期间创建的通用服务和管理组
    
- 域准备过程中在域根和内置容器中创建的访问控制项 (Ace)
    
- 通过 Grant_CsSetupPermission cmdlet 对 Active Directory 组织单位 (OU) 进行的更改。
    
- 通过 Grant_CsOUPermission cmdlet Active Directory OU 所做的更改。
