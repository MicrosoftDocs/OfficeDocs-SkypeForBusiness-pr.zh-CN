# <a name="schema-attributes-and-descriptions-in-skype-for-business-server"></a>架构属性和 Skype 中的业务服务器的说明
 
本节介绍用于通过 Skype 业务服务器的所有架构属性。 有关与属性关联的类，请参阅[Skype 业务服务器中的类的架构属性](schema-attributes-by-class.md)。
  
链接的对属性被指定为正向链接或反向链接。 指向另一个对象的属性是正向链接;反过来引用的第一个对象的其他对象的属性是反向链接。 正向链接可更新，，而反向链接为只读的。
  
有些属性具有位掩码值。 对于这些属性，由一个位表示每个设置和显示的十进制值表示位的位置。 与 0 位启动位的位置。 例如，1 （二进制） 是 0 位组和 10000 （二进制） 是 4 位组。 每个位表示的属性。 以下是一些示例： 
  
- 10000 （二进制） 的十进制值为 16 （即，设置第 4 位）。
    
- 100000000 （二进制） 的十进制值为 256 （即，设置 8 位）。
    
- 1100 （二进制） 的十进制值为 12 （即，设置 2 和 3 位; 由这两个位表示属性已启用）。
    
- 1111000001 （二进制） 的十进制值为 961 （即，设置 0、 6、 7、 8 和 9 位; 这些位的每个属性已启用）。
