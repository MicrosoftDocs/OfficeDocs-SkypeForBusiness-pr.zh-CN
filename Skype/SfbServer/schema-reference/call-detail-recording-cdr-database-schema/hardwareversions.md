# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>在业务服务器 2015年的 Skype 的 HardwareVersions 表
 
HardwareVersions 表是支持表。 每个记录都存储着一个设备硬件版本信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primary  <br/> |标识此硬件版本的唯一编号。  <br/> |
|**版本** <br/> |nvarchar(256)  <br/> | <br/> |硬件版本。  <br/> |
