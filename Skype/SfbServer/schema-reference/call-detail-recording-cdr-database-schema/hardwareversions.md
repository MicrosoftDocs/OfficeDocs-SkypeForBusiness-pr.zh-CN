# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>HardwareVersions 表中的业务服务器 2015 Skype
 
HardwareVersions 表是一个支持表。 每个记录存储设备硬件版本信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |Primary  <br/> |标识此硬件版本的唯一编号。  <br/> |
|**版本** <br/> |nvarchar(256)  <br/> | <br/> |硬件版本。  <br/> |
