# <a name="uritypes-table"></a>UriTypes 表
 
UriTypes 表中包含不同的 URI （统一资源标识符） 类型业务服务器 2015年的 Skype 进行监控。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**UriTypeId** <br/> |tinyint  <br/> |Primary  <br/> |分配给一个 URI 类型的唯一标识符。  <br/> |
|**UriType** <br/> |nvarchar(256)  <br/> || 不同的 URI 类型的说明。 允许的值包括： <br/>  0-电话 Uri <br/>  1-用户 Uri <br/> |
