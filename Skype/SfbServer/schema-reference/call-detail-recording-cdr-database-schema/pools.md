# <a name="pools-table"></a>Pools 表
 
Pool 表是存储有关各种池的信息的支持表。 表中的每条记录表示一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |Primary  <br/> |标识此池的唯一号码。  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |池 FQDN。  <br/> |
