# <a name="pools-table"></a>Pools 表
 
Pools 表是一个支持表，用于存储有关各种池的信息。 表中的每条记录代表一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**池 Id** <br/> |int  <br/> |Primary  <br/> |标识此池的唯一编号。  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |池 FQDN。  <br/> |
