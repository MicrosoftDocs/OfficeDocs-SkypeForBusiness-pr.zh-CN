# <a name="pools-table"></a>Pools 表
 
池表是一个支持的表来存储有关各种池的信息。 每个表中的记录表示一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**池 Id** <br/> |int  <br/> |Primary  <br/> |标识此池的唯一编号。  <br/> |
|**PoolFQDN** <br/> |nvarchar(256)  <br/> | <br/> |池的 FQDN。  <br/> |
