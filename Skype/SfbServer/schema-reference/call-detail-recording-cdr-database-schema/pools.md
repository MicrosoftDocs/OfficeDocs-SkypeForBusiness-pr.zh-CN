# <a name="pools-table"></a>Pools 表
 
Pools 表是一个支持表，用于存储有关各种池的信息。 表中的每条记录分别表示一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |主  <br/> |标识此池的唯一编号。  <br/> |
|**PoolFQDN** <br/> |nvarchar (256)   <br/> | <br/> |池 FQDN。  <br/> |
