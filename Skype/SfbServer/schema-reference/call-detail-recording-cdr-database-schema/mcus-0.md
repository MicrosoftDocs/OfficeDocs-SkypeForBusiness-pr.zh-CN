# <a name="mcus-view"></a>Mcus 视图
 
Mcus 视图存储参与会议会话的 MCUs 的相关信息。 此视图已在 Microsoft Lync Server 2013 中引入。
  
|**列**|**数据类型**|**详细信息**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |标识 MCU 的唯一号码。  <br/> |
|**McuUri** <br/> |nvarchar (450)  <br/> |MCU 的 URI。  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |MCU URI 的类型。 有关详细信息, 请参阅[UriTypes 表](uritypes.md)。 <br/> |
