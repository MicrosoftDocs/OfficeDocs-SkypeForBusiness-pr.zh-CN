# <a name="conferenceuris-view"></a>ConferenceUris 视图
 
ConfernceUris 视图存储已参与会议会话的 Uri 的信息。 此视图是在 Microsoft Lync Server 2013 中引入的。
  
|**列**|**数据类型**|**详细信息**|
|:-----|:-----|:-----|
|ConferenceUriId  <br/> |int  <br/> |标识此会议 URI 的唯一编号。  <br/> |
|ConferenceUri  <br/> |nvarchar(450)  <br/> |会议的 URI。  <br/> |
|ConferenceUriType  <br/> |nvarchar(256)  <br/> |会议 URI 的类型。 请参阅[UriTypes 表](uritypes.md)的详细信息。 <br/> |
