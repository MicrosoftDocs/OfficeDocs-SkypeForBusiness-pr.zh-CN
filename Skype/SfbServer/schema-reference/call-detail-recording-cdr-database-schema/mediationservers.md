# <a name="mediationservers-table"></a>MediationServers 表
 
MediationServers 表是支持表。 每个记录存储有关在数据库中具有记录的通话中涉及的一个中介服务器的信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |标识此中介服务器的唯一号码。  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |中介服务器名称。  <br/> |
