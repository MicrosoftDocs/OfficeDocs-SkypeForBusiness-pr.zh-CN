# <a name="mediationservers-table"></a>MediationServers 表
 
MediationServers 表是一个支持表。 每条记录都存储有关在数据库中具有记录的呼叫中涉及的一台中介服务器的信息。
  
|**列**|**数据类型**|**键/索引**|**Details**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |主  <br/> |标识此中介服务器的唯一编号。  <br/> |
|**MediationServer** <br/> |nvarchar (256)   <br/> | <br/> |中介服务器名称。  <br/> |
