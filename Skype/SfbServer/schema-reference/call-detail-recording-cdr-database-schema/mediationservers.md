# <a name="mediationservers-table"></a>MediationServers 表
 
MediationServers 表是支持表。 每个记录存储在数据库中具有记录的调用所涉及的一个中介服务器的信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**MediationServerId** <br/> |int  <br/> |Primary  <br/> |标识此中介服务器的唯一编号。  <br/> |
|**MediationServer** <br/> |nvarchar(256)  <br/> | <br/> |中介服务器的名称。  <br/> |
