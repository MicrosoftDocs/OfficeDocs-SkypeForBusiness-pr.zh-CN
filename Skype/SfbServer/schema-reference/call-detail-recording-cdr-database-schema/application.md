# <a name="application-table-in-skype-for-business-server-2015"></a>应用程序表中的业务服务器 2015 Skype
 
该表存储路由和连接涉及的业务服务器 2015年有关 Skype 内的各种进程的信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**ApplicationId** <br/> |int  <br/> |Primary  <br/> |标识此应用程序的唯一编号。  <br/> |
|**名称** - 按 WAN 链路进行筛选（筛选器位于图形右侧）。 <br/> |nvarchar(256)  <br/> | <br/> |服务器组件的名称。  <br/> |
