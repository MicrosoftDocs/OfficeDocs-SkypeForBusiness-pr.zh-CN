# <a name="servers-table"></a>Servers 表
 
服务器表是一个支持表，用于存储有关各种服务器的信息。 表中的每条记录代表一个服务器。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |Primary  <br/> |标识此服务器的唯一编号。  <br/> |
|**ServerFQDN** <br/> |nvarchar(256)  <br/> | <br/> |服务器 FQDN。  <br/> |
