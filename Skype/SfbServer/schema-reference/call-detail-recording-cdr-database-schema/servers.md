# <a name="servers-table"></a>Servers 表
 
Servers 表是一个支持表，用于存储有关各种服务器的信息。 表中的每条记录代表一台服务器。
  
|**列**|**数据类型**|**键/索引**|**Details**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |主  <br/> |标识此服务器的唯一编号。  <br/> |
|**ServerFQDN** <br/> |nvarchar (256)   <br/> | <br/> |服务器 FQDN。  <br/> |
