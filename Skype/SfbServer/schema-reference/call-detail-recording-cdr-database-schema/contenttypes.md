# <a name="contenttypes-table-in-skype-for-business-server-2015"></a>在业务服务器 2015年的 Skype 的内容类型表
 
内容类型表是一个支持的表来存储对等会话和会议会话中使用的内容类型的列表。 每个表中的记录表示一种内容类型。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**ContentTypeId** <br/> |int  <br/> |Primary  <br/> |标识内容类型的唯一编号。  <br/> |
|**ContentType** <br/> |nvarchar(256)  <br/> ||内容类型名称。  <br/> |
