# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>Manufacturers 表中的业务服务器 2015 Skype
 
Manufacturers 表是一个支持表。 每个记录存储设备 （桌面电话） 制造商的信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |标识此制造商的唯一编号。  <br/> |
|**制造商** <br/> |nvarchar(256)  <br/> | <br/> |制造商名称。  <br/> |
