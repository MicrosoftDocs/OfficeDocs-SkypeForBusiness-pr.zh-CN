# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>在业务服务器 2015年的 Skype 的制造商表
 
制造商表是一个支持的表。 每个记录都存储着一个设备 （桌面电话） 制造商联系的信息。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |Primary  <br/> |标识此制造商的唯一编号。  <br/> |
|**制造商** <br/> |nvarchar(256)  <br/> | <br/> |制造商的名称。  <br/> |
