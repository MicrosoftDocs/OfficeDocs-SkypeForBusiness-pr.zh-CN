# <a name="tblsiopwhitelist"></a>tblSiopWhiteList
 
tblSiopWhiteList 是已注册的外接程序可与节点相关联的列表。
  
**列**

|**列**|**类型**|**说明**|
|:-----|:-----|:-----|
|siopID  <br/> |GUID，不为空  <br/> |外接程序的 GUID。  <br/> |
|siopName  <br/> |nvarchar (50) 不为空  <br/> |外接程序的显示名称。  <br/> |
|siopUrl  <br/> |nvarchar (255) 不为空  <br/> |外接程序的 URL。  <br/> |
   
**密钥**

|**列**|**说明**|
|:-----|:-----|
|siopID  <br/> |为主键。  <br/> |
