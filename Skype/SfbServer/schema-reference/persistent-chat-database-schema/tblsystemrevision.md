# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含与 tblAdminLock 表一起使用的修订号，以实现多个管理员客户端之间的一致性。
  
**多**

|**列**|**类型**|**说明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint，not null  <br/> |修订号。  <br/> |
