# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含用于 tblAdminLock 表的修订号，以实现多个管理员客户端的一致性。
  
**Columns**

|**列**|**类型**|**说明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint，不为 null  <br/> |修订次数。  <br/> |
