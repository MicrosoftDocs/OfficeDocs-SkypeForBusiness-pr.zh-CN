# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含与 tblAdminLock 表一起用于跨多个管理员客户端实现一致性的修订号。
  
**列**

|**列**|**类型**|**说明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint，不为 null  <br/> |修订号。  <br/> |
