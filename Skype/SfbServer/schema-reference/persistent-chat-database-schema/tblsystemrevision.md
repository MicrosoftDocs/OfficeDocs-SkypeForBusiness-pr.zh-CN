# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含使用 tblAdminLock 表中用于跨多个管理员客户端实现一致性的修订号。
  
**列**

|**列**|**类型**|**说明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint，不为空  <br/> |修订号。  <br/> |
