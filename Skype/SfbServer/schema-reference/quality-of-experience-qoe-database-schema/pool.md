# <a name="pool-table"></a>Pool 表
 
池表是一个支持的表来存储有关各种前端池的信息。 每个表中的记录表示一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PoolKey** <br/> |int  <br/> |Primary  <br/> |标识此池的唯一编号。  <br/> |
|**PoolName** <br/> |nvarchar(256)  <br/> |唯一  <br/> |池的 FQDN。  <br/> |
