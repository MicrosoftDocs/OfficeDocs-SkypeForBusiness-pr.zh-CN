# <a name="pool-table"></a>Pool 表
 
Pool 表是存储有关各种前端池的信息的支持表。表中的每条记录分别表示一个池。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PoolKey** <br/> |int  <br/> |主  <br/> |标识此池的唯一编号。  <br/> |
|**PoolName** <br/> |nvarchar (256)   <br/> |独特  <br/> |池 FQDN。  <br/> |
