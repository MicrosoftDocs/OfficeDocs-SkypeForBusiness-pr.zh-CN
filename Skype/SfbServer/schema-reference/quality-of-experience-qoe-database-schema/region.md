# <a name="region-table"></a>Region 表
 
区域表是一个支持表。每条记录代表网络配置设置中定义的一个国家/区域。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |主  <br/> |标识国家/区域的唯一编号。  <br/> |
|**RegionName** <br/> |nvarchar (128)   <br/> |独特  <br/> |国家/区域的名称。  <br/> |
