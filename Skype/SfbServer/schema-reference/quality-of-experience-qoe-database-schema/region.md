# <a name="region-table"></a>Region 表
 
区域表是一个支持的表。 每个记录表示网络配置设置中定义的一个国家/地区。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Primary  <br/> |标识的国家/地区的唯一编号。  <br/> |
|**RegionName** <br/> |nvarchar(128)  <br/> |唯一  <br/> |国家/地区的名称。  <br/> |
