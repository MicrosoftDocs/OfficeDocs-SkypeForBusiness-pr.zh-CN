# <a name="region-table"></a>Region 表
 
区域表是支持表。 每条记录表示在 "网络配置" 设置中定义的一个国家/地区。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Primary  <br/> |标识国家/地区的唯一号码。  <br/> |
|**RegionName** <br/> |nvarchar  <br/> |唯一  <br/> |国家/地区的名称。  <br/> |
