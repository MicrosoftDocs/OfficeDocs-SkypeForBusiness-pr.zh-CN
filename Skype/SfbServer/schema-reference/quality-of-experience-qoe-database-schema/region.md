# <a name="region-table"></a>Region 表
 
区域表是一个支持表。 每条记录代表一个国家/地区网络配置设置中定义。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |Primary  <br/> |标识国家/地区的唯一编号。  <br/> |
|**RegionName** <br/> |nvarchar(128)  <br/> |唯一  <br/> |国家/地区的名称。  <br/> |
