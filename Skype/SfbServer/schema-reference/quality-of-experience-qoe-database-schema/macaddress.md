|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |主  <br/> |标识 Mac 地址的唯一编号。  <br/> |
|**MacAddress** <br/> |varchar (256)   <br/> |独特  <br/> |Mac 地址字符串。  <br/> |
