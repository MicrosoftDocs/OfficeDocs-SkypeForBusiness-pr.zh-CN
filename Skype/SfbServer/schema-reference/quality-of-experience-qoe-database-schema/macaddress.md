|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |Primary  <br/> |标识的 Mac 地址的唯一编号。  <br/> |
|**MacAddress** <br/> |varchar(256)  <br/> |唯一  <br/> |Mac 地址字符串。  <br/> |
