# <a name="devicedriver-table"></a>DeviceDriver 表
 
DeviceDriver 表是支持表。 每个记录都表示一个由捕获设备或呈现设备使用的驱动程序。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primary  <br/> |标识此设备驱动程序记录的唯一号码。  <br/> |
|**DeviceDriver** <br/> |varchar （256）  <br/> |唯一  <br/> |设备驱动程序名称。  <br/> |
