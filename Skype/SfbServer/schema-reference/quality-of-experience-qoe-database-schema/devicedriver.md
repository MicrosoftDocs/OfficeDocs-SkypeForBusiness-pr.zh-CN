# <a name="devicedriver-table"></a>DeviceDriver 表
 
DeviceDriver 表是支持表。 每个记录表示一个捕获设备使用的驱动程序，或导致设备。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primary  <br/> |标识此设备的驱动程序记录的唯一编号。  <br/> |
|**DeviceDriver** <br/> |varchar(256)  <br/> |唯一  <br/> |设备驱动程序的名称。  <br/> |
