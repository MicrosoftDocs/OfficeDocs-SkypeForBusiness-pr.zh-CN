# <a name="devicedriver-table"></a>DeviceDriver 表
 
DeviceDriver 表是一个支持表。 每条记录代表由捕获设备驱动程序或呈现设备。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |Primary  <br/> |标识此设备驱动程序记录的唯一编号。  <br/> |
|**DeviceDriver** <br/> |varchar(256)  <br/> |唯一  <br/> |设备驱动程序名称。  <br/> |
