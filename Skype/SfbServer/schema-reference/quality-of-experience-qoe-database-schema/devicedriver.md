# <a name="devicedriver-table"></a>DeviceDriver 表
 
DeviceDriver 表是一个支持表。每条记录都代表捕获设备或呈现设备所使用的一个驱动程序。
  
|**列**|**数据类型**|**键/索引**|**Details**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |主  <br/> |用于标识此设备驱动程序记录的唯一编号。  <br/> |
|**DeviceDriver** <br/> |varchar (256)   <br/> |unique  <br/> |设备驱动程序的名称。  <br/> |
