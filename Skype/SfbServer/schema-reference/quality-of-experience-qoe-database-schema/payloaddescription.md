# <a name="payloaddescription-table"></a>PayloadDescription 表
 
PayloadDescription 表是一个支持表。每条记录都代表一个音频或视频会话使用的编解码器。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |主  <br/> |用于标识该编解码器的唯一编号。  <br/> |
|**PayloadDescription** <br/> |nvarchar (256)   <br/> |独特  <br/> |编解码器的名称。  <br/> |
