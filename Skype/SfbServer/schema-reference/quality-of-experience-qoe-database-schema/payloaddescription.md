# <a name="payloaddescription-table"></a>PayloadDescription 表
 
自 PayloadDescription 表是一个支持表。 每条记录代表一个编解码器，用于在音频或视频会话。
  
|**列**|**数据类型**|**键/索引**|**详细信息**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |Primary  <br/> |标识该编解码器的唯一编号。  <br/> |
|**PayloadDescription** <br/> |nvarchar(256)  <br/> |唯一  <br/> |编解码器的名称。  <br/> |
