# <a name="media-quality-diagnostic-reports-in-skype-for-business-server"></a>Skype for Business Server 中的媒体质量诊断报告
 
**摘要：** 了解 Skype for Business Server 中的媒体质量诊断报告。
  
媒体质量诊断报告提供有关呼叫质量的信息以及失败的呼叫的诊断和疑难解答信息。
  
## <a name="in-this-section"></a>本节内容

- [Skype for Business Server 中的媒体质量摘要报告](summary.md) 提供不同终结点类型的整体质量数据，包括 企业语音 对等呼叫、企业语音 电话会议和至少部分依赖公用电话交换网 (PSTN) 的呼叫。
    
- [Skype for Business Server 中的媒体质量比较报告](comparison.md) 提供不同类型的音频呼叫的呼叫质量值的比较 (例如，通过无线网络拨打的呼叫与通过有线连接网络) 。
    
- [Skype for Business Server 中的服务器性能报告](server-performance.md) 根据诸如降级、数据包丢失和抖动等关键质量指标的度量，列出遇到最多问题的服务器。
    
- [Skype for Business Server 中的位置报告](location-report.md) 提供网络位置的列表以及每个位置发生的呼叫的媒体质量摘要。 对于此报告，位置基于 IP 子网。
    
- [Skype for Business Server 中的设备报告](device-report.md) 提供用于呼叫的设备的摘要企业语音包括按设备表示的呼叫的平均媒体质量。
    
- [Skype for Business Server 中的呼叫列表报告](call-list-report-0.md) 提供有关在组织中拨打或接听的电话呼叫的详细信息。
    
- [Skype for Business Server 中的呼叫详细信息报告](call-detail-report.md) 提供有关从组织内部拨打或接听的电话的详细信息。
    
- [Skype for Business Server 中的服务器媒体质量趋势报告](server-media-quality-trend-report.md) 提供了一种在用户体验质量指标（如呼叫量、质量欠佳的呼叫百分比、数据包丢失和抖动）上以图形方式比较最多五台服务器的方法。
    
- [Skype for Business Server 中的媒体质量指标分布报告](media-quality-metrics-distribution-report.md) 提供显示用户体验质量指标（如抖动或数据包丢失）的分布值的图形。
    
- [Skype for Business Server 中的位置趋势报告](location-trend-report.md) 提供网络位置的呼叫质量趋势信息。
