# <a name="conference-join-time-report-in-skype-for-business-server"></a>会议加入时间报告Skype for Business Server
 
**摘要：** 了解会议加入时间摘要报告中Skype for Business Server。
  
会议加入时间摘要使您能够确定用户加入会议所花的时间。 该报告显示了 (的平均加入时间（以毫秒) 为单位），并提供了一个细目，让您了解在 2 秒或更少时间内能够加入会议的用户数、需要 2 到 5 秒加入会议的用户数，等等。
  
## <a name="accessing-the-conference-join-time-report"></a>访问会议加入时间报告

会议加入时间报告从监控报告主页访问。
  
## <a name="filters"></a>筛选器

利用筛选器，您可以返回一组针对性更强的数据或通过不同的方式查看返回的数据。 下表列出了可用于会议加入时间报告的筛选器。
  
**会议加入时间报告筛选器**

|**名称**|**说明**|
|:-----|:-----|
|**From** <br/> |时间范围的开始日期/时间。若要按小时查看数据，请输入开始日期和时间，如下所示：  <br/> 7/7/2015 1：00 PM  <br/> 如果您未输入开始时间，该报告会自动将某个特定日的上午 12:00 作为开始时间。若要按日查看数据，请只输入日期：  <br/> 7/7/2015  <br/> 若要按周或按月查看，请输入您要查看的周或月中的任一日期（您不必输入周或月的第一天）：  <br/> 7/3/2015  <br/> 一周始终是从星期日开始至星期六结束。  <br/> |
|**To** <br/> |时间范围的结束日期/时间。若要按小时查看数据，请输入结束日期和时间，如下所示：  <br/> 7/7/2015 1：00 PM  <br/> 如果您未输入结束时间，该报告会自动将某个特定日的上午 12:00 作为结束时间。若要按日查看数据，请只输入日期：  <br/> 7/7/2015  <br/> 若要按周或按月查看，请输入您要查看的周或月中的任一日期（您不必输入周或月的第一天）：  <br/> 7/3/2015  <br/> 一周始终是从星期日开始至星期六结束。  <br/> |
|**Interval** <br/> | 时间间隔。选择下列选项之一： <br/>  每小时（最多可显示 25 个小时） <br/>  每天（最多可显示 31 天） <br/>  每周（最多可显示 12 周） <br/>  每月（最多可显示 12 个月） <br/>  如果开始日期和结束日期超出了所选间隔允许的最长时间，则仅显示最长时间（从开始日期开始）。 例如， 如果您选择开始日期为 2015/7/7、结束日期为 2015/2/28 的"每天"间隔，则会显示从 2015 年 8 月 7 日上午 12：00 到 2015 年 9 月 7 日上午 12：00 (即总共 31 天的数据) 。 <br/> |
|**Pool** <br/> |注册器池或边缘服务器的完全限定域名 (FQDN)。可以选择单个池，也可以单击“[所有]”查看所有池的数据。系统根据数据库中的记录自动为您填充该下拉列表。<br/> |
|**会议会话** <br/> | 会话类型。 允许的值包括： <br/>  [All] <br/>  焦点 (是联机会议的中心策略和状态管理器，协调会议的各个方面 <br/>  应用程序共享 <br/>  A/V 会议 <br/>  如果选择"[所有]"，则总会议加入时间将显示在报告的顶部。 请注意，这些总数仅适用于使用 Microsoft Exchange 或 Microsoft Outlook 安排的会议。 <br/> |
   
## <a name="metrics"></a>度量标准

下表列出了会议加入时间报告中提供的信息。
  
**会议加入时间报告指标**

|**名称**|**是否可按此项排序？**|**说明**|
|:-----|:-----|:-----|
|**Date** <br/> 此指标的实际标题将因选择的 Interval 而异。  <br/> |否  <br/> |会议的日期和时间。  <br/> |
|**会话总数** <br/> |否  <br/> |会话的总数，包括成功的会话、失败的会话（预期失败和意外失败）及未归类的会话。  <br/> |
|**平均 (毫秒)** <br/> |否  <br/> |参与者加入 (所用的) 的平均时间量（以毫秒为单位）。  <br/> |
|**会话 \< 2 seconds, Volume** <br/> |否  <br/> |在不到 2 秒钟内加入会议的参与者数量。  <br/> |
|**会话 \< 2 seconds, Percentage** <br/> |否  <br/> ||
|**会话 2-5 秒，音量** <br/> |否  <br/> |在 2 秒到 5 秒之间加入会议的参与者数量。  <br/> |
|**会话 2-5 秒，百分比** <br/> |否  <br/> |在 2 秒到 5 秒之间加入会议的总呼叫参与者百分比。  <br/> |
|**会话 5-10 秒，量** <br/> |否  <br/> |在 5 秒到 10 秒之间加入会议的参与者数量。  <br/> |
|**会话 5-10 秒，百分比** <br/> |否  <br/> |在 5 秒到 10 秒之间加入会议的总呼叫参与者百分比。  <br/> |
|**\>会话 10 秒，卷** <br/> |否  <br/> |需要 10 秒以上时间才能加入会议的参与者数量。  <br/> |
|**\>会话 10 秒，百分比** <br/> |否  <br/> |需要 10 秒以上时间加入会议的总呼叫参与者百分比。  <br/> |
