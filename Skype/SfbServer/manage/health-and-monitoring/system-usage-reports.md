# <a name="system-usage-reports-in-skype-for-business-server"></a>Skype 业务服务器中的系统使用情况报告
 
**摘要：** 了解业务服务器中 Skype 系统使用情况报告。
  
系统使用情况报告提供基于呼叫详细信息记录 (CDR) 数据收集 Skype 业务服务器的系统使用情况信息。
  
## <a name="in-this-section"></a>本节内容

- [Skype 中的企业服务器用户注册报告](user-registration-report.md)
    
    提供对业务服务器部署基于注册事件，如用户登录 Skype 用户连接的摘要。 该报告提供一种方法查看内部和外部登录，并比较登录到 Skype 业务服务器使用的实际使用该服务已登录时的用户数量的用户数。
    
- [对等活动中 Skype 业务服务器的摘要报告](peer-to-peer-activity-summary-report.md)
    
    提供对等即时消息 (IM)、音频、视频、文件传输和应用程序共享会话摘要。对等会话是指仅涉及两个用户的会话。
    
- [会议中 Skype 业务服务器的摘要报告](conference-summary-report.md)
    
    提供所有会议活动摘要。 会议是涉及三个或更多人的会话。
    
- [PSTN 会议摘要报告中 Skype 业务服务器](pstn-conference-summary-report.md)
    
    提供所有 PSTN 会议摘要。 这些会议是至少一个用户使用公用电话交换网 (PSTN) 拨入的会议，该会议又称为电话拨入式会议。
    
- [Skype 中的业务服务器的响应组使用情况报告](response-group-usage-report.md)
    
    提供响应组使用情况的摘要。 响应组应用程序提供了一种方法为您自动电话呼叫路由到实体如帮助支持或客户支持行。
    
- [IP 电话清单报告中 Skype 业务服务器](ip-phone-inventory-report.md)
    
    提供了有关组织中当前正在使用的 IP 电话的信息。 报表基于电话注册和登录。 它不应考虑的完整清单。 例如，您可能已删除因为他们至少一次登录仍列出在报表中的电话。 同样，您可能还需要新的电话不会显示在报表中只需因为用户具有到 Skype 业务服务器与其新的电话尚未登录。
    
- [为业务 Server Skype 中呼叫允许控制报告](call-admission-control-report.md)
    
    提供使用呼叫允许控制的对等活动和会议活动的列表。呼叫允许控制 (CAC) 是一种确定是否应根据带宽限制允许建立实时通信会话（如语音呼叫或视频呼叫）的方法。
