# <a name="system-usage-reports-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 中的系统使用情况报告
 
**摘要：**了解业务服务器 2015 Skype 在系统使用情况报告。
  
系统使用率报告提供系统基于呼叫详细记录 (CDR) 数据收集的业务服务器通过 Skype 的用法信息。
  
## <a name="in-this-section"></a>本节内容

- [用户注册报告在 Skype 业务服务器 2015](user-registration-report.md)
    
    提供业务服务器 2015年部署基于注册的事件，例如用户登录到 Skype 用户连接的摘要。 该报告提供一种方法以查看内部和外部登录，并比较的用户在登录 Skype 的业务服务器 2015年与实际使用该服务时，已登录的用户数。
    
- [对等活动在 Skype 业务服务器 2015年的摘要报告](peer-to-peer-activity-summary-report.md)
    
    提供对等即时消息 (IM)、音频、视频、文件传输和应用程序共享会话摘要。对等会话是指仅涉及两个用户的会话。
    
- [会议在 Skype 业务服务器 2015年的摘要报告](conference-summary-report.md)
    
    提供所有会议活动摘要。会议是涉及三个或更多人的会话。
    
- [PSTN 会议摘要报表在 Skype 业务服务器 2015](pstn-conference-summary-report.md)
    
    提供所有 PSTN 会议摘要。 这些是其中至少一个用户拨入使用公共交换的电话网络 (PSTN)，也指拨入会议会议。
    
- [在 Skype 业务服务器 2015年响应组使用情况报告](response-group-usage-report.md)
    
    提供响应组用法的摘要。 响应组应用程序的方式为您提供了自动路由到一个帮助等实体的电话联络到书桌或客户支持热线。
    
- [IP 电话库存报告在 Skype 业务服务器 2015](ip-phone-inventory-report.md)
    
    提供有关当前在组织中使用 IP 电话的信息。 该报告基于电话的注册和登录。 它不应视为完整的清单。 例如，您可能已删除了因为他们至少一次登录仍然列在报告的电话。 同样，您可能还有新的电话，不会显示在报告中只是因为用户已到 Skype 业务服务器用新手机尚未登录。
    
- [Skype 在调用业务服务器 2015年许可控制报告](call-admission-control-report.md)
    
    提供使用呼叫允许控制的对等活动和会议活动的列表。呼叫允许控制 (CAC) 是一种确定是否应根据带宽限制允许建立实时通信会话（如语音呼叫或视频呼叫）的方法。
