# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>Skype for Business Server 中的呼叫诊断报告
 
**摘要：** 了解 Skype for Business Server 中使用的多用户呼叫诊断报告。
  
呼叫诊断报告提供失败的点对点会话和会议会话的摘要信息及诊断数据。
  
## <a name="in-this-section"></a>本节内容

- [Skype for Business Server 中的呼叫诊断摘要报告](summary-report.md) 提供失败的点对点会话和会议会话的整体摘要。 对等会话是仅涉及两个参与者的会话。 会议会话涉及三个或多个参与者。
    
- Skype for Business Server 中的对等[活动诊断报告](peer-to-peer-activity-diagnostic-report.md)提供失败的对等会话的整体趋势视图。 对等会话仅涉及两个参与者。
    
- [Skype for Business Server 中的会议诊断报告](conference-diagnostic-report.md) 提供失败的会议会话的整体趋势视图和每种会议形式的趋势视图。 会议会话至少涉及三个参与者。
    
- [Skype for Business Server 中的热门故障报告](top-failures-report.md) 提供最常见的故障及其随着时间的推移的趋势的列表。
    
- [Skype for Business Server 中的故障分布报告](failure-distribution-report.md) 提供失败会话的分析。
    
- [Skype for Business Server 中的故障列表报告](failure-list-report.md) 提供有关失败的会议所涉及的各个参与者的详细信息。
    
- [Skype for Business Server 中的诊断报告](diagnostic-report.md) 提供诊断和疑难解答 (包括 SIP 响应代码和诊断标头以及失败) 的 ID。
