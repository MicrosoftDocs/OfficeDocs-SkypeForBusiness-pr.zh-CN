# <a name="call-diagnostic-reports-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 中的呼叫诊断报告
 
**摘要：**了解多用户调用诊断报告使用 Skype 业务服务器 2015年。
  
呼叫诊断报告提供失败的对等会话和会议会话的摘要信息及诊断数据。
  
## <a name="in-this-section"></a>本节内容

- [调用在 Skype 业务服务器 2015年诊断摘要报告](summary-report.md)提供了失败的对等会话和讲座的摘要信息。 对等会话是涉及两个参与者的会话。 会议会话涉及三个或更多的参与者。
    
- [对等活动的诊断报告在 Skype 业务服务器 2015](peer-to-peer-activity-diagnostic-report.md)提供总体趋势调查失败对等会话。 对等会话涉及两个参与者。
    
- [会议诊断报告在 Skype 业务服务器 2015](conference-diagnostic-report.md)提供每个会议模态失败的会议会话的总体趋势视图和趋势视图。 会议会话涉及至少三个参与者。
    
- [Top 错误报告在 Skype 业务服务器 2015](top-failures-report.md)随着时间的推移提供最常见的失败和其趋势列表。
    
- [在 Skype 业务服务器 2015年报告故障分布](failure-distribution-report.md)提供失败的会话分析。
    
- [在 Skype 业务服务器 2015年报告故障列表](failure-list-report.md)提供有关失败的会议中所涉及的各个参与者的详细的信息。
    
- [在 Skype 业务服务器 2015年的诊断报告](diagnostic-report.md)失败的会话提供诊断和故障排除信息 （包括 SIP 响应代码诊断页眉和 Id）。
