# <a name="peer-to-peer-activity-summary-report-in-skype-for-business-server"></a>Skype for Business Server 中的对等活动摘要Skype for Business Server
 
**摘要：** 了解 Skype for Business Server 中的对等活动摘要报告。
  
点对点活动摘要报告提供点对点通信会话的整体视图。 对等会话通常仅涉及两个用户，并且不需要使用 Skype for Business Server会议服务。 相比之下，会议通常涉及两名以上用户，并且要求使用Skype for Business Server会议服务。 会议活动在会议摘要报告上进行报告。
  
点对点活动摘要报告帮助您回答诸如下列问题：
  
- 在特定某一天，我的用户发送了多少点对点即时消息？
    
- 我的任何用户是否确实在利用Skype for Business Server和文件传输功能？
    
- 用户曾报怨在一天的特定时间网络看上去很慢。在此期间点对点音频和视频会话占用了多少分钟时间？
    
## <a name="accessing-the-peer-to-peer-activity-summary-report"></a>访问点对点活动摘要报告

可从监视报告主页访问点对点活动摘要报告。 通过单击[以下指标之一](im-report.md)，Skype for Business Server中打开对等 IM 报告：
  
- 对等 IM 会话总数
    
- 点对点 IM 消息总数
    
同样，通过单击以下任何指标可打开点对点语音和视频报告：
  
- 点对点音频会话总数
    
- 点对点音频会话分钟总数
    
- 点对点音频会话总数
    
- 点对点音频会话分钟总数
    
## <a name="making-the-best-use-of-the-peer-to-peer-activity-summary-report"></a>充分利用点对点活动摘要报告

在点对点活动摘要报告的底部，您将找到总标准数，例如，总点对点 IM 会话数和总点对点 IM 消息。这提供了报告正文中找到的详细信息的快速摘要。
  
## <a name="filters"></a>筛选器

利用筛选器，您可以返回一组针对性更强的数据或通过不同的方式查看返回的数据。例如，点对点活动摘要报告允许您选择数据的分组方式。在此示例中，将按小时、日、周或月对活动进行分组。
  
下表列出了可用于点对点活动摘要报告的筛选器。
  
**点对点活动摘要报告筛选器**

|**名称**|**说明**|
|:-----|:-----|
|**From** <br/> |时间范围的开始日期和时间。若要按小时查看数据，请输入开始日期和时间，如下所示：  <br/> 7/17/12015 1：00 PM  <br/> 如果您未输入开始时间，该报告会自动将某个特定日的上午 12:00 作为开始时间。若要按日查看数据，请只输入日期：  <br/> 7/17/12015  <br/> 若要按周或按月查看，请输入您要查看的周或月中的任一日期（您不必输入周或月的第一天）：  <br/> 7/13/2015  <br/> 一周始终是从星期日开始至星期六结束。  <br/> |
|**To** <br/> |时间范围的结束日期和时间。若要按小时查看数据，请输入结束日期和时间，如下所示：  <br/> 7/17/12015 1：00 PM  <br/> 如果您未输入结束时间，该报告会自动将某个特定日的上午 12:00 作为结束时间。若要按日查看数据，请只输入日期：  <br/> 7/17/12015  <br/> 若要按周或按月查看，请输入您要查看的周或月中的任一日期（您不必输入周或月的第一天）：  <br/> 7/13/2015  <br/> 一周始终是从星期日开始至星期六结束。  <br/> |
|**Interval** <br/> | 时间间隔。选择下列选项之一： <br/>  每小时（最多可显示 25 个小时） <br/>  每天（最多可显示 31 天） <br/>  每周（最多可显示 12 周） <br/>  每月（最多可显示 12 个月） <br/>  如果开始日期和结束日期超出了所选间隔允许的最长时间，则仅显示最长时间（从开始日期开始）。 例如，如果您选择开始日期为 2015/7/17、结束日期为 2015/2/28 的"每天"间隔， 显示 8/7/12015 12：00 AM 到 9/7/12015 12：00 AM (即总共 31 天的数据) 。 <br/> |
   
## <a name="metrics"></a>度量标准

下表列出了点对点活动摘要报告中提供的信息。
  
**点对点活动摘要报告指标**

|**名称**|**是否可按此项排序？**|**说明**|
|:-----|:-----|:-----|
|**每小时** <br/> **每天** <br/> **每周** <br/> **每月** <br/> |否  <br/> |指示在筛选器工具栏上选择的时间间隔。 如果适用，可单击某一给定的时间间隔以查看该间隔的详细信息。 例如，如果使用"每天"间隔并单击 7/17/12015，则会看到该日期的用户注册活动的每小时细分信息。  <br/> |
|**点对点会话总数** <br/> |否  <br/> |发起的点对点会话总数（无论是哪种会话类型）。  <br/> |
|**点对点 IM 会话总数** <br/> |否  <br/> |点对点即时消息 (IM) 会话总数。单击此项时，报告将显示所选时间段的点对点 IM 报告。  <br/> |
|**点对点 IM 消息总数** <br/> |否  <br/> |在点对点会话中发送的即时消息总数。单击此项时，报告将显示所选时间段的点对点 IM 报告。  <br/> |
|**点对点音频会话总数** <br/> |否  <br/> |点对点音频呼叫总数。单击此字段时，报告将显示所选时间段的点对点语音和视频报告。  <br/> |
|**点对点音频会话总分钟数** <br/> |否  <br/> |点对点音频会话持续的总时间。 单击此项时，报告将显示所选时间段的点对点语音和视频报告。  <br/> |
|**点对点音频会话平均分钟数** <br/> |否  <br/> |点对点音频会话持续的平均时间。计算方法是音频会话总时间除以音频会话总数。  <br/> |
|**点对点视频会话总数** <br/> |否  <br/> |点对点视频呼叫总数。请注意，视频会话还算作音频会话：每个视频会话算作一个视频会话和一个音频会话。单击此项时，报告将显示所选时间段的点对点语音和视频报告。  <br/> |
|**点对点视频会话总分钟数** <br/> |否  <br/> |点对点视频会话持续的总时间。单击此项时，报告将显示所选时间段的点对点语音和视频报告。  <br/> |
|**点对点视频会话平均分钟数** <br/> |否  <br/> |点对点视频会话持续的平均时间。计算方法是视频会话总时间除以视频会话总数。  <br/> |
|**点对点文件传输会话总数** <br/> |否  <br/> |包含文件传输的点对点会话总数。  <br/> |
|**点对点应用程序共享会话总数** <br/> |否  <br/> |包含应用程序共享的点对点会话总数。  <br/> |
