<span data-ttu-id="e73e5-591">服务质量 (QoS) 是一种网络技术，在某些组织中用于帮助为音频和视频通信提供最佳最终用户体验。</span><span class="sxs-lookup"><span data-stu-id="e73e5-591">Quality of Service (QoS) is a networking technology that is used in some organizations to help provide an optimal end-user experience for audio and video communications.</span></span> <span data-ttu-id="e73e5-592">QoS 最常用于带宽有限的网络：由于大量网络数据包与相当少的可用带宽相竞争，因此，利用 QoS，管理员可以为包含音频或视频数据的数据包分配更高的优先级。</span><span class="sxs-lookup"><span data-stu-id="e73e5-592">QoS is most frequently used on networks where bandwidth is limited: with a large number of network packets competing for a fairly small amount of available bandwidth, QoS enables administrators to assign higher priorities to packets carrying audio or video data.</span></span> <span data-ttu-id="e73e5-593">通过为这些数据包提供更高的优先级，与涉及文件传输、Web 浏览或数据库备份等内容的网络会话相比，音频和视频通信完成速度更快且中断更少。</span><span class="sxs-lookup"><span data-stu-id="e73e5-593">By giving these packets a higher priority, audio and video communications are likely to complete faster, and with less interruption, than network sessions involving things such as file transfers, web browsing, or database backups.</span></span> <span data-ttu-id="e73e5-594">这是因为用于文件传输或数据库备份的网络数据包分配有"最大努力"优先级。</span><span class="sxs-lookup"><span data-stu-id="e73e5-594">That's because network packets used for file transfers or database backups are assigned a "best effort" priority.</span></span>
