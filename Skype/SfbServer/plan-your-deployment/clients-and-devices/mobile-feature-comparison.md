# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>Skype 业务的移动客户端功能比较
 
**摘要：**同时规划为 Skype 业务服务器 2015年查看支持移动客户端的功能。
  
这篇文章比较的特性和功能之间业务移动客户端的 Skype 和 Skype 的业务按以下类别的桌面客户端：
  
- 登录、推送通知和常规功能
    
- 增强状态
    
- 联系人和联系人组
    
- 即时消息 (IM)
    
- Skype 为 Skype 业务的音频和视频的业务的
    
- 会议
    
- 电话
    
- 外部用户
    
- 存档与合规性
    
-  新式验证
    
下表列出了可用于 Skype 适用于业务用户的 Skype 业务服务器 2015年为内部部署中的功能。 相同的功能还有 Skype 的在线业务和 Microsoft Office 365 的用户，除非另有说明表脚注中。
  
> [!NOTE]
> 联机帮助和最终用户的资源，请参阅[为业务发现 Skype](https://go.microsoft.com/fwlink/p/?LinkId=528686)。 
  
> [!NOTE]
> 要比较的业务客户端中其他 Skype 提供的功能，请参阅[Skype 业务为桌面客户端功能比较](desktop-feature-comparison.md)。
