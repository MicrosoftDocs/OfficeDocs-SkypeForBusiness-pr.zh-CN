# <a name="microsoft-teams-room-app-version-support"></a>Microsoft 团队会议室应用程序版本支持
 
Microsoft 计划的 Microsoft 团队聊天室每年支持从正式发布 (GA) 发行日期十二 12 个月每次更新几次发布更新。 将为整个十二 12 个月提供技术支持。 但是，我们支持结构现动态，到两个不同的服务阶段依赖于最新版本的可用性的发展。

**服务和关键更新服务阶段**\-运行 Microsoft 团队会议室的最新版本，您会收到包含安全和服务更新的定期更新。


**安全更新 （仅限） 服务阶段**\-发布新版本之后，支持的旧分支将降低安全更新仅用于十二 （12） 个月的服务更新的其余部分支持生命周期。


> [!NOTE]
> 最新版本始终是在服务和服务阶段的关键更新。 这意味着，在遇到代码缺陷，这样可保证关键更新，您必须已经安装即可接收修复的最新版本。 所有其他受支持的版本将只能可以接收安全更新。

所有支持结束十二 （12） 个月生命周期版本已过期后，或从那时起发布两个以上的更新。 此时，客户必须更新到支持的版本。

所有版本的[Microsoft 团队聊天室发行说明](srs2-release-note.md)中都列出。 
 
# <a name="os-version-support"></a>操作系统版本支持
运行 Microsoft 团队聊天室的设备的 Windows 10 功能更新未提供的 6 个月从 Windows 时进行更新的版本的时间。 通过将一个适用于 Microsoft 团队聊天室设备特殊块放在 Windows Update 上用于业务通道 （即半年通道），通过应用程序设置达到此目的。 在 Microsoft 执行此阻止期间各种测试包括在公司内部和通过设备 OEM 合作伙伴以确保该新版本的 Windows 10 功能正常使用的 Microsoft 团队聊天室应用程序和外围设备连接到它的协调。 这是重要同时确保设备安全，一致的用户体验，并确保体验质量提供通过 Microsoft 团队聊天室应用程序。   

从块的时间提升 (即提供功能更新这些设备上下载的窗口 10)、 Microsoft 团队聊天室支持特定的 Windows 10 功能发行版的 12 个月嵌入到应用程序支持策略。 由于 Windows 10 功能更新时提供有关每 6 个月，这也意味着 Microsoft 团队将具有两个其他版本，通过来测试时支持的结束当前版本。 这也意味着，Windows 10 版本都将解除对所有 Microsoft 团队聊天室客户每 6 个月。 随着我们持续进行应用程序更改和开发针对最后一个未被阻止的 Windows 版本，在某些情况下需要承担的 Windows 和它很难 Microsoft 进行更改事实较新版本中可用的 Windows 功能依赖项12-18 月旧版本。 若要确保您获得应用程序修复问题您遇到 Microsoft 团队聊天室设备上，我们敦促所有客户提供支持的 windows 版本指南内保持最新 Windows 10 功能更新到升级这些设备。

因此，Microsoft 团队聊天室需要 Windows 10 启动 2019 年 5 的支持作为最低的版本 1709年版本。 没有新的应用程序版本提供给系统上 Windows 10 版本 1703年或下方。
