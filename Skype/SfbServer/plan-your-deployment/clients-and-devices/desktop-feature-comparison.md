# <a name="desktop-client-feature-comparison-for-skype-for-business"></a>Skype 业务为桌面客户端功能比较
 
**摘要：**Skype 的业务服务器 2015年或 Skype 的在线业务管理员可以使用这些表来了解哪些客户端上支持哪些功能。
  
 部署或升级到 Skype 的业务之前，应检查哪些客户端都已在您的组织中使用。 使用下表来了解对这些客户端的功能支持影响。 这有助于你与用户交流更改、确定执行推出过程的进度以及完全了解升级到最新客户端的好处。
  
具有的业务服务器 2015 Skype 可用的一些功能在中不可用 Skype 的在线业务，有关的详细信息，请参阅[联机或混合用户帐户限制](desktop-feature-comparison.md#Online-Hybrid)。 Skype 业务联机管理员可能想要对它们的这些不同计划的信息，请参阅[Skype 业务联机服务说明](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx)。
  
下表显示了可用于每个客户端可以使用 Skype 与业务服务器 2015年或 Skype 的在线业务的功能。 您可能还想要的智能手机和 tablet 客户端功能比较，请参阅[Skype 业务的移动客户端功能比较](mobile-feature-comparison.md)。 你的组织购买的客户端访问许可证或用户订阅许可证也将影响用户可以使用的功能。 向用户部署完整版还是基本版客户端取决于贵组织选择购买的许可证或计划。 请参阅[授权指南](https://products.office.com/en-us/skype-for-business/it-pros)有关更多详细信息。
  
> [!IMPORTANT]
> Skype 的业务服务器 2015年和 Skype 的在线业务支持以前发布的以下客户端： Lync 2013、 Lync 2010、 Lync 2010 移动，Lync 电话版和 Lync 2010 助理。 有关这些客户端与其他服务器一起使用时的信息，请参阅[客户端的 Lync Server 2013 的比较表](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.15%29.aspx)和[Lync Server 2010 中的客户端比较表](https://technet.microsoft.com/en-us/library/gg425836%28v=ocs.14%29.aspx)。 
  
> [!NOTE]
> **Lync 2010 助理**客户端不支持 Skype 的在线业务。
  
> [!NOTE]
> Skype 业务 Web 应用程序浏览器客户端和 Skype 会议应用程序窗口 10 应用程序仅提供[的会议支持](desktop-feature-comparison.md#BKMK_Conferencing)。 有关这些客户端的详细信息请参阅[计划会议客户端 （Web 应用程序和会议的应用程序）](meetings-clients.md) 。

## <a name="client-limitations"></a>客户端限制 
<a name="Types"> </a>

### <a name="basic-client-limitations"></a>基本客户端限制
<a name="Full-Basic"> </a>

以下功能在使用完整客户端时可用，在使用基本客户端时不可用： 
  
- 管理团队呼叫设置
    
- 管理代理人
    
- 代表其他联系人发起呼叫（经理/代理人方案）
    
- 如果配置为代理人，则处理其他人的呼叫
    
- 管理大量呼叫
    
- 发起针对响应组的呼叫
    
- 呼叫寄存
    
- 更改问候
    
- 组内呼叫应答
    
### <a name="online-or-hybrid-user-account-limitations"></a>联机或混合用户帐户限制
<a name="Online-Hybrid"> </a>

用户帐户可以联机形式存在或在本地存在，这将影响用户可以使用的功能。 在 Skype 的在线业务上拥有帐户的用户不会访问以下功能，即使有了完整的客户端： 
  
- 增强状态：对“我的图片”使用任何公共网站中的照片
    
- 联系人：搜索响应组
    
- IM 支持：持久聊天（群聊）集成
    
- IM 的支持： 提升到 Skype 持久聊天房间商务会议通过一次单击
    
- 外部用户：进行与外部用户的双方或多方呼叫
