# <a name="system-requirements-for-skype-for-business-for-windows-phone"></a>Windows Phone 版 Skype for Business 的系统要求
 
**摘要：**同时规划为 Skype 业务服务器 2015年查看客户端系统设备要求。
  
从智能手机或 Windows 专业移动设备连接的用户在您的组织中业务为 Windows Phone 的 Microsoft Skype 提供即时消息 (IM)、 增强的状态显示和电话服务。 移动设备，用户可以将 Skype 业务的范围扩大。 本主题介绍规划注意事项的 Skype 业务为 Windows Phone 包含标识系统必备组件和技术要求、 所需的组件和部署指南。
  
## <a name="skype-for-business-for-windows-phone-prerequisites"></a>Windows Phone 版 Skype for Business 先决条件

以下是 Skype 业务为 Windows Phone 系统必备组件。
  
- Windows Phone 8.1 或更高版本。
    
- Windows Phone 设备必须已安装 Microsoft 发布的最新更新。 有关详细信息，请参见 Windows Phone 8.1 在[Windows Phone 8 更新历史记录](https://go.microsoft.com/fwlink/p/?LinkID=281961)。
    
- 该设备必须有 22 MB 的可用磁盘空间。
    
- 用户必须从运营商处开通了语音和数据资费套餐。
