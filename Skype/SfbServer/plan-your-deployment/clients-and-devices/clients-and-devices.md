# <a name="plan-for-clients-and-devices"></a>规划客户端和设备

**摘要：** 查看支持的 Skype for Business 客户端和应用。

如今的员工在不断迁移。 员工需要沟通和协作，无论是从公司办公室工作、在地区办公室工作、在家庭办公室工作，还是路过路。 Skype for Business Server 通过一组可以部署到组织用户的客户端接口支持这些需求。 周密的规划可确保员工获得所需的信息，并确保无论他们身在何处，Skype for Business 都可供他们使用。

## <a name="available-clients"></a>可用客户端

Skype for Business Server 支持多种类型的客户端，包括计算机安装的客户端软件、基于 Web 的客户端和移动设备的客户端。 本部分中引入了主要客户端，有关所有受支持的客户端的详细列表，请参阅 Skype for [Business Server 2015](desktop-feature-comparison.md) 的桌面客户端功能比较或 [Skype for Business Server 2019](../../../SfBServer2019/plan/feature-comparison.md)的桌面客户端功能比较。 如果你之前使用过 Lync 客户端的组合，请注意，存在与 Skype for [](clients-and-devices.md#Legacy) Business Server 2019 不兼容的不受支持的旧客户端。 更新会定期进行，因此请定期查看本主题，了解最新的客户端信息。

### <a name="skype-for-business-2019"></a>Skype for Business (2019) 

Skype for Business (2019) 是 Skype for Business Server 2015 和 2019 的推荐功能齐全的客户端。 有关 [新功能的说明，请参阅"关注 Skype for Business](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 中的最新更新"。 Skype [for Business](desktop-feature-comparison.md)的桌面客户端功能比较中详细说明了客户端功能支持，用户文档位于 Skype for [Business 帮助中](https://support.office.com/Skype-for-business)。 此客户端包含在用户安装 Microsoft 365 或 Office 365 时。

也提供支持较少功能的免费基本客户端。 这两个版本均可在所有设备的 [下载 Skype for Business 上下载](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 "基本客户端限制"部分介绍了完整客户端和 [基本客户端之间的差异](desktop-feature-comparison.md#Full-Basic) 。

### <a name="skype-for-business-2016"></a>Skype for Business 2016

Skype for Business 2016 是 Skype for Business Server 2015 或 2019 功能齐全的客户端。 有关 [新功能的说明，请参阅 Skype for Business 2016](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781) 中的新增功能。 Skype [for Business](desktop-feature-comparison.md)的桌面客户端功能比较中详细说明了客户端功能支持，用户文档位于 Skype for [Business 帮助中](https://support.office.com/Skype-for-business)。 此客户端包含在用户安装 Office 365 时。

也提供支持较少功能的免费基本客户端。 这两个版本均可在所有设备的 [下载 Skype for Business 上下载](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 "基本客户端限制"部分介绍了完整客户端和 [基本客户端之间的差异](desktop-feature-comparison.md#Full-Basic) 。

### <a name="skype-for-business-2015"></a>Skype for Business 2015

Skype for Business 2015 是 Skype for Business Server 2015 或 2019 功能齐全的客户端。 Skype for Business 用户界面已完全重新设计，包括新集成的功能，如呼叫监视器、Skype 目录集成、表情符号等。 有关更改的摘要，请参阅 [Lync 现在是 Skype for Business - 查看新增功能](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)。 Skype [for Business](desktop-feature-comparison.md)的桌面客户端功能比较中详细说明了客户端功能支持，用户文档位于 Skype for [Business 帮助中](https://support.office.com/Skype-for-business)。 此客户端包含在用户安装 Office 365 时。

### <a name="skype-for-business-on-mac"></a>Mac 版 Skype for Business

Mac [上的 Skype for Business](https://www.microsoft.com/download/details.aspx?id=54108) 客户端可供下载。 请参阅 [Mac 上的 Skype for Business 客户端要求](mac-requirements.md) 以查看先决条件。

### <a name="skype-for-business-for-mobile-devices"></a>适用于移动设备的 Skype for Business

客户端适用于 Windows Phone、iPhone/iPad 和 Android。 用户可以在所有设备上通过 [下载 Skype for Business 获取它们](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 Skype for Business 的移动客户端功能比较中详细介绍 [了这些客户端的功能支持](mobile-feature-comparison.md)。

> [!NOTE]
> SKYPE (Server 2019 中不再提供对) 移动客户端的 MCX 移动服务支持。 所有当前的 Skype for Business 移动客户端已使用统一通信 Web API (UCWA) 来支持即时消息 (IM) 、状态和联系人。 使用 MCX 的旧客户端的用户将需要升级到当前客户端。

### <a name="online-meeting-add-in-for-skype-for-business"></a>适用于 Skype for Business 的联机会议外接程序

适用于 Skype for Business 的联机会议外接程序支持从 Windows 上的 Microsoft Outlook 消息和协作客户端内管理会议。 Skype for Business 联机会议外接程序软件随 Skype for Business 一起自动安装。

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>Skype for Business Web App 和 Skype 会议应用

如果未在用户计算机上安装 Skype for Business，并且用户单击 Windows 计算机上会议请求中的会议链接，将安装并打开 Skype 会议应用或 Skype for Business Web App。  Skype 会议应用是组织外部的参与者选择的客户端。  (请注意，在 Mac 上，将安装并打开 Mac 上的 Skype for Business。) 请参阅规划会议客户端 [ (Web App 和会议应用) ](meetings-clients.md) 了解使用这些客户端的要求。


### <a name="skype-for-business-web-scheduler"></a>Skype for Business Web 计划程序

[Skype for Business Web 计划](https://sched.lync.com) 程序是一种基于 Web 的会议计划和管理工具，适用于无法访问 Microsoft Outlook 或操作系统（不基于 Windows）的 Skype for Business Online 用户。 使用 Skype for Business Web 计划程序，用户可以创建新的会议、修改现有会议以及使用其首选的电子邮件计划发送邀请。 Skype for Business Web 计划 [程序文档](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)  提供了更多详细信息。

### <a name="vdi-plugins"></a>VDI 插件

VDI (虚拟桌面基础结构) 安全与合规性问题特别敏感的某些组织使用。 在类似连接上将 Skype for Business 与完整的音频和视频一同使用需要在虚拟桌面上的客户端上执行大量音频和视频处理。 其他 VDI 插件软件可用于将处理卸载到最终用户的本地计算机，并减少虚拟桌面上的负载。 有关 [使用这些插件的详细信息](vdi-environments.md) ，请参阅 VDI 环境中规划 Skype for Business。

### <a name="microsoft-teams-rooms"></a>Microsoft Teams 会议室

Microsoft Teams 会议室是 Microsoft 最新的会议解决方案，它使用熟悉的界面，并且易于部署和管理，利用像 MICROSOFT 面板这样的现有设备以便于安装。 Microsoft Teams 会议室使用在控制台模式下在 Surface Pro 4 或 Surface Pro 上运行的有用途的 UWP 应用 (在部署 UWP 应用后，该应用是唯一将在设备) 上运行的应用，并且它需要自己的设备帐户才能实现。 软件通过 Windows 应用商店和 Windows 更新进行更新。 有关 https://aka.ms/MTRDocs 在部署中使用这些会议室控制台的详细信息，请参阅。 

### <a name="skype-for-business-on-surface-hub"></a>Surface Hub 上的 Skype for Business

Microsoft Surface Hub 是一个一体式生产力设备，旨在进行集体讨论、协作和演示。 它有自己的 Skype for Business 客户端迭代，记录在 [Microsoft Surface Hub 管理指南中](https://docs.microsoft.com/surface-hub/)。

## <a name="choosing-your-organizations-preferred-client"></a>选择组织的首选客户端
<a name="BK_client_choose"> </a>

如果组织购买了相应的许可证，请选择"完整客户端"，否则选择"基本"客户端。

你的用户可以在所有设备上通过下载 Skype [for Business 自行安装客户端](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 当用户在 Windows 上安装 Microsoft 365 或 Office 365 时，也会安装客户端。 如果你的一些用户拥有 Mac，则这些用户将具有一组不同的功能，如前面部分所述。

Skype for Business Server 2015 中提供某些功能在 Skype for Business Online 或 Skype for Business Server 2019 中不可用，请参阅 [2015 或 2015](desktop-feature-comparison.md#Online-Hybrid) 的联机或混合用户帐户限制，有关具体内容，请参阅 [2019](desktop-feature-comparison.md#Online-Hybrid) 的 Online 或混合用户帐户限制。 Skype for Business Online 管理员可能想要参考 [Skype for Business Online 服务说明](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx) ，了解他们可用的不同计划。

 在部署或升级到 Skype for Business 之前，请检查组织中已在使用哪些客户端。 使用 [Skype for Business 的桌面](desktop-feature-comparison.md) 客户端功能比较来了解功能支持对这些客户端的影响。 这可以帮助您将更改传达给用户、加快推出过程，并完全了解升级到最新客户端的好处。

## <a name="ways-to-deploy-the-client-to-your-users"></a>向用户部署客户端的方法
<a name="BK_User_Deploy"> </a>

客户端安装程序可用于 MSI 和即点即用类型安装程序。 Skype for Business 客户端维持策略可能会影响你的选择，因此你应该了解以下几点：

- 一般情况下，Skype for Business 不会向之前发布的客户端添加新功能

- 一般情况下，Skype for Business 不会计划在其初始发布后交付 Skype for Business MSI 中的新功能。 两次发布之间的 MSI 改进主要是质量/安全性方面的改进。

- Skype for Business 2019 即点即用安装程序中提供最新、最丰富的 Skype for Business 客户端体验。

你可以执行客户端的自定义部署，如 Skype for [Business Server 中的自定义 Windows 客户端安装中所述](../../deploy/deploy-clients/customize-windows-client-installation.md)。 Skype for Business Server 的部署客户端中详细介绍 [了安装方法](../../deploy/deploy-clients/deploy-clients.md)

### <a name="click-to-run"></a>Click-to-run

即点即用是一种 Microsoft 流式传输和虚拟化技术，可用于安装和更新 Office 产品（包括 Skype for Business）。 这些流式传输和虚拟化功能基于 Microsoft Application Virtualization (App-V) 。 单击即可运行具有以下优点：

- Office 套件的流式安装，导致安装时间短

- 整合更新和修补程序

- 占用更少的磁盘空间

- 基于用户的许可：每个用户 5 次安装

- 通过 XML 编辑器进行自定义，用于安装独立程序

您可能需要将 [Office 部署工具用于](https://www.microsoft.com/download/details.aspx?id=49117) 这种类型的安装程序。

基本和完整的客户端版本 (（选择 32 位和 64 位版本) ）均使用即点即用安装程序提供，用户可以在所有设备上下载[Skype for Business。](https://products.office.com/skype-for-business/download-app?tab=tabs-3)

### <a name="msi"></a>MSI

MSI 是一种较传统的安装方法，用于 Skype for Business 2015 和 2016 客户端。 它允许你手动安装更新和修补程序、使用批量许可和激活，并且可通过 [Office 自定义工具进行自定义](https://www.microsoft.com/download/details.aspx?id=49030)。 可以通过应用组策略、使用 Microsoft Endpoint Configuration Manager 或第三方工具分发客户端。
