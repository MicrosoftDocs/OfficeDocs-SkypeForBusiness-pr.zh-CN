# <a name="requirements-for-your-skype-for-business-2015-environment"></a>Skype for Business 2015 环境的要求
 
**摘要：** 有关 Skype for Business Server 2015 的系统要求和网络或基础结构要求的主题概述。 如果你需要安装或配置任何内容作为安装和部署 Skype for Business Server 2015 的先决条件，它将在此处。
  
这是有关 Skype for Business Server 2015 部署要求的文章的登陆页面。 这包括服务器要求和要准备的更多常规内容，包括域和网络结构以及证书。

环境要求部分中的两篇文章概述了在组织服务器环境中成功部署或升级到 Skype for Business Server 2015 之前需要拥有和/或配置的内容。
  
### <a name="server-requirements"></a>服务器要求

[Skype for Business Server 2015](server-requirements.md) 的服务器要求是你需要在打算用于部署的个人服务器上做好准备。
  
### <a name="environmental-requirements"></a>环境要求

[Skype for Business Server 2015](environmental-requirements.md) 的环境要求是部署 Skype for Business Server 2015 之前需要在基础结构中准备的更广泛的内容。
