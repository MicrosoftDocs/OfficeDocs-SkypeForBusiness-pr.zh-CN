# <a name="capacity-planning-for-persistent-chat-server-in-skype-for-business-server-2015"></a>Skype for Business Server 2015 中持久聊天服务器的容量规划
 
**摘要：**阅读本主题以了解有关规划在 Skype 的持久聊天服务器的业务服务器 2015年的容量。
  
持久的聊天服务器可以执行可持久保存的多用户、 实时聊天以后检索和搜索。 与不同的组和即时消息 (IM)，将保存在用户的邮箱如果配置对话历史记录、 持久聊天服务器会话保持打开状态，并且该内容保存在服务器上，以及邮件、 文件、 Url 和其他数据的一部分正在进行的对话。
  
容量规划是准备持久聊天服务器部署的重要组成部分。 本主题提供了可用于确定您的部署的最佳配置的容量计划表。 它还介绍了如何最好地管理持久聊天服务器的部署，在高峰时间需要更大的容量。
  
在阅读本节之前，应熟悉持久聊天拓扑。 有关详细信息，请参阅[计划持续聊天服务器拓扑结构](topology.md)。
  
## <a name="persistent-chat-server-capacity-planning"></a>持久的聊天服务器容量规划

下表可帮助您通过建模各种持久的聊天服务器设置如何影响容量规划持久聊天服务器的容量。
  
- 用户数量容量规划
    
- 聊天室访问容量规划
    
- 规划容量为聊天室的访问，通过邀请
    
- 性能容量规划
