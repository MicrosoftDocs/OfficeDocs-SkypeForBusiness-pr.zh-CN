# <a name="plan-for-pstn-connectivity-in-skype-for-business-server"></a>在 Skype for Business Server 中规划 PSTN 连接
 
在 Skype for Business Server 企业语音 PSTN 连接。
  
企业级 VoIP 解决方案必须以始终如一的服务质量 (QoS) 提供来往于公用电话交换网 (PSTN) 的呼叫。 拨打和接听电话的用户不应了解基础技术：从用户的角度来看，企业语音 基础结构和 PSTN 之间的呼叫应该就像是另一个电话呼叫。
  
Skype for Business Server 通过以下选项提供可靠、可扩展的 PSTN 连接：
  
- 到 Internet 电话服务提供商 (ITSP) 的 **SIP 中继**
    
- 到 PSTN 网关的 **直接 SIP 连接**
    
- 到 PBX 的 **直接 SIP 连接**
    
根据企业大小、地理范围和现有语音基础结构，企业可能会在不同位置使用一个、两个甚至所有三个选项。 有关这些选项的详细信息，请参阅以下部分。
