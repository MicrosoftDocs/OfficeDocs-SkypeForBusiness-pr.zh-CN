# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>为 Skype for Business Server 选择 E9-1-1 服务提供商
 
在 Skype for Business Server 企业语音 中为 E9-1-1 部署选择服务提供商。
  
E9-1-1 服务提供商根据呼叫中包含的位置信息，将来自 Skype for Business Server 的紧急呼叫路由到正确的公共安全应答点 (PSAP) 。 
  
要支持 E9-1-1 作为 Skype for Business Server 部署的一部分，你必须从 Lync 开放互操作性计划合格的 E9-1-1 服务提供商获取 E9-1-1 路由服务。 选择最能满足组织要求的提供商。

To see the list of E9-1-1 routing services for Lync， see [E-911 Services qualified for Lync](../../../SfbPartnerCertification/lync-cert/e-911-service-providers.md).
