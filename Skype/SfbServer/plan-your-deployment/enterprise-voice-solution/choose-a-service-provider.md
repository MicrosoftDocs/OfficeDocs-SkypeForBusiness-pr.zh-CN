# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>选择 Skype for Business Server 的 E9-1-1 服务提供商
 
选择在 Skype E9-1-1 部署的服务提供商业务 Server 企业语音。
  
E9-1-1 服务提供程序紧急呼叫路由源自 Skype 业务服务器到正确公共安全应答点 (PSAP) 基于呼叫中包含的位置信息。 
  
支持 E9-1-1 Skype Business Server 部署的一部分，您必须获得 E9-1-1 从 Lync 打开互操作性程序路由服务限定 E9-1-1 服务提供商。 选择最能满足组织要求的提供商。

若要查看 lync E9-1-1 路由服务列表，请参阅[E-911 服务符合 Lync 要求](https://docs.microsoft.com/en-us/skypeforbusiness/lync-cert/e-911-service-providers)。
