# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>为 Skype for Business Server 选择 E9-1-1 服务提供商
 
在 Skype for Business Server 中为 E9-1-1 部署选择企业语音。
  
E9-1-1 服务提供商根据呼叫中包含的位置信息将来自 Skype for Business Server 的紧急呼叫路由到正确的公共安全应答点 (PSAP) 。 
  
若要在 Skype for Business Server 部署中支持 E9-1-1，必须从 Lync 开放互操作性计划合格的 E9-1-1 服务提供商获取 E9-1-1 路由服务。 选择最能满足组织要求的提供商。

若要查看 Lync 的 E9-1-1 路由服务列表，请参阅限定为 Lync 的 [E-911 服务](https://docs.microsoft.com/skypeforbusiness/lync-cert/e-911-service-providers)。
