# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server-2015"></a>选择 Skype for Business Server 2015 的 E9-1-1 服务提供商
 
选择业务服务器企业语音为 Skype E9-1-1 部署的服务供应商。
  
在 E9-1-1 服务提供程序路由紧急调用源自 Skype 业务服务器到正确公共安全应答点 (PSAP) 根据包含在调用中的位置信息。 
  
为支持 E9-1-1 为 Skype 业务服务器部署的一部分，您必须获取 E9-1-1 Lync 开放互操作性程序中的路由服务限定 E9-1-1 的服务提供商。 选择最能满足组织要求的提供商。
