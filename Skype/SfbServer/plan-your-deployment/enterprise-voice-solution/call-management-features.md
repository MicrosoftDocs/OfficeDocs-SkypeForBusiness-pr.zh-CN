# <a name="plan-for-call-management-features-in-skype-for-business-2015"></a>在 Skype for Business 2015 中规划呼叫管理功能
 
Overview of Business Server 企业语音的 Skype 中的受支持的呼叫管理功能。
  
企业语音呼叫管理功能控制传入呼叫的路由和应答。 Skype 业务服务器提供以下呼叫管理功能： 
  
- **呼叫寄存**： 允许语音用户暂时寄存呼叫，然后将其从相同或另一拿起电话。
    
- **组分拣**： 允许语音用户拿起响铃的呼叫分拣组分配给其他语音用户的呼叫。
    
- **响应组**： 传入呼叫路由至使用的代理组的智能寻线或互动语音响应 (IVR) 问题和答案。 
    
- **通知**：向对未分配号码发出的呼叫播放消息，或将其路由至其他位置，或同时执行两者。
    
如果计划部署企业语音，则可以选择实现任一或所有这些呼叫管理功能。
