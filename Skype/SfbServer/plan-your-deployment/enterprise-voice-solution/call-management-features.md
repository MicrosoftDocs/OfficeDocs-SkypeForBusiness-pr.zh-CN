# <a name="plan-for-call-management-features-in-skype-for-business"></a>在 Skype for Business 中规划呼叫管理功能

Skype for Business Server 服务中支持的呼叫管理功能企业语音。

企业语音呼叫管理功能控制如何路由和应答传入呼叫。 Skype for Business Server 提供以下呼叫管理功能：

- **呼叫停止**：使语音用户能够暂时将呼叫暂停，然后从同一电话或其他电话接听。

- **组分** 拣：使语音用户能够接听为分配给呼叫接听组的其他语音用户响铃的呼叫。

- **响应组**：使用搜索组或互动语音响应将传入呼叫路由到代理组， (IVR) 和答案。

- **通知**：向对未分配号码发出的呼叫播放消息，或将其路由至其他位置，或同时执行两者。

如果计划部署企业语音，则可以选择实现任一或所有这些呼叫管理功能。
