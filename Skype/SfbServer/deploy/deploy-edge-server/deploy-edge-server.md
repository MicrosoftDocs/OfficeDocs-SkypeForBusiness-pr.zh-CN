# <a name="deploy-edge-server-in-skype-for-business-server"></a>部署边缘服务器在 Skype 业务服务器
 
**摘要：** 了解如何为您 Skype 业务服务器环境中部署边缘服务器或边缘池。
  
为什么到您 Skype 业务服务器环境中部署边缘服务器或边缘池？ 如果需要不登录到组织内部网络的外部用户能够与内部用户交互，那就有必要这样做。 这些外部用户可能是经过身份验证的远程用户和匿名远程用户、联盟伙伴或其他移动客户端。
