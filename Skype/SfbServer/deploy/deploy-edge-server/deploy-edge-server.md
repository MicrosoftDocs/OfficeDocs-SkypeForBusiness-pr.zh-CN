# <a name="deploy-edge-server-in-skype-for-business-server"></a>在 Skype for Business Server 中部署边缘服务器
 
**摘要：** 了解如何将边缘服务器或边缘池部署到 Skype for Business Server 环境中。
  
为什么要将边缘服务器或边缘池部署到 Skype for Business Server 环境？ 如果需要未登录到组织内部网络的外部用户能够与内部用户进行交互，则有必要。 这些外部用户可能是经过身份验证的匿名远程用户、联盟伙伴或其他移动客户端。
  
## <a name="deployment-checklist-for-the-edge-for-skype-for-business-server"></a>适用于 Skype for Business Server 的边缘部署清单

如上所述，Skype for Business Server 的边缘服务器部署有许多方面。 此清单概述了需要执行的任务，以及指向更详细步骤的链接。
  
我们希望你已开始在 [Skype for Business Server 中规划边缘服务器部署](../../plan-your-deployment/edge-server-deployments/edge-server-deployments.md) 部分。 如果没有，我们将在其中详细介绍我们引用的许多内容。 部署部分仅包含过程，因此，如果您想了解这些步骤背后的原因，可以开始规划。
  
本文档还基于你已完成 Skype for Business Server 的基本部署。 你可能正在与边缘并行执行该部署，但你需要先执行这些步骤，然后你将能够对此处记录的边缘进行拓扑更改。
