# <a name="deploy-edge-server-in-skype-for-business-server"></a>部署边缘服务器Skype for Business Server
 
**摘要：** 了解如何将边缘服务器或边缘池部署到Skype for Business Server环境中。
  
为什么要将边缘服务器或边缘池部署到Skype for Business Server环境中？ 如果需要未登录到组织内部网络的外部用户能够与内部用户进行交互，则有必要。 这些外部用户可能是经过身份验证的匿名远程用户、联盟伙伴或其他移动客户端。
