# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>在 Skype for Business Server 中部署 VIS 服务器角色
 
**摘要：** 在 Skype for Business Server (VIS) 角色部署视频互操作服务器。
  
若要在拓扑生成器中刚刚创建的服务器上设置 VIS 服务，请启动 Skype for Business Server 部署向导，按"安装"或"更新 **Skype for Business Server 系统** "并按照向导中的以下步骤操作：
  
1.  选择 **"安装本地配置存储"。**
    
2. 选择 **"安装"或删除"Skype for Business Server 组件"。**
    
3. 选择 **"请求、安装或分配证书"。**
    
4. 选择 **"启动服务"。**
    
此服务的软件现已安装并运行。 你可以打开服务 mmc 工具，查看 **Skype for Business Server 视频互操作服务器** 服务是否与其他 Skype for Business Server 服务一起运行。 接下来，必须配置 VIS 服务器或池。
