# <a name="deploy-the-vis-server-role-in-skype-for-business-server-2015"></a>在 Skype for Business Server 2015 中部署 VIS 服务器角色
 
**摘要：**为业务服务器 2015年部署在 Skype 的视频互操作服务器 (VIS) 角色。
  
以设置 VIS 服务只是拓扑生成器中创建的服务器上，启动业务服务器部署向导 Skype、 按**安装或更新 Skype 业务服务器系统**请按照下列步骤在向导中：
  
1.  选择**安装本地配置存储**。
    
2. 选择**安装或删除 Skype 业务服务器组件**。
    
3. 选择**请求、安装或分配证书**。
    
4. 选择**启动服务**。
    
现已安装和运行此服务的软件。 您可以打开服务 mmc 工具**Skype 业务服务器视频互操作服务器**服务是否正在运行以及其他 Skype 业务服务器服务。 接下来，你必须配置 VIS 服务器或池。
