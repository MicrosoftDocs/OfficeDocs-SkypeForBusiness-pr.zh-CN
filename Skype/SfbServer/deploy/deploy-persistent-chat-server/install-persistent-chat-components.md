# <a name="install-persistent-chat-components-in-skype-for-business-server-2015"></a>在 Skype for Business Server 2015 中安装持久聊天组件
 
**摘要：**阅读本主题，以了解如何使用 Skype 业务服务器部署向导来安装 Skype 业务服务器 2015年组件和服务。
  
安装永久聊天组件之前，请确保您已经安装的系统必备组件的硬件和软件，并创建适当的拓扑以支持持久聊天服务器。 有关计划和要求的详细信息，请参阅[规划业务服务器 2015年的 Skype 的持久聊天服务器](../../plan-your-deployment/persistent-chat-server/persistent-chat-server.md)和[您的业务环境的 Skype 要求](../../plan-your-deployment/requirements-for-your-environment/requirements-for-your-environment.md)。 有关如何更新并发布拓扑包括持久聊天服务器的信息，请参阅[添加持久聊天服务器到您 Skype 业务服务器 2015年拓扑](add-persistent-chat-server.md)。
  
## <a name="install-and-start-services"></a>安装和启动服务

使用 Skype 业务服务器部署向导，请选择安装或更新 Skype 业务服务器系统，请执行以下步骤： 
  
1. 安装本地配置存储
    
2. 安装或删除 Skype for Business Server 组件
    
3. 请求、安装或分配证书
    
4. 启动服务
    
如何使用部署向导来安装组件的详细信息，分配证书，并启动服务，请参阅[有关业务服务器 2015年安装 Skype](../../deploy/install/install.md)和[用于在拓扑中的服务器上的业务服务器 2015年安装 Skype](../../deploy/install/install-skype-for-business-server.md)。
