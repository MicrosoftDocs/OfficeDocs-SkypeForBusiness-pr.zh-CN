# <a name="deploy-call-management-features-in-skype-for-business-2015"></a>在 Skype for Business 2015 中部署呼叫管理功能
 
为业务服务器企业语音部署在 Skype 呼叫管理功能。
  
企业语音调用管理功能控制方式的传入呼叫的路由和解答。 Skype 业务服务器提供以下调用管理功能： 
  
- **调用公园**： 使语音用户可以暂时寄存呼叫，然后拿从同一电话线或另一个电话。
    
- **组调用拾取**： 使用户能够回答对另一个用户的拨入呼叫分拣组号码分配给分拣组进行调用。
    
- **响应组**： 组通过使用代理的路由传入呼叫查寻组或交互式语音响应 (IVR) 问题和回答。
    
- **通知**：向对未分配号码发出的呼叫播放消息，或将其路由至其他位置，或同时执行两者。
    
本部分介绍如何在企业语音部署期间配置这些调用的管理功能。
