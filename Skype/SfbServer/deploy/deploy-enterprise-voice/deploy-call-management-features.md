# <a name="deploy-call-management-features-in-skype-for-business"></a>部署 Skype for Business 中的呼叫管理功能
 
为业务 Server 企业语音部署 Skype 中的呼叫管理功能。
  
企业语音呼叫管理功能控制传入呼叫的路由和应答。 Skype 业务服务器提供以下呼叫管理功能： 
  
- **呼叫寄存**： 允许语音用户暂时寄存呼叫，然后将其从同一电话或另一个电话拿起。
    
- **组呼叫分拣**： 允许用户应答其他用户通过拨打的呼叫分拣组号分配给分拣组发出呼叫。
    
- **响应组**： 传入呼叫路由至使用的代理组的智能寻线或互动语音响应 (IVR) 问题和答案。
    
- **通知**：向对未分配号码发出的呼叫播放消息，或将其路由至其他位置，或同时执行两者。
    
本节介绍如何在企业语音部署期间配置这些呼叫管理功能。
