# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>（可选）Skype for Business 中的验证响应组部署
 
验证您的响应组部署成功，业务 Server 企业语音的 Skype 中。
  
配置响应组后，您需要验证配置以确保您的响应组按预期方式工作。 至少应使用以下用户类型验证以下方案：
  
 **用户**
  
- 用户驻留在 for Business 的 Skype 上
    
- 使用公用电话交换网 (PSTN) 的外部用户
    
- 驻留在 for Business 的 Skype 上代理
    
  **Scenarios**
  
- Skype 业务用户呼叫响应组。
    
- 外部用户呼叫响应组。
    
- 用户在代理处理其他呼叫时呼叫响应组，并进入队列。
