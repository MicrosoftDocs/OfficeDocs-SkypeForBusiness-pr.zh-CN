# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a> (Skype for Business) 验证通知部署时可选
 
验证 Skype for Business Server 应用程序中通知应用程序的企业语音。 这会影响如何处理对未分配号码的呼叫。
  
安装并配置通知后，需要验证该配置以确保可按预期呼叫未分配的号码。至少必须验证以下内容：
  
- 呼叫对您的组织有效但为未分配号码的号码。
    
- 呼叫未分配的号码并验证是否播放正确的通知。
