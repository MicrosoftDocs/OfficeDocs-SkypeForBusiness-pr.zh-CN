# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>（可选）验证通知部署中的业务的 Skype
 
验证通知中的应用程序 Skype 业务 Server 企业语音部署。 这将影响如何处理打给未分配号码的呼叫。
  
安装并配置通知后，需要验证该配置以确保可按预期呼叫未分配的号码。至少必须验证以下内容：
  
- 呼叫可供组织使用但未分配的号码。
    
- 呼叫未分配的号码并验证是否播放正确的通知。
