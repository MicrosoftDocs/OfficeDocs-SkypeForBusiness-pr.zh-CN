# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>为业务服务器在 Skype 安装 SQL Server Reporting Services 
 
**摘要：** 了解在何处查找有关 SQL Server Reporting Services for Business Server 使用的 Skype 的信息。
  
Skype 业务服务器可用于查看和监控报告的 SQL Server Reporting Services (SSRS)。 为了使用此功能，你需要安装和配置 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安装 SQL Server Reporting Services

如果您打算使用 Skype 的业务服务器监控报告 （请参阅本文档的详细信息的下一篇文章） 必须首先安装 SQL Server Reporting Services;可以在安装 Microsoft SQL Server 的同一时间或任何时间安装 SQL Server 之后安装 reporting Services。 如果你未安装 SQL Server，则请按照本文档之前提供的说明操作。 安装 SQL Server 时，确保在“功能选择”页上选择 Reporting Services。 这将安装 SQL Server Reporting Services。
  
若要了解如何安装 SQL Server Reporting Services，请参阅[SQL Server Reporting Services (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx)。
  
如果你已安装 SQL Server 但未安装 SQL Server Reporting Services，则可按照 SQL Server Reporting Services 的相应说明集添加此功能。
