# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>在SQL Server Reporting Services安装Skype for Business Server 
 
**摘要：** 了解在何处查找有关用户SQL Server Reporting Services的信息Skype for Business Server。
  
Skype for Business Server SSRS SQL Server Reporting Services (SSRS) 查看和监控报告。 若要使用此功能，需要安装并配置 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安装 SQL Server Reporting Services

如果您打算使用监控Skype for Business Server报告 (请参阅本文档的下一篇文章，了解详细信息) 您必须首先安装SQL Server Reporting Services;可以在安装 Reporting Services 的同时安装Microsoft SQL Server安装报告服务SQL Server安装。 如果您未安装 SQL Server，则请按照本文档之前提供的说明操作。 安装 SQL Server 时，确保在“功能选择”页上选择 Reporting Services。 这将安装 SQL Server Reporting Services。
  
若要了解如何安装SQL Server Reporting Services，请参阅SQL Server Reporting Services ([SSRS) 。 ](/sql/reporting-services/create-deploy-and-manage-mobile-and-paginated-reports)
  
如果已安装SQL Server但没有安装SQL Server Reporting Services可以按照一组适当的说明添加该功能SQL Server Reporting Services。
