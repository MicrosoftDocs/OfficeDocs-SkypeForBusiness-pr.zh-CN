# <a name="install-sql-server-reporting-services-in-skype-for-business-server-2015"></a>在 Skype 业务服务器 2015年安装 SQL Server 报表服务
 
**摘要：**了解从何处去寻找 SQL Server 报告服务业务服务器 2015年的 Skype 所使用的信息。
  
Skype 的业务服务器 2015年可以使用 SQL Server 报告服务 (SSRS) 来查看和监视报告。 为了使用此功能，你需要安装和配置 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安装 SQL Server Reporting Services

如果您打算使用 Skype 业务服务器 2015年监视报告 （请参阅本文档的详细信息的下一篇文章） 必须首先安装 SQL Server 报表服务;报告服务可以安装在安装 Microsoft SQL Server 在同一时间或在安装 SQL Server 之后的任何时间。 如果你未安装 SQL Server，则请按照本文档之前提供的说明操作。 安装 SQL Server 时，确保在“功能选择”页上选择 Reporting Services。 这将安装 SQL Server Reporting Services。
  
若要了解如何安装 SQL Server 报表服务，请参阅[SQL Server 报告服务 (SSRS)](https://technet.microsoft.com/en-us/library/ms159106.aspx)。
  
如果你已安装 SQL Server 但未安装 SQL Server Reporting Services，则可按照 SQL Server Reporting Services 的相应说明集添加此功能。
