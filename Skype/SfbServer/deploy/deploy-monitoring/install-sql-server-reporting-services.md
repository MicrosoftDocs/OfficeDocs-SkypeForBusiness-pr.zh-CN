# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>在 Skype for Business Server 中安装 SQL Server Reporting Services 
 
**摘要：** 了解在哪里可以找到有关 Skype for Business Server 使用的 SQL Server Reporting Services 的信息。
  
Skype for Business Server 可以使用 SQL Server Reporting Services （SSRS）查看和监控报告。 若要使用此功能，您将需要安装和配置 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安装 SQL Server Reporting Services

如果您打算使用 Skype for Business Server Monitoring Reports （有关详细信息，请参阅本文档的下一篇文章），必须首先安装 SQL Server Reporting Services;可以在安装 Microsoft SQL Server 的同时安装 Reporting Services，也可以在安装 SQL Server 之后安装任何时间。 如果您未安装 SQL Server，则请按照本文档之前提供的说明操作。 安装 SQL Server 时，确保在“功能选择”页上选择 Reporting Services。 这将安装 SQL Server Reporting Services。
  
若要了解如何安装 SQL Server Reporting Services，请参阅[Sql Server Reporting services （SSRS）](https://technet.microsoft.com/library/ms159106.aspx)。
  
如果已安装 SQL Server 但未安装 SQL Server Reporting Services，则可以通过遵循 SQL Server Reporting Services 的相应指令集来添加该功能。
