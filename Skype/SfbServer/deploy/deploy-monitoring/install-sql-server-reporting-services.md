# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>在 skype for Business Server SQL Server Reporting Services 
 
**摘要：** 了解在哪里可以找到有关 Skype for Business Server SQL Server Reporting Services 的信息。
  
Skype for Business Server 可以使用 SQL Server Reporting Services (SSRS) 查看和监视报告。 若要使用此功能，您需要安装并配置 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安装 SQL Server Reporting Services

如果打算使用 Skype for Business Server 监控报告 (请参阅本文档的下一篇文章，了解详细信息) 必须先安装 SQL Server Reporting Services;可以在安装报告服务的同时安装Microsoft SQL Server安装报告服务SQL Server安装报告服务。 如果您未安装 SQL Server，则请按照本文档之前提供的说明操作。 安装 SQL Server 时，确保在“功能选择”页上选择 Reporting Services。 这将安装 SQL Server Reporting Services。
  
若要了解如何安装 Reporting Services SQL Server，请参阅[SQL Server Reporting Services (SSRS) 。 ](https://technet.microsoft.com/library/ms159106.aspx)
  
如果您已经安装了 SQL Server 但没有安装 SQL Server Reporting Services，则可以通过按照适用于 Reporting Services 的一组相应说明SQL Server该功能。
