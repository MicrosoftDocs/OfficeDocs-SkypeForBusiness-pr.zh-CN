# <a name="provisioning-of-skype-room-system-exchange-and-skype-accounts"></a>预配 Skype 会议室系统 Exchange 和 Skype 帐户
 
阅读这些主题，了解如何为 Skype 会议室系统设置 Exchange 和 Skype 帐户。 

> [!NOTE]
> Microsoft Teams 会议室是具有不同的依赖项和部署过程的不同产品。 有关 Microsoft Teams 会议室的信息，请参阅 Microsoft Teams 会议室 [部署概述](/MicrosoftTeams/rooms/rooms-deploy)。
  
> [!NOTE]
> Skype 会议室系统帐户设置取决于你的组织具有的拓扑类型。 若要详细了解 Active Directory 拓扑，请参阅[Environmental requirements for Skype for Business Server 2015。](../../plan-your-deployment/requirements-for-your-environment/environmental-requirements.md) 
  
## <a name="provisioning-of-skype-room-system-exchange-amp-skype-for-business-accounts"></a>设置 Skype 会议室系统 Exchange &amp; Skype for Business 帐户

以下主题介绍如何为以下帐户设置和管理 Skype 会议室系统 Exchange 和 Skype for Business 帐户：
  
- 单个林本地部署
    
- 多个林本地部署
    
- Microsoft 365 或 Office 365
    
- 混合部署
    
- Skype 会议室系统和 Skype for Business 联盟伙伴
    
- 管理 Skype 会议室系统账户
