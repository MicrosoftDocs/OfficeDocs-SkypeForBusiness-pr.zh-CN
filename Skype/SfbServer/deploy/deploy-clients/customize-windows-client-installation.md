# <a name="customize-windows-client-installation-in-skype-for-business-server"></a>自定义 Windows 客户端中 Skype 业务服务器安装
 
**摘要：** 安装方法和工具的 Skype for Business 概述。
  
> [!NOTE]
> 有关 Office 365 附带的业务的 Skype 安装信息，请参阅[部署 Office 365 中的商业客户端 Skype](https://support.office.com/article/8c563b81-22c9-4024-9efe-9fe28c7bbc96)。 
  
企业管理员可以使用本节中讨论的方法自定义基于 Windows Installer (.msi) 安装的 Skype for Business 的批量许可版本。 因为没有单个工具提供了所有自定义选项，您将业务部署，在您 Skype 中可能使用这些方法的组合。 您可以使用以下章节中所述的工具：
  
- [使用 Office 自定义工具 (OCT) 中的业务服务器 Skype](use-the-office-customization-tool-oct.md)自定义安装程序选项和功能的 Skype 业务和其他 Office 程序。
    
- [使用 Config.xml 执行安装任务 Skype 业务服务器中的](use-config-xml-to-perform-installation-tasks.md)，指定网络安装点的路径并执行无提示安装。
    
- [Skype 业务服务器中的使用 Setup command-line options](use-setup-command-line-options.md)指定要在安装过程中使用的 Config.xml 文件。
    
- 使用组策略对象编辑器 mmc 管理单元[中的业务服务器 Skype 的配置客户端引导策略](configure-client-bootstrapping-policies.md)。
    
可能将您需要配置的产品的 Office 套件部署时其他的选项。 本节中的主题概要介绍这些自定义工具，并讨论特定到 Skype 的注意事项业务。 包含的是指向每个工具的详细 Office 帮助的链接。
