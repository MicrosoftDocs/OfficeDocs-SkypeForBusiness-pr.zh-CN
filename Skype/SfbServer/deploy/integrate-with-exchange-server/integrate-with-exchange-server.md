# <a name="integrate-skype-for-business-server-with-exchange-server"></a>将Skype for Business Server与Exchange Server

**摘要：** 查看 Exchange Server 2013 或更高版本以及 Skype for Business Server。

Exchange Server 2013 或更高版本Skype for Business Server兼容且集成良好。 例如，Skype for Business Microsoft Outlook 中报告用户状态信息;同样，Skype for Business访问用户的 Outlook 日历，注意用户已安排会议，并且会议期间将用户状态显示为忙碌。 尽管您不一Exchange Server运行 Skype for Business Server (反之亦然) 两种产品共同增强彼此的用户体验。

本文档提供有关集成 Skype for Business Server 和 Exchange Server 2016 或 Exchange Server 2013 的信息，但它假定这两个产品的初始安装和配置已经发生。 有关部署部署Skype for Business Server请参阅Skype for Business Server[技术中心](../../../Hub/index.yml)。 有关部署部署Exchange Server请参阅你的部署版本的部署Exchange。

如果要将本地安装的 Skype for Business Server 与 Microsoft Exchange Online 集成，请参阅配置本地 Skype for Business Server 与 Outlook Web App[之间的集成](outlook-web-app.md)。

如果要将 Skype for Business Online 与本地Exchange Server集成，请参阅在 Skype for Business Online 和 Exchange[内部部署 之间配置 OAuth。](oauth-with-online-and-on-premises.md)
