# <a name="mobility-planning-tool"></a>Mobility (Planning Tool) 
 
Skype for Business Server 2015 中的移动功能支持移动设备上的 Skype for Business 功能。
  
移动功能与 Lync Server 2013 中的功能相同。 有关为移动环境做准备的更多详细信息，请参阅 Plan [for Mobility for Skype for Business Server。](../../plan-your-deployment/mobility.md)
