# <a name="mobility-planning-tool"></a>移动规划工具
 
Skype for Business Server 2015 中的移动功能支持Skype for Business移动功能。
  
移动功能与 Lync Server 2013 中的功能相同。 有关准备环境实现移动功能的信息，请参阅规划移动[Skype for Business Server。](../../plan-your-deployment/mobility.md)
