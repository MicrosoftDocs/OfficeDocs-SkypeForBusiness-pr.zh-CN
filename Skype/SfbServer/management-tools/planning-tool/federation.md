# <a name="federation-planning-tool"></a>联合身份验证（规划工具）
 
可以将边缘服务器配置为允许您的内部和外部用户访问合作伙伴组织或服务的联系人。
  
 联合身份验证的工作方式与 Microsoft Lync 2013 一样。 有关为联合准备环境的更多详细信息，请参阅下列主题：
  
- [规划 Lync Server 2013 和 Office 通信服务器联盟](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [在 Lync Server 2013 中规划公共即时消息连接](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [在 Lync Server 2013 中规划可扩展消息和状态协议（XMPP）联盟](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
