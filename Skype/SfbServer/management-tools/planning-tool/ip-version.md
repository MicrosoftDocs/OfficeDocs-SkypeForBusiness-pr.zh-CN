# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
在 Skype for Business Server 2015 中配置 IP 地址时，你有三个选项。 你可以将 Skype for business Server 2015 配置为仅支持 IP 版本4（IPv4）、仅支持 IP 版本6（IPv6）或二者（称为双重堆栈）的组合。
  
有关选择 IP 基础结构的更多详细信息，请参阅[在 Skype For business 中规划 IPv6](../../plan-your-deployment/network-requirements/ipv6.md)。
