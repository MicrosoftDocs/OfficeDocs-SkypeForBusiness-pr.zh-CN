# <a name="ip-version-planning-tool"></a>IP Version (Planning Tool)
 
时为业务服务器 2015 Skype 中配置 IP 地址，有三个选项。 您可以配置的业务服务器 2015 支持仅 IP 版本 4 (IPv4)，仅 IP 版本 6 (IPv6) 的 Skype 或二者的组合 （称为双协议栈）。
  
有关选择 IP 基础结构的详细信息，请参阅[Plan for Business 的 Skype 中的 IPv6 的](../../plan-your-deployment/network-requirements/ipv6.md)。
