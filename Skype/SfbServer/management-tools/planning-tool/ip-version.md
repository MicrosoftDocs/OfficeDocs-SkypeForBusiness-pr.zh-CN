# <a name="ip-version-planning-tool"></a>IP 版本 （规划工具）
 
您有三个选项时的业务服务器 2015 Skype 在配置 IP 地址。 您可以配置业务服务器 2015 支持仅 IP 版本 4 (IPv4)，唯一的 IP 版本 6 (IPv6) 的 Skype 或两者的组合 （即所谓的双堆栈）。
  
选择您的 IP 基础结构的详细信息，请参阅[规划业务的 Skype 在 IPv6](../../plan-your-deployment/network-requirements/ipv6.md)。
