# <a name="voice-traffic-planning-tool"></a>语音流量规划工具
 
规划工具使用有关用户流量的信息来确定部署中要使用的端口数。 此数字反过来确定所需的中介服务器和网关的数量。 有关详细信息，请参阅[Estimating voice usage and traffic for Skype for Business Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md)。
