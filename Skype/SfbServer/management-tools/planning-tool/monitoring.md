# <a name="monitoring-planning-tool"></a>监视规划工具

在 Skype for Business Server 2015 中，监控用于收集有关用户参与的通信会话的使用信息和用户体验质量 (QoE) 数据。

有关规划监控服务器的更多详细信息，请参阅 Planning [for Monitoring。](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring)
