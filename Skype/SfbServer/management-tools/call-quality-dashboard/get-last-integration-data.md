# <a name="get-last-integration-data"></a>获取上次集成数据
 
**摘要：** 了解有关获取最后一个集成数据操作，它是用于呼叫的质量仪表板的数据 API 的一部分。 呼叫质量仪表板是用于业务服务器 Skype 的工具。
  
获取上次集成数据操作是用于呼叫的质量仪表板的数据 API 的一部分。
  
## <a name="get-last-integration-data"></a>获取上次集成数据

获取上次集成数据操作返回上次 5 成功/的存档和多维数据集处理失败的列表。
  
默认情况下禁用此功能，并且它需要启用通过配置数据 API。
  

|**方法**|**请求 URI**|**HTTP 版本**|
|:-----|:-----|:-----|
|获取  <br/> |https://\<门户\>/QoEDataService/IntegrationLog/Status  <br/> |HTTP/1.1  <br/> |
   
 **URI 参数**-无。
  
 **请求标头**的任何其他的标头。
  
 **请求正文**-无。
  
 **响应**-响应包括 HTTP 状态代码和一响应标头。
  
 **状态代码**-成功的操作返回状态代码 200 （确定)。
  
 **响应标头**的任何其他的标头。
  
 **响应正文**-下面是示例日志状态。
