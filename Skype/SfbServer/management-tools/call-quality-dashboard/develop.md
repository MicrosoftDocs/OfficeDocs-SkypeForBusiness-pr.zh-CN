# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>开发 Skype for business 服务器的通话质量仪表板
 
**摘要:** 了解如何开发通话质量仪表板的解决方案。 通话质量仪表板是 Skype for business 服务器的工具。
  
本主题提供适用于 Skype for business 服务器的通话质量仪表板 (CQD) 的开发文档。
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>开发 Skype for business 服务器的通话质量仪表板

可以使用自定义开发将 CQD 扩展并集成到你的报告环境中。 
  
有两组 web Api 可用, 一个用于访问多维数据集中的数据 (通过数据 Api), 另一个用于访问报表元数据 (通过存储库 API)。 
  
- [Skype for Business Server 中的呼叫质量仪表板 (CQD) 的数据 API](data-api.md)
    
- [Skype for Business Server 中的呼叫质量仪表板 (CQD) 的知识库 API](repository-api.md)
    
- [CQD 开发示例](cqd-development-samples.md)
