# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>为 Skype for Business Server 开发通话质量仪表板
 
**摘要：** 了解如何为通话质量仪表板开发解决方案。 通话质量仪表板是 Skype for Business Server 的工具。
  
本主题提供 Skype for Business Server 的呼叫质量仪表板 (CQD) 文档。
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>为 Skype for Business Server 开发通话质量仪表板

可以使用自定义开发将 CQD 扩展并集成到报告环境。 
  
有两组 Web API 可用，一组用于通过数据 API (访问多维数据集) ，另一组用于通过存储库 API (访问报告元数据) 。 
  
- [Skype for Business Server 中用于呼叫质量仪表板 (CQD) 的数据 API](data-api.md)
    
- [Skype for Business Server 中用于呼叫质量仪表板 (CQD) API](repository-api.md)
    
- [CQD 开发示例](cqd-development-samples.md)
