# <a name="call-quality-dashboard-for-skype-for-business-server-2015"></a>呼叫质量的业务服务器 2015 Skype 仪表板
 
**摘要：** 了解有关呼叫质量仪表板，它是用于业务服务器 2015年的 Skype 的报告工具。
 
Skype 的业务服务器 2015年呼叫质量仪表板 (CQD) 是在质量的体验中的数据库中的业务服务器 2015年和 Lync Server 2013 的 Skype 的监控服务器报告层。 CQD 使用 Microsoft SQL Server Analysis Services 提供聚合的用法和呼叫质量的信息，以及筛选和数据集上透视。
