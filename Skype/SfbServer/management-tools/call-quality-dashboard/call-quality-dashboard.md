# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Skype for Business Server 的通话质量仪表板
 
**摘要：** 了解通话质量仪表板，它是 Skype for Business Server 的报告工具。
 
Skype for Business Server 呼叫质量仪表板 (CQD) 是 Skype for Business Server 和 Lync Server 2013 中监控服务器中用户体验质量数据库顶部的报告层。 CQD 使用Microsoft SQL Server Analysis Services聚合使用情况和呼叫质量信息，以及筛选和透视数据集。
  
本节中包含以下主题，应在部署呼叫质量仪表板之前查看这些主题。

- [为 Skype for Business Server 部署通话质量仪表板](deploy-0.md)
- [使用 Skype for Business Server 的呼叫质量仪表板](use.md)
- [为 Skype for Business Server 开发通话质量仪表板](develop.md)
