# <a name="call-quality-dashboard-for-skype-for-business-server"></a>Skype for Business Server 的通话质量仪表板
 
**摘要：** 了解呼叫质量仪表板，它是一个报告工具Skype for Business Server。
 
Skype for Business Server呼叫质量仪表板 (CQD) 是一个报告层，位于 Skype for Business Server 和 Lync Server 2013 中监控服务器中用户体验质量数据库的顶部。 CQD 使用Microsoft SQL Server Analysis Services聚合使用情况和呼叫质量信息，以及筛选和透视数据集。
