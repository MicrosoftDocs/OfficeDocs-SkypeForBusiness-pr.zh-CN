# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015 管理工具
 
**摘要：** 为业务服务器 2015年了解 Skype 中的服务管理工具。
  
Skype 业务服务器 2015年通信软件 (以前称为 Lync Server)，提供了即时消息 (IM)、 状态、 会议和电话服务解决方案支持企业级协作要求。 用于管理这些服务的工具灵活而强大。
