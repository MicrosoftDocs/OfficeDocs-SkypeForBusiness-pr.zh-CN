> [!IMPORTANT]

通过使用Microsoft PowerShellSkype for Business Online的远程管理运行以下操作系统之一的 64 位计算机上仅支持：
  
- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012 R2
    
- Windows Server 2012
    
- Windows Server 2008
    
- Windows 7
    
除了受支持的操作系统，计算机必须也运行下列：
  
- PowerShell
    
- Skype for Business Online 连接器模块
    
您可以找到有关下载和安装这些项目下列主题中的信息：
  
- [下载并安装 Skype for Business Online 连接器模块](download-and-install-the-skype-for-business-online-connector-module.md)
    
- [下载并安装 Windows PowerShell 3.0](download-and-install-windows-powershell-3-0.md)
