# <a name="phone-number-management-for-belgium"></a>比利时的电话号码管理

使用此表查找有关获取和管理适用于 Skype for Business Online 的比利时电话号码的信息。 
  
有关详细信息，请参阅[为你的组织管理电话号码](manage-phone-numbers-for-your-organization.md)。
  
|**你要做什么？**|**下载正确的表单并发送给我们**|
|:-----|:-----|
|获取新的用户电话号码。  <br/> |在比利时，你无法使用 Skype for Business 管理中心为你的组织获取电话号码，你必须提交新的电话号码请求。 下载下列表单之一： <br/> [比利时的新电话号码请求（地理号码）(v.3.0)(de.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(de.be).pdf) <br/> [比利时的新电话号码请求（地理号码）(v.3.0)(fr.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(fr.be).pdf) <br/> [比利时的新电话号码请求（地理号码）(v.3.0)(nl.be)](../../downloads/new-number-request-forms/new-phone-number-request-for-belgium-(geographic-numbers)-(v.3.0)-(nl.be).pdf) <br/>  填写此请求并[将其发送给我们](mailto:ptneu@microsoft.com)。  <br/> |
|你需要将电话号码转移给我们。  <br/> | 下载下列表单之一： <br/> [比利时授权书 (LOA)（地理号码）(v.1.0)(nl.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(nl.be).pdf) <br/> [比利时授权书 (LOA)（地理号码）(v.1.0)(fr.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(fr.be).pdf) <br/> [比利时授权书 (LOA)（地理号码）(v.1.0)(de.be)](../../downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(geographic-numbers)-(v.1.0)-(de.be).pdf) <br/>  填写此请求并[将其发送给我们](mailto:ptneu@microsoft.com)。 <br/> |
|你需要在国家电话目录中发布/取消发布电话号码。  <br/> |[向我们发送](mailto:ptneu@microsoft.com)你想要发布/取消发布的电话号码列表。      <br/> |
|你需要将用户的号码更改为服务号码。  <br/> |[向我们发送](mailto:ptneu@microsoft.com)你想要转换为服务号码的用户号码列表。 <br/> |
