# 我的移植订单的状态是什么？

> [!IMPORTANT]
> 本文是由机器翻译的，请参阅[免责声明](f651e82d-4237-4e3d-ad74-40fdf11fa8d1.md#MT_Footer)。请在 [此处](https://support.office.com/en-us/article/f651e82d-4237-4e3d-ad74-40fdf11fa8d1) 中查找本文的英文版本以便参考。
  
您可以通过转到 **Skype for Business 管理中心**查看端口订单的状态 > **语音**> **端口订单**。在 **状态**列中，将列出每个端口订单状态。如果您帮助[联系 Office 365 商业版支持人员 - 管理员帮助](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
下表列出了端口顺序状态，以及如果需要您可以采取的操作。
  
|**状态**|**是否可以查看订单？**|**是否可以编辑订单？**|**是否可以取消订单？**|**是否可以删除订单？**|**说明**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**处理** <br/> |是  <br/> |不支持  <br/> |支持  <br/> |不支持  <br/> |管理员已经创建了顺序，并已收到 microsoft。  <br/> |
|**联系运营商** <br/> |是  <br/> |不支持  <br/> |支持  <br/> |不支持  <br/> |收到的订单，将其由 Microsoft、 审批和我们正在与丢失运营商以获取其批准。  <br/> |
|**转移已获批准** <br/> |是  <br/> |确认订单承诺 (FOC)  <br/> |是  <br/> |不支持  <br/> |订单已被接受丢失运营商，并且已设置 FOC 日期。  <br/> |
|**转移待定** <br/> |是  <br/> |否  <br/> |否  <br/> |不支持  <br/> |传输是不超过 24 小时，因此顺序不再可编辑或取消。  <br/> |
|**错误** <br/> |否  <br/> |是  <br/> |支持  <br/> |是 （此时，您无法删除的端口顺序如果出现错误。端口顺序需要重新创建，或需要[联系 Office 365 商业版支持人员 - 管理员帮助](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。  <br/> |被弃用的运营商已拒绝该订单。  <br/> |
|**已完成** <br/> |是  <br/> |否  <br/> |否  <br/> |否  <br/> |号码已成功转移。  <br/> |
|**已取消** <br/> |否  <br/> |支持  <br/> |否  <br/> |否  <br/> |管理员已取消订单。  <br/> |
   
 **完成的分步说明，请参阅[将电话号码转移到 Office 365](transfer-phone-numbers-to-office-365.md)。**
