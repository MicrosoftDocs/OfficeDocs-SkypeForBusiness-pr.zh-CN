# 更改组织的 Skype 会议直播设置

> [!IMPORTANT]
> 本文是由机器翻译的，请参阅[免责声明]。
  
你可以启用 Skype 会议直播并对这些会议的设置和策略进行更改。
  
- **启用 Skype 会议直播** 启用 Skype 会议直播。启用 Skype 会议直播后，你需要[设置 Skype 会议直播网络](set-up-your-network-for-skype-meeting-broadcast.md)。如果希望面向企业外部的人员举办网络研讨会和其他直播活动，请执行此步骤。
    
- **我的组织启用 Skype 会议广播预览功能**Skype for Business 客户计划为您提供了早期访问新产品和功能。这样，您的组织快速浏览了什么问题并有机会在您自己的环境中测试新功能和提供反馈，我们发布给公众的产品版本之前。Skype for Business 预览版https://www.skypepreview.com/[]()
    
- **允许组织者可以安排匿名会议**这样，不要关闭字幕和翻译直播会议组织者。
    
- **允许直播的会议记录**这样，您必须演示者或组织者录制的任何会议。
    
- **技术支持人员支持与会者的 URL**输入会议广播与会者使用如果他们需要帮助连接或参加广播的会议的链接。
