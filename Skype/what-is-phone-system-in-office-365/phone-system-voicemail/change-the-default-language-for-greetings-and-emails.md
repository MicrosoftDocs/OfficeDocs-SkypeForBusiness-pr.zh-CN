# <a name="change-the-default-language-for-greetings-and-emails"></a>更改贺卡和电子邮件的默认语言

如果您是[Office 365 全局管理员](https://support.office.com/en-us/article/da585eea-f576-4f55-a1e0-87090b6aaa9d)，您可以设置 Skype 为玩另一种语言问候其默认语音邮件的业务。 默认系统问候语是类似，"请为 John Smith 将一条消息。 后音，请记录您的消息。 完成录制后，挂上，或井按键查看更多选项。 
  
 **首先，请阅读以下重要信息：**
  
- **对您可用的语言取决于组织的位置**。 例如，如果您的公司位于美国，您可以设置默认语言为英语或西班牙语。 如果您的组织是在加拿大，您可以选择英语和法语。 有关支持的语言的列表，请参阅[语音邮件问候语和来自 Skype 业务邮件的语言](languages-for-voicemail-greetings-and-messages.md)。
    
- **无法仅为组织中的一个用户更改系统语言。** 只能为组织中的所有用户更改问候语言。
    
    > [!NOTE]
    > 用户可以更改自己的问候到其设置的语言，他们签名后。 
  
- **记录传出语音邮件消息吗？** [检查的 Skype 业务语音邮件和选项](https://support.office.com/en-us/article/2deea7f8-831f-4e85-a0d4-b34da55945a8)，请参阅。
    
## <a name="change-the-system-language-for-everyone-in-your-organization"></a>更改组织内所有用户的系统语言

1. 使用[https://portal.office.com/adminportal/home](https://portal.office.com/adminportal/home)在[Office 365 提供全局管理员](https://support.office.com/en-us/article/da585eea-f576-4f55-a1e0-87090b6aaa9d)帐户登录。 
    
2. 在管理中心，选择**设置** > **组织配置文件**。 
    
     ![Choose Settings and then choose Organization profile.](../../images/9d9de520-bb84-409f-9417-96bd8ec86c48.png)
  
3. 选择" **编辑**"。
    
    ![Choose Edit.](../../images/e4a0b09d-2b68-4bc8-a0d3-230939843ee2.png)
  
4. 为您的组织中的每个人都从**首选语言**列表中选择一种语言。
    
5. 选择" **保存**"。
