# <a name="skype-for-business-server-2019-management-tools"></a>Skype 的业务 Server 2019 管理工具
 
**摘要：** 为业务服务器 2019年了解 Skype 中的服务管理工具。
  
Skype 的业务服务器 2019年提供即时消息 (IM)、 状态、 会议和电话服务解决方案支持企业级协作要求。 用于管理这些服务的工具灵活而强大。
