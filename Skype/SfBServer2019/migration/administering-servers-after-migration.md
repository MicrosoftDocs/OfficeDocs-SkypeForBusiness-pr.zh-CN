# <a name="administering-servers-after-migration"></a>迁移后管理服务器

一般情况下，您必须使用与要管理的服务器版本相对应的管理工具。 不能从不同版本的相同计算机上安装管理工具。
  
> [!IMPORTANT]
> Skype 的业务 Server 试点池部署后，不能使用拓扑生成器或控制面板管理任何 2019年资源。 您必须使用 2019年工具管理当前和以前版本的所有资源。
