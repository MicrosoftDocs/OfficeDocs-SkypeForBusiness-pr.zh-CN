# <a name="administering-servers-after-migration"></a>迁移后服务器的管理

通常, 你必须使用与要管理的服务器版本对应的管理工具。 不能在同一台计算机上安装不同版本的管理工具。
  
> [!IMPORTANT]
> 部署 Skype for Business 服务器试验池后, 您不能使用拓扑生成器或控制面板管理任何2019资源。 必须使用2019工具管理所有当前和以前的版本资源。
