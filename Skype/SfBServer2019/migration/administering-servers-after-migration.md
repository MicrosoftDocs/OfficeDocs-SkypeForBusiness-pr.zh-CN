# <a name="administering-servers-after-migration"></a>迁移后服务器的管理

通常，使用的管理工具必须与要管理的服务器版本对应。 不能在同一台计算机上安装不同版本的管理工具。
  
> [!IMPORTANT]
> 部署 Skype for Business Server 试点池后，不能使用拓扑生成器或控制面板来管理任何2019资源。 必须使用2019工具来管理所有当前和以前的版本资源。
