# <a name="user-migration"></a>用户迁移

迁移普遍接受最佳做法是创建多个测试用户并使用它们来执行系统测试。 您已成功移动并测试这些帐户之后，您应确定一组试生产用户移动其帐户并执行这些验证测试。 时获得满意的结果，您可以将其余的用户移动到新的部署。
  
有关为用户启用的业务服务器 2019 Skype 的其他信息，请参阅业务服务器 2019年文档的部署 Skype。
