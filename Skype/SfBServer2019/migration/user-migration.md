# <a name="user-migration"></a>用户迁移

通常接受的迁移最佳做法是创建多个测试用户并使用它们来执行系统测试。 成功移动并测试这些帐户后, 您应标识一组试验性生产用户并在其上移动其帐户并执行验证测试。 获得满意的结果后, 您可以将其他用户移动到新部署。
  
有关为 Skype for business Server 2019 启用用户的其他信息, 请参阅部署 Skype for business Server 2019 文档。
