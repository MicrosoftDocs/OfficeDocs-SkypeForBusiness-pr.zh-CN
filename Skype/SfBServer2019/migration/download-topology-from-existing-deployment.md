# <a name="download-topology-from-existing-deployment"></a>从现有部署下载拓扑

创建业务服务器 2019年池 Skype 时, 将使用中央管理存储的与旧安装相关联。 启动时拓扑生成器在首次使用和后续编辑会话，则会提示您希望拓扑生成器以加载当前配置文档的位置。 您已有定义拓扑，并建立中央管理存储，因为您应选择从现有部署下载拓扑。 拓扑生成器会读取数据库，并检索当前定义。 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>若要从现有部署下载拓扑

1. 打开 Skype 业务 Server 部署向导。
    
2. 从**Skype 的业务服务器 2019-部署向导**页上，单击**安装管理工具**。
    
3. 启动拓扑生成器： 单击**开始**，单击**所有程序**，都单击**Microsoft Skype 的业务服务器 2019年**，，然后都单击**业务 Server 拓扑生成器的 Skype**。
    
4. 选择**从现有部署下载拓扑**。
  
5. 选择文件名并保存具有默认.tbxml 文件类型的拓扑。
    
6. 展开企业服务器节点 Skype，以显示部署中的各个服务器角色。
