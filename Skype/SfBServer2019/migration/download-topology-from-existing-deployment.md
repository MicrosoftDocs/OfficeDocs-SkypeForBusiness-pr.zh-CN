# <a name="download-topology-from-existing-deployment"></a>从现有部署下载拓扑

创建 Skype for Business Server 2019 池时，将使用与旧安装关联的中央管理存储。 因第一次使用而启动拓扑生成器并随后编辑会话时，系统会提示您要使拓扑生成器加载当前配置文档的位置。 由于已定义了一个拓扑，并且已建立了中央管理存储，因此应选择从现有部署下载拓扑。 拓扑生成器将读取数据库并检索当前的定义。 
  
### <a name="to-download-a-topology-from-an-existing-deployment"></a>从现有部署下载拓扑

1. 打开 Skype for Business Server 部署向导。
    
2. 在 " **Skype For Business Server 2019-部署向导**" 页上，单击 "**安装管理工具**"。
    
3. 启动拓扑生成器：依次单击 "**开始**"、"**所有程序**"、" **Microsoft Skype for business server 2019**"，然后单击 " **Skype for business server 拓扑生成器**"。
    
4. 选择“从现有部署下载拓扑”****。
  
5. 选择一个文件名，并使用默认的 tbxml 文件类型保存该拓扑。
    
6. 展开 Skype for Business Server 节点以显示部署中的各种服务器角色。
