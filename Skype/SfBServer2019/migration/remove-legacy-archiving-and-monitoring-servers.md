# <a name="remove-legacy-archiving-and-monitoring-servers"></a>删除旧存档和监控服务器

如果旧部署包含存档服务器或监控服务器，则迁移到 Skype for Business Server 2019 后，可以从旧环境中删除这些服务器，只要所有用户都从其余任何旧池中删除。 可以任意顺序删除存档服务器或监控服务器。 关键要求是所有用户均已从其余任何旧池中删除。
  
您可以按照阶段[4：](phase-4-move-test-users-to-the-pilot-pool.md)将测试用户移至试点池中中概述的过程将用户移动到 2019 Skype for Business Server 2019。
  
确认所有用户已从任何剩余池中删除后，取消服务器通信并删除角色。 一个过时但相关的示例是"卸载 Microsoft Lync Server 和删除服务器角色"，可从 下载 [https://go.microsoft.com/fwlink/p/?linkId=246227](https://go.microsoft.com/fwlink/p/?linkId=246227) 。
