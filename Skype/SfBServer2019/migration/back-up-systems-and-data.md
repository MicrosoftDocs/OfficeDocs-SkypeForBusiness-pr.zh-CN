# <a name="back-up-systems-and-data"></a>备份系统和数据

开始迁移之前，请执行完整系统备份并记录现有系统，以便可以回滚有必要如果包括驻留在每个池的用户帐户清单。 为备份和还原数据、 设置和系统提供了多种工具和程序。

有关详细信息和过程，请参阅备份和还原过程的版本。
