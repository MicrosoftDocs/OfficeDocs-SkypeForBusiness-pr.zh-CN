# <a name="back-up-systems-and-data"></a>备份系统和数据

在开始迁移之前，请执行完整的系统备份并记录现有系统，包括驻留在每个池中的用户帐户的清单，以便您可以在需要时回滚。 可以使用多种工具和程序来备份和还原数据、设置和系统。

有关详细信息和过程，请参阅版本的备份和还原过程。
