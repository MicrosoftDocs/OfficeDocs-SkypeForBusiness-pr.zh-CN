# <a name="view-current-users-in-legacy-pool"></a>查看旧池中的当前用户

在了解在池之间移动用户的各种方法之前，我们必须先确定哪些用户存在于旧池中。 " **注册器池** "列标识为旧池配置的用户。 这些是我们将移至 2019 年Skype for Business Server用户。
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>查看旧池中的用户列表

1. 使用 RTCUniversalServerAdmins 组或 CsAdministrator 或 CsUserAdministrator 管理角色 成员的帐户登录到旧版前端服务器。
    
2. 打开 **Skype for Business Server控制面板"。**
    
3. 依次单击“用户”、“搜索”和“查找”。
