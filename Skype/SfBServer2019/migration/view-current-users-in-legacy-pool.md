# <a name="view-current-users-in-legacy-pool"></a>查看当前用户在旧池

之前学习可以将用户移动池之间的各种方式，我们首先必须确定哪些用户存在于旧池中。 下图中，在**注册器池**列标识为旧池中配置了六个用户。 这些是我们将移动到 Skype 业务服务器 2019年池的测试用户。
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>若要在旧池中的用户的列表，请参阅

1. 登录到旧版前端服务器使用具有 RTCUniversalServerAdmins 组的成员身份或 CsAdministrator 或 CsUserAdministrator 管理角色的成员的帐户。
    
2. 打开**Skype 的业务 Server Control Panel**。
    
3. 单击**用户**，单击**搜索**，然后单击**查找**。
