# <a name="prepare-active-directory-for-skype-for-business-server"></a>为 Skype for business 服务器准备 Active Directory

在共存状态中部署 Skype for Business Server 2019 之前, 必须执行一些其他 Active Directory 任务, 以便为 Skype for business Server 2019 配置架构、林和域。 架构扩展添加 Skype for business Server 2019 所需的 Active Directory 类和属性。 <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>为 Skype for business Server 2019 准备 Active Directory

1. 在 Skype for business 服务器2019前端服务器上, 运行 Skype for business Server 2019 安装程序。
    
2. 选择 "**准备 Active Directory**"。
  
3. 完成向导中的步骤1至步骤5。
