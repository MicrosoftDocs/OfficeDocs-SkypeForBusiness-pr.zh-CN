# <a name="skype-for-business-server-prepare-active-directory-for-skype"></a>Skype for Business Server：为 Active Directory 准备Skype

在以共存Skype for Business Server部署 2019 之前，您必须执行更多 Active Directory 任务来配置 Skype for Business Server 2019 的架构、林和域。 架构扩展添加了 2019 年 10 月 Skype for Business Server所需的 Active Directory 类和属性。 <!-- For more information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>为 2019 Skype for Business Server Active Directory

1. 在 Skype for Business Server 2019 前端服务器上，运行 Skype for Business Server 2019 安装程序。
    
2. 选择“准备 Active Directory”。
  
3. 完成向导中的步骤 1 至 5。
