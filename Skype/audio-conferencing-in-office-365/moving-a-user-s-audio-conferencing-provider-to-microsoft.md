# <a name="moving-a-users-audio-conferencing-provider-to-microsoft"></a>移动到 Microsoft 的用户的音频会议提供商

要为业务和 Microsoft 小组使用 Skype 与 Office 365 音频会议，用户需要有分配给他们和他们的音频会议提供商**音频会议**许可证您自己组织中必须设置给 Microsoft。 查看[试用或购买 Office 365 中的音频会议](try-or-purchase-audio-conferencing-in-office-365.md)若要获得许可和多少它的成本的详细信息。
  
## <a name="to-move-a-users-audio-conferencing-provider-to-microsoft"></a>若要向微软移动用户的音频会议提供商

如果**音频会议**许可分配给不具备音频会议提供商的用户，用户的提供商将被自动设置为 Microsoft，您不必执行任何其他操作。 如果该用户已经具有音频会议提供商，必须将它们分配**音频会议**许可证后到 Microsoft 更改用户的提供商。
  
将 Microsoft 设置为具有**音频会议**许可的用户音频会议提供商分配，但正在使用另一个音频会议提供商，请执行此操作：
  
1. 使用你的工作或学校帐户登录 Office 365。
    
2. 请转到**Office 365 管理中心** > **业务的 Skype**。
    
3. 在**Skype 的业务管理中心**，转到**音频会议**。
    
4. 如果您看到一个横幅通知您存在具有**音频会议**的用户许可证分配但没有 Microsoft 设置为其音频会议提供商，但单击**以将其移动，请单击此处**。 如果您看不到标语，在**业务管理中心的 Skype**单击**用户**，然后选择**用户可以转移到会议音频**筛选器。
    
5. 选择您想要移动的用户，然后在操作窗格中，单击**编辑**。
    
6. 在**提供程序名称**列表中选择**Microsoft** 。
    
    > [!NOTE]
    > 当更改为 Microsoft 用户的音频会议提供商时，会更改用户的音频会议信息。 如果您需要保留此信息，请在更改任何用户的提供商之前某个位置记录。 
  
7. 单击" **保存**"。
    
## <a name="what-else-should-i-know"></a>我还需了解哪些信息？

- 如果选择用户列表中，您可以查看用户的默认音频会议信息，但您将无法查看音频会议针。 通过单击**重置**，可以重置用户的 PIN 音频会议。
    
- 如果您想要更改的用户的音频会议设置，您可以从列表中选择用户然后单击**编辑**并在**属性**页上进行更改。
