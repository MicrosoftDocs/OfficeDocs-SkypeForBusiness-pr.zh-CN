# <a name="compare-audio-conferencing-providers"></a>比较音频会议提供商

当您设置音频会议为 Skype 业务和 Microsoft 团队时，您需要选择音频会议提供商。 您可以选择为您的音频会议提供商的 Microsoft 或第三方。 
  
> [!IMPORTANT]
> Microsoft 的团队的用户不能使用第三方音频会议提供商。 
  
音频会议提供商执行以下操作： 
  
- 提供*音频会议桥*。 会议桥将设置拨入电话号码、 针，并对会议的会议 Id。
    
- 提供收费电话，免费电话，并向拨入用户的国际电话号码。 它们包括在会议邀请中的电话号码。
    
- 设置的语言的用户将听到当他们拨号加入会议。
    
- 创建在用户加入会议时所使用的会议 ID。
    
- 为您提供针通过电话开始会议的组织者。
