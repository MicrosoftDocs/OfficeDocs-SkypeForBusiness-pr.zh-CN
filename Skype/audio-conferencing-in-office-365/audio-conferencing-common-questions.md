# 音频会议常见问题

> [!IMPORTANT]
> 本文是由机器翻译的，请参阅[免责声明](a90ea695-aabf-4f10-ae92-24b3f6b27c56.md#MT_Footer)。请在 [此处](https://support.office.com/en-us/article/a90ea695-aabf-4f10-ae92-24b3f6b27c56) 中查找本文的英文版本以便参考。
  
下面是一些最重要的问题，我们获得从我们的客户想要使用的音频会议。
  
设置音频会议的分步说明，请参阅[设置音频会议 for Skype Business 和 Microsoft 团队](set-up-audio-conferencing-for-skype-for-business-and-microsoft-teams.md)。
  
## 音频会议的优点是什么？

拨入会议时非常有用人员是在旅途中，例如，和不能参加会议在其便携式计算机或移动设备上使用Skype for Business或 Microsoft 小组应用程序。但那里其他情况下，在其中使用电话参加Skype for Business或者 Microsoft 小组会议也可能是更好的选择比计算机上使用应用程序：
  
- Internet 连接受限。
    
- 会议只有音频。
    
- 用户尝试加入 Skype for Business 会议，但是失败。
    
- 呼叫质量是更好的如果他们拨入。
    
- 用户可以使用蓝牙设备采用"免提"方式加入会议。
    
- 用户发现这样更加简便，且更加适合他们的情况。
    
## 谁可以参加会议的音频会议？我可以听到谁？

拨入号码和会议 ID 的任何人可以加入Skype for Business或 Microsoft 小组会议如果会议组织者已被锁定会议。
  
无论您企业版或 Microsoft 小组的应用程序中使用电话或 Skype 呼叫，您将能够听到调用时，其他所有人都和他们可以听到您的声音。会议组织者具有"静音"与会者如果他们不想听到它们的能力。
  
## 可以为我的音频会议用户添加免费电话号码吗？

可以，我们提供免费电话号码（服务号码），但是这些号码只在某些国家/地区提供。要查看可用的号码列表，请访问[Where can I get toll-free numbers?](https://support.office.com/article/448bb563-c122-48ce-aab9-a97157a17bd3)。
  
## 目前支持多少个本地拨入号码？

有时您的音频会议购买的许可证分配给您的本地拨入号码。会议邀请中将包含拨入号码。这些本地电话号码将仅适用于您的组织。电话分配给您的组织，通过该组织内的启用了音频会议的用户共享该数字。这样，由用户 A 和另一个用户 B 计划的业务或 Microsoft 小组会议 Skype 将都具有相同的拨入号码。
  
本地拨入号码以及在某些情况下你的组织所在的国家或地区的国际拨入号码，将包含在会议邀请中。如果一名与会者使用了邀请中包含的不同号码，它将成为共享的电话号码。

## 什么是音频会议的最大长度？

最长持续时间取决于与会者以及他们用于加入会议的身份验证类型。
  
|**与会者**|**会议结束时间**|
|:-----|:-----|
|有用户使用 Skype for Business 或 Microsoft 团队应用加入或拨入会议。  <br/> |如果 24 小时后与会者列表没有任何更改，会议结束。  <br/> |
|所有用户都是拨号加入会议，但是部分用户使用了 PIN 进入会议。  <br/> |24 小时后会议结束。  <br/> |
|所有用户都拨号加入会议，但是没有人使用 PIN 进入会议。  <br/> |4 小时后会议结束。  <br/> |
   
## 在会议中可以有多少个总共参与者？

音频会议允许达 250 个电话与会者。
  
若要了解会议限制，请参阅 [Skype for Business Online 限制](https://technet.microsoft.com/zh-cn/library/skype-for-business-online-limits.aspx#bkmk_Meeting_LyncOnlineLimits)。
  
## 为什么未用户开始接收电子邮件与他们音频会议的信息？

我们添加一项新功能，使您的[在 Office 365 中分配管理员角色](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)，发送并更新电子邮件中的音频会议信息和 PIN。若要了解详细信息，包括如何禁用该功能，请参阅[启用或禁用发送电子邮件时更改音频会议设置](enable-or-disable-sending-emails-when-audio-conferencing-settings-change.md)。
  
## 可以使用通过是 Skype for Business Server 2015 本地部署的一部分的用户的音频会议？

我们不太。 但是，您可以继续使用 Skype for Business Server 2015 中以及 PSTN 网关为您的本地用户可用的音频会议。
  
## 用户可以获取个人会议 ID 吗？

企业版和 Microsoft 小组的用户的 Skype 将随机分配给会议 Id，无法保留或设置只有他们能够使用的静态会议 ID。
  
## 可以与 Skype 会议直播一起使用音频会议？

不支持当前为用户启用音频会议加入 Skype 会议直播。
  
## 会议期间，用户可以获得操作员帮助吗？

否，用户无法获得任何操作员帮助或支持通过按 * 会议期间 0。如果存在与音频会议的问题，组织的管理员可以联系[联系 Office 365 商业版支持人员 - 管理员帮助](https://support.office.com/article/32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
## 用户如何访问或更改他们的会议 ID？

为业务或 Microsoft 小组的用户的 Skype 可以找到分配给他们的安排会议 Outlook 和 Outlook 网页版中的会议 ID。 此外，用户可以找到后设置将发送给他们的电子邮件中的会议 ID。
  
> [!NOTE]
> 用户将无法重置其会议 id。 仅可以由您[在 Office 365 中分配管理员角色](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)，组织重置会议 ID。

## 支持哪些会议中拨号盘命令？

- *6（将自己静音/取消静音）
    
- * 1 （播放可用的拨号盘命令的说明）
    
## 可以与会者拨出国际电话号码时 Skype for Business 或 Microsoft 小组会议？

是，与会者可以国际拨出并邀请其他呼叫者加入 Skype for Business 或 Microsoft 小组会议。请参阅[从会议拨出，以便其他人可以加入会议](dialing-out-from-a-meeting-so-other-people-can-join-it.md)。
  
## 为业务或 Microsoft 小组的用户的 Skype 如何安排与音频会议会议详细信息的会议？

如果为用户分配的 **音频会议**许可证，用户在 Outlook 或 Outlook 网页版中创建新的 Skype for Business 或 Microsoft 小组会议，拨入电话号码和会议 Id 会自动添加到会议邀请。
  
## 当所有与会者将使用电话拨入时，用户如何安排和启动会议？

为用户分配的 **音频会议**许可证后，为用户分配了所谓预约式会议 id。可以与会议与会者，而无需安排会议的用户的会议访问电话号码以及共享此会议 ID。
  
下面是当组织者和所有与会者都拨入加入会议时阐明方案：
  
|**用户的设置**|**组织者步骤**|**与会者步骤**|
|:-----|:-----|:-----|
|不允许匿名用户启动会议。  <br/> | 拨打音频会议电话号码。 <br/>  输入会议 ID。 <br/>  输入会议的 PIN。 <br/> | 拨打音频会议电话号码。 <br/>  输入会议 ID。 <br/>  如果会议组织者尚未加入会议，与会者将在会议厅中收听音乐，直到组织者加入会议。 <br/> |
|允许匿名用户启动会议。  <br/> | 拨打音频会议电话号码。 <br/>  输入会议 ID。 <br/> | 拨打音频会议电话号码。 <br/>  输入会议 ID。 <br/>  如果与会者在组织者之前加入会议，则每个与会者都可以启动会议。 <br/> |
