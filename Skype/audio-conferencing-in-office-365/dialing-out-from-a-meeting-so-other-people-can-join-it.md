# <a name="dialing-out-from-a-meeting-so-other-people-can-join-it"></a>从会议拨出，以便其他人可以加入会议

作为会议组织者，您可以拨出使用 Skype 业务或 Microsoft 小组应用程序来让其他人加入同一会议使用他们的电话。 虽然它并不是必需，正在给人拨出时，建议您拨全部号码，包括国家/地区代码。 若要获取拨出工作：
  
- 您可以拨出只有当您加入会议的业务或 Microsoft 小组应用程序使用 Skype。
    
- 作为会议组织者您已启用音频会议的。

 **第 1 步：**在邀请会议中，**邀请更多人员**选项用于拨出的电话号码。
  
![Dial-out with Skype for Business.](../images/9896abec-7a6f-4148-ad09-76a1cf4b56e1.png)
  
 **第 2 步：**输入完整的电话号码，在框中包含国家/地区代码。
  
![Dial-out phone number in Skype for Business](../images/084b4fb5-21eb-4f10-9a1a-c92b919084b9.png)
