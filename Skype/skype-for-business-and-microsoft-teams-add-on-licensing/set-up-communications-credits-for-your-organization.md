# <a name="set-up-communications-credits-for-your-organization"></a>为您的组织设置通信片尾

您将需要设置通信片尾，如果您想要使用 Skype 业务和 Microsoft 团队提供免费电话号码。 此外，我们建议您设置通信贷 （国内或国际） 调用计划和音频会议需要拨出**任何目标**的能力的用户。 许多国家/地区都包括在内，但有些目标可能不包含在调用计划或音频会议预订。 如果不设置通信贷记帐和分派给您的用户的**通信贷**许可证并且运行出组织分钟 （取决于您调用计划或音频会议的计划在您的国家/地区），这些用户将无法进行呼叫或从音频会议的会议拨出。 可以获取详细信息，包括建议融资金额，通过读取[通信片尾是什么？](what-are-communications-credits.md)
  
> [!NOTE]
> 若要找出多少它的成本，[看到这里的速率](https://go.microsoft.com/fwlink/p/?LinkId=799523 )。 
  
## <a name="step-1-assign-an-audio-conferencing-and-calling-plan-license-to-your-users"></a>步骤 1： 将音频会议和调用规划许可证分配给用户

如果您注册，具体取决于您的国家/地区获得一定数量的分钟。 您可以看到您将获得搜索的国家/地区或地区 [此处] 的分钟数。(../ country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md））。 在使用这一时间之后，调用将被断开。 若要防止这种情况发生，您需要设置通信贷。
  
为此，请向您的用户**需要将音频会议或电话系统许可分配**。
  
- 将**音频会议**许可证分配给用户。 请参阅[指派的 Skype 业务和 Microsoft 小组的许可证](assign-skype-for-business-and-microsoft-teams-licenses.md)。
    
    分配此许可证后，您将需要设置音频会议。 有关分步说明，请参阅[设置用于商业和 Microsoft 小组 Skype 的音频会议](../audio-conferencing-in-office-365/set-up-audio-conferencing.md)。
    
- 将**电话系统**和国内或国内和国际调用规划许可证分配给用户。 请参阅[指派的 Skype 业务和 Microsoft 小组的许可证](assign-skype-for-business-and-microsoft-teams-licenses.md)。
    
    > [!NOTE]
    > 它具有不需要的通信片尾，虽然仍需要还将分配**计划调用国内**或**国内，并调用计划国际**许可证。
  
    在分配这些许可证后，需要为你的组织获取电话号码，然后将这些号码分配给组织中的人。 有关分步说明，请参阅[设置调用计划](../what-are-calling-plans-in-office-365/set-up-calling-plans.md)。
    
有关详细信息，请参阅[附加业务和 Microsoft 小组授权的 Skype](skype-for-business-and-microsoft-teams-add-on-licensing.md)
  
## <a name="step-2-set-up-communications-credits-for-your-organization"></a>第 2 步： 为您的组织设置通信片尾

1. 使用你的工作或学校帐户登录 Office 365。
    
2. 在 Office 365 管理中心左侧导航，请转到**收费** > **订阅** > **加载项** > **购买的加载项**，然后选择**通信信用** > **立即购买**。
    
3. 在**通信片尾**订阅页中，填写您的信息，，然后单击**下一步**:
    
  - **添加资金**输入您想要添加到您的帐户的金额。 如果不启用自动充电的这些资金都用完时，调用启用信用卡通信的能力将会中断 （如入站免费电话服务）。 为了避免手动补充您的余额达到 0 （零） 每次通信的贷方余额，我们建议您启用自动充电功能。
    
  - **自动充值**：启用自动充值将在余额低于你设置的阈值时自动补充你的帐户余额。
    
    建议使用**自动充电**设置以避免任何中断服务的通信的贷方余额应达到 0 （零）。 充电交易成功，充电事务失败 （例如，信用卡已过期），以及通信的贷方余额达到 0 （零） 时，您将会发送一封电子邮件。
    
  - **为充电量**您想添加到您的帐户，下面的触发器量到达**重新充电**框中输入的金额。
    
  - **触发器的金额**到*触发器*自动充电，**余额低于时**将使用的框中输入的金额。 一旦您的余额低于这个数量，将自动将充电量添加到您的帐户。

      > [!NOTE]
    > 基金将仅可用于在 Microsoft 通讯信用发布率使用服务时。 在 12 个月的购买日期内不使用任何资金将过期并被没收。 
    
4. 输入您的付款信息并单击**下订单**。
    >[!IMPORTANT]
    >如果您不是批量许可证客户，您可以选择您企业协议编号的付款。 如果您有多个企业协议号，您将能够选择您想要使用的付款的哪些企业协议。 您会有机会指定采购订单编号相关联的企业协议编号 （如果适用）。
    
每个组织必须调用计划卷和费率，要考虑的不同用法。 您需要从您当前的服务提供商获取这种类型的使用情况数据。 公司已经在使用 Skype 在线业务已为其服务提供商可以通过它查看**业务管理中心的 Skype**在获得使用率数据 > **报告** > **PSTN 用法详细信息**报告。
  
设置通信片尾，需要调查的您的组织可以确定您将需要将放入的金额的调用使用。 您可以调用使用信息通过检查**PSTN 用法详细信息**报告。 此报表中，可以将呼叫数据记录导出到 Excel 中，如果您想要导出的数据存储或创建自定义报表。 若要查看使用情况，请参阅[Skype 业务 PSTN 使用情况报告](../skype-for-business-online-reporting/pstn-usage-report.md)
  
## <a name="step-3-assign-a-communications-credits-license-to-users"></a>步骤 3： 将通信片尾许可证分配给用户

1. 使用你的工作或学校帐户登录 Office 365。
    
2. 在 Office 365 管理中心左侧导航，请转到**用户** > **活动用户**，然后从列表中选择一个用户。
    
3. 在"操作"窗格中的" **产品许可证**"下，单击" **编辑**"。
    
4. 在**产品许可证**页面切换 * * 通信片尾 * * 到**上**指定此许可证，然后单击**保存**。
    
    > [!NOTE]
    > 即使有**企业 E5**许可证分配的用户，但仍建议您这样做。
