# 停止 Lync Server 2010 服务

 

_**上一次修改主题：** 2012-10-04_

您可以使用 Lync Server 控制面板启动或停止在特定计算机上运行的所有 Lync Server 2010 服务，或者启动或停止特定的 Lync Server 2010 服务。

## 启动或停止计算机上的所有 Lync Server 服务

1.  打开 Lync Server 控制面板。

2.  在左侧导航栏中，单击“拓扑”，然后单击“状态”。

3.  在“状态”页上，根据需要对列表进行排序或搜索列表，以查找正在运行要启动或停止的服务的计算机，然后单击该计算机。

4.  单击“操作”。

5.  单击“启动所有服务”或“停止所有服务”。

## 启动或停止特定服务

1.  打开 Lync Server 控制面板。

2.  在左侧导航栏中，单击“拓扑”，然后单击“状态”。

3.  在“状态”页上，根据需要对列表进行排序或搜索列表，以查找正在运行要启动或停止的服务的计算机，然后单击该计算机。

4.  单击“属性”。

5.  如有必要，对服务列表进行排序，然后单击要启动或停止的服务。

6.  单击“操作”。

7.  单击“启动服务”或“停止服务”。

8.  单击“关闭”。
