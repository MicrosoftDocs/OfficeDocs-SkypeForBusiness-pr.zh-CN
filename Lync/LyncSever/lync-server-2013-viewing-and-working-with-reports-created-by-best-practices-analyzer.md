# 查看和使用由最佳做法分析器创建的报告

 

_**上一次修改主题：** 2012-06-14_

当使用最佳做法分析器扫描您的环境时，最佳做法分析器将创建用于确定有关您的部署的问题和其他信息的报告。您可以使用最佳做法分析器查看包含扫描结果的报告并了解报告中确定的问题。
