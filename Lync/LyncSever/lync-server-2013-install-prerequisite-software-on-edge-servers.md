# 在 Lync Server 2013 边缘服务器上安装必备软件

 

_**上一次修改主题：** 2012-09-08_

在安装 Lync Server 2013 之前，需要在部署的每台边缘服务器上部署必备软件。其中包括在满足系统要求的服务器上安装操作系统。有关系统要求的详细信息，包括支持的操作系统，请参阅 [Lync Server 2013 的外部用户访问组件的系统要求](lync-server-2013-system-requirements-for-external-user-access-components.md)。
