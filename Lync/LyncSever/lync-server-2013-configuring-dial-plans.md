# 在 Lync Server 2013 中配置拨号计划

 

_**上一次修改主题：** 2013-02-22_

Lync Server 2013 拨号计划是一组指定的规范化规则，可转换指定位置、单个用户或联系人对象的电话号码，以进行电话授权和呼叫路由。
