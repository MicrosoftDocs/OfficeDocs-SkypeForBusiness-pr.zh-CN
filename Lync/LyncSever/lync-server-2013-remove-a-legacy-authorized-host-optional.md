# 在 Lync Server 2013 中删除旧版授权主机（可选）

 

_**上一次修改主题：** 2012-09-05_

将远程呼叫控制迁移到 Lync Server 部署时，您必须删除旧版部署中所有 SIP/CSTA 网关的旧版授权主机条目（在 Lync Server 中称为 *受信任应用程序条目* ）。您必须使用管理工具来分别删除这些部署中的授权主机条目。
