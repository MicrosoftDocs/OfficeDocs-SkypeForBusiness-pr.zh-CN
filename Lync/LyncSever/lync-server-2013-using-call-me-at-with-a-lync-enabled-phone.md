# 将“通过以下方式呼叫我”功能与启用 Lync 的电话一起使用

 

_**上一次修改主题：** 2013-08-09_

Lync 中的“通过以下方式呼叫我”功能向用户提供了一种方法来使用移动电话、“有线电话”或连接到公用电话交换网 (PSTN) 的其他设备加入会议的音频部分。当您尝试使用 Lync 加入会议时，系统通常会向您显示“加入会议音频”对话框：

![“使用 Lync 加入会议音频”窗口的屏幕截图](images/Dn383570.e28f17f0-9f17-44ef-b893-f4ef132f47ac(OCS.15).png "“使用 Lync 加入会议音频”窗口的屏幕截图")

如果选择“通过以下方式呼叫我”，您可以从下拉列表中选择电话号码。Lync Server 2013 将向所选号码发出电话呼叫，您可以使用该电话参与会议的音频部分。

下拉列表将填充您在“Lync – 选项”对话框的“电话”选项卡上输入的电话号码（移动电话、家庭电话或其他电话）：

![Lync“电话设置选项”的屏幕截图](images/Dn383570.03d2f25d-49e2-47b4-b1e9-b1614fc0c11c(OCS.15).png "Lync“电话设置选项”的屏幕截图")

如果您尚未在“电话”选项卡上输入任何电话号码，那么您在下拉框中没有任何号码可用。但是，您始终可以单击“新号码”并让 Lync Server 拨打您希望的任何电话号码：

![Lync“加入会议音频、呼叫我”窗口的屏幕截图](images/Dn383570.27f2ac7a-cc1c-465c-b145-202ad03af4f2(OCS.15).png "Lync“加入会议音频、呼叫我”窗口的屏幕截图")

如果您按预期方式使用“通过以下方式呼叫我”功能，那么该功能表现非常出色：让 Lync Server 呼叫 PSTN 电话号码。但是，如果您让 Lync Server 通过启用 Lync 的终结点（例如，会议室中的电话）呼叫您，那么您可能遭遇不太好的体验。下面介绍了您可能遇到的问题和解决该问题的两种方法。

首先，这里介绍了当您向启用 Lync 的电话的电话号码提供“通过以下方式呼叫我”功能时将会发生的情况。假设 Ken Myer 让 Lync Server 通过 1-206-555-1219（启用 Lync Server 的电话号码）呼叫他来尝试加入会议。Ken 最初将连接到会议，但是数秒后，Lync 将显示消息“呼叫未完成或已结束”，Ken 将显示为已从会议断开：

![Lync 呼叫会议窗口的屏幕截图](images/Dn383570.c2a81727-8751-41b5-946a-03a1b75b9d95(OCS.15).png "Lync 呼叫会议窗口的屏幕截图")

但是请注意，Lync 对话窗口中存在一些差异。Ken Myer 是“参与者”标题下列出的唯一名称。但是，如果您查看窗口的标题栏，将看到该会议总共包含 4 个参与者。

同样，如果您查看任何其他会议参与者的对话窗口，也不会看到任何位置列出 Ken Myer：

![Lync“参与者列表”窗口的屏幕截图](images/Dn383570.fa5990cf-2694-402c-ac06-946aa66b6837(OCS.15).png "Lync“参与者列表”窗口的屏幕截图")

同时，Ken Myer 能够使用其启用 Lync 的电话参与呼叫的音频部分。“通过以下方式呼叫我”功能实际上已发挥效用，但是用户体验不佳。

至少有两种方法可解决此问题。如果您已经通过这种方法加入会议（就像 Ken Myer 那样），您通常可以通过采用不同形式来解决问题。例如，如果您发送即时消息，对话窗口现在将显示所有会议参与者（包括您自己）：

![Lync“对话和参与者列表”的屏幕截图](images/Dn383570.9b5ff6d6-9f73-467c-99a7-ef3aa8bd7e7a(OCS.15).png "Lync“对话和参与者列表”的屏幕截图")

此时，您应能够按预期方式参与会议。

或者，您可以通过更改加入会议的方式来避免此问题。如果您需要让 Lync Server 呼叫启用 Lync Server 的电话，那么您应该首先加入会议但不要连接音频。为此，当出现“加入会议音频”对话框时，请选择“不加入音频”：

![Lync“不加入会议音频”窗口的屏幕截图](images/Dn383570.280a148d-cce5-4b02-87f9-9f78f17a81c1(OCS.15).png "Lync“不加入会议音频”窗口的屏幕截图")

成功连接到会议后，您现在可以“邀请”启用 Lync Server 的电话加入会议。为此，请将鼠标置于“人员”图标上方，然后单击“邀请更多人”：

![Lync“邀请更多参与者”窗口的屏幕截图](images/Dn383570.69b81b29-d1d2-4ed3-acb6-e37dd18e3d86(OCS.15).png "Lync“邀请更多参与者”窗口的屏幕截图")

此时将显示“发送 IM”对话框。请忽略对话框标题（您不会实际发送即时消息）并键入启用 Lync 的电话的电话号码：

![“发送即时消息”对话框的屏幕截图](images/Dn383570.cd67a3f0-06d8-41ba-a808-c067f64bec9f(OCS.15).png "“发送即时消息”对话框的屏幕截图")

单击“确定”后，Lync Server 将呼叫在对话框中输入的号码。建立连接后，您将通过启用 Lync 的电话使用完整的音频功能，能够查看完整的会议名单并与其交互。
