# 在 Lync Server 2013 中使用 Windows PowerShell Cmdlet 排查持久聊天服务器配置故障

 

_**上一次修改主题：** 2012-10-06_

使用以下解决方法可通过 Windows PowerShell 命令行接口 cmdlet 解决 持久聊天服务器配置问题。

  - 如果出现“主体未设置”错误消息，则说明您尝试添加的主体不可用。
    
    > [!TIP]
    > 解决方法 – 无解决方法。此错误消息表示主体不可用。
