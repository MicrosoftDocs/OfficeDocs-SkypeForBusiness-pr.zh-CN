# 在 Lync Server 2013 中规划和部署统一联系人存储

 

_**上一次修改主题：** 2012-06-14_

统一联系人存储是一项功能，它提供跨 Microsoft Office 产品的一致的联系人体验。利用此功能，用户可以将所有联系人信息存储在 Exchange 2013 中，以便能通过 Lync、 Exchange、Outlook 和 Outlook Web Access 全局访问这些信息。
