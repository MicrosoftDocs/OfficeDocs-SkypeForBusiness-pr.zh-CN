# 在 Lync Server 2013 中拓扑生成器支持 SQL Server 镜像

 

_**上一次修改主题：** 2012-09-26_

Lync Server 2013 支持后端服务器镜像以实现高可用性。您可以使用 拓扑生成器设置镜像，包括设置见证。
