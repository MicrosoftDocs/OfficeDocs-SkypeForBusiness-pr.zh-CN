# 为 Lync Server 2013 准备 Active Directory 域服务

 

_**上一次修改主题：** 2013-02-21_

在部署和运行 Lync Server 2013 之前，必须通过扩展架构然后创建并配置对象来准备 Active Directory 域服务。架构扩展会添加 Lync Server 所需的 Active Directory 类和属性。

本节中的主题介绍如何准备 AD DS 以部署 Lync Server，以及如何分配设置和组织单位 (OU) 权限。有关 Lync Server 所需的架构更改的详细信息，请参阅 [Lync Server 2013 使用的 Active Directory 架构扩展、类和属性](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md)。
