# 部署 Lync 2013 的联机会议加载项

 

_**上一次修改主题：** 2012-09-30_

部署 Lync 2013 联机会议外接程序 以便用户可以安排支持电话拨入式会议的会议。

安装 Lync 2013 时会自动安装 Lync 2013 联机会议外接程序。有关自定义 Lync 的 Lync 2013 联机会议外接程序 的详细信息，请参阅 [配置会议邀请](lync-server-2013-configuring-the-meeting-invitation.md)。
