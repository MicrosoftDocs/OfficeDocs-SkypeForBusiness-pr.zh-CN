# 运行最佳做法分析器的先决条件

 

_**上一次修改主题：** 2012-06-25_

可以使用最佳做法分析器来扫描您的 Lync Server 2013 环境中的问题并创建报告，以及查看当前或之前扫描的结果。在运行最佳做法分析器扫描环境之前，您需要确认计划用于运行扫描的帐户具有必需的权限，并且满足所有安装先决条件。
