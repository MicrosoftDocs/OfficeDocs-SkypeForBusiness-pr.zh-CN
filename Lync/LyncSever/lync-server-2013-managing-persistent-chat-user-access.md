# 在 Lync Server 2013 中管理持久聊天用户访问

 

_**上一次修改主题：** 2012-10-06_

以下主题介绍如何使用 Lync Server 2013 控制面板和“持久聊天”页管理持久聊天中的用户访问。若要管理持久聊天服务器，您必须具有系统上的持久聊天管理员权限。安装持久聊天服务器时，可指定一个或多个具有必需的持久聊天管理员权限的用户。若要向其他用户帐户授予管理员权限，请使用“管理用户和用户组”。有关详细信息，请参阅[允许用户在 Lync Server 2013 中管理类别、聊天室和用户权限](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md)。
