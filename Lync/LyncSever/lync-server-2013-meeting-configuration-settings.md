# 会议配置设置

 

_**上一次修改主题：** 2012-11-01_

在 Lync Server 2013 中，会议策略定义用户计划和参与体验，而位于会议配置页上的与会设置定义如下内容：

  - 从公用电话交换网 (PSTN) 拨入的用户是否进入会议厅

  - 谁能成为演示者

  - 默认情况下是否分配会议类型

  - 默认情况下是否允许匿名（未经身份验证）用户加入会议

本节中的主题介绍如何配置与会设置。
