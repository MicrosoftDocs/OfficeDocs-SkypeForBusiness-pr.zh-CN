# （可选）在 Lync Server 2013 中验证通知部署

 

_**上一次修改主题：** 2013-02-25_

安装并配置通知后，需要验证该配置以确保可按预期呼叫未分配的号码。至少必须验证以下内容：

  - 呼叫可供组织使用但未分配的号码。

  - 呼叫未分配的号码并验证是否播放正确的通知。
