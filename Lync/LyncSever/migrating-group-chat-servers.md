# 迁移群聊服务器

 

_**上一次修改主题：** 2012-09-29_

如果已在旧版 Lync Server 2010 环境中部署 群聊服务器，则您必须部署 Lync Server 2013持久聊天服务器。 群聊服务器和 持久聊天服务器可以共存，但内容（如聊天室等）并不在这些服务器间共享。若要从 持久聊天服务器中访问旧版 群聊服务器内容，您必须将 群聊服务器迁移至 持久聊天服务器。有关迁移至 持久聊天服务器的详细信息，请参阅迁移文档中的 [从 Lync Server 2010 群聊或 Office Communications Server 2007 R2 群聊迁移到 Lync Server 2013 持久聊天服务器](migration-from-lync-server-2010-group-chat-or-office-communications-server-2007-r2-group-chat-to-lync-server-2013-persistent-chat-server.md)。
