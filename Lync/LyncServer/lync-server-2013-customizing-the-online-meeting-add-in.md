# 自定义联机会议外接程序

 

_**上一次修改主题：** 2012-06-28_

Lync 2013 联机会议外接程序支持从 Outlook 消息传送和协作客户端内管理会议。有几种可以自定义外接程序行为的办法。其中一些是新选项，用来将徽标和文本添加到联机会议邀请正文中。
