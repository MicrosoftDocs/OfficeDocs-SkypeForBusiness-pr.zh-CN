# <a name="reviewing-the-summary-report-in-lync-server-2013"></a>在 Lync Server 2013 中查看摘要报告

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-21_

Lync Server 管理员报告是详细记录设计的四个有价值的报告中的第一个报告。此报告和其他三个相关报告中的信息是信息技术团队的优秀文档：

![常规摘要管理报告](images/Gg558626.9c529ef7-cb1b-4ce1-a8bc-3ec79aba2377(OCS.15).jpg "常规摘要管理报告")

  - 证书报告

  - 防火墙报告

  - DNS 报告

摘要报告列出了与边缘网络关联的常规配置信息。 将记录位置、完全限定的域名 (FQDN) 和 IP 地址、网络类型以及特定于指定角色的注释。

设计人员和每个要部署、管理和维护基础结构的团队应检查摘要报告的准确性，并确保至少有错误。
