# <a name="configure-dial-in-conferencing-personal-identification-number-pin-rules-in-lync-server-2013"></a>在 Lync Server 2013 中配置电话拨入式会议个人识别号（PIN）规则

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-06-19_

在你的组织中拥有 Active Directory 域服务（AD DS）凭据的 Lync Server 2013 用户可以使用个人识别码（PIN）将电话拨入式会议作为已验证用户加入电话拨入式会议。 PIN 策略定义电话拨入式会议 PIN 工作方式的规则。

如果要将特定策略应用于某个站点或某个用户组，可以创建新的 PIN 策略。 如果要在整个组织中使用相同的 PIN 策略，则可以使用全局 PIN 策略并根据需要对其进行修改。 PIN 策略既可以应用于少数用户，也可以应用于众多用户。 如果为用户分配用户级别的 PIN 策略，则优先应用这些设置。 如果没有分配用户策略，则将应用站点级别的 PIN 策略（如果存在）。 如果未应用用户策略或站点策略，则全局 PIN 策略会提供默认设置。
