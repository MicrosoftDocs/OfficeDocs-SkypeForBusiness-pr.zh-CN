# <a name="optional-verify-the-group-call-pickup-deployment-in-lync-server-2013"></a>Optional在 Lync Server 2013 中验证组内的 "呼叫装货" 部署

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-01-30_

为用户启用组呼叫应答后，验证配置是否按预期运行。 至少应验证以下各项：

  - 呼叫启用了组间电话装货的用户，并让另一个用户检索该呼叫。 其他用户可以在同一组中，也可以位于不同的组中，或者不启用组内呼叫应答。

  - 呼叫启用了组间电话装货的用户，但不应答呼叫。
