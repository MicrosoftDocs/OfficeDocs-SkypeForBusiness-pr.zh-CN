</div>

<div id="mainSection">

_**主题上次修改时间：** 2012-06-20_

将企业语音设置为 Microsoft Lync Server 2013 实现的一部分包括创建必须全部协同工作的路由、策略和规则，以确保传入和传出呼叫按预期完成。 Lync Server Management Shell 包含可用于测试连接和路径的 cmdlet，以及用于解决在实现期间可能出现的问题。
