# <a name="planning-for-monitoring-in-lync-server-2013"></a>在 Lync Server 2013 中规划监视

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-09-05_

Microsoft Lync Server 2013 中的监控服务为管理员提供了一种方法，用于收集组织中发生的通信会话的使用率、趋势和服务质量数据。 Lync Server 2013 中的监控不再需要单独的服务器角色;相反，监视服务内置在每个前端服务器中。 但是，默认情况下，在 Lync Server 2013 中不启用监控功能。 本文档将帮助您确定是否应在贵组织中启用该功能。
