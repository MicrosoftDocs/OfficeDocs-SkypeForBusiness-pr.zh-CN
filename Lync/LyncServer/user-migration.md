# 使用迁移

 

_**上一次修改主题：** 2012-10-19_

一种普遍认可的迁移最佳实践是，创建几个测试用户，然后利用这些用户执行系统测试。在成功移动并测试这些帐户之后，应确定一组试生产用户，移动这些用户的帐户，然后对它们执行验证测试。如果获得的结果令人满意，则可以将其余的用户移动到新部署。

有关对用户启用 Lync Server 2013 的更多信息，请参阅“部署 Lync Server 2013”文档中的[禁用或重新启用 Lync Server 的用户帐户](lync-server-2013-disable-or-re-enable-user-account-for-lync-server.md)主题。
