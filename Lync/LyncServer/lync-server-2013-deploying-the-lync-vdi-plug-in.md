# 在 Lync Server 2013 中部署 Lync VDI 插件

 

_**上一次修改主题：** 2012-10-03_

Lync 2013 客户端支持虚拟桌面基础结构 (VDI) 环境中的音频和视频。用户可以将音频或视频设备（例如，耳机或网络摄像头）连接到本地计算机（例如，瘦客户端或已调整用途的计算机）。用户可以连接到虚拟机，登录到在虚拟机上运行的 Lync 2013 客户端，并参与实时音频和视频通信，就像客户端在本地运行一样。

Lync VDI 插件是在本地计算机上安装的独立应用程序，允许使用本地音频和视频设备，同时 Lync 2013 客户端在虚拟机上运行。该插件不需要在本地计算机上安装 Lync。在用户登录到在虚拟机上运行的 Lync 2013 客户端后， Lync 将提示用户重新输入其凭据以便与在本地计算机上运行的 Lync VDI 插件建立连接。建立此连接后，用户即可发起和接收音频和视频呼叫。
