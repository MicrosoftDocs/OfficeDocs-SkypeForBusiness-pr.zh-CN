# <a name="create-response-group-workflows-in-lync-server-2013"></a>在 Lync Server 2013 中创建响应组工作流

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2012-09-12_

工作流定义了从电话响铃到有人接听电话这段时间内呼叫的行为。工作流指定用于保留呼叫的队列，并指定用于智能寻线的路由方法或用于互动响应组的问题和答案。工作流还定义了诸如欢迎消息、保持音乐、工作时间和假日等设置。

使用 "响应组配置" 工具创建工作流。 您可以从 Lync Server 控制面板的 "响应组" 页面访问 "响应组配置" 工具。

<div>


> [!NOTE]  
> 必须先创建代理组和队列，然后再创建使用这些组和队列的工作流。
