# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>通过 System Center Operations Manager 监视 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2015-05-06_

Lync Server 管理包 (MP) 是监视任何 Lync Server 部署的首选监视解决方案。

MP 实现传统的事件日志和基于性能计数器的检测, 并在 Lync Server 中启用新可用的检测, 如几个关键运行状况指示器的配对事件 (失败/成功), 同时还完全实现新的合成事务 (Test-Cs\* Windows PowerShell cmdlet)。

您可以在上[http://go.microsoft.com/fwlink/p/?LinkId=400468](http://go.microsoft.com/fwlink/p/?linkid=400468)找到 Lync Server 2013 管理包及其相关文档。 如果你运行的是 System Center Operations Manager 2012, 则建议这样做。
