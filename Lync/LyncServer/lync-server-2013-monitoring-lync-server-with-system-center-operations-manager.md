# <a name="monitoring-lync-server-2013-with-system-center-operations-manager"></a>使用 System Center Operations Manager 监视 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2015-05-06_

Lync Server 管理包（MP）是监视任何 Lync Server 部署的选择的监视解决方案。

MP 实现传统的事件日志和基于性能计数器的检测，并在 Lync Server 中启用新可用的规范，例如多个关键运行状况指示器的配对事件（故障/成功），同时还完全实现新的合成事务\* （Test-Cs Windows PowerShell cmdlet）。

您可以在[https://go.microsoft.com/fwlink/p/?LinkId=400468](https://go.microsoft.com/fwlink/p/?linkid=400468)中找到 Lync Server 2013 管理包及其相关文档。 如果您运行的是 System Center Operations Manager 2012，建议这样做。
