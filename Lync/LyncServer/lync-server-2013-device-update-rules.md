# <a name="device-update-rules-in-lync-server-2013"></a>Lync Server 2013 中的设备更新规则

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-20_

Microsoft 会定期为 Lync Phone Edition 发布一组新的设备固件更新。 *设备更新规则*将固件更新与硬件设备（电话和其他运行 Lync Phone Edition 的设备）相关联。

若要获取最新的设备更新规则集，请转到 Microsoft 网站上的 "帮助和支持" 页，并搜索 "Phone Edition"。 下载更新程序包，并将文件解压缩到要上载更新的计算机上的某个文件夹中。 提取文件后，导入在提取的中找到的设备更新规则。CAB 文件（其名称为为 ucupdates-r2.）。 然后，使用 Lync Server 控制面板或 Windows PowerShell cmdlet 查看和管理组织设备的这些规则。

以下主题介绍如何导入、查看和管理设备更新规则。
