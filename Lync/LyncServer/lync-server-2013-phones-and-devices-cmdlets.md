Microsoft Lync Server 2013 提供了许多 cmdlet，使您能够管理电话和其他硬件设备。 这包括诸如 IP 语音 (VoIP) 电话之类的内容;常见的区域电话 (例如，在大楼大厅、自助版或其他公共场所) 的电话;甚至模拟电话，不能运行 Lync Phone Edition 的电话。
