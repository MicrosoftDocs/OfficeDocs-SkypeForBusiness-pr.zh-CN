# <a name="determining-your-system-requirements-for-lync-server-2013"></a>确定 Lync Server 2013 的系统要求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2014-01-02_

运行 Lync Server 的所有服务器必须满足一定的最低系统要求。 Lync Server 的系统要求包括服务器硬件、要在每台服务器上安装的操作系统以及相关软件要求 (如 Windows 更新和必须安装在服务器上的其他软件)。

<div>


> [!IMPORTANT]  
> Lync Server 仅在64位版本中可用, 这需要64位硬件和一个64位的 Windows Server 版本。 例外情况是 Microsoft Lync Server 2013、计划工具, 可在32位版本中使用。



</div>

<div>

> 有关 Active Directory 支持、支持的拓扑、服务器 collocation 和其他支持问题的详细信息, 请参阅<A href="lync-server-2013-supportability.md">Lync server 2013 的支持支持</A>。



</div>
