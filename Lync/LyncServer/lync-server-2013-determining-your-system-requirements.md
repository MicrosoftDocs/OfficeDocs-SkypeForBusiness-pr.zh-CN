# <a name="determining-your-system-requirements-for-lync-server-2013"></a>确定 Lync Server 2013 的系统要求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2014-01-02_

运行 Lync Server 的所有服务器都必须满足特定的最低系统要求。 Lync Server 的系统要求包括服务器硬件、要在每台服务器上安装的操作系统以及相关的软件要求（如 Windows 更新和必须安装在服务器上的其他软件）。

<div>


> [!IMPORTANT]  
> Lync Server 仅可用于64位版本，这需要64位硬件和一个64位版本的 Windows Server。 例外是 Microsoft Lync Server 2013，即规划工具，可在32位版本中使用。



</div>

<div>


> [!NOTE]  
> 有关 Active Directory 支持、支持的拓扑、服务器并置和其他可支持性问题的详细信息，请参阅<A href="lync-server-2013-supportability.md">Lync server 2013 的可支持</A>性。



</div>
