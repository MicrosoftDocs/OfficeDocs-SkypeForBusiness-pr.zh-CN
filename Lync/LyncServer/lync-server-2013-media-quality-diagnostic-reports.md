# Lync Server 2013 中的媒体质量诊断报告

 

_**上一次修改主题：** 2013-02-22_

媒体质量诊断报告提供有关呼叫质量的信息以及失败的呼叫的诊断和疑难解答信息。

## 本部分内容

  - [Lync Server 2013 中的媒体质量摘要报告](lync-server-2013-media-quality-summary-report.md) 提供不同终结点类型（包括企业语音点对点呼叫、企业语音会议呼叫和依赖（至少部分依赖）公用电话交换网 (PSTN) 的呼叫）的整体质量数据。

  - [媒体质量比较报告](lync-server-2013-media-quality-comparison-report.md) 为不同类型的音频呼叫提供呼叫质量值的比较（例如，通过无线网络进行的呼叫与通过有线连接进行的呼叫）。

  - [Lync Server 2013 中的服务器性能报告](lync-server-2013-server-performance-report.md) 列出遇到最多问题的服务器，由测量诸如降级、数据包丢失和抖动的关键质量指标得出。

  - [Lync Server 2013 中的位置报告](lync-server-2013-location-report.md) 提供网络位置的列表和在每个位置发生的呼叫的媒体质量的摘要。对于该报告，位置是基于 IP 子网的。

  - [Lync Server 2013 中的设备报告](lync-server-2013-device-report.md) 提供用于企业语音呼叫的设备的摘要，它包含按设备列出的呼叫的平均媒体质量。

  - [Lync Server 2013 中的呼叫列表报告](lync-server-2013-call-list-report.md) 提供有关在贵组织中拨打或接收的电话呼叫的详细信息。

  - [Lync Server 2013 中的呼叫详情报告](lync-server-2013-call-detail-report.md) 提供有关在贵组织中拨打或接收的电话呼叫的详细信息。

  - [服务器媒体质量趋势报告](lync-server-2013-server-media-quality-trend-report.md)   为您提供一种方法来以图形方式比较多达 5 台服务器的用户体验质量指标（如呼叫量、质量欠佳的呼叫百分比、数据包丢失和抖动）。

  - [媒体质量标准分发报告](lync-server-2013-media-quality-metrics-distribution-report.md)   提供显示用户体验质量指标（如抖动或数据包丢失）分布值的图形。

  - [位置趋势报告](lync-server-2013-location-trend-report.md)   提供网络位置的呼叫质量趋势信息。
