# Web 服务器和服务 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

许多 Microsoft Lync Server 2013 组件都基于 Web：这些组件使用 Web 服务或网页来完成其任务。通过 Web 服务器和 Web 服务 cmdlet，您可以执行配置 Web 服务器设置等操作，还可以管理简单 URL。简单 URL 既便于用户参加会议，又便于管理员登录到 Lync Server 2013 控制面板。
