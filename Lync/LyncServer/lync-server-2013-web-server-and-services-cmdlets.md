# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的 Web 服务器和服务 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-18_

许多 Microsoft Lync Server 2013 组件都是基于 web 的：这些组件可以使用 Web 服务或网页来执行任务。 通过 Web 服务器和 Web 服务 cmdlet，您可以执行配置 Web 服务器设置等操作，还可以管理简单 URL。 简单 Url 使用户可以更轻松地加入会议和会议，并使管理员能够更轻松地登录到 Lync Server 2013 控制面板。
