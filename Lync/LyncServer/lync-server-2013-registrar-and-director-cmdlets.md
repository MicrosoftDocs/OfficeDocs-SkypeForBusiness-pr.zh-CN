# 注册器和控制器 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

注册器和控制器用于对登录请求进行身份验证并维护有关用户状态和可用性的信息。通过注册器和控制器 cmdlet，可以管理这些服务器的配置设置。
