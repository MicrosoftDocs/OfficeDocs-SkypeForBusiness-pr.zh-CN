即时消息 (IM) 和状态始终在 Enterprise Edition 前端池和 Standard Edition 服务器上运行。 有关受支持的硬件、操作系统和数据库软件的信息, 请参阅以下内容:

## <a name="supported-collocation"></a>支持的 Collocation

前端服务器角色可以与中介服务器 collocated。 您也可以在前端服务器上运行监视和存档。 前端服务器无法与 Edge 服务器或 Director collocated。
