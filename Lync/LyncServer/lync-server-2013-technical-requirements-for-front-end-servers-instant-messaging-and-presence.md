# Lync Server 2013 中的前端服务器、即时消息和状态的技术要求

 

_**上一次修改主题：** 2012-09-18_

即时消息 (IM) 和状态设置始终在 企业版前端池和 标准版 服务器上运行。有关受支持的硬件、操作系统和数据库软件的信息，请参阅以下内容：

  - [支持的适用于 Lync Server 2013 的硬件](lync-server-2013-supported-hardware.md)

  - [Lync Server 2013 中的服务器软件和基础结构支持](lync-server-2013-server-software-and-infrastructure-support.md)

## 支持的并置

前端服务器角色可以与中介服务器并置。您还可以在前端服务器上运行监控和存档功能。前端服务器不能与边缘服务器或控制器并置。
