# 新的组内呼叫应答功能

 

_**上一次修改主题：** 2013-02-12_

Lync Server 2013 累积更新 2013 年 2 月版引入了组内呼叫应答作为新的企业语音功能。使用组内呼叫应答，可以将用户分配到组，以便其他用户可以从自己的电话应答传入到组内的用户的呼叫。

组内呼叫应答基于呼叫寄存应用程序。对于组内呼叫应答，与呼叫寄存一样，您设置要用作组号码的虚拟号码范围。用户拨打组号码来应答为组内另一位用户响铃的呼叫。
