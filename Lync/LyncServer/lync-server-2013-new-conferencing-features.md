    <span data-ttu-id="42c71-106">Lync Server 2013 更新联接启动器, 以便在启动客户端之前验证每个会议, 并为在以下客户端中打开会议提供支持:</span><span class="sxs-lookup"><span data-stu-id="42c71-106">Lync Server 2013 updates the Join launcher to validate each meeting before launching a client, and to provide support for opening a meeting in the following clients:</span></span>

      - <span data-ttu-id="42c71-108">Android 设备</span><span class="sxs-lookup"><span data-stu-id="42c71-108">Android devices</span></span>

      - <span data-ttu-id="42c71-110">Windows 8</span><span class="sxs-lookup"><span data-stu-id="42c71-110">Windows 8</span></span>
    
      - <span data-ttu-id="42c71-111">Internet Explorer 10</span><span class="sxs-lookup"><span data-stu-id="42c71-111">Internet Explorer 10</span></span>

  - <span data-ttu-id="42c71-112">**更新的 PowerPoint 共享**</span><span class="sxs-lookup"><span data-stu-id="42c71-112">**Updated PowerPoint Sharing**</span></span>
    
    <span data-ttu-id="42c71-113">Lync Server 2013 现在使用 Office Web Apps 和 Office Web Apps 服务器 (以前称为 WAC Server) 处理 PowerPoint 演示文稿。</span><span class="sxs-lookup"><span data-stu-id="42c71-113">Lync Server 2013 now uses Office Web Apps and the Office Web Apps Server (formerly known as WAC Server) to handle PowerPoint presentations.</span></span> <span data-ttu-id="42c71-114">Office Web Apps 服务器的使用允许高分辨率显示和更好地支持 PowerPoint 功能、访问更多类型的移动设备 (Lync Server 2013 使用标准 DHTML 和 JavaScript 广播 PowerPoint 演示文稿), 以及具有相应权限的用户能够在与演示文稿本身无关的 PowerPoint 演示文稿之间滚动。</span><span class="sxs-lookup"><span data-stu-id="42c71-114">The use of Office Web Apps Server allows for higher-resolution displays and better support for PowerPoint capabilities, access to more types of mobile devices (Lync Server 2013 uses standard DHTML and JavaScript to broadcast PowerPoint presentations), and the ability for users with the appropriate privileges to scroll through a PowerPoint presentation independent of the presentation itself.</span></span>

  - <span data-ttu-id="42c71-115">**库视图和 HD 视频会议**</span><span class="sxs-lookup"><span data-stu-id="42c71-115">**Gallery View and HD Video Conferencing**</span></span>
    
    <span data-ttu-id="42c71-116">在视频会议中, 用户可以同时查看多达五个会议参与者的视频。</span><span class="sxs-lookup"><span data-stu-id="42c71-116">In video conferences, users can see videos of up to five conference participants at the same time.</span></span>

    </div>

  - <span data-ttu-id="42c71-119">**HD 视频**</span><span class="sxs-lookup"><span data-stu-id="42c71-119">**HD Video**</span></span>
    
    <span data-ttu-id="42c71-120">用户可以在两方呼叫和多方会议中体验高达 HD 1080P 的分辨率。</span><span class="sxs-lookup"><span data-stu-id="42c71-120">Users can experience resolutions up to HD 1080P in two-party calls and multiparty conferences.</span></span>

  - <span data-ttu-id="42c71-121">**仅演示者视频模式**</span><span class="sxs-lookup"><span data-stu-id="42c71-121">**Presenter Only Video Mode**</span></span>

  - <span data-ttu-id="42c71-125">**视频聚焦**</span><span class="sxs-lookup"><span data-stu-id="42c71-125">**Video Spotlight**</span></span>
    
    <span data-ttu-id="42c71-126">演示者可以配置会议, 以便会议中的每个人只能看到来自所选参与者的视频。</span><span class="sxs-lookup"><span data-stu-id="42c71-126">Presenters can configure the conference so that only the video from a selected participant who is a video source is seen by everyone in the conference.</span></span> <span data-ttu-id="42c71-127">此模式还适用于通过全景视频的会议设备捕获和提供的视频。</span><span class="sxs-lookup"><span data-stu-id="42c71-127">This mode also applies to video captured and provided by conferencing devices for panoramic video.</span></span>

  - <span data-ttu-id="42c71-128">**非企业语音用户拨出式会议**</span><span class="sxs-lookup"><span data-stu-id="42c71-128">**Dial-out Conferencing for non-Enterprise Voice users**</span></span>
    
    <span data-ttu-id="42c71-129">Lync Server 2013 现允许未启用企业语音的参与者从会议会议启动拨出呼叫。</span><span class="sxs-lookup"><span data-stu-id="42c71-129">Lync Server 2013 now allows participants that are not Enterprise Voice enabled to initiate dial-out calls from a meeting conference.</span></span> <span data-ttu-id="42c71-130">管理员可对此功能进行配置。</span><span class="sxs-lookup"><span data-stu-id="42c71-130">This feature is configurable by the administrator.</span></span>

  - <span data-ttu-id="42c71-131">**存档**</span><span class="sxs-lookup"><span data-stu-id="42c71-131">**Archiving**</span></span>
    
    <span data-ttu-id="42c71-132">如果 Exchange Server 集成已启用存档, 则在会议期间共享的任何文档将存档到 Exchange 2013 数据存储中。</span><span class="sxs-lookup"><span data-stu-id="42c71-132">Any document that is shared during a conference is archived into Exchange 2013 data storage if Exchange Server integration is enabled with Archiving.</span></span> <span data-ttu-id="42c71-133">这包括 PowerPoint 演示文稿、附件、白板和投票。</span><span class="sxs-lookup"><span data-stu-id="42c71-133">This includes PowerPoint presentations, attachments, whiteboards and polls.</span></span>
