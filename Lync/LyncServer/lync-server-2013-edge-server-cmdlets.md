# <a name="edge-server-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的边缘服务器 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-10-07_

边缘服务器提供了一种将 Microsoft Lync Server 2013 的功能扩展到未登录到内部网络的用户的方法。 例如，如果您有通过 Internet 而不是通过内部网络登录到 Lync Server 2013 的远程用户-已通过身份验证的用户-您将需要设置运行 Lync Server 访问边缘服务的边缘服务器。 此外，如果要建立与其他组织的联盟，或者要向用户授予与具有公用即时消息服务（如 Yahoo\!、AOL 或 MSN）的帐户进行通信的用户的权利，则需要边缘服务器。
