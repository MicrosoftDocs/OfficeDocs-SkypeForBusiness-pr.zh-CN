# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>Optional在 Lync Server 2013 中验证通知部署

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-25_

安装并配置通知后，需要验证该配置以确保可按预期呼叫未分配的号码。至少必须验证以下内容：

  - 拨打对您的组织有效的号码，但它是未分配的号码。

  - 呼叫未分配的号码并验证是否播放正确的通知。
