# Lync Server 2013 中的会议

 

_**上一次修改主题：** 2012-09-11_

通过 Lync Server 2013 中的统一会议，用户可以实时进行协作、共享信息和协调工作。所有用户都可以使用广泛的自发协作、预定会议和会议工具。可以从任何有 Internet 连接的位置使用语音和视频会议功能，不在计算机旁边的用户可以通过公用电话交换网 (PSTN ) 电话拨入来参加音频会议。

集成到 Outlook 中的会议工具使组织者可以通过一次单击来安排会议或启动临时会议，并方便与会者加入会议。通过 Web 客户端，没有运行桌面版 Lync 的参与者也可以使用丰富的会议功能。

## 音频和视频会议

Lync Server 提供了一种传统音频桥服务（包括使用按键呼叫控制命令的 PSTN 拨入服务）用户所熟悉的用户体验。同时，还融合了仅由集成的统一通信平台提供的强大的安排、加入和管理功能。

用户可以通过一次单击从 Outlook 安排会议。详细信息（如会议时间、位置和与会者）遵循常用的 Outlook 模板。此外，还会自动填充特定于电话会议的信息，如拨入号码、会议 ID 和个人标识号 (PIN) 提醒。

为了帮助确保只有授权的用户参加通话， Lync Server 为参与者提供了多个级别的身份验证。使用 Lync 加入会议的用户已通过 Active Directory 域服务进行身份验证，不需要输入 PIN、密码或会议 ID。

通过将视频合并到统一客户端， Lync 简化了视频会议用户体验，从而可以无缝且轻松地安排视频会议或自发升级到视频会议。

通过 Lync Server，只需一次单击即可轻松向标准电话呼叫中添加视频。当视频呼叫或会议中有多个参与者时，每个用户最多能同时观看来自五个其他用户的视频，或者演示者只能选择一个要被所有人以独占方式观看的视频源。

在高端计算机上运行 Lync 的用户之间进行的对等呼叫支持高清视频（分辨率为 1270 x 720；纵横比为 16:9）和 VGA 视频（分辨率为 640 x 480；纵横比为 4:3）。单次对话中每个参与者所查看的分辨率可能有所不同，具体取决于每个用户的相应硬件的视频功能。

IT 管理员可以设置策略，限制或禁止在客户端上使用高清或 VGA 视频，具体取决于计算机功能、网络带宽以及可提供所需分辨率的摄像机的状态。通过带内设置实施这些策略。

## Web 会议

Lync Server 将会议共享功能（如桌面、应用程序、附件、白板、轮询和 PowerPoint）集成到简化的 Lync 中。通过与音频或视频会议结合使用，会话变得更加生动，协作更加顺畅。

为了改善用户演示或查看 PowerPoint 演示文稿的整体体验， Lync Server 2013 使用了 Office Web Apps 来处理 PowerPoint 演示文稿。用户可使用 Lync 会议中的白板共享图片或复制并粘贴文本。演示者可在 Lync 会议中执行轮询以请求与会者的反馈。

桌面共享使演示者可以直接从 Lync 实时将任何视频、应用程序、网页、文档、软件或部分桌面内容广播到远程参与者。受众成员可以跟着一起进行鼠标移动和键盘输入。演示者可以选择共享整个屏幕或仅共享部分屏幕。通过共享桌面，演示者可以从任何位置与其受众进行产品或软件互动演示。

应用程序共享使演示者可以共享其桌面上的软件的控制权，同时还可兼顾参与者的反馈或文本问题。演示者还可以将应用程序的控制权委派给会议参与者。

## 拨入会议

不使用个人计算机的用户可以通过多种方法加入 Lync Server 电话会议。PSTN 用户可以拨打访问号码，访问会议桥，然后输入会议 ID。对于安全性更高的会议，还可以要求用户输入其 PIN 以针对 Active Directory 进行身份验证。 Lync Server 还支持 Lync Phone Edition 设备，这种设备是由 Microsoft 合作伙伴提供的独立 IP 电话设备。
