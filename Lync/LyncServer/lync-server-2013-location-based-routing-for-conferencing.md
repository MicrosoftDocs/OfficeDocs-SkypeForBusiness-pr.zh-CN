# <a name="location-based-routing-for-conferencing-in-lync-server-2013"></a>Lync Server 2013 中的会议基于位置的路由

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2013-07-31_

基于位置的路由使您能够根据呼叫中各方的位置限制 VoIP 终结点和 PSTN 终结点之间的通话路由。 使用 Lync Server 2013 的累积更新2，基于位置的路由规则可在 Lync 会议（即会议）上强制实施，以防止 PSTN 免绕过。 应用程序监视活动会议，并根据参与用户的位置强制执行基于位置的路由限制。 基于位置的路由会议应用程序还支持将基于位置的路由限制强制转换为涉及 PSTN 终结点的咨询式转移。
