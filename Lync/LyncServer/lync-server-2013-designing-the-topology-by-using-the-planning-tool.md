# 使用规划工具设计适用于 Lync Server 2013 的拓扑

 

_**上一次修改主题：** 2016-12-08_

Microsoft Lync Server 2013 规划工具是一个向导驱动的访谈类工具，会提问有关您正在设计的 Lync Server 2013 拓扑的问题。规划工具使用提供的信息，结合用于拓扑设计和容量的首选做法，并根据提供的答案呈现推荐的拓扑。您可以从 Microsoft 下载中心下载 规划工具，网址为 [http://go.microsoft.com/fwlink/?LinkID=282725](http://go.microsoft.com/fwlink/?linkid=282725)。

规划工具的最终目标是尽可能降低设计完整 Lync Server 2013 拓扑的复杂性。如果可以通过 Internet 连接来连接到 Microsoft TechNet 网站，该工具还会提供对工具内规划文档和部署文档的上下文引用。

使用基础结构的 TCP/IP 地址和完全限定域名 (FQDN) 自定义拓扑后， 规划工具将提供一系列报告，其中包括域名系统 (DNS) 命名、防火墙规则、证书等。

规划工具也提供以两种格式导出信息的能力：

  - Microsoft Excel

  - Microsoft Visio
