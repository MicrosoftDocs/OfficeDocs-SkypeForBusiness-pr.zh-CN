# 配置 Lync Phone Edition 的语音服务质量

 

_**上一次修改主题：** 2012-09-29_

可通过设置连接到 Lync Server 2013 的 IP 电话的服务质量 (QoS) 级别，来配置池中 Lync Phone Edition 设备的语音 QoS 要求。

## 配置 Lync Phone Edition 的语音服务质量

1.  打开浏览器窗口，然后输入管理 URL 以打开 Lync Server 控制面板。有关可以用于启动 Lync Server 控制面板的不同方法的详细信息，请参阅[打开 Lync Server 管理工具](lync-server-2013-open-lync-server-administrative-tools.md)。

2.  在左侧导航栏中，单击“客户端”，然后单击“设备配置”。

3.  在“设备配置”页上的设备配置列表中，双击要更改 QoS 设置的配置。

4.  在“编辑设备配置”中的“语音服务质量”下，指定 QoS 级别。默认级别为“40”。
