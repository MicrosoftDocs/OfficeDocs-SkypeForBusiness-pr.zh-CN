</div>

<div id="mainSection">

集中日志记录 cmdlet 为管理员提供了一种管理和配置 Microsoft Lync Server 2013 中引入的集中日志记录功能的方法。 集中日志记录允许管理员同时在多台计算机上启用或禁用事件跟踪。
