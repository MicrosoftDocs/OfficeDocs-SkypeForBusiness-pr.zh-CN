# 集中日志记录 Cmdlet

 

_**上一次修改主题：** 2012-10-08_

利用集中日志记录 cmdlet，管理员可以管理和配置 Microsoft Lync Server 2013 中引入的集中日志记录功能。集中日志记录允许管理员在多台计算机上同时启用或禁用事件跟踪。

## 集中日志记录 cmdlet

可利用集中日志记录 cmdlet 管理 Lync Server 2013 中引入的集中日志记录服务：

**集中日志记录 cmdlet**
