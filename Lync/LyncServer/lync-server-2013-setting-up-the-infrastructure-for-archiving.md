# 设置存档的基础结构

 

_**上一次修改主题：** 2012-10-01_

存档的基础结构要求与 Lync Server 部署的基础结构要求相同（存储除外）。除了设置使用 Exchange 2013 存储和/或存档数据库的存储之外，不需要额外的基础结构设置。有关 Lync Server 2013 的基础结构要求的详细信息，请参阅规划文档中的[确定 Lync Server 2013 的基础结构要求](lync-server-2013-determining-your-infrastructure-requirements.md)和部署文档中的[为 Lync Server 2013 准备基础结构和系统](lync-server-2013-preparing-the-infrastructure-and-systems.md)。有关存档的存储要求的详细信息，请参阅规划文档中的[Lync Server 2013 中的存档技术要求](lync-server-2013-technical-requirements-for-archiving.md)、部署文档中的[为存档设置系统平台](lync-server-2013-setting-up-system-platforms-for-archiving.md)以及部署文档中的[设置存档存储](lync-server-2013-setting-up-storage-for-archiving.md)。
