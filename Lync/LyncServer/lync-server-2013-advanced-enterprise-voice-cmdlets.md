# 高级企业语音 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

在组织中实现企业语音时，还可以选择实现可改善用户体验和提高组织效率的其他功能和应用程序。这些功能包括呼叫允许控制、增强型 9-1-1 (E9-1-1) 紧急呼叫和媒体配置设置。
