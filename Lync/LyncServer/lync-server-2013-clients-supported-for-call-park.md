# <a name="clients-supported-for-call-park-in-lync-server-2013"></a>Lync Server 2013 中支持呼叫寄存的客户端

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-09-13_

本节介绍可用于寄存呼叫的客户端和可用于取回寄存呼叫的客户端。

<div>

## <a name="clients-supported-for-parking-calls"></a>支持寄存呼叫的客户端

可寄存来自任意 IP、专用交换机 (PBX)、公用电话交换网 (PSTN) 或移动电话的呼叫。

<div>


> [!NOTE]  
> 只能寄存音频呼叫。不能寄存即时消息和会议。



</div>

以下客户端可以使用寄存呼叫的呼叫寄存：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

<div>


> [!NOTE]  
> 移动电话无法将呼叫寄存用于寄存呼叫。



</div>

</div>

<div>

## <a name="clients-supported-for-retrieving-calls"></a>支持取回呼叫的客户端

将通道范围配置为虚拟分机块（未分配用户或电话的分机）。将通道配置为虚拟分机时，移动电话和 PSTN 电话无法取回寄存的呼叫。

联盟用户无法取回寄存的呼叫。

以下客户端可以检索寄存在呼叫寄存上的呼叫：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

  - IP 公用区域电话

  - 连接到 Lync Server 2013 基础结构的非 IP 电话，包括公用区域电话和专用交换机（PBX）电话

</div>
