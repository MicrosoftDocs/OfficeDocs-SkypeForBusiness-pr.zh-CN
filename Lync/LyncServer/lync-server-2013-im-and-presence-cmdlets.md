# IM 和状态 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

使用即时消息 (IM) 和状态 cmdlet，您可以通过 Windows PowerShell 管理这些客户端功能。可以在 global、site 或每用户作用域设置应用于用户的状态策略。还可以配置各种隐私和 IM 功能。

## IM 和状态 Cmdlet

要配置 IM 和状态，请使用以下 cmdlet：
