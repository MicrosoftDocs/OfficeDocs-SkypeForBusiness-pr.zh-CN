# 管理响应组工作流

 

_**上一次修改主题：** 2012-10-01_

响应组工作流定义了从电话响铃到代理应答呼叫这段时间内呼叫的行为。工作流包括队列和路由信息，还包括智能寻线或互动语音响应 (IVR) 信息。

本节中的主题确定了用于设计 IVR 工作流的最佳实践，并说明了如何创建自定义的工作时间和假日集、如何创建或修改工作流以及如何删除工作组。
