# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>使用户能够在 Lync Server 2013 中管理类别、聊天室和用户权限

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-11-01_

**CsPersistentChatAdministrator**角色的成员 (持久聊天管理员) 可以向其他用户和他们自己授予聊天室管理员权限。

持久聊天管理员可以执行以下操作：

  - 创建聊天类别和聊天室。

  - 设置所有类别和聊天室的成员身份。

  - 管理所有类别和聊天室的所有设置。

  - 为持久聊天服务器启用或禁用策略。

  - 设置和管理持久聊天服务器池上的配置设置。

有关详细信息，请参阅部署文档中的在 [Lync Server 2013 中添加持久聊天管理员](lync-server-2013-adding-a-persistent-chat-administrator.md) 。
