# <a name="mcujoinsandleaves-view-in-lync-server-2013"></a>Lync Server 2013 中的 McuJoinsAndLeaves 视图

</div>

<div id="mainSection">

<div id="mainBody">

_**主题上次修改时间：** 2012-10-01_
