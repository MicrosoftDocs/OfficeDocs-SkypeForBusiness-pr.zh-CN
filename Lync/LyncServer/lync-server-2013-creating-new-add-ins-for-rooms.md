# 在 Lync Server 2013 中创建新的聊天室外接程序

 

_**上一次修改主题：** 2012-10-06_

若要为 持久聊天聊天室创建外接程序，请参阅部署文档中的 [在 Lync Server 2013 中配置聊天室外接程序](lync-server-2013-configure-add-ins-for-rooms.md)。如果您是 持久聊天管理员，则可以使用 Lync Server 控制面板或 Windows PowerShell cmdlet 创建外接程序。
