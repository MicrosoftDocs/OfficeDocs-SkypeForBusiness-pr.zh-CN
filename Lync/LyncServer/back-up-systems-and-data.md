_**上次修改的主题：** 2012-09-18_

在开始迁移到 Lync Server 2013 之前，强烈建议您执行完整的系统备份并记录现有系统，包括驻留在每个池中的用户帐户的清单，以便您可以在需要时回滚到 Lync Server 2010。 可以使用多种工具和程序来备份和还原数据、设置和系统。

有关详细信息和过程，请参阅 [备份和还原 Lync Server 2010](https://go.microsoft.com/fwlink/p/?linkid=265417) 。
