# Microsoft Lync Server 2013

 

_**上一次修改主题：** 2016-12-08_

Microsoft Lync Server 2013 通信软件可以提供即时消息 (IM)、状态、会议及电话服务解决方案，这些解决方案可以支持企业级协作要求。

想要下载 Lync Server 2013 文档副本？您可以从 Microsoft 下载中心下载 **LyncServer2013\_ITPro** 编译帮助 (.chm) 文件，网址为 [http://go.microsoft.com/fwlink/p/?LinkID=331103](http://go.microsoft.com/fwlink/p/?linkid=331103)。
