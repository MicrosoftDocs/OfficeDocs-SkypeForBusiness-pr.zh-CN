# Lync Server 2013 中的大型会议支持

 

_**上一次修改主题：** 2012-10-01_

Lync Server 2013 可支持使用音频/视频 (A/V) 会议功能（包括共享 PowerPoint 演示文稿）且包含最多 1000 个参与者的会议。若要实现此支持，需要配置一个专用池以支持大型会议，并且需要以某种方式管理该池以确保一次仅主持一个大型会议。

本节介绍了如何使用专用 Lync Server 2013 池来支持大型会议，并介绍了专用池的扩展性注意事项和实现要求，包括拓扑、硬件、软件和配置要求。此外，本节还提供了一组用于支持大型会议的最佳做法建议、测试方法以及由 Lync Server 工程团队执行的服务器扩展性测试的结果的摘要，以及有关大型会议支持的常见问题的解答。
