# <a name="planning-for-persistent-chat-server-in-lync-server-2013"></a>在 Lync Server 2013 中规划持久聊天服务器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-11_

您可以使用 Lync Server 2013 和持久聊天服务器，使多个用户可以参与对话，在其中发布和访问有关特定主题（包括文本、链接和文件）的内容。 尽管用户可以在会话期间实时进行通信，但每个会话的内容都可以持久保存，也就是说，在会话结束后依然可以获得这些内容。

本节介绍 Lync Server 2013、持久聊天服务器部署中的规划注意事项，包括定义要求、识别组件和受支持的拓扑以及部署建议。
