# <a name="removing-lync-for-iphone-and-ipad-in-lync-server-2013"></a>在 Lync Server 2013 中删除适用于 iPhone 和 iPad 的 Lync

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-19_

若要从设备中删除 Lync 2013 for iPhone 或 Microsoft Lync 2010 for iPad，请执行以下步骤：

1.  在移动设备主屏幕上，点击并按住 Lync 图标。

2.  当拼贴开始晃动并显示“X”**** 时，请点击“X”**** 删除该应用程序。
