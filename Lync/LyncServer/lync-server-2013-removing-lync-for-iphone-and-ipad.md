# 删除 Lync for iPhone 和 Lync for iPad

 

_**上一次修改主题：** 2013-02-19_

若要从设备中删除 Lync 2013 for iPhone 或 Microsoft Lync 2010 for iPad，请执行以下步骤：

1.  在移动设备主屏幕上，点击并按住 Lync 图标。

2.  当拼贴开始晃动并显示“X”时，请点击“X”删除该应用程序。
