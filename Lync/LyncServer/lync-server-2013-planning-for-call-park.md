# 在 Lync Server 2013 中规划呼叫寄存

 

_**上一次修改主题：** 2012-09-07_

Lync Server呼叫寄存应用程序使企业语音用户可以将呼叫置于保持状态，稍后从任何电话取回该呼叫。寄存呼叫的用户可以通过拨打呼叫寄存提供的通道号来取回寄存的呼叫，也可以使用外部机制（如即时消息或寻呼系统）向其他人通知该通道号以取回呼叫。本部分包括特定于 呼叫寄存的规划信息。
