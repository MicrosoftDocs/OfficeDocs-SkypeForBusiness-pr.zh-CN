# 在 Lync Server 2013 中设置控制器

 

_**上一次修改主题：** 2014-05-05_

如果通过部署边缘服务器来启用外部用户访问，则还可以选择部署控制器。控制器是运行 Microsoft Lync Server 2013 的服务器，它对用户请求进行身份验证但不驻留任何用户帐户。现在，这不是必需的，但是如果您担心性能问题并且希望帮助简化身份验证请求，则它很有帮助。如果您认为它适合您的组织，则设置控制器或控制器池的步骤与设置 Enterprise Edition 前端池或 Standard Edition 服务器的步骤类似。在拓扑生成器中定义控制器后，需执行本节中的步骤。
