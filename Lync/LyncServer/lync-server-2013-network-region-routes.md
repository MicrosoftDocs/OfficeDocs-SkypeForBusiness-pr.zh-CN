# <a name="network-region-routes-in-lync-server-2013"></a>Lync Server 2013 中的网络区域路由

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-21_

*网络区域路由* 定义一对网络区域之间的路由。呼叫允许控制部署中的每对网络区域均需要网络区域路由。这样部署中的每个网络区域便能够访问其他每个区域。使用本节中的过程可查看、创建、修改或删除网络区域路由。
