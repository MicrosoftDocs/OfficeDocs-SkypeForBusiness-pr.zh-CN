# <a name="choosing-an-e9-1-1-service-provider-for-lync-server-2013"></a>为 Lync Server 2013 选择 E9-1-1 服务提供商

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-06-06_

E9-1-1 服务提供商根据呼叫中包含的位置信息，将来自 Lync Server 的紧急呼叫路由到正确的公共安全应答点（PSAP）。

若要支持 E9-1-1 作为 Lync Server 部署的一部分，您必须从 Lync Open 互操作性计划合格的 E9-1-1 服务提供商获取 E9-1-1 路由服务。 选择最能满足组织要求的提供商。
