# <a name="replication-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的复制 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-10-09_

复制 cmdlet 提供了一种监视和管理 Lync Server 复制的方式。 你可以使用这些 cmdlet 配置复制设置;监视复制进度;并在服务器上手动强制执行复制。
