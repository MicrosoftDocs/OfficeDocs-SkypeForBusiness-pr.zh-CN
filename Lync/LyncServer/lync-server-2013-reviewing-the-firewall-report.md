# 在 Lync Server 2013 中查看防火墙报告

 

_**上一次修改主题：** 2013-02-21_

Lync Server 2013 有一组可能很复杂的防火墙规则。规划工具通过根据设计师的输入条件生成详细定义所有防火墙要求的报告来降低此复杂度。IT 防火墙管理员将能够使用此报告配置和定义所需的规则。

从防火墙管理的角度考虑，应仔细查看此报告以确保与现有防火墙规则没有冲突，并且没有违反任何策略或过程。
