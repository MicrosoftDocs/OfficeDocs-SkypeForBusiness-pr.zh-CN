# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>在 Lync Server 2013 中删除旧版授权主机（可选）

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-09-05_

将远程呼叫控制迁移到 Lync Server 部署时，必须在旧版部署中删除适用于任何 SIP/CSTA 网关的旧版授权主机条目（在 Lync Server 中称为 "*受信任的应用程序条目*"）。 必须使用管理工具分别从这些部署中删除授权的主机条目。
