# 在 Lync Server 2013 中配置高级 E9-1-1 功能

 

_**上一次修改主题：** 2012-06-06_

Lync Server 2013 包括以下可用于自定义 E9-1-1 部署的功能：

  - 用于将位置信息服务器连接到 SNMP 应用程序的 Web 服务接口。

  - 用于连接到辅助位置源数据库的 Web 服务接口。
