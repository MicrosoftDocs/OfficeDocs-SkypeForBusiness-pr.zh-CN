# Lync Server 2013 中持久聊天管理工具的集成

 

_**上一次修改主题：** 2012-08-16_

Lync Server 2013 通过将 持久聊天服务器的管理工具与用于 Lync Server 的其余部分的工具集成来简化持久聊天服务器的管理。

持久聊天服务器包括已与 Lync Server 控制面板集成的管理用户界面体验。此外， 持久聊天服务器包括一组 Windows PowerShell cmdlet 以便管理 持久聊天服务器类别、聊天室（包括删除聊天室和清除过时内容）和外接程序。
