_**上次修改的主题：** 2013-02-21_

在部署和操作 Lync Server 2013 之前，必须先扩展架构，然后创建并配置对象，以准备 Active Directory 域服务。 架构扩展添加了 Lync Server 所需的 Active Directory 类和属性。

本节中的主题介绍如何准备 AD DS 以部署 Lync Server，以及如何将安装和组织单位 (OU) 权限。 有关 Lync Server 所需的架构更改的详细信息，请参阅 [Lync server 2013 所使用的 Active Directory 架构扩展、类和属性](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md)。
