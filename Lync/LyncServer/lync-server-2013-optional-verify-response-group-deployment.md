# （可选）验证响应组部署

 

_**上一次修改主题：** 2012-09-11_

在配置响应组后，您需要验证该配置以确保您的响应组正常工作。至少应使用以下用户类型验证以下方案：

**用户**

  - 驻留在 Lync Server 2013 上的用户

  - 使用公用电话交换网 (PSTN) 的外部用户

  - 驻留在 Lync Server 2013 上的代理

**方案**

  - Lync Server 2013 用户呼叫响应组。

  - 外部用户呼叫响应组。

  - 用户在代理处理其他呼叫时呼叫响应组，并进入队列。
