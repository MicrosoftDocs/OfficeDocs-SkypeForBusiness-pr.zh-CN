# <a name="install-lync-server-2013-on-the-director"></a>在控制器上安装 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2014-05-05_

使用这些步骤在控制器上安装 Lync Server 2013 组件。

<div>

## <a name="to-install-lync-server-components-on-a-director"></a>在控制器上安装 Lync Server 组件

1.  在 Lync Server 部署向导中，在 "Lync Server 2013" 页上的 "**步骤2：设置" 或 "删除 Lync Server 组件**" 旁边，单击 "**运行**"。

2.  在 "**安装程序 Lync Server 组件**" 页上，单击 "**下一步**"，设置在已发布拓扑中定义的组件。

3.  在 Lync Server 组件安装程序完成后，单击 "**完成**"。
