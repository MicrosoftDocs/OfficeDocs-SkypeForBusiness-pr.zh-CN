# <a name="voice-normalization-rules-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的语音规范化规则 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-06-20_

语音规范化规则用于将电话拨号要求（例如，拨打9的外线）转换为 Microsoft Lync Server 2013 使用的 E：164个电话号码格式。

  - <span></span>
