# <a name="direct-sip-connections-support-in-lync-server-2013"></a>Lync Server 2013 中的直接 SIP 连接支持

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-06-29_

Lync Server 2013 支持使用直接 SIP 连接将 Lync Server 2013 连接到以下任一项：

  - IP-PBX

  - PSTN 网关

Lync Server 2013 池中的中介服务器可以控制多个网关、由电话服务提供商提供的会话边界控制器 (SBCs) ，或其中的一些组合。 此外，池中的多台中介服务器可以与单个网关进行交互。

可以使用面向企业电话基础结构的 Microsoft 统一通信开放互操作性计划查找合格的 PSTN 网关、IP-PBX 和 SIP 中继服务。 有关详细信息，请参阅 Microsoft 统一通信开放式互操作性计划网站，网址为 [https://go.microsoft.com/fwlink/p/?linkId=203309](https://go.microsoft.com/fwlink/p/?linkid=203309) 。

有关直接 SIP 连接的拓扑和部署选项的详细信息，请参阅规划文档中的 [Lync Server 2013 中的直接 SIP 连接](lync-server-2013-direct-sip-connections.md) 。
