# 在 Lync Server 2013 中监控、启动和停止持久聊天服务

 

_**上一次修改主题：** 2012-10-06_

持久聊天服务和 持久聊天合规性服务是 Lync Server 2013 拓扑的一部分，因此可以分别使用 Windows PowerShell cmdlet **get-CsWindowsService**、 **stop-CsWindowsService** 和 **start-CsWindowsService** 进行监控、停止和启动。
