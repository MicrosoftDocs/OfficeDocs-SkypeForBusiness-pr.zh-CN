# 证书和身份验证 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

证书和身份验证 cmdlet 涉及大量任务，其中包括：服务器和客户端证书的管理；用户 PIN（个人标识号）的管理；SIP 域和与 Internet 信息服务一起使用的 Kerberos 帐户的管理。
