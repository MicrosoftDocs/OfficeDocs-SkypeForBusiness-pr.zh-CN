# CDR 视图的列表

 

_**上一次修改主题：** 2015-03-09_

可通过视图轻松访问有关用于从 CDR 数据库返回数据的最常见方案的信息。建议您使用视图生成自定义报表，而不要使用实际的 CDR 数据库表；这是因为数据库视图更可能与未来版本的 Lync Server 保持向后兼容。
