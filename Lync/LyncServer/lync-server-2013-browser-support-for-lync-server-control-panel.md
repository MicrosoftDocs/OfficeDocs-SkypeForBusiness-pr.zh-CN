# Lync Server 2013 控制面板的浏览器支持

 

_**上一次修改主题：** 2012-09-28_

Lync Server 2013 支持使用 拓扑生成器，并且在使用以下浏览器时支持 Lync Server 2013 控制面板：

  - Internet Explorer 10（32 位）

  - Internet Explorer 9（32 位）

  - Internet Explorer 8（32 位）
