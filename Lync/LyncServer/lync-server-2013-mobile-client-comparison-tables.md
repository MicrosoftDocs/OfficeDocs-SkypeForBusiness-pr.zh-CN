# <a name="mobile-client-comparison-tables-for-lync-server-2013"></a>Lync Server 2013 的移动客户端比较表

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2015-09-21_

下表比较了 Lync 2013 移动客户端和 Lync 2013 桌面客户端之间的功能和功能:

  - 登录、推送通知和常规功能

  - 增强状态

  - 联系人和联系人组

  - 即时消息 (IM)

  - Lync 至 Lync 音频和视频

  - 会议

  - 电话

  - 外部用户

  - 存档与合规性

这些表指示 lync Server 2013 的本地部署中的 Lync 用户可用的功能。 Skype for Business Online 和 Microsoft Office 365 用户也可以使用相同的功能, 除非表脚注中另有指示。
