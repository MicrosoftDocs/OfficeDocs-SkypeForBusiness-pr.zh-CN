# <a name="planning-for-announcements-in-lync-server-2013"></a>在 Lync Server 2013 中规划通知

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-09-13_

通过 Lync Server 通知应用程序，可以配置在拨入的电话号码对组织有效，但未分配给用户或电话时对传入电话呼叫的处理。 可以将这些呼叫转换到预确定的目标（电话号码、SIP URI 或语音邮件）和/或播放音频通知。 该通知应用程序帮助您可以避免呼叫者拨错号并听到忙音或 SIP 客户端收到错误消息的情况。 本节包含特定于通知应用程序的规划信息。
