# 在 Lync Server 2013 中规划通知

 

_**上一次修改主题：** 2012-09-13_

当已拨号码对您的组织有效，但没有分配到用户或电话时，Lync Server通知应用程序 可用于配置传入电话呼叫的处理。可以将这些呼叫转换到预确定的目标（电话号码、SIP URI 或语音邮件）和/或播放音频通知。该通知应用程序帮助您可以避免呼叫者拨错号并听到忙音或 SIP 客户端收到错误消息的情况。本节包括特定于 通知应用程序的规划信息。
