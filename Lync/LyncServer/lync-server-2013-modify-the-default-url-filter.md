# 修改默认 URL 筛选器

 

_**上一次修改主题：** 2012-06-26_

通过使用即时消息 (IM) 筛选器，Lync Server 2013 可提供全局 URL 筛选器，用以阻止 Lync Server 2013 部署中用户间的 IM 对话中包含的特定 URL。使用 Lync Server 控制面板可以执行下列操作：

  - 阻止即时消息对话中的所有或部分 URL。

  - 允许所有 URL。作为一个选项，可以创建通知，将其插入每个包含 URL 的即时消息的开头。

  - 允许特定 URL，并对每个包含 URL 的即时消息发出警告。

此外，可选择阻止包含特定文件类型的 URL，也可以允许服务器本地 Intranet 区域内部的 URL (Intranet URL) 通过服务器，从而仅阻止 Internet URL。有关 URL 筛选的详细信息，请参阅[为即时消息 (IM) 配置文件传输和 URL 筛选](lync-server-2013-configuring-file-transfer-and-url-filtering-for-instant-messaging-im.md)。
