# <a name="planning-for-devices-in-lync-server-2013"></a>在 Lync Server 2013 中规划设备

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2013-02-28_

Lync Server 2013 包括 Lync Phone Edition、在合格设备上运行的软件，并提供传统和高级电话功能、集成安全性、可管理性等。 Lync Phone Edition 与 lync server 2013 的工作方式相同，与 Lync Server 2010 相同。 有关规划设备的详细信息，请参阅 Lync Server 2010 TechNet 库中[的 "规划设备](http://go.microsoft.com/fwlink/p/?linkid=285880)"。
