# 管理响应组队列

 

_**上一次修改主题：** 2012-10-02_

队列将保留对响应组的呼叫，直到代理应答呼叫。管理队列时，需向队列分配一个或多个代理组并指定队列设置，如执行溢出操作前队列可以保留的呼叫数，以及执行超时操作前呼叫等待代理应答的时间长度。响应组应用程序搜索空闲代理时，将按照代理组的列出顺序进行搜索。
