# <a name="remove-legacy-archiving-and-monitoring-servers"></a>删除旧存档和监控服务器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2012-09-26_

如果您的 Office 通信服务器 2007 R2 部署包含存档服务器或监视服务器, 迁移到 Lync Server 2013 之后, 可以从旧环境中删除这些服务器, 前提是所有用户都已从任何剩余的计算机中删除。Office 通信服务器 2007 R2 池。 你可以按任意顺序删除存档服务器或监视服务器。 密钥要求是所有用户都已从任何剩余的 Office 通信服务器 2007 R2 池删除。

你可以按照第6阶段中概述的过程将用户从 Office 通信服务器 2007 R2 移动到 Lync Server 2013 [: 将用户移动到试验池](phase-6-move-users-to-the-pilot-pool.md)。

确认所有用户都已从任何剩余的池中删除后, 请按照 "删除服务器和服务器角色" 中的过程进行操作[http://go.microsoft.com/fwlink/p/?linkId=205887](http://go.microsoft.com/fwlink/p/?linkid=205887)。
