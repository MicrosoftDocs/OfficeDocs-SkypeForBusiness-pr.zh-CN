# <a name="administering-servers-after-migration"></a>迁移后管理服务器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-09-29_

通常，使用的管理工具必须与要管理的服务器版本对应。 您不能在同一台计算机上安装 Lync Server 2010 和 Lync Server 2013 管理工具。 此外，不会在每台服务器上自动安装 Lync Server 2013 控制面板。 若要安装 Lync Server 2013 控制面板，请按照部署文档中的 "[安装 Lync Server 2013 管理工具](lync-server-2013-install-lync-server-administrative-tools.md)" 主题中的过程操作。

<div>


> [!IMPORTANT]  
> 在部署 Lync Server 2013 的试点池之后，不能使用 Lync Server 2010 拓扑生成器或 Lync Server 2010 控制面板管理任何 Lync Server 2013 资源。 您必须使用 Lync Server 2013 工具来管理 Lync Server 2013 和 Lync Server 2010 资源。
