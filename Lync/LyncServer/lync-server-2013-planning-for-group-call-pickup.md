# 规划组内呼叫应答

 

_**上一次修改主题：** 2013-02-01_

Lync Server 2013 累积更新 2013 年 2 月版引入了组内呼叫应答作为新的企业语音功能。组内呼叫应答让用户通过拨打呼叫应答组号码来应答为其他用户响铃的呼叫。本节包括特定于组内呼叫应答的规划信息。
