# <a name="adding-a-persistent-chat-administrator-in-lync-server-2013"></a>在 Lync Server 2013 中添加持久聊天管理员

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-06_

在 Lync Server 2013 中，执行特定任务的用户必须作为一个或多个特定组的成员进行分配。 基于角色的访问控制 (RBAC) 也可用于通过将用户分配给预定义的 Lync Server 2013 管理角色来授予权限。

在配置和管理持久聊天服务器之前，请确保适当的用户权限和权限已准备就绪，并将任何被分类为持久聊天管理员的用户添加到 CsPersistentChatAdministrator 安全组中。
