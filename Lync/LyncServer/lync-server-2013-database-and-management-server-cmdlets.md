</div>

<div id="mainSection">

<div id="mainBody">

_**上次修改的主题：** 2012-10-09_

数据库和管理服务器 cmdlet 用于管理 Microsoft Lync Server 2013 后端数据库和前端管理服务。 除了为中央管理存储配置 Active Directory 服务控制点之外，您还可以使用这些 cmdlet 来安装或卸载 Lync Server 2013 使用的任何数据库。
