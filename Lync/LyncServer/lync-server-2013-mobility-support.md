# <a name="mobility-support-in-lync-server-2013"></a>Lync Server 2013 中的移动支持

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-06-29_

借助 Lync Server 2013 移动性，用户可以在其移动设备上访问 Lync 2013 功能。 受支持的移动设备包括以下这些：

  - 受支持的 Apple iOS 设备

  - Windows Phone

  - Android

  - Nokia

有关移动功能和要求的详细信息，请参阅规划文档中的在[Lync Server 2013 中规划移动](lync-server-2013-planning-for-mobility.md)功能。
