# 部署

 

_**上一次修改主题：** 2012-10-18_

Lync Server 2013通信软件部署包括准备 Active Directory 域服务，部署前端服务器及其他核心 Lync Server 2013 内部组件，然后部署组织可能需要的其他任何服务器角色和功能，如外部用户访问和企业语音。

本文档介绍了部署 Lync Server 2013 的三种方案：

  - 新建 Lync Server 2013 Enterprise Edition 部署

  - 新建 Lync Server 2013 Standard Edition 部署

  - 在现有 Lync Server 2013 Standard Edition 或 Enterprise Edition 部署基础上新建 Lync Server 2010 Standard Edition 或 Enterprise Edition 部署

有关在现有 Microsoft Office Communications Server 2007 或 Microsoft Office Communications Server 2007 R2 环境中部署 Lync Server 2013 的信息，请参阅[迁移](migration.md)文档。
