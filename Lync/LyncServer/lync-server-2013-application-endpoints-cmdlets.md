# <a name="application-endpoints-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的应用程序终结点 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-06-20_

应用程序终结点是 Active Directory 域服务中的联系人对象。 你可以使用 Lync Server Management Shell 中的 cmdlet 来管理 Microsoft Lync Server 2013 的这些联系人。

<div>

## <a name="application-endpoints-cmdlets"></a>应用程序终结点 Cmdlet

使用以下 cmdlet 管理应用程序终结点。

**应用程序终结点**

  - <span></span>  
    [Get-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398655(v=OCS.15))

  - <span></span>  
    [Move-CsApplicationEndpoint](https://technet.microsoft.com/en-us/library/Gg398188(v=OCS.15))

</div>
