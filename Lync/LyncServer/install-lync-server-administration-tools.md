<div>

## <a name="to-install-the-lync-server-2013-administrative-tools"></a>安装 Lync Server 2013 管理工具

1.  在 Lync Server 2013 前端服务器上，运行 Lync Server 2013 安装程序。

2.  在 "Lync Server 2013 部署向导" 页上，选择 "**安装管理工具**"
