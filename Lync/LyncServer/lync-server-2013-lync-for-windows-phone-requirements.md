# <a name="lync-for-windows-phone-requirements-in-lync-server-2013"></a>Lync Server 2013 中的 Windows Phone 要求的 lync

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-20_

Microsoft Lync 2013 for Windows Phone 为组织中从智能手机或 Windows 专业移动设备连接的用户提供即时消息（IM）、增强状态和电话服务。 移动设备使用户能够扩展 Lync 2013 的范围。 本主题介绍了适用于 Windows Phone 的 Lync 2013 规划注意事项，其中包括确定先决条件和技术要求、必需的组件以及部署指南。

<div>

## <a name="lync-for-windows-phone-prerequisites"></a>适用于 Windows Phone 的 Lync 先决条件

以下是适用于 Windows Phone 必备组件的 Lync 2013。

  - Windows Phone 8、codenamed "Apollo" 或最新版本。

  - Windows Phone 设备必须具有 Microsoft 提供的最新更新。 有关详细信息，请参阅中的 Windows Phone [https://go.microsoft.com/fwlink/p/?LinkID=281961](https://go.microsoft.com/fwlink/p/?linkid=281961)8 更新历史记录。

  - 设备必须具有 12 MB 的可用磁盘空间。

  - 用户必须具有来自运营商的语音和数据计划。
