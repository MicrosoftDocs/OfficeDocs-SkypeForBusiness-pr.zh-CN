# 在 Lync Server 2013 中规划企业语音恢复能力

 

_**上一次修改主题：** 2014-10-17_

语音恢复能力是指在承载 Lync Server 2013 的中央站点因为广域网 (WAN) 故障或其他原因变得不可用时，用户继续发出和接收呼叫的能力。如果中央站点发生故障，企业语音服务必须通过无缝故障转移至备份站点以保持不中断。如果发生 WAN 故障，分支站点的呼叫必须重定向到本地 PSTN 网关。本节讨论了如何规划在中央站点或 WAN 发生故障时的语音恢复能力。
