# Lync Server 2013 中持久聊天服务器的证书要求

 

_**上一次修改主题：** 2012-10-03_

要安装 持久聊天服务器，对于运行文件上载/下载的 持久聊天 Web 服务的每台服务器，您必须拥有由 Lync Server 2013 内部服务器所使用的同一 CA 所颁发的证书。在开始 持久聊天安装之前，确保拥有所需的证书，尤其是使用外部 CA 时。
