# 在 Lync Server 2013 中创建或编辑新类别

 

_**上一次修改主题：** 2012-10-06_

若要创建新类别，请参阅部署文档中的 [在 Lync Server 2013 中配置类别](lync-server-2013-configure-categories.md)。如果您是 持久聊天管理员，则可以使用 Lync Server 控制面板或 Windows PowerShell cmdlet 创建类别。
