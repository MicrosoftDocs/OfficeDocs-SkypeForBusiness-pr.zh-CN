# 执行 ABC 前端池故障转移

 

_**上一次修改主题：** 2013-02-21_

本节的两个主题介绍了在 Lync Server 2013 中执行 ABC 池故障转移的过程，其中 Lync Server 前端池 A 和 B 配对，池 A 变得不可恢复。使用此过程，将用一个新的完全限定域名 (FQDN) 创建一个新的前端池 C。池 C 根据来自发生故障的池 A 的信息来构造。此过程还包括将池 B 和 C 配对在一起。

  - [ABC 池故障转移的备份先决条件](lync-server-2013-backup-prerequisites-for-abc-pool-failover.md)

  - [Lync Server 2013 中的前端池 ABC 故障转移过程](lync-server-2013-front-end-pool-abc-failover-procedure.md)
