# PSTN 连接 Cmdlet

 

_**上一次修改主题：** 2016-12-08_

企业语音以始终如一的服务质量 (QoS) 提供允许呼叫来往于公用电话交换网 (PSTN) 的设置。您可以通过 Lync Server 命令行管理程序提供的 cmdlet 配置这些设置。
