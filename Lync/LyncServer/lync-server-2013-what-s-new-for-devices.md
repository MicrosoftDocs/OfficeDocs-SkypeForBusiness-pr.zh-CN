# <a name="whats-new-for-devices-in-lync-server-2013"></a>Lync Server 2013 中面向设备的新内容

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-06-22_

Lync Server 2013 包括 Lync Phone Edition、在合格设备上运行的软件，并提供传统和高级电话功能、集成安全性、可管理性等。 Lync Phone Edition 与 lync server 2013 的工作方式相同，与 Lync Server 2010 相同。 有关与设备相关的最新功能的详细信息，请参阅 Lync Server 2010 TechNet 库中[设备的新增](http://go.microsoft.com/fwlink/p/?linkid=256490)功能。
