# <a name="setting-up-permissions-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中设置存档权限

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-01_

在 Lync Server 2013 中，特定任务仍要求执行这些任务的用户是一个或多个特定组的成员。 但是，也可以使用基于角色的访问控制（RBAC），通过将用户分配给预定义的 Lync Server 管理角色来授予权限。在部署存档之前，请确保适当的用户权限和权限已准备就绪，并且要分配到特定 RBAC 角色的任何用户已分配给该角色。 有关用于部署对存档的支持的用户权限、权限和角色的详细信息，请参阅[Lync Server 2013 中的存档的部署清单](lync-server-2013-deployment-checklist-for-archiving.md)（在规划文档和部署文档中提供）。 有关 RBAC 的详细信息，请参阅规划文档中的在[Lync Server 2013 中规划基于角色的访问控制](lync-server-2013-planning-for-role-based-access-control.md)。
