# <a name="new-virtualization-features-in-lync-server-2013"></a>Lync Server 2013 中新的虚拟化功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2013-11-07_

Lync Server 2013 支持 Windows Server 2012、Windows Server 2012 R2 和 Windows Server 2008 R2 上的虚拟化。 Windows Server 2012 和 Windows Server 2012 R2 上的支持包括对单个根 i/o 虚拟化（SR-IOV）功能的支持。 使用 SR-IOV，物理网络适配器的虚函数直接分配给虚拟机。 这会增加网络吞吐量并减少网络延迟，同时减少处理网络流量所需的主机 CPU 开销。 若要充分利用 SR-IOV，你必须使用具有支持 SR-IOV 的 BIOS 的主机服务器，以及使用支持 SR-IOV 的网络适配器。
