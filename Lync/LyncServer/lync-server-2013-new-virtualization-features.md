# <a name="new-virtualization-features-in-lync-server-2013"></a>Lync Server 2013 中的新的虚拟化功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-11-07_

Lync Server 2013 支持在 Windows Server 2012、Windows Server 2012 R2 和 Windows Server 2008 R2 上实现虚拟化。 对 Windows Server 2012 和 Windows Server 2012 R2 的支持包括对单个根 i/o 虚拟化（SR-IOV）功能的支持。 借助 SR-IOV，可直接将物理网络适配器的虚拟功能分配到虚拟机。 这增加了网络吞吐量并减少的网络延迟，同时还减少了处理网络流量所需的主机 CPU 开销。 要充分利用 SR-IOV，必须使用主机服务器，该服务器具有支持 SR-IOV 的 BIOS，并使用支持 SR-IOV 的网络适配器。
