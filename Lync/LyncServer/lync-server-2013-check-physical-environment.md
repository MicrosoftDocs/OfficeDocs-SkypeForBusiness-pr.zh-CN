# <a name="performing-physical-environmental-checks"></a>执行物理环境检查

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**上次修改的主题：** 2014-04-30_

在检查 Lync Server 2013 部署的性能、可用性和功能之前，应检查物理环境。 例如，服务器的室内温度可能必须降低，或者可能需要更换网络电缆。 为获得最佳结果，请执行以下物理环境检查：

  - **物理安全措施**    物理安全保护（如锁、门和受限制访问的聊天室）必须受到保护。 检查是否有未授权和强制的设备损坏迹象。

  - **温度和湿度**    高温、空气流量差和湿度可能导致硬件组件过热。 检查温度和湿度，以确保环境系统（如加热和空调）可以在硬件制造商的规范中维护可接受的条件和功能。 最近安装新设备时，还应检查服务器的通风流是否不受支持，并符合制造商的规范。

  - **设备和组件**    Lync Server 2013 组织依赖于正常运行的物理网络和相关硬件。 请确保路由器、交换机、集线器、物理电缆和连接器正常运行。

有关如何执行这些检查的具体说明将很大程度上取决于您的安装网站和所选的服务器硬件。 第一次执行此检查时，请参阅硬件文档，并记下所需的参数以供将来参考。
