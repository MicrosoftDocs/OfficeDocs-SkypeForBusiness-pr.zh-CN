# Lync Server 2013 中的设备更新 Web 服务

 

_**上一次修改主题：** 2016-12-08_

Lync Server 包括 设备更新 Web 服务，它自动作为 Web 服务 角色的一部分安装。此服务使您能够从 Microsoft 下载更新，对其进行测试，然后为组织中的 IP 电话部署更新。您还可以使用 设备更新 Web 服务 将设备回滚到之前的软件版本。

此部分提供有关如何使用设备更新日志、规则（Lync Phone Edition 使用*规则*将固件版本更新与硬件设备相关联）和配置设置管理 设备更新 Web 服务 和已部署更新的详细信息。

有关 设备更新 Web 服务 过程和功能的详细信息，请参阅 Lync Server 2010 TechNet 库中的[更新设备](http://technet.microsoft.com/en-us/library/gg412864\(v=ocs.14\).aspx)。（请注意，设备更新 Web 服务 与所有 Lync Phone Edition 组件一样，在 Lync Server 2013 上的工作方式与在 Lync Server 2010 上的工作方式是相同的。）
