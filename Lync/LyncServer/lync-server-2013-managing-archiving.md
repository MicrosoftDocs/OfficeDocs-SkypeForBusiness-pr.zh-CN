# 管理 Lync Server 2013 存档

 

_**上一次修改主题：** 2012-10-10_

在为组织部署存档时，可以在部署期间指定初始配置。但是，有时您希望针对日常管理或为了满足组织的新需要而更改实施存档支持的方式。例如，您可能需要为组织中的特定站点、池或用户单独设置存档支持。对于驻留在 Lync Server 2013 上的用户，您可以通过创建和自定义存档策略和配置来执行此操作。如果使用 Microsoft Exchange 集成，则还必须配置 Exchange 2013 设置。本节提供允许您更改存档部署的相关信息和过程。
