# <a name="call-diagnostic-reports-in-lync-server-2013"></a>在 Lync Server 2013 中呼叫诊断报告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-21_

呼叫诊断报告提供失败的点对点会话和会议会话的摘要信息及诊断数据。

<div>

## <a name="in-this-section"></a>本部分内容

  - [Lync Server 2013](lync-server-2013-call-diagnostic-summary-report.md)   中的呼叫诊断摘要报告提供失败的对等会话和会议会话的总体摘要。 对等会话是仅涉及两个参与者的会话。 会议会话涉及三个或更多参与者。

  - [Lync Server 2013](lync-server-2013-peer-to-peer-activity-diagnostic-report.md)   中的对等活动诊断报告提供失败的对等会话的总体趋势视图。 对等会话仅涉及两个参与者。

  - [Lync Server 2013](lync-server-2013-conference-diagnostic-report.md)   中的会议诊断报告提供了每个会议模态的失败会议会话和趋势视图的总体趋势视图。 会议会话至少涉及三个参与者。

  - [Lync Server 2013](lync-server-2013-top-failures-report.md)   中的热门故障报告提供了一系列最常见的故障及其随时间发展的趋势。

  - [Lync Server 2013](lync-server-2013-failure-distribution-report.md)   中的故障分布报告提供了失败会话的分析。

  - [Lync Server 2013](lync-server-2013-failure-list-report.md)   中的 "故障列表报告" 提供有关失败的会议中涉及的各个参与者的详细信息。

  - [Lync Server 2013](lync-server-2013-diagnostic-report.md)   中的诊断报告为失败的会话提供诊断和疑难解答信息（包括 SIP 响应代码和诊断头和 id）。
