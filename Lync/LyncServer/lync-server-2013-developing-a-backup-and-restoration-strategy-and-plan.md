# 制定备份和还原策略及计划

 

_**上一次修改主题：** 2013-02-17_

Lync Server 备份和还原操作的效率取决于备份和还原策略和计划。您应制订一个适应组织总体策略的 Lync Server 备份和还原策略，以及一个全面、简洁的计划，以便备份数据和设置并在中断时还原服务。

为确保前端池的灾难恢复最可靠，请使用 Lync Server 2013 中介绍的配对池灾难恢复拓扑。有关详细信息，请参阅[在 Lync Server 2013 中规划高可用性和灾难恢复](lync-server-2013-planning-for-high-availability-and-disaster-recovery.md)。
