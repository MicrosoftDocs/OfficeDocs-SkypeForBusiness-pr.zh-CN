# <a name="new-im-and-presence-features-in-lync-server-2013"></a>Lync Server 2013 中的新 IM 和状态功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-19_

Microsoft Lync Server 2013 添加了以下新的即时消息（IM）和状态功能，以丰富用户的 Lync 体验。

  - 如果您的组织也运行 Exchange 2013，则用户可以利用统一的联系人存储库。 用户可以在 Outlook 2013、Outlook Web App 以及 Lync 2013 中管理其联系人。

  - 由于 Lync Server 2013 的**XMPP 集成**功能，用户可以使用可扩展消息和状态协议（如 Google 谈话）的公共 IM 提供商的用户交换即时消息和状态信息。 XMPP 集成内置于前端服务器和边缘服务器中，您可以启用它并将其配置为允许此功能。
