# <a name="configuring-group-call-pickup-in-lync-server-2013"></a>在 Lync Server 2013 中配置组呼叫装货

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2013-02-01_

Lync Server 2013 的累积更新：2月2013将组呼叫应答引入为新的企业语音功能。 组内呼叫应答允许用户通过拨打呼叫应答组号码来接听其他用户拨打的电话。

在部署企业语音时，会自动在前端服务器或 Standard Edition 服务器上安装和启用组呼叫应答使用的组件。 但是，必须先配置组呼叫应答，然后才能供用户使用。

本部分将指导您完成组呼叫应答的配置。
