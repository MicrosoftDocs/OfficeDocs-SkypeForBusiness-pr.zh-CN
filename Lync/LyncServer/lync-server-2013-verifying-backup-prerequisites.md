# <a name="verifying-backup-prerequisites-in-lync-server-2013"></a>在 Lync Server 2013 中验证备份先决条件

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间:** 2013-02-17_

在开始备份 Lync Server 之前, 请验证你是否已准备好以下内容:

  - 备份工具。 有关详细信息, 请参阅[Lync Server 2013 中的备份和还原要求: 工具和权限](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md)。

  - 授权. 有关详细信息, 请参阅[Lync Server 2013 中的备份和还原要求: 工具和权限](lync-server-2013-backup-and-restoration-requirements-tools-and-permissions.md)。

  - 用于存储备份的位置。 有关详细信息, 请参阅[设置 Lync Server 2013 的备份位置](lync-server-2013-setting-up-a-backup-location.md)。

  - 媒体用于备份。 有关详细信息, 请参阅[设置 Lync Server 2013 的备份位置](lync-server-2013-setting-up-a-backup-location.md)。
