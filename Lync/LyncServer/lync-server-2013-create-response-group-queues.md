# 在 Lync Server 2013 中创建响应组队列

 

_**上一次修改主题：** 2012-01-18_

队列使呼叫者处于保持状态，直到有代理应答呼叫为止。 响应组应用程序搜索空闲代理时，将按照代理组的列出顺序进行搜索。可以选择分配给队列的代理组并指定队列的行为，如限制队列可容纳的呼叫数，以及呼叫等待代理应答呼叫的时间长度。
