# Lync Server 2013 中新的企业语音功能

 

_**上一次修改主题：** 2013-05-01_

Lync Server 2013 引入了多个新的用于增强 企业语音的路由和呼叫管理功能。

Lync Server 2013 支持 中介服务器和网关之间的多个中继。 *中继* 是端口号与 中介服务器之间以及端口号与网关之间的逻辑关联。这意味着一个 中介服务器可具有多个至不同网关的中继，一个网关可具有多个至不同中介服务器的中继。中继间路由使 Lync Server 2013 可将 IP-PBX 互连到公用电话交换网 (PSTN) 网关或互连多个 IP-PBX 系统。 Lync Server 2013 服务器充当不同拓扑系统间的粘附（即，互连）。

Microsoft Lync Server 2013 在呼叫转接、同时响铃、语音邮件处理和来电显示等方面进行了改进。这些功能丰富了 企业语音的呼叫体验。
