# 拨入式会议访问数

 

_**上一次修改主题：** 2012-09-18_

要允许用户从公用电话交换网 (PSTN) 通过电话拨入加入内部会议的音频部分，必须配置电话拨入式会议访问号码。电话拨入式会议访问号码是用户为加入会议而呼叫的号码。

电话拨入式访问号码显示在会议邀请和电话拨入式会议设置网页上。

> [!NOTE]  
> 在 Active Directory 完成对一个新拨入访问号码的复制前，无法使用该拨入访问号码。复制可能需要几个小时。
