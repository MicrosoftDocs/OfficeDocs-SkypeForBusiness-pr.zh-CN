# <a name="removing-lync-for-windows-phone-in-lync-server-2013"></a>在 Lync Server 2013 中删除适用于 Windows Phone 的 Lync

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**上次修改的主题：** 2013-02-19_

若要从移动设备中删除适用于 Windows Phone 应用程序的 Lync 2013，请执行以下步骤：

1.  在移动设备上，从开始屏幕上滑动手指以查看应用程序列表。

2.  按住 Lync 应用程序，然后选择“卸载”****。
