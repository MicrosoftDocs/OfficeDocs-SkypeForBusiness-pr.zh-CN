# <a name="manage-xmpp-federated-partners-in-lync-server-2013"></a>在 Lync Server 2013 中管理 XMPP 联盟伙伴

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-10-19_

这是初步文档，可能会发生变更。 空白主题作为占位符包含在内。

要管理对 XMPP 联盟域用户的支持，需执行以下操作：

  - 配置一个或多个外部访问策略以支持 XMPP 联盟域用户。

  - 配置访问边缘配置策略以支持联盟。

  - 创建 XMPP 联盟伙伴定义。

  - 了解可用于 XMPP 联盟的协商设置。

要执行这些任务，请使用本节中的过程。
