# 在 Lync Server 2013 中管理组织的 XMPP 联盟伙伴

 

_**上一次修改主题：** 2012-10-19_

本文档是预备文档，可能随时更改。空白主题均以占位符的形式包含在内。

要管理对 XMPP 联盟域用户的支持，需执行以下操作：

  - 配置一个或多个外部访问策略以支持 XMPP 联盟域用户。

  - 配置访问边缘配置策略以支持联盟。

  - 创建 XMPP 联盟伙伴定义。

  - 了解可用于 XMPP 联盟的协商设置。

要执行这些任务，请使用本节中的步骤。
