# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Lync Server 2013 中新的统一联系人存储功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改的主题：** 2012-09-13_

Lync Server 2013 引入了统一联系人存储。 统一联系人存储允许用户将其所有联系人信息保留在 Microsoft Exchange Server 2013 中。 将用户的 Lync 联系人迁移到 Exchange 2013 后，用户可以从 Lync 2013、Outlook 或 Outlook Web App 访问和管理其联系人，并且他们的收藏夹保持同步。 例如，如果用户将联系人添加到 Outlook 中的 "收藏夹"，则该联系人将显示在 Lync 2013 的 "收藏夹" 组中。 用户无需登录到 Lync 即可从 Outlook 或 Outlook Web App 管理其联系人。

默认情况下将启用统一的联系人存储。您可以全局、按站点、按租户或者按个人或个人组对用户启用或禁用统一的联系人存储。
