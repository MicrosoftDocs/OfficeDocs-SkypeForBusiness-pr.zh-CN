# <a name="call-park-application-cmdlets-in-lync-server-2013"></a>在 Lync Server 2013 中调用寄存应用程序 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主题上次修改时间：** 2012-03-21_

呼叫驻留应用程序允许用户将呼叫置于保持状态，然后从其他电话检索该呼叫。 使用这些 cmdlet 配置呼叫寄存轨道式和呼叫寄存应用程序的设置。

<div>

## <a name="call-park-application-cmdlets"></a>呼叫寄存应用程序 Cmdlet

以下 cmdlet 可用于管理呼叫驻留应用程序。

**呼叫寄存应用程序**

    [CsCallParkOrbit](https://technet.microsoft.com/en-us/library/Gg398554(v=OCS.15))

  - <span></span>

  - <span></span>  
    [Remove-CsCallParkOrbit](https://technet.microsoft.com/en-us/library/Gg412901(v=OCS.15))

  - <span></span>  
    [Set-CsCallParkOrbit](https://technet.microsoft.com/en-us/library/Gg398796(v=OCS.15))
