<a name="troubleshoot-connectivity-issues-with-the-microsoft-teams-client"></a>使用 Microsoft Teams 客户端解决连接性问题
==============================================================

使用 Microsoft 团队客户端发现的大多数问题可以追溯到防火墙或代理的连接。 验证此必需的 Url、 IP 地址和防火墙中打开端口或代理将减少不必要的故障排除。 有关 Url 和 Ip 的 Microsoft 团队所需的特定信息，请参阅[Office 365 Url 和 IP 地址](https://support.office.com/article/Office-365-URLs-and-IP-address-ranges-8548a211-3fe7-47cb-abb1-355ea5aa88a2)支持文章。 以下方案需要特定的 Url，在防火墙中打开端口。

-   身份验证

-   Microsoft 团队客户端连接

-   协作

-   媒体

-   共享的服务

-   第三方集成

-   Skype 的业务互操作性

-   Skype 的业务客户端互操作性
