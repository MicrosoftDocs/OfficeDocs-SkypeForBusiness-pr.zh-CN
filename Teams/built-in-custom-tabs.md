<a name="use-built-in-and-custom-tabs-in-microsoft-teams"></a>在 Microsoft Teams 中使用内置选项卡和自定义选项卡
==================================================

选项卡允许工作组成员访问在专用画布中通道或聊天服务。 这样的团队可以直接使用工具和数据和对话有关其上下文中的通道或聊天。 

所有者和团队成员可以向通道、 私人聊天和群聊以帮助集成其云服务添加选项卡。 可以添加选项卡来帮助用户轻松访问和管理它们需要或最与之交互的数据。 这可以是 Power BI 报表、仪表板，甚至可以是 [Microsoft Stream](https://go.microsoft.com/fwlink/?linkid=855785) 视频频道，你可以在此发布培训视频。

![三个显示选项卡中各种内容的屏幕截图。](media/Use_built-in_and_custom_tabs_in_Microsoft_Teams_image4.png)

## <a name="work-with-tabs"></a>使用选项卡

- 与每个新的通道，默认情况下，对话和文件设置两个选项卡。 

    ![Marketing 团队的“对话”部分屏幕截图。](media/Use_built-in_and_custom_tabs_in_Microsoft_Teams_image1.png)
- 与每个私人聊天，默认情况下，对话、 文件、 组织和活动设置四个选项卡。

    ![聊天中的选项卡的屏幕截图。](media/Use_built-in_and_custom_tabs_add_tabs_to_a_chat.png)

- 所有者和团队成员可以添加更多选项卡通道或聊天通过单击**添加选项卡**![屏幕截图显示添加选项卡上按钮，加号。](media/Use_built-in_and_custom_tabs_add_a_tab_button.png) 在通道或聊天顶部。

- 它们可以转换为选项卡之前，必须将 Excel、 PowerPoint、 Word 和 PDF 文件上载到**文件**选项卡。 此外，任何现有上载的文件可以转换为选项卡上，单击一下，如下所示。

    ![选择了一个 PowerPoint 文件的“文件”选项卡屏幕截图。](media/Use_built-in_and_custom_tabs_in_Microsoft_Teams_image2.png)

- 若要添加网站，URL 必须启动**https**前缀使信息交换保持安全。

- 当工作组成员尝试将自定义选项卡添加到其通道或聊天时提供了详细的说明。 自定义选项卡添加到通道后，会创建**选项卡对话**允许工作组成员已关注讨论有关内容。

    ![窗口右侧有对话选项卡的自定义选项卡屏幕截图。](media/Use_built-in_and_custom_tabs_in_Microsoft_Teams_image3.png)

## <a name="develop-custom-tabs"></a>开发自定义选项卡

除了内置选项卡，您可以设计和开发您自己的选项卡可集成到团队或共享与社区的其余部分。

在 Microsoft Developer Network 提供了[详细的说明](https://go.microsoft.com/fwlink/?linkid=855786)来设计和构建您自己的选项卡和可下载的 Microsoft 开发的[示例选项卡](https://go.microsoft.com/fwlink/?linkid=855787)。

![Microsoft Teams 中的示例自定义选项卡屏幕截图。](media/Use_built-in_and_custom_tabs_in_Microsoft_Teams_image5.png)
