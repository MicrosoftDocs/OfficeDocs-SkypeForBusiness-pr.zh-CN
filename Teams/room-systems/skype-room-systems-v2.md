# <a name="microsoft-teams-rooms-licenses"></a>Microsoft Teams 会议室许可证
<a name="bkmk_srs"> </a>

下表列出了 Microsoft 团队聊天室中提供的与授权相关的功能以及为获得这些功能需要购买哪些许可证。
  
> [!NOTE]
> [!注释] 要设置的会议室需为用户对象，并且要为其分配这些许可证。
> 现在, Office 365 中提供了新的会议室许可证。

|Microsoft 团队会议室方案 |您目前拥有 Office 365 Premium、Office 365 专业增强版或 Skype for business 独立计划2。  <br/> 需要购买的项目：  |你当前使用的是基于企业的计划。  <br/> 需要购买的项目： |已有 Skype for Business Server 2015（本地或混合）。 <br/> 需要购买的项目：|
|:-----|:-----|:-----|:-----|
|加入计划会议。  |Skype for Business 独立计划 1 |E1、3、4 或 5  |Skype for Business Server 标准 CAL  |
|启动临时会议。 |Skype for Business 独立计划 2 |E1、3、4 或 5 |Skype for Business Server 标准 CAL  <br/> Skype for Business Server 企业 CAL|
|启动临时会议并从会议拨出到电话号码。 |带有音频会议的 Skype for business 独立计划2  <br/> **注意：** 通信点数可选。 |具有音频会议的 E1 或 E3  <br/> 或者  <br/> E5  <br/> |Skype for Business 标准 CAL  <br/> Skype for Business Server 企业 CAL|
|为聊天室提供一个电话号码, 使用电话号码拨打或接听会议室或加入音频会议。  |Skype for Business 独立版计划 2 ，带 Office 365 电话系统和通话套餐  <br/> **注意：** 通信点数可选。           |E1 或 E3，带 Office 365 电话系统与 Office 365 通话套餐  <br/> 或者  <br/> E5   |Skype for Business Server 标准 CAL  <br/> Skype for Business Server Plus CAL  |

 **使用正确版本的 Windows 10**: 对于想要将 windows 10 映像部署到其设备的客户, 请参阅[配置 Microsoft 团队聊天室控制台](/Skypeforbusiness/deploy/deploy-clients/console.md)。
