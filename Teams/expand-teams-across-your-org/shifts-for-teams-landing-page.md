# <a name="shifts-for-teams"></a>Teams 中的排班

团队为你的组织中的一线工作人员提供了他们进行通信和高效协作所需的工具，并执行最佳工作。 在此处，你将找到你需要设置和管理团队中的倒班、计划管理工具的管理指南。

### <a name="shifts-extensions"></a>班次延长

|               |               |
| ------------- | ------------- |
| ![#c1](../media/api-small.svg) | **[移动图形 api](/graph/api/resources/shift?view=graph-rest-1.0)** 通过移动 Api，你可以将倒班数据与外部劳动力管理系统集成，让你能够灵活地在后端构建自定义倒班体验，同时为用户提供团队中的丰富的前端体验。             |
| ![#c1](../media/api-small.svg) | **[劳动力管理集成](https://github.com/OfficeDev/Microsoft-Teams-Shifts-WFM-Connectors)** 如果您使用的是第三方劳动力管理系统（如 Kronos 和 JDA）来安排、时间和出勤，则可以通过开放源代码集成，与倒班直接集成。 |
| ![#c1](../media/process-flow-teams-small.svg) | **[自动切换 + 电源](https://github.com/OfficeDev/Microsoft-Teams-Shifts-Power-Automate-Templates)** 倒班 + Power "自动处理" 使你可以从倒班中获取信息，并使用其他应用创建自定义工作流，并以比例执行操作。 只需少量代码即可自动化关键流程。 触发器和模板支持各种方案，例如，在不需要经理批准时为倒班请求启用自动审批。 |
