# <a name="view-guest-users-in-a-team"></a>在团队中查看来宾用户

团队中的任何人都可以通过团队中的管理团队查看团队的所有成员，包括Microsoft Teams。  有关详细信息 [，请参阅](https://support.office.com/article/see-who-s-on-a-team-or-in-a-channel-5c6be9be-9c45-4a0f-a1a0-f332b23cb6b7) 查看团队成员或频道中的用户。

标识来宾用户：

1.  登录到 Microsoft 365[管理中心](https://admin.microsoft.com)。
2.  选择 **"管理中心**  >  **Teams"。**
3.  在管理Teams，选择"用户 **"。** 会看到 **来宾附加到** 任何来宾显示名称的后面。
  
还可以从你拥有的团队内标识来宾用户。
