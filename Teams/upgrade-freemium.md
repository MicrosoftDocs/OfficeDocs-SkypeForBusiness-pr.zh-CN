# <a name="upgrade-microsoft-teams-free-to-subscription-version"></a>将Microsoft Teams免费升级到订阅版本

如果你的组织正在使用免费版本的 Microsoft Teams，则可以通过为用户购买订阅计划Microsoft 365或Office 365升级到完整版本。 完整版本提供免费Teams不提供的其他附加功能，例如计划、音频会议、增强的管理和安全功能。 Microsoft 365 和 Office 365 将熟悉的 Microsoft Office 桌面套件与基于云的 Microsoft 下一代通信和协作服务（包括 Exchange Online、SharePoint Online 和 Office）相结合，帮助用户从几乎任何位置通过 Internet 高效工作。 升级Teams时，Teams数据不会丢失;所有团队、频道、聊天、文件和权限都随你一起提供。 

> [!NOTE]
> 如果已有 Microsoft 365 或 Office 365 订阅，则有资格使用企业标识 Teams (的) 版本，而不是免费版本。 此Teams试用版提供有限Teams的完整版本。 有关详细信息，请参阅管理[Microsoft Teams商业云试用产品/服务](./teams-exploratory.md)。

## <a name="how-does-teams-free-compare-to-the-full-version-of-teams"></a>与Teams版本相比，免费版Teams？

Teams免费计划专为中小型企业设计，具有以下功能：

- 最多 500，000 个用户
- 无限制聊天消息和搜索
- 来宾访问
- 与应用和服务集成，包括 Word、Excel、PowerPoint 和 OneNote
- 每个用户 2 GB 存储，共享存储 10 GB
- 1：1 和群组在线音频和视频通话
- 频道会议
- 屏幕共享

订阅或订阅Teams的完整版本Microsoft 365 Office 365提供以下功能，以及免费版Teams功能：

- 使用企业 (没有用户限制) 
- Exchange电子邮件托管和自定义电子邮件域
- OneDrive、SharePoint、Planner、Yammer和其他Microsoft 365 Office 365服务
- 每个用户 1 TB 的存储空间
- 计划的会议
- 音频会议
- 增强的安全性和符合性功能，包括多重身份验证、单一登录以及高级审核和报告
- 管理控制和支持功能，包括 24 x 7 电话和 Web 支持、用于管理用户和应用的管理工具、Microsoft 365 或 Office 365 服务的使用情况报告、服务级别协议以及可配置的用户设置和策略

有关免费功能和Teams功能Teams比较，请参阅[比较Teams计划](https://products.office.com/microsoft-teams/free)。

## <a name="upgrade-requirements"></a>升级要求

如果满足以下要求，Teams升级到完整版本的版本：

- 你是注册现有免费订阅Teams用户。
- 如果你要自带域，则它尚未通过试用版或Azure Active Directory (订阅或订阅Microsoft 365 Office 365关联) 。

> [!NOTE]
> 若要升级和传输数据，必须完成升级过程在 Teams 购买订阅。 如果在未Microsoft 365 Office 365的情况下Teams或带租户，则不能传输数据，因为已有单独的租户。

## <a name="limitations"></a>限制

请记住以下限制：

- 升级后无法切换回Teams免费版。
- 不能将多个免费Teams合并到单个付费租户中。
- 所有用户必须在同一个域中。  (所有用户都将使用 .domain.com 格式登录 @ *)*
- 所有用户都必须升级：不支持在同一租户Teams混合使用免费订阅和付费订阅用户。

## <a name="how-do-i-upgrade-my-organization"></a>如何升级我的组织？

若要升级到完整版本的 Teams，请选择"**在 Teams"。**

![显示"升级"按钮的屏幕截图。](media/teams-freemium-upgrade-image1.png)

输入用于登录帐户的电子邮件地址，Teams购买Microsoft 365 商业标准版计划。 若要购买版本或Microsoft 365 商业基础版版本Enterprise，Office 365[联系支持人员](https://portal.office.com/support/altusupport.aspx?app=teamsfreeupgrade)。
