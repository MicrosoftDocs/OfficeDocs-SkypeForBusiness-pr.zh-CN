# <a name="set-up-microsoft-teams-calling-plans"></a>设置Microsoft Teams 通话套餐

呼叫其他 Teams 用户是免费的，但如果你希望你的用户能够在公司外部拨打电话，请根据需要在 Microsoft 365 中获取国内通话套餐、国际通话套餐或即用即付通话套餐。 为企业设置Microsoft Teams 通话套餐非常简单。  有关通话套餐的详细信息，请参阅 [哪种通话套餐适合你？](calling-plan-landing-page.md)。

## <a name="step-1-find-out-if-calling-plans-are-available-in-your-countryregion"></a>步骤 1：了解通话套餐是否在你的国家/地区可用

转到 [音频会议和通话套餐的国家和地区可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) ，然后选择你的国家或地区以获取有关通话套餐的可用性信息，以及有关音频会议、Teams 电话、收费和免费号码以及通信额度的信息。

如果通话套餐不适用于你的国家或地区，请参阅 [PSTN 连接选项](pstn-connectivity.md) 了解所有可用选项。
  
## <a name="step-2-buy-and-assign-licenses"></a>步骤 2：购买和分配许可证

1. 如果 Microsoft 365 计划未包含 Teams 电话功能，则可能需要购买 **电话系统** 附加许可证。 拥有 **电话系统** 许可证后，请购买 [适用于 Microsoft 365 的通话套餐](calling-plans-for-office-365.md)。 请参阅 [Microsoft Teams 附加许可](./teams-add-on-licensing/microsoft-teams-add-on-licensing.md)，并购买许可证和计划。

    > [!TIP]
    > Microsoft 365 中的 **电话系统** 许可证和通话套餐一起，因此若要查看购买通话套餐的选项，必须首先拥有 **电话系统** 许可证。
  
2. 首先分配许可证，然后将通话套餐许可证分配给组织中的人员。 请参阅 [分配 Microsoft Teams 附加许可证](./teams-add-on-licensing/assign-teams-add-on-licenses.md)。
    1. 如果购买即用即付通话套餐，可能需要购买通信点数。 若要确定是否需要购买通信额度，请参阅 [如何为即用即付通话套餐提供资金](calling-plans-for-office-365.md#how-to-fund-a-pay-as-you-go-calling-plan)。

## <a name="step-3-get-phone-numbers"></a>步骤 3：获取电话号码

有三种方法，以获得新的用户数：

- **使用 Teams 管理中心。** 对于某些国家/地区，可以使用 Teams 管理中心获取用户的号码，请参阅 [获取用户的电话号码](getting-phone-numbers-for-your-users.md)。

- **转网现有的号码。** 可以将现有号码从当前服务提供商或电话运营商移植或转移到 Microsoft 365。 有关详细信息，请参阅 [将电话号码转移到 Teams](phone-number-calling-plans/transfer-phone-numbers-to-teams.md) 或 [管理组织的电话号码](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)。
  
- **使用新号码的请求表单。** 有时 (取决于你的国家/地区) 你无法使用 Teams 管理中心获取新的电话号码，或者你将需要特定的电话号码或区号。 如果是这样，你需要下载表单并将其寄回给我们。 有关详细信息，请参阅[为你的组织管理电话号码](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)。

## <a name="step-4-add-emergency-addresses-and-locations-for-your-organization"></a>步骤 4：为组织添加紧急地址和位置
<a name="bkmk_add_addresses"> </a>

紧急地址必须与电话号码相关联。 发生此关联时可能因国家和地区而异。 例如在美国，向用户分配电话号码时需要关联紧急地址。 在英国，当你从 Microsoft 365 获取电话号码时，或者从当前服务提供商转移电话号码时，需要将紧急地址关联到电话号码。

有关紧急呼叫和管理紧急地址的信息，请参阅 [管理紧急呼叫](what-are-emergency-locations-addresses-and-call-routing.md) 和 [为组织添加、更改或删除紧急位置](add-change-remove-emergency-location-organization.md)。

## <a name="step-5-assign-an-emergency-address-and-a-phone-number-to-a-user"></a>步骤 5：向用户分配紧急地址和电话号码
<a name="bkmk_add_addresses"> </a>

在 Microsoft 365 中设置通话套餐时，必须为每个用户分配电话号码和紧急地址。 必须先创建紧急地址，然后才能将其与电话号码关联。 有关详细信息，请参阅 [分配或更改紧急地址](assign-change-emergency-location-user.md)。

> [!TIP]
> [!提示] 如果在执行此步骤前刚刚向企业添加了更多人员，这些人员可能需要 **几小时** 才会显示在" **语音用户**"页面。

## <a name="step-6-tell-your-users-about-their-new-phone-numbers"></a>步骤 6：告知用户新电话号码

Microsoft 建议发送邮件或使用你企业的首选通信方法告诉用户他们的新电话号码。

在 **Microsoft Teams** 中，用户通过单击左侧导航窗格中的 **呼叫**，可以看到他们的电话号码。 电话号码显示在拨号板上方。

![单击“呼叫”后可用选项的屏幕截图。](media/teams-phone-number.png)

## <a name="do-you-want-to-automate-assigning-phone-numbers"></a>想要自动分配电话号码？
<a name="bkmk_add_addresses"> </a>

如果了解 Windows PowerShell，可以使用以下 cmdlet 为用户自动分配电话号码。
  
- [Get-CsPhoneNumberAssignment](/powershell/module/teams/Get-CsPhoneNumberAssignment)：从租户检索电话号码。

- [Set-CsPhoneNumberAssignment](/powershell/module/teams/Set-CsPhoneNumberAssignment)：设置电话号码。

若要了解详细信息，请参阅 [Teams PowerShell 概述](teams-powershell-overview.md)。
