## <a name="dial-plans"></a><span data-ttu-id="cf686-700">拨号计划</span><span class="sxs-lookup"><span data-stu-id="cf686-700">Dial plans</span></span>

<span data-ttu-id="cf686-701">Office 365 的电话系统功能中的[拨号计划](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans)是一组规范化规则，用于将拨打的电话号码转换为替代格式（通常为 [E.164](https://go.microsoft.com/fwlink/?linkid=859014) 格式）以便进行呼叫授权和呼叫路由。</span><span class="sxs-lookup"><span data-stu-id="cf686-701">A [Dial Plan](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans), in the Phone System feature of Office 365, is a set of normalization rules that translates dialed phone numbers into an alternate format (typically [E.164](https://go.microsoft.com/fwlink/?linkid=859014) format) for call authorization and call routing.</span></span> <span data-ttu-id="cf686-702">音频会议服务利用电话系统使用的相同功能在会议拨出应用场景中转换拨打的电话号码。</span><span class="sxs-lookup"><span data-stu-id="cf686-702">Audio Conferencing service leverages the same capabilities used by Phone System to translate dialed phone numbers in conference dial out scenarios.</span></span>

<span data-ttu-id="cf686-703">拨号计划允许用户按其习惯的方式（例如，对于本地呼叫省略区号，对于国内呼叫省略国家/地区代码，甚至在进行会议拨出时使用简短数字拨号）拨打电话号码。</span><span class="sxs-lookup"><span data-stu-id="cf686-703">A dial plan allows users to dial phone numbers the way they are accustomed to, such as omitting area code for local calls, omitting country code for domestic calls, or even using short digit dialing when performing conference dial out.</span></span>

<span data-ttu-id="cf686-704">在 Office 365 的电话系统功能中，有两种类型的拨号计划：</span><span class="sxs-lookup"><span data-stu-id="cf686-704">Within the Phone System feature of Office 365, there are two types of dial plans:</span></span>

-   <span data-ttu-id="cf686-705">**服务拨号计划**。</span><span class="sxs-lookup"><span data-stu-id="cf686-705">**Service dial plan**.</span></span> <span data-ttu-id="cf686-706">这是默认的拨号计划，基于 Office 365 使用位置适用于用户，无法修改。</span><span class="sxs-lookup"><span data-stu-id="cf686-706">This is the default dial plan and applied to users based on Office 365 usage location, and it cannot be modified.</span></span>
-   <span data-ttu-id="cf686-707">**租户拨号计划**。</span><span class="sxs-lookup"><span data-stu-id="cf686-707">**Tenant dial plan**.</span></span> <span data-ttu-id="cf686-708">这是租户中的自定义拨号计划，进一步分成两种类型：</span><span class="sxs-lookup"><span data-stu-id="cf686-708">This is a customizable dial plan within a tenant, and further divided into two types:</span></span>
    -   <span data-ttu-id="cf686-709">**租户-全局拨号计划** - 该拨号计划适用于租户中的所有用户。</span><span class="sxs-lookup"><span data-stu-id="cf686-709">**Tenant-global dial plan**—the dial plan applies to all users within the tenant.</span></span>
    -   <span data-ttu-id="cf686-710">**租户-用户拨号计划** - 该拨号计划仅适用于特定用户。</span><span class="sxs-lookup"><span data-stu-id="cf686-710">**Tenant-user dial plan**—the dial plan applies only to specific users.</span></span>

> [!NOTE]
> <span data-ttu-id="cf686-711">有关进一步详细信息和示例，请参阅[什么是拨号计划？](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans)文档。</span><span class="sxs-lookup"><span data-stu-id="cf686-711">Check out the [What are dial plans?](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/what-are-dial-plans) documentation for further details and examples.</span></span>

<span data-ttu-id="cf686-712">分配给用户的有效拨号计划是服务拨号计划（基于用户的 Office 365 使用位置）和租户拨号计划（可以是租户-全局拨号计划或租户-用户拨号计划）的组合。</span><span class="sxs-lookup"><span data-stu-id="cf686-712">The effective dial plan assigned to users is the combination of service dial plan (based on user’s Office 365 usage location) and tenant dial plan (can be either tenant-global dial plan or tenant-user dial plan).</span></span>
