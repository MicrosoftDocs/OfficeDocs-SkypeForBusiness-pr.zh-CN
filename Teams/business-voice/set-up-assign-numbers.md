<span data-ttu-id="53f27-104">在用户可以使用 Teams拨打或接听常规电话线路的电话呼叫之前，你需要为其分配电话号码。</span><span class="sxs-lookup"><span data-stu-id="53f27-104">Before users can use Teams to make or receive phone calls to or from regular phone lines, you need to assign phone numbers to them.</span></span> <span data-ttu-id="53f27-105">在Microsoft Teams，当用户单击"呼叫"时，将列出你 **分配的电话号码**。</span><span class="sxs-lookup"><span data-stu-id="53f27-105">In Microsoft Teams, the phone number that you assign is listed when a user clicks **Calls**.</span></span> <span data-ttu-id="53f27-106">为需要电话号码的每个用户执行以下操作。</span><span class="sxs-lookup"><span data-stu-id="53f27-106">Do the following for each user that needs a phone number.</span></span>

> [!NOTE]
> <span data-ttu-id="53f27-108">如果看不到任何电话号码，请稍候。</span><span class="sxs-lookup"><span data-stu-id="53f27-108">If you don't see any phone numbers, please wait.</span></span> <span data-ttu-id="53f27-109">新电话号码可能需要几个小时才能在 Teams。</span><span class="sxs-lookup"><span data-stu-id="53f27-109">It may take several hours for your new phone numbers to become available in Teams.</span></span>

3. <span data-ttu-id="53f27-112">在"**编辑"** 窗格的 **"** 分配到"下，按显示名称或用户名搜索用户，然后单击"分配 **"。**</span><span class="sxs-lookup"><span data-stu-id="53f27-112">In the **Edit** pane, under **Assigned to**, search for the user by display name or user name, and then click **Assign**.</span></span>

5. <span data-ttu-id="53f27-114">确保 **已选择"电子邮件用户，** 包含电话号码信息"，以便用户收到包含其新电话号码的欢迎电子邮件。</span><span class="sxs-lookup"><span data-stu-id="53f27-114">Make sure **Email user with telephone number information** is selected so the user receives a welcome email with their new phone number.</span></span>
