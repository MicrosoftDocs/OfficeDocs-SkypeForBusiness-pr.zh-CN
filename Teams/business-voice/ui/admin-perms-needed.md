# <a name="admin-permissions-needed-to-buy-microsoft-365-business-voice"></a>购买 Microsoft 365 Business Voice 所需的管理员权限

你好！ 你已被发送到此页面，因为你尝试购买[Microsoft 365 Business Voice](../whats-business-voice.md)许可证，但没有执行此操作所需的权限。

首先，如果您意外地单击了某个链接，并且不意味着试用并购买商业语音许可证，您只需关闭此页面即可。 如果你**想要**购买商业语音许可证（或者你有兴趣了解有关业务语音的详细信息），请继续阅读！

## <a name="how-can-i-buy-business-voice"></a>如何购买业务语音？

商业语音是 Microsoft 365 和 Office 365 的加载项。 您需要先注册 Microsoft 365 或 Office 365，然后才能购买业务语音。 注册后，或者如果你已有 microsoft 365 或 Office 365 订阅，则可以通过转到[microsoft 365 管理中心](https://admin.microsoft.com)  >  **Billing**  >  **购买服务**  >  **加载项**（滚动到页面底部）来购买商业语音附加许可证，> **Microsoft 365 商业版语音**。

## <a name="why-dont-i-see-an-option-to-buy-business-voice"></a>为什么我看不到购买业务语音的选项？

如果您无法访问 Microsoft 365 管理中心，如果看不到用于购买业务语音的选项，或者如果您收到错误，则可能是由于以下原因之一：

- **权限不足**-若要向 Microsoft 365 或 Office 365 组织添加许可证（包括业务语音），您的帐户需要是全局管理员或需要分配有帐单管理员角色。 请与您的 Microsoft 365 或 Office 365 管理员联系，获取合适的权限或为您添加业务语音许可证。
- **登录到非管理员帐户**-如果你是 Microsoft 365 或 Office 365 组织的全局管理员或帐单管理员，并且按照建议的做法，让 "常规" 用户帐户与管理员帐户分开，请注销你的常规用户帐户并登录到你的管理员帐户。
- **未登录**-确保你已登录到要管理的 Microsoft 365 或 Office 365 组织中的全局管理员帐户或帐单管理员帐户。
- **登录了错误的组织**-如果你有权访问多个 Microsoft 365 或 Office 365 组织，请确保你已登录到正确的组织。 您需要注销当前的 Office 365 帐户，并登录到要管理的组织中的帐户。
- 目前，**您**的企业语音目前在有限数量的国家和地区可用。 查看什么是业务[语音](../whats-business-voice.md)，查看您所在的企业语音是否可用。

## <a name="what-if-im-still-having-trouble"></a>如果仍然遇到问题，该怎么办？

如果您已检查上述所有原因，但仍无法购买业务语音，请参阅[联系业务产品支持-管理员帮助](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products)。
