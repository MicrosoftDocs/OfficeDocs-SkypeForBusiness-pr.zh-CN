# <a name="set-up-the-teams-phone-system-with-calling-plan-bundle-for-small-to-medium-businesses"></a>使用适用于Teams 电话系统企业的呼叫计划捆绑包设置套餐

"Teams 电话系统套餐"捆绑包是一个完整的电话系统，可以替换现有的电话提供商。 无论您是第一次设置电话号码的新企业，还是从旧版本地电话提供商发展的业务，这些文章中的步骤都可以帮助你通过呼叫计划捆绑包启动并运行 电话系统。 完成设置后：

* 你将能够接听主要公司电话线路上的收费或免费电话呼叫。 您甚至可以设置呼叫菜单（如果需要）。
* 使用"呼叫Teams 电话套餐许可证设置套餐的用户将具有其自己的直接拨号电话号码，他们可以使用这些号码从任何安装了 Teams 的设备拨打和接听电话。
* 如果会议参与者无法从客户端加入会议，他们将能够使用常规电话Teams会议。
* 如果您有现有电话号码，那么在将电话号码移动到呼叫套餐电话系统继续使用它们。

若要详细了解适用于中小型企业的电话系统套餐套餐，请查看适用于Teams 电话系统[和中型企业的指南](whats-business-voice.md)。

> [!IMPORTANT]
> Microsoft Teams和电话系统仅在用户的邮箱位于 Microsoft 365 (（Microsoft 365 Teams Essentials 订阅者) ）时) 。 不支持本地 Exchange Server 上的邮箱。

## <a name="before-you-begin"></a>开始之前

在设置套餐电话系统套餐之前，需要执行一些操作。 以下任务将确保您的组织准备就绪。

* 购买 **Teams 电话** 套餐许可证的套餐，如果要获取免费电话号码、进行远程电话呼叫或购买通信积分，请参阅使用呼叫计划捆绑许可证购买 [Teams 电话](whats-business-voice.md#purchase-teams-phone-with-calling-plan-bundle-licenses)。
* **请确保 Internet 连接可以支持电话系统**。 有关详细信息，请参阅[检查 Internet 连接Teams 电话系统](get-ready-internet.md)。

使用"呼叫计划"捆绑包为组织电话系统，选择"下一步：开始使用呼叫计划Teams 电话系统 **套餐**。
