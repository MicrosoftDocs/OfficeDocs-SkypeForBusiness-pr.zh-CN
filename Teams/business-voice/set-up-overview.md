# <a name="set-up-the-teams-phone-system-with-calling-plan-bundle-for-small-to-medium-businesses"></a>为中小型企业设置“通话套餐”捆绑包的Teams 电话系统

通话套餐捆绑包的Teams 电话系统是一个完整的电话系统，可以替换现有的电话提供商。 无论是首次设置电话号码的新业务，还是从旧式本地电话提供商迁移的已建立业务，这些文章中的步骤都可以帮助你通过通话套餐捆绑包启动和运行电话系统。 设置完成后：

* 你将能够在公司主电话线路上收到收费或免费电话呼叫。 如果需要，甚至可以设置呼叫菜单。
* 使用呼叫计划捆绑包许可证设置Teams 电话的用户将拥有自己的直拨电话号码，这些电话号码可用于从安装了Teams的任何设备拨打和接听电话。
* 如果会议参与者无法从Teams客户端加入，他们将能够使用常规电话呼叫会议。
* 如果你有现有的电话号码，你将能够继续使用它们后，他们被移动到电话系统呼叫计划。

若要了解有关中小型企业通话套餐捆绑包电话系统的详细信息，[请查看Teams 电话系统针对中小企业的指导](whats-business-voice.md)。

> [!IMPORTANT]
> Microsoft Teams和电话系统仅当用户的邮箱位于Microsoft 365 (（不包括 Microsoft 365 Teams Essentials 订阅者) ）时才起作用。 不支持本地 Exchange Server 上的邮箱。

## <a name="before-you-begin"></a>开始之前

在使用通话套餐捆绑包设置电话系统之前，需要执行一些操作。 以下任务将确保你的组织已准备就绪。

* **使用通话套餐捆绑许可证购买Teams 电话**，如果想要获取免费号码、拨打长途电话或购买通信额度，请参阅 [“使用通话套餐捆绑包许可证购买Teams 电话](whats-business-voice.md#how-do-i-purchase-teams-phone-with-calling-plan-bundle-licenses)”。
* **确保 Internet 连接可以支持电话系统**。 有关详细信息，请[参阅检查 Internet 连接是否Teams 电话系统](get-ready-internet.md)。

为组织准备呼叫计划捆绑包的电话系统后，选择 **“下一步："开始"菜单使用呼叫计划设置Teams 电话系统**。
