# <a name="step-6-set-up-an-auto-attendant-for-your-companys-main-phone-number"></a>步骤 6：为公司主电话号码设置自动助理

使用自动助理可以设置呼叫菜单，以便呼叫主电话号码的呼叫者可以从选项列表中选择。 例如，你可以选择连接到销售或支持部门、特定个人、切换面板，甚至其他自动助理。
