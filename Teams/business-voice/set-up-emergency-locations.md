# <a name="step-1-set-up-a-teams-phone-system-emergency-location"></a>步骤 1：设置 Teams 电话系统紧急位置

当组织中的某人呼叫消防、警察或救护车等紧急服务时，将使用紧急位置。 当一个人调用紧急服务时，配置为组织紧急地址的地址将发送到服务。 此步骤为组织设置主要紧急位置。 在后续步骤中，此位置将与公司的主要电话号码相关联。

如果你有用户在多个位置，如家庭办公室或办公室在其他城市，你可以配置其他紧急位置。 甚至可以在某个位置内配置特定位置。 位置可以是不同的建筑物、楼层、办公室或用户可能位于某个位置的其他位置。 使用呼叫计划完成 Teams 电话系统的初始设置后，可以添加其他位置和位置。

## <a name="add-an-emergency-location"></a>添加紧急位置

1. 打开 Microsoft Teams 管理中心，并使用全局管理员的用户帐户登录。此帐户通常是用于注册 Microsoft 365 的帐户。
2. 在左侧导航窗格中，转到 <a href="https://admin.teams.microsoft.com/locations" target="_blank">**“位置** > **紧急”地址**</a>。
3. 单击“**添加**”。
4. 输入位置的名称和说明。
5. 选择国家或地区，然后输入地址。

   > [!NOTE]
   > 在比利时、法国、德国、爱尔兰、荷兰和西班牙，请务必了解，若要在 Microsoft 365 或Office 365中成功激活电话号码，在紧急位置设置的地址（用于获取号码）必须与电话号码的区域代码匹配。

6. 如果找不到该地址，并且要手动编辑地址，请 **手动打开“编辑地址**”。
7. 单击“**保存**”。
