# <a name="whats-new-for-operator-connect-operators"></a>运算符连接运算符的新增功能

本文介绍运算符连接中的新增功能。 请经常回头查看更新。

## <a name="dr-to-oc-migration-api"></a>DR 到 OC 迁移 API

2022 年 7 月 1 日，Microsoft 发布了一个适用于操作员的 API，可以在其中将数字从直接路由迁移到 Operator Connect。 

有关详细信息，请参阅 [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#tag/TnMigration-(New))。

## <a name="number-status"></a>数字状态

2022 年 7 月 1 日，Microsoft 发布了一个适用于操作员的 API，该 API 用于提供有关同意租户的给定数字的操作员状态信息。

有关详细信息，请参阅 [Operator Connect Swagger](https://operatorconnect.microsoft.com/api/swagger#operation/NumberStatus_GetNumberStatus)。
