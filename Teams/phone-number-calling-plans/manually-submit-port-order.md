# <a name="manually-submit-a-port-order"></a>手动提交转网订单

在某些国家和地区，可能需要手动提交服务请求以获取电话号码、转接电话号码、释放电话号码或更改地址。 若要查看每个国家/地区需要什么，或了解有关号码移植的详细信息，请参阅 [管理组织的电话号码](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)。

如果国家或地区未在 [Microsoft 的移植向导 ](transfer-phone-numbers-to-teams.md)中列出，请使用本文中的步骤手动提交端口订单。

## <a name="manually-submit-a-new-port-order-request"></a>手动提交新的端口订单请求

若要手动提交新的端口订单，请将已完成的授权书 (LOA) 表单直接发送到区域的 TNS 服务台。

1. 下载所在国家或地区的 [LOA](../manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md) 。
2. 完成窗体。
3. 将其发送到 [区域的 TNS 服务台](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md) 。

## <a name="what-else-should-you-know-about-number-porting"></a>关于数字移植，还应了解哪些信息

- 若要使用通话套餐，必须购买许可证并向用户分配许可证。 请参阅 [Teams 加载项许可](../teams-add-on-licensing/microsoft-teams-add-on-licensing.md)。

- 你必须将这些新电话号码分配给你的每个用户。 请参阅 [为用户分配、更改或删除电话号码](../assign-change-or-remove-a-phone-number-for-a-user.md)。
