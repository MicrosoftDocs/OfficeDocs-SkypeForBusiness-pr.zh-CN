# <a name="whats-the-status-of-your-port-orders"></a>你的转网订单的状态是什么？

若要查看端口订单的状态，请在 Microsoft Teams 管理中心的左侧导航栏中，转到> **语音** > **端口订单**，然后单击 **订单历史记录**。 状态 **列中** 列出了每个端口订单状态。 查看端口号了解订单过程 [需要多长时间](../phone-number-calling-plans/port-order-overview.md#how-long-does-it-take-to-port-numbers) 。 

下表列出了端口订单状态，以及可以根据需要执行的操作。

|**地位**|**是否可以查看订单？**|**是否可以编辑订单？**|**是否可以取消订单？**|**是否可以删除订单？**|**说明**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**处理** <br/> |是  <br/> |否  <br/> |是  <br/> |否  <br/> |管理员创建了订单，并且已由 Microsoft 接收。  <br/> |
|**联系运营商** <br/> |是  <br/> |否  <br/> |是  <br/> |否  <br/> |该订单已获得 Microsoft 的接收和批准，我们正在与丢失的运营商合作以获得批准。  <br/> |
|**已批准的转让** <br/> |是  <br/> |否  <br/> |是  <br/> |否  <br/> |损失的运营商已接受该订单，并且已设置公司订单承诺 (FOC) 日期。  <br/> |
|**传输挂起** <br/> |是  <br/> |否  <br/> |否  <br/> |否  <br/> |传输距离不到 24 小时，因此无法再编辑或取消订单。  <br/> |
|**Error** <br/> |否  <br/> |是  <br/> |是  <br/> |是 (此时，如果出现错误，则无法删除端口顺序。 需要重新创建端口订单，或者需要联系 [TNS 服务台](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md)。  <br/> |丢失的运营商拒绝了订单。  <br/> |
|**完成** <br/> |是  <br/> |否  <br/> |否  <br/> |否  <br/> |数字已成功转移。  <br/> |
|**取消** <br/> |否  <br/> |是  <br/> |否  <br/> |否  <br/> |管理员取消了订单。  <br/> |

有关完整的分步说明，请参阅 [将电话号码传输到 Teams](transfer-phone-numbers-to-teams.md)。

如果需要帮助或需要获取更多电话号码，请联系 [TNS 服务台](../manage-phone-numbers-for-your-organization/contact-tns-service-desk.md)。
