# <a name="supported-rate-centers-and-coverage-for-calling-plans"></a>通话套餐支持的费率中心和覆盖范围

本文列出了北美地区通话套餐支持的费率中心和覆盖范围。 费率中心是一个地理区域的行业术语，传统上，该地理区域定义公共电话交换网 PSTN (的本地呼叫、计费费率和电话号码) 。 在许多情况下，随着行业转向所有非独占计划或分钟数捆绑包，费率中心对于计费已变得不太重要，但仍被一些用户使用。

北美 [的费率中心和覆盖区域矩阵列出了](https://www.microsoft.com/download/details.aspx?id=102534) 我们支持的费率中心。 收到新电话号码时，或者将电话号码从现有提供商转移给提供商Teams下载电子表格，并使用它查找费率中心。 如果不知道费率中心，可以在 Internet 上根据 NXX 的区号 (NXX) 和 (前缀) 它。
如果你收到新号码，但我们没有你请求的号码，我们将尝试从同一个费率中心提供号码。

如果您收到新号码，但我们没有您请求的数字，我们将提供来自同一费率中心的数字。
