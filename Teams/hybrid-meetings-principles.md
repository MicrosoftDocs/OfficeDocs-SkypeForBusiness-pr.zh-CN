# <a name="teams-hybrid-meeting-principles"></a>Teams 混合会议原则

:::image type="content" source="media/hybrid-audience-itpro-small.png" alt-text="IT Pro 受众" border="false":::

无论使用何种技术，会议原则都是包容性会议的核心。 这些原则定义会议参与者如何相互交互、如何共享信息以及如何启用有效的协作。 使用以下原则作为你为组织定义的原则的基础。 在组织中与领导和个人参与者（亲自和远程、有残疾和无残疾）一起查看你的原则。

其中每个原则都有助于定义步骤 2 中概述的会议最佳 [做法 - 宣传会议最佳做法](hybrid-meetings-educate-participants.md)。

## <a name="equality-and-inclusivity"></a>相等性和包容性

所有参与者，无论是亲自还是远程，都期望并值得倾听，并有机会参与协作。 残疾参与者需要获得技术和材料，使他们能够充分参加会议。

以下 Teams 功能可以帮助支持此原则：

* Teams 聊天进行旁对话
* 前行以查看其他位置的参与者
* 残障人士的会议脚本、实时隐藏式字幕和录音

## <a name="be-seen-and-heard"></a>被看到和听到

所有参与者，特别是那些远程参与者，都应该感到有权让他们在会议中了解自己的存在，并参与对话。

以下 Teams 功能可以帮助支持此原则：

* 室内伴侣加入，以便会议室参与者可以在设备上参与会议聊天、使用反应等
* 相机支持，包括对会议室内的支持，以便参与者可以显示他们的状态和对他人的反应
* “举手”反应，以便参与者可以指示他们想参与讨论
* 对内容和讨论表现出情绪的反应

## <a name="see-and-listen"></a>查看和侦听

所有参与者，特别是当面参与者，都应确保有机会，以便任何想参与对话的人都能。

以下 Teams 功能可以帮助支持此原则：

* 室内伴侣加入，以便会议室参与者可以在设备上关注会议聊天、使用反应等
* 前排，以便这些会议室内的参与者可以轻松查看远程参与者
* 针对残障参与者的实时隐藏式字幕
* “举手”反应，以便所有参与者都能看到谁想要参与对话
* 查看内容和讨论情绪的反应

## <a name="collaborate"></a>合作

所有参与者都应努力使用虚拟协作工具，以确保每个人都能够参与讨论。

以下 Teams 功能可以帮助支持此原则：

* 分组讨论的分组会议室
* 从设备共享内容
* 用于无缝集思广益的虚拟白板
* 会议聊天，以旁讨论所呈现的内容
* 用于跟踪会议分钟数和操作的共享笔记

## <a name="request-and-provide-feedback"></a>请求并提供反馈

会议组织者和与会者之间的公开和诚实沟通，无论是在会议期间还是之后，都对会议的成功至关重要。

* 会议聊天，使组织者和参与者了解会议中的问题
* “举手”操作以引起组织者或演示者的注意
* 会议投票，让演示者和组织者向参与者提问
* 会后调查，以便在会议结束后发送详细反馈
