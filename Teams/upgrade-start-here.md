# <a name="getting-started-with-your-microsoft-teams-upgrade"></a>您的 Microsoft 团队入门升级

## <a name="why-upgrade-to-microsoft-teams"></a>为什么要升级到 Microsoft 团队？

Microsoft 团队提供中心位置的团队协作。 扩展的 Skype for Business 的功能，团队融合聊天，会议，调用，协作、 应用程序集成和文件存储单一界面，简化用户获取，又完成任务的方式，可帮助驱动器用户满意度并加快业务产出。 为业务客户现有 Skype，我们邀请您升级到 Microsoft 团队，以体验此全套中的单个客户端体验的通信和协作功能。 

## <a name="when-should-my-organization-upgrade-to-teams"></a>何时我的组织应向工作组升级？

若要优化团队的福利总的说来，我们鼓励客户尽快升级到团队。 我们了解更改可能需要花费时间而，即使不准备刚刚尚未升级，您仍可以启动今天实现团队的好处。 部署团队旁 for Business 的 Skype 允许用户熟悉团队现在可帮助加快随着时间的推移升级到团队。 事实上，我们的许多客户看到该库到团队 shift 其通信活动的用户。  
 
## <a name="how-is-microsoft-helping-customers-with-their-upgrade-to-teams"></a>如何为 Microsoft 帮助客户使用其升级到团队？ 

Microsoft 致力于您成功，并提供了稳固的指南，围绕可帮助规划和执行升级到团队的经验证、 端到端升级成功框架。 我们指南旨在为升级旅程，每个阶段刚入门规划，已在运行旁 for Business 的 Skype 团队还是准备好升级到团队。  
 
## <a name="why-do-i-need-upgrade-guidance-cant-i-just-deletedecommission-skype-for-business"></a>为什么需要升级指南？ 不能我刚删除/停用 Skype for Business？ 

从 for Business 的 Skype 升级到团队是多个技术迁移。 它表示在用户如何进行通信和协作，转换，并更改并不总是轻松。 理想的升级方法应升级的技术方面的地址以及鼓励用户接受和采用团队，驱动良好的用户体验和业务结果实现。 

## <a name="where-do-i-start-planning-for-teamsmy-upgrade-to-teams"></a>从开始规划团队 / 我升级到团队？ 

首先您旅程您熟悉我们升级成功的框架。 此框架，以及关联的资源，充当用于导航 for Business 的 Skype 从您迁移到团队的基础。
