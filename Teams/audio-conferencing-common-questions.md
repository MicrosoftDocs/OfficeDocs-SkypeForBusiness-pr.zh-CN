# <a name="audio-conferencing-common-questions"></a>音频会议常见问题

以下是我们从想要使用音频会议的客户那里得到的一些首要问题。
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>音频会议有哪些好处？

例如，当人们在路上（例如，无法使用笔记本电脑或移动设备上的 Microsoft Teams 应用参加会议）时，呼叫加入会议非常有用。 但在其他情况下，使用电话参加 Teams 会议可能比在计算机上使用应用更好：
  
- Internet 连接受限。

- 会议只有音频。

- 此人尝试加入 Teams 会议，但失败。

- 如果拨入，呼叫质量会更好。

- 用户可以使用蓝牙设备“免提”加入会议。

- 用户发现对于他们的情况而言，这样更加简便。

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>谁可以参加音频会议？ 我能听到谁的声音？

具有拨入号码和会议 ID 的任何人都可以加入 Teams 会议，除非会议组织者已锁定会议。
  
无论你是使用电话还是 Teams 应用来电，你都可以听到呼叫中的其他人，并且他们可以听到你的声音。 如果会议组织者不想听到他们的声音，他们能够“静音”会议与会者。
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>是否可以为音频会议用户添加免费号码？

是的，免费电话号码 (服务号码) 可用，但仅适用于某些国家/地区。 有关可用号码的列表，请参阅 [音频会议和通话套餐的国家和地区可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>目前支持多少个本地拨入号码？

购买音频会议许可证时，会为你分配本地拨入号码。 拨入号码将包含在会议邀请之中。 这些本地号码仅可供你的组织使用。 分配给你的组织的电话和该号码由该组织内启用了音频会议的用户共享。 这意味着用户 A 和另一个用户 B 安排的 Teams 会议将具有相同的拨入号码。
  
本地拨入号码以及在某些情况下你的组织所在的国家或地区的国际拨入号码，将包含在会议邀请中。 如果与会者使用邀请中包含的其他号码，该号码将是共享电话号码。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Microsoft 365 或 Office 365 中的音频会议支持多少国际电话拨入号码？

有关国家/地区的当前列表，请参阅 [Microsoft Teams 中音频会议的电话号码](phone-numbers-for-audio-conferencing-in-teams.md)。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>音频会议的最大长度是多少？

最长持续时间取决于与会者以及他们用于加入会议的身份验证类型。
  
|**与会者**|**会议结束时间**|
|:-----|:-----|
|用户已使用 Teams 应用加入或已拨入会议。  <br/> |如果 24 小时后与会者列表没有任何更改，会议结束。  <br/> |
|所有用户都是拨号加入会议，但是部分用户使用了 PIN 进入会议。  <br/> |24 小时后会议结束。  <br/> |
|所有用户都拨入会议，但没有人使用 PIN 进入会议。  <br/> | 如果允许匿名用户开始会议，则会议将在四小时后结束。 如果不允许匿名用户启动会议，并且所有经过身份验证的参与者都已离开会议，则会议将在最后一个经过身份验证的参与者离开后 90 分钟结束。 <br/> |

## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>会议中可以有多少个电话参与者？

请参阅 [限制和规范](limits-specifications-teams.md) ，了解会议中允许的与会者总数。

若要了解会议限制，请参阅 [会议和通话](limits-specifications-teams.md#meetings-and-calls)。
  
## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>为什么用户开始收到包含其音频会议信息的电子邮件？

我们添加了一项新功能，允许 [管理员](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)在电子邮件中发送和更新音频会议信息和 PIN。 若要详细了解它（包括如何禁用它），请参阅[在 Microsoft Teams 中更改音频会议设置时启用或禁用发送电子邮件](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md)。
  
## <a name="can-a-user-get-a-personal-conference-id"></a>用户可以获取个人会议 ID 吗？

Teams 用户将随机分配会议 ID。 用户无法保留或设置只有他们可以使用的静态会议 ID。
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>会议期间，用户可以获得操作员帮助吗？

不可以，会议期间，用户不能按 *0 获得任何操作员帮助或支持。 如果音频会议出现问题，组织的管理员可以联系[Microsoft支持Office 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>用户如何访问或更改他们的会议 ID？

Teams 用户可以通过在 Outlook 中安排会议来查找分配给他们的会议 ID，并Outlook 网页版。 此外，用户可以在设置后发送给他们的电子邮件中找到会议 ID。
  
> [!NOTE]
> [!注释] 用户无法重置他们的会议 ID。 会议 ID 只能由组织 [管理员](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)重置。
  
我们正在开发一种解决方案，使用户无需组织管理员的帮助即可访问和重置会议 ID。

## <a name="how-do-users-access-or-change-their-pin"></a>用户如何访问或更改其 PIN？

Teams 用户可以在设置后发送给他们的电子邮件中找到 PIN。

> [!NOTE]
> Teams 用户可以更改其 PIN。 会议邀请底部有一个用于重置其 PIN 的链接。
  
我们正在开发一种解决方案，该解决方案将允许用户访问和重置 PIN，而无需组织管理员的帮助。

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-teams-meeting"></a>与会者在参加 Teams 会议时是否可以拨打国际电话号码？

是的，与会者可以拨打国际电话，并邀请其他呼叫者加入 Teams 会议。 请参阅[从Microsoft Teams 会议拨出，以便其他人可以加入会议](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md)。
  
## <a name="how-does-a-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Teams 用户如何安排包含音频会议会议详细信息的会议？

向用户分配音频 **会议** 许可证，并且用户在 Outlook 或 Outlook 网页版 中创建新的 Teams 会议时，电话拨入电话号码和会议 ID 会自动添加到会议邀请中。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>当所有与会者都使用电话拨入时，用户如何安排和启动会议？

安排将由所有与会者使用电话拨入加入的会议与安排常规联机会议并无不同。 但是，有两种方法可以启动会议，所有参与者都使用电话拨入会议：

- **选项 1**：默认情况下，如果会议组织者和所有参与者都使用电话加入会议，会议组织者需要输入她的音频会议 PIN 才能启动会议。 当呼叫者拨打在线会议的电话号码时，系统会询问他们是否希望以给定会议的组织者的身份进行身份验证。 在组织者开始之前通过拨入方式加入会议的所有参与者都将放置在大厅中，并会收听保留的音乐。 对于 Teams 会议，参与者将根据组织者会议策略中自动允许人员设置的值加入会议。

- **选项 2**：默认禁用“允许未经身份验证的呼叫者成为会议中的第一个人员”设置。 如果为给定组织者启用了设置，则可以在组织者输入音频会议 PIN 之前启动该组织者安排的所有会议。 启用此设置后，第一个参与者通过拨入电话号码加入会议后，会议将立即开始，并且第一个参与者不会放在大厅中。 有关详细信息，请参阅[管理 Microsoft Teams 中用户的音频会议设置](manage-the-audio-conferencing-settings-for-a-user-in-teams.md)。
