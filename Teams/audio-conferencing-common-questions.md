# <a name="audio-conferencing-common-questions"></a>音频会议常见问题

以下是我们从想要使用音频会议的客户那里得到的一些最常见问题。
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>音频会议有什么好处？

例如，当用户在路上，并且无法在其笔记本电脑或移动设备上使用Skype for Business或 Microsoft Teams 应用参加会议时，呼叫会议是有益的。 但还有其他一些情况，使用手机参加 Skype for Business 或 Microsoft Teams 会议可能比在计算机上使用应用更好：
  
- Internet 连接受限。

- 会议只有音频。

- 用户尝试加入 Skype for Business 会议，但是失败。

- 如果他们拨入，呼叫质量会更好。

- 用户可以使用蓝牙设备“免提”加入会议。

- 用户发现对于他们的情况而言，这样更加简便。

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>谁可以参加音频会议？ 我能听到谁的声音？

拥有拨入号码和会议 ID 的任何人都可以加入Skype for Business或 Microsoft Teams 会议，除非会议组织者锁定了会议。
  
无论是使用电话、Skype for Business还是 Microsoft Teams 应用进行呼叫，你都可以听到其他所有人的呼叫，他们可以听到你的声音。 如果会议组织者不想听到他们的声音，他们能够“静音”会议与会者。
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>是否可以为音频会议用户添加免费号码？

是的，免费电话号码 (服务号码) 可用，但仅在某些国家/地区。 有关可用数字的列表，请参阅 [音频会议和通话套餐的国家/地区可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>目前支持多少个本地拨入号码？

购买音频会议许可证时，会向你分配本地拨入号码。 拨入号码将包含在会议邀请之中。 这些本地号码仅可供你的组织使用。 分配给组织的电话和该号码由为音频会议启用的组织内的用户共享。 因此，用户 A 和其他用户 B 安排的Skype for Business或 Microsoft Teams 会议都将具有相同的拨入号码。
  
本地拨入号码以及在某些情况下你的组织所在的国家或地区的国际拨入号码，将包含在会议邀请中。 如果会议与会者使用邀请中包含的不同号码，则该号码将是共享电话号码。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Microsoft 365 或 Office 365 中的音频会议支持多少个国际电话拨入号码？

有关国家/地区的当前列表，请参阅 [Microsoft Teams 中音频会议的电话号码](phone-numbers-for-audio-conferencing-in-teams.md)或 [Skype for Business Online 中音频会议的电话号码](/SkypeForBusiness/audio-conferencing-in-office-365/phone-numbers-for-audio-conferencing)。
  
## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>是否可以设置来自全国其他城市的音频会议的本地号码？

如果音频会议的电话号码在你所在区域不可用或不能满足组织的需求，请在 [SkypeFeedback 论坛上](https://techcommunity.microsoft.com/t5/skype-for-business-blog/welcome-to-the-skype-for-business-community/ba-p/89273)向我们发送反馈。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>音频会议的最大长度是哪个？

最长持续时间取决于与会者以及他们用于加入会议的身份验证类型。
  
|**与会者**|**会议结束时间**|
|:-----|:-----|
|用户已使用 Skype for Business 或 Microsoft Teams 应用加入，或已拨入会议。  <br/> |如果 24 小时后与会者列表没有任何更改，会议结束。  <br/> |
|所有用户都是拨号加入会议，但是部分用户使用了 PIN 进入会议。  <br/> |24 小时后会议结束。  <br/> |
|所有用户都拨入会议，但没有任何用户使用 PIN 进入会议。  <br/> | 如果允许匿名用户开始会议，会议将在四小时后结束。 如果不允许匿名用户开始会议，并且所有经过身份验证的参与者都离开了会议，则会议将在最后一个经过身份验证的参与者离开后 90 分钟结束。 <br/> |

## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>在会议中，我可以拥有多少个电话参与者？

请参阅会议允许的与会者总数 [的限制和规范](limits-specifications-teams.md) 。
  
若要了解会议限制，请参阅 [Skype for Business Online 限制](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-limits#meeting-limits)。
  
## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>为什么用户开始接收包含其音频会议信息的电子邮件？

我们添加了一项新功能，使 [管理员](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)能够在电子邮件中发送和更新音频会议信息和 PIN。 若要了解有关它的详细信息，包括如何禁用它，请参阅在 [Microsoft Teams 中音频会议设置更改时启用或禁用发送电子邮件](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md)，[或在音频会议设置在 Skype for Business Online 中更改时启用或禁用发送电子邮件](/SkypeForBusiness/audio-conferencing-in-office-365/enable-or-disable-sending-emails-when-their-settings-change)。
  
## <a name="can-audio-conferencing-be-used-by-the-users-who-are-part-of-an-on-premises-deployment-of-skype-for-business-server"></a>作为本地部署Skype for Business Server的一部分的用户是否可以使用音频会议？

借[助“会议第一](meetings-first.md)”，Skype for Business Server用户可以从 Teams 会议和 Teams 音频会议中受益。
  
## <a name="can-a-user-get-a-personal-conference-id"></a>用户可以获取个人会议 ID 吗？

Skype for Business和 Microsoft Teams 用户将被随机分配会议 ID。 用户不能保留或设置只有他们才能使用的静态会议 ID。
  
## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>是否可以将音频会议与Skype 会议广播配合使用？

目前不支持为音频会议启用的用户加入Skype 会议广播。
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>会议期间，用户可以获得操作员帮助吗？

不可以，会议期间，用户不能按 *0 获得任何操作员帮助或支持。 如果音频会议出现问题，组织管理员可以联系 [Microsoft 支持人员进行Office 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>用户如何访问或更改他们的会议 ID？

Skype for Business或 Microsoft Teams 用户可以通过在 Outlook 和 Outlook 网页版 中安排会议来查找分配给他们的会议 ID。 此外，用户可以在设置后将发送给他们的电子邮件中找到会议 ID。
  
> [!NOTE]
> [!注释] 用户无法重置他们的会议 ID。 只能由组织 [管理员](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)重置会议 ID。
  
我们正在研究一种解决方案，该解决方案允许用户在没有组织管理员帮助的情况下访问和重置会议 ID。
  
## <a name="how-do-users-access-or-change-their-pin"></a>用户如何访问或更改其 PIN？

Skype for Business或 Microsoft Teams 用户可以在设置后将发送给他们的电子邮件中找到 PIN。
  
> [!NOTE]
> Teams 用户可以更改其引脚。 会议邀请底部有一个链接，用于重置其引脚。
  
我们正在研究一种解决方案，该解决方案允许用户在没有组织管理员帮助的情况下访问和重置 PIN。
  
## <a name="what-in-meeting-dial-pad-commands-are-supported"></a>支持哪些会议中拨号盘命令？

对于会议参与者和组织者：

| 命令名称    | 命令密钥 | 说明                                                                 |
|--------------   |--------------|-----------------------------------------------------------------------------|
| 命令菜单    | \*1          | 按 \*1 播放可用拨号板命令的说明。 |
| 举手/下手| \*5          | 按 \*5 以举手或下手。                                           |  
| 静音/取消静音     | \*6          | 按 \*6 将自身静音或取消静音。                                     |

仅对于会议组织者：

| 命令名称                   | 命令密钥 | 说明                                                 |
|--------------------------------|--------------|-------------------------------------------------------------|
| 全部允许                      | \*21         | 按 \*21 允许在大厅中等待的所有参与者。  |
| 将全部静音                       | \*22         | 按 \*22 将所有参与者静音 (组织者) 除外。 |
| 切换 Enter/Exit 公告 | \*23         | 按 \*23 启用或禁用 Enter/Exit 公告。    |

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-skype-for-business-or-microsoft-teams-meeting"></a>与会者在参加Skype for Business或 Microsoft Teams 会议时能否拨打国际电话号码？

是的，与会者可以在国际上拨打电话，邀请其他呼叫者参加Skype for Business或 Microsoft Teams 会议。 请参阅[从 Microsoft Teams 会议中拨出，以便其他人可以加入或](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md)[从Skype for Business联机会议中拨出，以便其他人可以加入。](/SkypeForBusiness/audio-conferencing-in-office-365/dialing-out-from-a-meeting-so-other-people-can-join-it)
  
## <a name="how-does-a-skype-for-business-or-microsoft-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>Skype for Business或 Microsoft Teams 用户如何安排与音频会议会议的会议详细信息？

为用户分配 **音频会议** 许可证，并且用户在 Outlook 或 Outlook 网页版 中创建新的Skype for Business或 Microsoft Teams 会议时，电话拨入电话号码和会议 ID 会自动添加到会议邀请中。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>当所有与会者都将使用电话拨入时，用户如何安排和启动会议？

安排所有与会者将使用电话拨入的会议，与安排定期联机会议并无不同。 但是，有两种方法可以启动会议，所有参与者都使用电话拨入：

- **选项 #1**：默认情况下，如果会议组织者和所有参与者都使用电话加入会议，则会议组织者需要输入她的音频会议 PIN 来启动它。 当呼叫者拨打在线会议的电话号码时，系统会询问他们是否要作为给定会议的组织者进行身份验证。 所有在组织者开始前通过电话拨入加入会议的参与者都将被放置在大厅中，并会按住收听音乐。 对于Skype for Business会议，一旦组织者通过输入其音频会议 PIN 启动会议，大厅中的所有参与者将自动加入会议。 对于 Microsoft Teams 会议，参与者将根据组织者会议策略中自动允许人员设置的值加入会议。

- **选项 #2**：默认禁用“允许未经身份验证的呼叫者成为会议中的第一个人”设置。 如果为给定组织者启用了设置，则可以在组织者输入音频会议 PIN 之前启动由该组织者安排的所有会议。 启用此设置后，一旦第一个参与者通过拨入电话号码加入会议，会议就会开始，第一个参与者将不会放在大厅中。 有关详细信息，请参阅：[在 Microsoft Teams 中管理用户的音频会议设置](manage-the-audio-conferencing-settings-for-a-user-in-teams.md)，或[在 Skype for Business Online 中管理用户的音频会议设置](/SkypeForBusiness/audio-conferencing-in-office-365/manage-the-audio-conferencing-settings-for-a-user)。
