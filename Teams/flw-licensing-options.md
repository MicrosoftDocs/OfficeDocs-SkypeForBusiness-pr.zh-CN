# <a name="understand-frontline-worker-user-types-and-licensing"></a>了解一线工作人员用户类型和许可

许多行业（如零售业、制造业、酒店业等）中的组织都具有混合的用户类型：

- 信息工作者，他们可能拥有台式机或笔记本电脑，并且处理文档方面的工作可能多于直接与客户打交道，例如总部员工。
- 一线员工，他们经常在平板电脑或手机上工作，并且直接与客户或普通公众交流。 他们提供服务、支持和销售产品，或者是直接参与产品和服务的制造和分发的员工。 例如：零售助理、医疗保健临床医生和护理人员、工厂工人等。

Microsoft 365 和 Microsoft Teams 通过其功能和许可模型为组织向这两种类型用户提供支持。 [适用于一线员工的 Microsoft 365](https://www.microsoft.com/microsoft-365/enterprise/frontline) 针对经常移动的员工进行了优化，这些员工主要与客户交互，但也需要与组织的其他成员保持联系。

## <a name="assign-appropriate-licenses-to-support-frontline-worker-and-information-worker-user-types"></a>分配适当的许可证以支持一线员工和信息工作者用户类型

如果这一点适用于你的组织，则可以向这些不同类型的用户分配不同的许可证。 下图显示了为零售员工应用许可证和策略的一种方法，例如：

:::image type="content" source="./media/flw-license-options.png" alt-text="混合许可证类型，以支持组织中的不同角色。为信息工作者（例如总部员工）使用企业许可证，并为移动员工（例如商店经理、销售助理、送货员、库管员和拣货员）使用一线员工许可证。":::

使用此示例，可以为组织中两种类型的工作人员混合使用许可证类型，如下表所示：

| 一线员工需要快速访问可帮助他们与客户协作的工具： | 信息工作者需要工具来支持其任务，例如： |
| ----- | ----- |
| 一线员工需要随时随地访问信息 - 他们在手机或平板电脑上拥有 Office 应用。 | 信息工作者编写、编辑和共享文档，并在台式机或笔记本电脑上拥有 Office 应用，以及手机和平板电脑上的移动应用。 |
| 一线员工随时随地使用 Teams 聊天进行通信、使用 Shifts 安排工作、、使用 Tasks 签出待办事项、跟踪 Lists 中的项目，以及其他 Teams 功能。  | 信息工作者通过 Teams 使用聊天、开会和通话进行通信，并使用其他 Teams 应用（具体取决于特定许可证）。 |
| 一线员工可以使用 SharePoint、OneDrive、Yammer 和 Stream 查找组织范围的资源、通信或培训。 | 信息工作者使用随附的 Microsoft 365 服务创建或使用团队或组织范围的资源和通信（具体取决于特定的许可证）。 |

有关配备各种许可证的 Microsoft 365 所包含内容的详细比较，请参阅此 [比较表](https://go.microsoft.com/fwlink/?linkid=2139145)。
