# <a name="microsoft-teams-displays"></a>Microsoft 团队显示

Microsoft 团队显示是一种由多人专用团队设备组成的类别，这些设备提供了一个环境触摸屏和一个由 Cortana 提供支持的无人参与体验。 本文提供团队显示的概述，并可帮助你规划、交付和管理组织中的团队显示。

团队显示将你喜爱的团队的功能汇集 &ndash; &ndash; 到一个设备中。 随着团队的显示，用户可以使用麦克风、相机和扬声器 (或蓝牙耳机) 进行可靠的通话和会议体验。 团队显示与用户的 Windows 电脑集成，以提供支持无缝的跨设备交互的配套体验。

若要了解详细信息，请查看 [团队显示入门](https://support.microsoft.com/office/get-started-with-teams-displays-ff299825-7f13-4528-96c2-1d3437e6d4e6)。

## <a name="features-supported-by-teams-displays"></a>团队支持的功能显示

除了 [团队手机支持的功能](phones-for-teams.md#features-supported-by-teams-phones)之外，以下功能对于团队显示是唯一的：

- **团队的专用显示** 用户可以访问所有核心团队功能，包括聊天、会议、通话、团队和频道、文件等。
- **环境体验** 用户可以使用 "始终启用" 和 "glanceable" 显示屏轻松地保持其工作，以便查看重要的活动和通知，而不会在其主要工作设备上进行上下文切换。 用户还可以通过使用 "设置" 自定义背景来对团队显示进行个性化设置。
- **与 Cortana 一起免提** 用户可以通过语音与团队的语音进行交互，以轻松加入会议、在会议中演示和演示，决定团队聊天的答复，检查日历上的内容等。
- **在锁屏界面上留下笔记** 来宾可以选择离开音频、视频和文本笔记，用户可以查看来宾留下的笔记并查看谁停止了。  

## <a name="required-licenses"></a>所需的许可证

团队许可证可以作为 [Microsoft 365 和 Office 365 订阅](https://docs.microsoft.com/office365/servicedescriptions/teams-service-description)的一部分进行购买。 若要了解有关使用团队显示所需许可证的详细信息，请参阅 [使用 Microsoft 团队进行语音和视频通话](https://products.office.com/microsoft-teams/voice-calling)。

有关如何获取团队的详细信息，请查看 [如何获取 Microsoft 团队的访问权限？](https://support.office.com/article/fc7f1634-abd3-4f26-a597-9df16e4ca65b)

## <a name="deploy-teams-displays-using-intune"></a>使用 Intune 显示部署团队

若要了解有关如何使用 Intune 部署团队的详细信息，请参阅 [部署团队手机和团队显示](phones-displays-deploy.md)。

## <a name="manage-teams-displays-in-your-organization"></a>管理组织中的团队显示

若要管理团队显示设备，请在 Microsoft 团队管理中心的左侧导航中，转到 " **团队显示** "。 在此处，你可以更改设备配置文件、管理更新、重启设备、添加和删除设备标记等。 有关详细信息，请参阅 [管理团队中的设备](device-management.md)。

## <a name="upgrade-teams-phones-to-teams-displays"></a>将团队的电话升级到团队显示

团队显示是团队手机的演变。 你可以使用 Microsoft 团队管理中心将你的组织中的团队电话升级到团队。 此选项仅适用于支持升级到团队的电话。 若要了解详细信息，请参阅 [将团队手机升级到团队显示](upgrade-phones-to-displays.md)。
