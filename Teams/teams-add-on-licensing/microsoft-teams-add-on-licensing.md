# <a name="microsoft-teams-add-on-licenses"></a>Microsoft Teams 附加许可证

## <a name="what-are-add-on-licenses"></a>什么是附加许可证？

附加许可证是针对特定 Microsoft Teams 功能的许可证。 它们可让你灵活地仅为组织中需要这些功能的用户添加功能。 若要添加功能，为每个将使用该功能的用户购买一个附加许可证。

## <a name="what-voice-features-are-available-with-my-plan"></a>我的计划提供哪些语音功能？

查看在 Teams 中获取语音功能所需的附加许可证，具体取决于你拥有的计划。

根据你需要的语音功能，我们建议比较切换到包含这些功能的计划是否更具成本效益。 例如，对于大多数大型组织，在 Microsoft 365 企业版计划中购买一系列软件可以降低总体成本。 通常，单独购买多个附加许可证而不是作为计划的一部分购买时，最终总费用可能会更高。

#### <a name="small-business"></a>[**小型企业**](#tab/small-business/)

如果你是小型或中型企业用户 (用户不到 300) ，则有四种方法可获取语音功能：

- 购买 Microsoft 365 商业版计划和 Microsoft 365 商业语音。
- 购买企业版 E1 或 E3 计划以及 Microsoft 365 商业语音。
- 购买企业版 E1 或 E3 计划并单独添加语音功能。
- 购买包括语音功能的企业 E5 计划。

对于 Microsoft 365 商业语音，有两个选项可供选择：

- **具有呼叫计划的 Microsoft 365 商业** 语音：如果希望由 Microsoft 处理所有电话号码管理和计费，请获取此套餐。
- 没有呼叫计划的 **Microsoft 365 Business Voice (直接路由) ：** 如果您希望您的第三方电话提供商处理所有电话号码管理和计费，请获取此语音。 需要设置和部署直接路由，以将[](../direct-routing-landing-page.md)本地会话边界控制器 (SDC) Microsoft 365 或 Office 365。

若要了解有关详细信息，请参阅 ["使用 Microsoft 365 商业](../business-voice/what-to-buy.md)语音需要购买什么？"。

下面是基于许可计划的摘要。

  ![小型企业计划选项](../media/licensing-smallbusiness-options.png)

如果音频会议在你的国家/地区不受支持，你可选择不带音频会议计划的 Microsoft 365 企业版 E5 或不带音频会议计划的 Office 365 企业版 E5。

如果已有 Microsoft 365 企业版或 Office 365 企业版计划，仍可以购买单个语音功能的附加许可证。 但是，可以选择切换到 [Microsoft 365 商业](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab:primaryr2) 版计划和 Microsoft [365 商业语音](https://www.microsoft.com/microsoft-365/business/business-voice)。 在某些情况下，根据你需要的功能，使用 Microsoft 365 商业语音获取 Microsoft 365 商业版计划可能更具成本效益。

> [!NOTE]
> 如果需要使用第三方电话服务，直接路由部署在你的[](../direct-routing-landing-page.md)组织中，或者如果你的通话计划在你的国家/地区不可用，则需要使用直接路由作为语音功能。
