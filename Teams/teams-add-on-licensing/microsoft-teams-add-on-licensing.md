# <a name="microsoft-teams-add-on-licenses"></a>Microsoft Teams 附加许可证

## <a name="what-are-add-on-licenses"></a>什么是附加许可证？

附加许可证是针对特定 Microsoft Teams 功能的许可证。 它们可让你灵活地仅为组织中需要这些功能的用户添加功能。 若要添加功能，为每个将使用该功能的用户购买一个附加许可证。

## <a name="what-voice-features-are-available-with-my-plan"></a>我的计划提供哪些语音功能？

查看在 Teams 中获取语音功能所需的附加许可证，具体取决于你拥有的计划。

根据你需要的语音功能，我们建议比较切换到包含这些功能的计划是否更具成本效益。 例如，对于大多数大型组织而言，在 Microsoft 365 企业版计划中购买一系列软件可以降低总体成本。 通常，单独购买多个附加许可证而不是作为计划的一部分购买时，最终综合成本可能会更高。

#### <a name="small-business"></a>[**小型企业**](#tab/small-business/)

如果你是一家小型或中型企业， (用户不到 300) ，则有四种方法可获取语音功能：

- 购买 Microsoft 365 商业版计划和 Microsoft 365 商业语音。
- 购买企业版 E1 或 E3 计划和 Microsoft 365 商业语音。
- 购买企业版 E1 或 E3 计划并单独添加语音功能。
- 购买包括语音功能的企业 E5 计划。

- **具有呼叫计划的 Microsoft 365 商业** 语音：如果希望由 Microsoft 处理所有电话号码管理和计费，请获取此号码。

若要了解有关详细信息，请参阅我需要购买哪些[产品以使用 Microsoft 365 商业语音？。](../business-voice/what-to-buy.md)
