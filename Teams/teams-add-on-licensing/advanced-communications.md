# <a name="advanced-communications-add-on-for-microsoft-teams"></a>Microsoft Teams 的高级通信加载项

如果你的组织希望进一步增强你的通信体验，请获取高级通信以使用高级会议自定义和工具来管理组织的通信策略。

## <a name="what-is-advanced-communications"></a>什么是高级通信？

高级通信包括高级会议自定义和用于管理组织通信策略的工具。 其他功能包括自定义品牌会议大厅来定制会议，以及用于监视、跟踪和分析用户和设备上数据的工具。

- **定制和自定义会议：** 使用自定义品牌会议大厅等功能，为内部方案和面向自定义的方案促进跨会议的标准化。 在组织各个部门之间灵活实施。
- **管理组织通信**：监视、跟踪和分析用户和设备上的数据，以确保顺畅的体验。

高级通信作为任何付费 Microsoft 365 或 Office 365 订阅的附加内容在所有 Microsoft 365 市场中提供。 高级通信中包含的功能不会添加到其他 Microsoft 365 或 Office 365 计划，并且无法单独购买。 若要获取这些功能，必须购买附加内容。

> [!NOTE]
> 功能可用时将发布。 有关最新功能时间，请访问 [Microsoft 365](https://www.microsoft.com/microsoft-365/roadmap?filters=Microsoft%20Teams) 公共路线图。

> [!NOTE]
> 高级通信尚不适用于 Microsoft 365 GCC 高级版或国防部 (DoD) 环境。

## <a name="trial-offer"></a>试用产品/服务

从 2020 年 8 月 4 日开始提供 60 天的试用版。 可以通过 Microsoft 365 管理中心获取试用版。
