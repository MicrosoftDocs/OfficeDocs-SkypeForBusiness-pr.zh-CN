# <a name="advanced-communications-add-on-for-microsoft-teams"></a>Microsoft Teams 的高级通信加载项

如果你的组织希望进一步增强你的通信体验，请获取高级通信以进行高级会议自定义和工具来管理组织的通信策略。

## <a name="what-is-advanced-communications"></a>什么是高级通信？

高级通信包括高级会议自定义、用于管理组织的通信策略的工具、使用自定义策略包实现的高级可管理性，以及定制会议的自定义品牌会议大厅。

- 在预加入和大厅中使用公司徽标自定义会议预加入体验。 
- 使用专为公司和会议需求设计的自定义“一起模式”场景创建统一会议体验。
- 使用 Teams 管理员中心或 PowerShell 为组织中具有类似角色的用户捆绑自己的一组策略，为自定义策略包定制体验。 每个用户都需要许可证才能接收自定义策略包分配。 

高级通信可作为任何付费 Microsoft 365 或Office 365订阅的附加内容在所有 Microsoft 365 市场中使用。 高级通信中包含的功能不会添加到其他 Microsoft 365 或Office 365计划中，也无法单独购买。 这些功能暂时在公共预览版中提供给所有 Microsoft Teams 客户。 若要在预览后获取功能，将提前通知用户如何通过加载项许可证继续其服务。

> [!NOTE]
> 功能在可用时将发布。 有关最新功能计时，请访问 [Microsoft 365 公共路线图](https://www.microsoft.com/microsoft-365/roadmap?filters=Microsoft%20Teams) 。

> [!NOTE]
> 高级通信尚不适用于 Microsoft 365 GCC High 或国防部 (DoD) 环境。
