# <a name="messaging-policies-licensing"></a>消息策略许可

我们将听取全球各地客户的来信，COVID （Coronavirus） pandemic 在其日常运营中处理重大变动的方式带来了新的挑战。

在较快的条件下，能够直接和快速地与团队成员保持联系是维护业务连续性的关键，因此我们正在将促销访问扩展到 Microsoft 团队中的关键消息功能。

最初在2019年6月发布的情况下，团队的[优先级通知](https://support.microsoft.com/article/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462)授予用户在所有标记为*紧急*的邮件上最多20分钟持续发送两分钟的通知的能力。 在发布时，我们已将优先级通知提供给所有团队用户在2020年4月1日之前的促销活动。 

**为帮助我们的客户和社区在其危机响应工作中，我们将在2020的第二部分扩展优先级通知促销。** 对优先级通知的持续访问将在危机响应和恢复工作中提供。 提升期结束后，许可用户将能够根据其订阅条款发送优先级通知。

管理员将获得有关优先级通知使用情况的报告，以支持相应的用户许可。 无论许可证如何，用户都可以通过优先级通知收到不受限制的紧急邮件。 [了解有关优先级通知的详细信息](../messaging-policies-in-teams.md)。
