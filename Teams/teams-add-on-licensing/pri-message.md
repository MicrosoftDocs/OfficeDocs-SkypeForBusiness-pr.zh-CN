# <a name="messaging-policies-licensing"></a>消息策略许可

我们将听取全球各地客户的来信，COVID （Coronavirus） pandemic 在其日常运营中处理重大变动的方式带来了新的挑战。 在较快的条件中，能够直接快速地与团队成员保持联系对于在危机响应过程中维护业务连续性非常重要，并且许多人返回到工作场所。

在2019年6月发布的情况下，Microsoft 团队的[优先级通知](https://support.microsoft.com/article/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462)允许用户在所有标记为紧急的邮件上持续发送最多20分钟的通知，每隔两分钟重复。 已将优先级通知提供给所有团队客户<sup>1</sup> ，直到2020的第二一半。

现在，我们已结束本次促销，<sup>并为所有团队客户提供</sup>了优先级通知，这是一种基本功能。 用户现在可以通过优先级通知发送和接收不受限制的紧急邮件。 [了解有关管理优先级通知的详细信息](../messaging-policies-in-teams.md)。

<sup>1</sup>优先级通知当前不可用于 GCC、GCC 高或 DoD 客户。
