# <a name="messaging-policies-licensing"></a>消息策略许可

在2020年4月1日之前，Microsoft 团队中的无限制优先级通知可供所有用户使用。 此促销已延长为2020年4月31日（2019）的原始结束日期。 4月1日之后，许可用户将能够根据其订阅条款发送优先级通知。

如6月2019日宣布的，当用户有权访问 "优先级通知" 功能时，用户可以将其标记为 "紧急邮件"。 将使用优先级通知发送紧急邮件，通知收件人每隔两（2分钟）分钟长达20分钟，或者收件人阅读邮件。 此功能最大程度地提高了邮件的处理可能性。

IT 管理员将获得有关优先级通知使用情况的报告，以支持相应的用户许可。 无论许可证如何，用户都可以通过优先级通知收到不受限制的紧急邮件。 [了解有关优先级通知的详细信息](https://docs.microsoft.com/MicrosoftTeams/messaging-policies-in-teams)。
