## <a name="introduction-to-qos-queues"></a><span data-ttu-id="1a8c7-137">QoS 队列简介</span><span class="sxs-lookup"><span data-stu-id="1a8c7-137">Introduction to QoS queues</span></span>

<span data-ttu-id="1a8c7-138">为了提供 QoS, 网络设备必须能够对流量进行分类, 并且必须能够将语音或视频与其他网络流量区分开。</span><span class="sxs-lookup"><span data-stu-id="1a8c7-138">To provide QoS, network devices must have a way to classify traffic and must be able to distinguish voice or video from other network traffic.</span></span>

<span data-ttu-id="1a8c7-139">当网络流量进入路由器时, 流量将放入队列中。</span><span class="sxs-lookup"><span data-stu-id="1a8c7-139">When network traffic enters a router, the traffic is placed into a queue.</span></span> <span data-ttu-id="1a8c7-140">如果未配置 QoS 策略, 则仅有一个队列, 所有数据将按具有相同优先级的第一项处理。</span><span class="sxs-lookup"><span data-stu-id="1a8c7-140">If a QoS policy isn't configured, there is only one queue, and all data is treated as first-in, first-out with the same priority.</span></span> <span data-ttu-id="1a8c7-141">这意味着语音流量 (对延迟非常敏感) 可能会陷入通信量不足的情况下, 延迟几个额外的毫秒不会出现问题。</span><span class="sxs-lookup"><span data-stu-id="1a8c7-141">That means voice traffic (which is very sensitive to delays) might get stuck behind traffic where a delay of a few extra milliseconds wouldn't be a problem.</span></span>

<span data-ttu-id="1a8c7-142">当你实现 QoS 时, 你可以使用以下几个拥塞管理功能之一定义多个队列 (如 Cisco 的优先级队列和基于类的加权公平队列[CBWFQ](https://www.cisco.com/en/US/docs/ios/12_0t/12_0t5/feature/guide/cbwfq.html#wp17641)) 和拥塞避免功能 (如加权随机的提前检测[WRED](https://en.wikipedia.org/wiki/Weighted_random_early_detection))。</span><span class="sxs-lookup"><span data-stu-id="1a8c7-142">When you implement QoS, you define multiple queues using one of several congestion management features (such as Cisco’s priority queuing and Class-Based Weighted Fair Queueing [CBWFQ](https://www.cisco.com/en/US/docs/ios/12_0t/12_0t5/feature/guide/cbwfq.html#wp17641)) and congestion avoidance features (such as weighted random early detection [WRED](https://en.wikipedia.org/wiki/Weighted_random_early_detection)).</span></span>
