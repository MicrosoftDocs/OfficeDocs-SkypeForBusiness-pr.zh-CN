# <a name="manage-the-whiteboard-in-microsoft-teams"></a>在 Microsoft Teams 中管理白板

Microsoft Whiteboard 是一种自由形式的数字画布，人们、内容和创意汇集在一起。 Microsoft Teams 会议中的 Whiteboard 集成由 Whiteboard Web 应用提供支持，该应用允许 Teams 会议参与者在共享的数字画布上一起绘制、草图和写入。

用户可以共享一个白板，使其可供 Teams 会议中的所有参与者使用。 同一白板在 Windows 10、iOS 和 Web 应用上的所有 Whiteboard 应用程序中同时可用。

若要打开或关闭 Whiteboard 应用，请参阅 [为组织启用 Microsoft Whiteboard](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24)。 请记住，此设置将为整个组织启用或禁用 Whiteboard，而不仅仅是 Teams。

## <a name="whiteboard-on-surface-hub"></a>Surface Hub 上的白板

在尝试使用 Microsoft Whiteboard 之前，请确保在 Surface Hub 设备上安装了 Whiteboard 应用。 如果未安装 Whiteboard，请在 Surface Hub 设备上转到 Microsoft Store 应用并获取 [Microsoft Whiteboard](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab)。 有关详细信息，请参阅 [在 Surface Hub 上启用 Microsoft Whiteboard](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627)。
