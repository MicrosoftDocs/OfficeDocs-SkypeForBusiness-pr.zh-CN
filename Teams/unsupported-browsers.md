<span data-ttu-id="7b36a-104">某些浏览器（如 Internet Explorer 11、Safari 和 Firefox）支持 Microsoft 团队 web 应用，但不支持某些团队通话和会议功能。</span><span class="sxs-lookup"><span data-stu-id="7b36a-104">Some browsers, such as Internet Explorer 11, Safari, and Firefox, support the Microsoft Teams web app but don't support some of the Teams calling and meeting features.</span></span> <span data-ttu-id="7b36a-105">为了解决此限制，"团队" web 应用允许用户通过 PSTN 连接接收音频，并让用户以更低的显示费率查看所显示的内容（屏幕共享）。</span><span class="sxs-lookup"><span data-stu-id="7b36a-105">To work around this limitation, the Teams web app lets users receive audio through a PSTN connection and lets them view presented content (screen share) at a reduced display rate.</span></span>

<span data-ttu-id="7b36a-106">当团队检测到不受支持的浏览器时，它会自动显示一条消息，说明问题和会话限制。</span><span class="sxs-lookup"><span data-stu-id="7b36a-106">When Teams detects an unsupported browser, it automatically displays a message explaining the issue and the session limitations.</span></span> <span data-ttu-id="7b36a-107">该消息提供有关访问会议音频的进一步说明，例如，建议用户保留回拨号码，以便团队可以呼叫用户，或指示用户呼叫会议邀请中包含的会议号码。</span><span class="sxs-lookup"><span data-stu-id="7b36a-107">The message provides further instructions for accessing the meeting audio, such as advising the user to leave a call back number so that Teams can call the user, or instructing the user to call the conference number included in the meeting invitation.</span></span> <span data-ttu-id="7b36a-108">该消息还鼓励用户下载和使用[团队桌面客户端](https://teams.microsoft.com/downloads)，以获得完整的团队体验。</span><span class="sxs-lookup"><span data-stu-id="7b36a-108">The message also encourages the user to download and use the [Teams desktop client](https://teams.microsoft.com/downloads) for the full Teams experience.</span></span>
