# <a name="phone-number-management-for-australia"></a>澳大利亚的电话号码管理

在澳大利亚，我们已与 Telstra 合作，为具有澳大利亚帐单地址的客户购买和预配呼叫计划。 Telstra 将销售其通话套餐并提供澳大利亚用户号码。 Telstra 为自动助理和呼叫队列的用户和服务号码提供电话号码管理支持。 但是，音频会议的服务号码和许可证仍由 Microsoft 365 和 Office 365 出售和管理。

有关 [Telstra 呼叫详细信息](https://aka.ms/TelstraVoicePlan)。

使用此表查找有关获取和管理澳大利亚 Microsoft Teams 的电话号码的信息。

有关详细信息，请参阅["管理组织的电话号码"。](manage-phone-numbers-for-your-organization.md)
  
|**你要做什么？**|**执行下列步骤并下载正确的表单**|
|:-----|:-----|
|获取澳大利亚通话计划的新用户电话号码。   <br/> |请联系 [Telstra Calling，](https://aka.ms/TelstraVoicePlan) 为用户提供新电话号码。 然后，他们将这些号码添加到你的组织，以便你可以将它们分配给你的用户。 <br/>
|如果使用 Telstra 呼叫，可能会获得可用于云自动助理的新服务电话号码，以及来自 Telstra 的澳大利亚呼叫队列。 <br/> |有关[新电话号码，请联系 Telstra Calling。](https://aka.ms/TelstraVoicePlan) 然后，他们将这些号码添加到你的组织，以便你可以将其分配给自动助理和呼叫队列。 如果不 **使用** Telstra 呼叫，可以联系我们的 Microsoft 号码管理服务台，从 Microsoft 获取服务号码 [。请与我们联系](mailto:ptnapac@microsoft.com)。 <br/>|
|获取新的澳大利亚音频会议服务电话号码。   <br/> |[请联系我们](mailto:ptnapac@microsoft.com)。|
|获取转接到 Microsoft 365 或 Office 365 的用户、自动助理和呼叫队列的电话号码。  <br/> | 请联系 [Telstra 呼叫](https://aka.ms/TelstraVoicePlan) 以转移电话号码。 然后，他们将这些数字添加到你的组织。  <br/> |
|获取转移到 Microsoft 365 或 Office 365 以用于音频会议的电话号码。  |[联系我们](mailto:ptnapac@microsoft.com) |
