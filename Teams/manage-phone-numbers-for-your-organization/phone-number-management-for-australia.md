# <a name="phone-number-management-for-australia"></a>澳大利亚的电话号码管理
在澳大利亚，我们已与 Telstra 合作，为使用澳大利亚帐单地址的客户购买和预配通话计划。 Telstra 将销售通话计划、提供澳大利亚用户号码，同时为自动助理和呼叫队列提供用户和服务号码的电话号码管理支持。 但是，音频会议的服务号码和许可证仍将由 Microsoft 365 和 Office 365 进行销售和管理。

有关[Telstra 通话](https://aka.ms/TelstraVoicePlan)的详细信息。

使用此表查找有关在澳大利亚 for Microsoft 团队中获取和管理电话号码的信息。

有关详细信息，请参阅[管理你的组织的电话号码](manage-phone-numbers-for-your-organization.md)。
  
|**你要做什么？**|**执行下列步骤并下载正确的表单**|
|:-----|:-----|
|获取澳大利亚通话计划的新用户电话号码。   <br/> |请与[Telstra 通话，拨打](https://aka.ms/TelstraVoicePlan)您的用户的新电话号码。 稍后将相关号码添加至您所属组织，以便您向您的用户进行分发。 <br/>
|如果您使用的是 Telstra 呼叫，您可能会收到新的服务电话号码，可与云自动助理一起使用，并可通过 Telstra 通话 "澳大利亚通话队列"。 <br/> |请联系[Telstra 拨打](https://aka.ms/TelstraVoicePlan)新电话号码。 稍后将相关号码添加至您所属组织，以便您向您的自动语音和呼叫队列进行分发。 如果您使用的不是 Telstra 通话，您可以联系我们的 Microsoft 号码 ManagementService 桌面获取来自 Microsoft 的服务号码，[请](mailto:ptnapac@microsoft.com)与我们联系。 <br/>|
|获取新的澳大利亚音频会议服务电话号码。   <br/> |[请联系我们](mailto:ptnapac@microsoft.com)。|
|获取为用户、自动助理和呼叫队列转移到 Microsoft 365 或 Office 365 的电话号码。  <br/> |请联系[Telstra 通话](https://aka.ms/TelstraVoicePlan)以转移电话号码。 稍后将相关号码添加到您所在组织。  <br/> |
|获取转接到 Microsoft 365 或 Office 365 的电话号码，以便与音频会议配合使用。  |[请联系我们](mailto:ptnapac@microsoft.com) |
|更改已分配给加入呼叫计划电话号码的用户的紧急呼叫地址。 |请联系[Telstra 通话](https://aka.ms/TelstraVoicePlan)以进行紧急地址更改。 他们可以为您更新相关信息。|
|将用户号码更改为服务号码。 |使用 Telstra 通话时受支持。
