# <a name="manage-phone-numbers-for-your-organization"></a>管理组织的电话号码

有三种方法可以为您的组织获取用户和服务（收费和免费）号码。
- 在 Microsoft 团队管理中心中获取数字。
- 把您现有的号码转移或“迁移”给我们。
- 对未在 Microsoft 团队管理中心中列出的新号码使用请求表单。

> [!IMPORTANT]
> 在比利时、法国、德国、爱尔兰、荷兰和西班牙, 您不能使用 Microsoft 团队管理中心获取电话新电话号码。 您必须下载并填写表单。 在澳大利亚的新电话号码，请参阅[Telstra 语音服务](https://aka.ms/TelstraVoicePlan)。

## <a name="heres-a-bit-more-information-about-these-number-request-forms"></a>下面是有关这些号码申请表的更多信息

### <a name="request-forms-for-new-phone-numbers"></a>新电话号码的申请表

每个国家或地区都有不同的说明、不同类型的号码 (地理/非地域和服务 (收费/免费电话) 和用于获取电话号码的规则/规章, 以便在团队中使用。

有时 (取决于您所在的国家/地区) 您将无法使用 Microsoft 团队管理中心获取新用户 (订阅者) 或服务 (收费或免费) 号码, 或者您可能需要特定的电话号码或区号。

如果是这样, 您将需要下载正确的 PDF 表单 (每个国家/地区和号码的类型), 并将其发送给我们。 我们的服务台将从这里获取, 并将让你了解进度或你的订单。

### <a name="letters-of-authorization-loas-for-transferring-numbers"></a>转让号码的授权书（LOAs）

如果您需要转移电话号码，您将下载  **授权书（LOA）**   ，这使我们可以代表您请求在发生争议时从不同的服务提供商处转移现有号码。 请使用 LOA 如果有以下情况：
- 您已从其他运营商那里获得用户号码。
- 音频会议网桥、自动助理或呼叫队列服务 （收费）号码。
- 服务 （免费）电话号码。
- 您需要获得超过999个用户电话号码, 但无法使用 "使用本地号码移植向导" 访问 Microsoft 团队管理中心。

## <a name="if-you-still-need-help-and-arent-sure-what-to-do"></a>如果你仍然需要帮助并且不知道该怎么办……。

如果您仍然需要帮助, 请告诉我们我们可以为您提供哪些帮助。 在您的帮助请求中，包括您的组织 ID、域、您需要帮助的号码类型、所需数量、授权人帐户上以及有关您的问题的其他重要详细信息。
- 有关美国内部电话号码的帮助， [发送您的请求](mailto:ptn@microsoft.com)  。
- 有关美国以外的电话号码的帮助，  [发送您的请求](mailto:ptneu@microsoft.com) 。
