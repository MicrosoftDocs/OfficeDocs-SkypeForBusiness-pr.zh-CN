| **我想要转移/移植现有号码**  | **我想订购的新号码** |
|:-----:|:------:|
| [**美国或加拿大**境内，通过电子邮件发送 ptn@microsoft.com](mailto:ptn@microsoft.com)| [**比利时、法国、德国、英国、爱尔兰、荷兰、西班牙、瑞典、瑞士**、通过电子邮件发送给 ptneu@microsoft.com](mailto:ptneu@microsoft.com)|
|[在**美国或加拿大以外**，通过电子邮件发送给我们的 ptneu@microsoft.com](mailto:ptneu@microsoft.com)| [**世界各地的其他**人通过电子邮件 gcstnmsd@microsoft.com](mailto:gcstnmsd@microsoft.com)|

## <a name="transfer-or-port-your-existing-phone-numbers-to-microsoft-teams"></a>将现有电话号码转移或移植到 Microsoft 团队
将电子邮件发送到上表中相应的电子邮件别名。 包括尽可能多的详细信息，例如：
  - 你的 [组织 id (租户 ID) 或当前活动的 Microsoft 域](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)。
  - 您需要帮助的电话号码类型;如用户、服务收费或免费服务。
  - 基于你的请求的其他重要详细信息。

## <a name="get-new-phone-numbers-for-microsoft-teams"></a>为 Microsoft 团队获取新的电话号码
将电子邮件发送到上表中相应的电子邮件别名。 包括尽可能多的详细信息，例如：
  - 你的 [组织 id (租户 ID) 或当前活动的 Microsoft 域。](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)
  - 您需要帮助的电话号码类型;如用户、服务收费或免费服务。
  - 您所需的电话号码的数量、区号 (s) 或国家或地区，您希望从这些号码发出新电话号码。
