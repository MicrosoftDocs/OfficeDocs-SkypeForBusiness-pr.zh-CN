# <a name="what-are-communications-credits"></a>什么是通信点数？

通信点数是支付音频会议和通话套餐分钟数的便捷方式。 它有助于确保你和你的用户在无法：
  
- 添加免费号码以用于音频会议、自动助理或呼叫队列。 免费呼叫按分钟计费，需要通信积分余额为正数。

- 从音频会议中拨出，以从世界任何地方添加其他人。

- 使用 Microsoft Teams 应用或安装Skype for Business应用从音频会议会议拨出到移动电话，这些应用未包含在订阅中。

- 当你拥有 **国内通话套餐** 订阅时，请拨打任何国际电话号码。

- 拨打国内和国际 **通话套餐** 订阅中包含的号码以外的国际电话号码。

- 一旦用完每月一分钟的分配，请拨出并按分钟付费。

- 如果你有即用即付通话套餐，则拨出并按分钟支付所有传出呼叫。

## <a name="can-i-use-communication-credits"></a>是否可以使用通信点数？

并非所有客户都可以或应该使用通信额度。 下面是不能或不应使用通信点数的几个原因。

### <a name="call-destinations-may-be-included-in-your-subscription"></a>你的订阅中可能包含呼叫目标

对某些目标的出站呼叫可能包含在音频会议或通话套餐订阅中。 在购买不需要的通信点数之前，请查看订阅信息以了解详细信息。
  
### <a name="conflicting-organization-addresses"></a>冲突的组织地址

如果组织所在的区域与 企业协议 (EA) 的帐单邮寄地址不同，则可能无法购买通信点数。

如果无法购买通信点数，请从门户创建支持事件，我们将与你一起缓解此问题，直到永久解决方案到位。

### <a name="customers-with-new-commerce-experience-calling-subscriptions"></a>具有新商务体验呼叫订阅的客户

NCE)  (新的商业体验允许客户在使用服务后支付服务费用，也称为使用后计费。

由于通信额度是用于支持传出分钟数的预付费预算，因此无法使用 NCE 呼叫订阅的客户购买。

相反，NCE 客户将在使用超额传出几分钟后支付费用。 不需要通信点数池。

有关呼叫订阅的新商业体验的详细信息，请参阅 [为订阅启用即用即付](/microsoft-365/commerce/subscriptions/manage-pay-as-you-go-services) 和 [电信即用即付的新商业超额](/partner-center/new-commerce-telco-payg)。

## <a name="what-is-it"></a>它是什么？

若要使用免费号码，需要通信额度，因为这些呼叫按分钟计费。 此外，建议为需要拨出到 **任何 PSTN 目标的** 通话套餐和音频会议用户设置通信额度。 包括许多国家/地区，但某些目标可能未包含在通话套餐或音频会议订阅中。 如果未设置通信点数或向用户分配许可证，并且组织 (的通话套餐、音频会议计划或国家/地区) ，则这些用户将无法拨打电话或拨出在线音频会议。
  
> [!NOTE]
> 必须设置通信点数，然后为 **Microsoft 365 管理中心** > **用户** > **活动** 用户中的每个用户分配通信点数许可证。 通信点数许可证将显示为无限数量，因为它们用于授予你向其分配通信点数许可证的任何用户的访问权限。
  
建议仔细评估组织的 PSTN 服务使用情况，以便了解通信额度的最佳设置。 可以通过阅读[电话系统和通话套餐以及](calling-plan-landing-page.md)[音频会议常见问题](Audio-Conferencing-common-questions.md)来详细了解 PSTN 服务。
  
可以使用预付费余额的一 **次性资金** 设置消耗计费， **或者** 通过设置 **自动充值** ( **建议** 的) 金额和最小余额来触发该充值金额的购买。 一开始，建议你设置自动充值金额，以便你监视实际的使用量并找到满足你的需求和使用模式的合适的自动充值阈值。 应随着时间的推移监视通信信用额度，然后根据需要调整最小余额和充值金额，以满足每个组织的需求。
  
> [!NOTE]

在 [Microsoft 365 管理中心](https://go.microsoft.com/fwlink/p/?linkid=2024339)中，转到 **“计费** > **购买服务** > **附加** 项”，你可以选择“通信信用额度”来添加信用额度，并且可以随时手动添加信用额度。 You can also enable the **Auto-recharge** option, which will allow automatic account refills when the balance falls below the threshold that you set. 如果选择使用一次性金额为通信点数余额提供资金，然后余额降至零，则上述其他通话方案将不再有效，包括免费电话号码。 因此，建议使用 **“自动充值** ”设置，以避免在通信点数余额达到 0 (零) 时中断服务。 可以通过转到 **计费** > **产品** > 通信额度来查看当前 **通信额度** 余额。
