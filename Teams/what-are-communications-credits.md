# <a name="what-are-communications-credits"></a>什么是通信点数？

通信额度是支付音频会议和通话套餐分钟数的便捷方式。 它有助于确保你和你的用户在不能够的情况下永远不会被捕获：
  
- 添加用于音频会议、自动助理或呼叫队列的免费号码。 免费呼叫每分钟计费，需要一个正通信额度余额。

- 从音频会议中拨出，从世界任何地方添加其他人。

- 使用安装到订阅中尚未包含的 Microsoft Teams 应用或Skype for Business应用，从音频会议到移动电话。

- 拥有 **国内通话套餐** 订阅时，请拨打任何国际电话号码。

- 拨打国际电话号码，超出 **国内和国际通话套餐** 订阅中包含的内容。

- 用尽每月的分钟分配后，拨打电话，每分钟付款一次。

- 如果有即用即付呼叫计划，则拨打电话并每分钟支付所有传出呼叫费用。

> [!NOTE]
> 对某些目标的出站呼叫可能包含在音频会议订阅中。 有关详细信息，请查看订阅信息。
  
> [!NOTE]
> 如果组织位于与企业协议 (EA) 的计费地址不同的区域中，则可能无法购买通信额度。 如果无法获取通信额度，请从门户中打开支持事件，我们将与你合作，在永久解决方案到位之前缓解此问题。
  
## <a name="what-are-the-communications-credits-rates"></a>什么是通信额度率？

如果想知道通话套餐的通话费用是多少，请参阅 [基于云的电话系统进行语音呼叫](https://go.microsoft.com/fwlink/p/?LinkId=799523) (向下滚动到“查看要呼叫的位置的费率”) 。
  
## <a name="what-is-it"></a>它是什么？

若要使用免费号码，需要通信额度，因为这些呼叫每分钟计费。 此外，我们建议为需要拨出到 **任何 PSTN 目标** 的呼叫计划和音频会议用户设置通信额度。 包括许多国家/地区，但某些目标可能不包括在通话套餐或音频会议订阅中。 如果未设置通信额度或为用户分配许可证，并且组织 (会根据通话计划、音频会议计划或国家/地区) 运行分钟数，则这些用户将无法拨打电话或从联机音频会议拨号。
  
> [!NOTE]
> 必须设置通信额度，然后为 **Microsoft 365 管理中心** > **Users** > **Active Users** 中的每个用户分配通信信用额度许可证。 通信信用额度许可证在数量上将显示为无限数量，因为它们用于为分配通信信用额度许可证的任何用户授予对通信信用额度余额的访问权限。
  
建议仔细评估组织的 PSTN 服务使用情况，以了解通信额度的最佳设置。 可以通过阅读[电话系统、通话套餐和](calling-plan-landing-page.md)[音频会议常见问题](Audio-Conferencing-common-questions.md)来了解有关 PSTN 服务的详细信息。
  
可以通过设置 **自动充值** (**建议** 的) 金额（最小余额）来设置使用预付费余额 **或** 的一 **次性资金** 来设置消费计费，这将触发该充值金额的购买。 一开始，建议你设置自动充值金额，以便你监视实际的使用量并找到满足你的需求和使用模式的合适的自动充值阈值。 通信额度应随时间推移进行监视，然后根据需要根据每个组织的需求调整最小余额和充值金额。
  
> [!NOTE]
> 可以通过 [登录到Microsoft 365 管理中心](https://portal.office.com/adminportal/home?add=sub&amp;adminportal=1#/catalog)并转到计 **费** > **订阅** > **添加订阅** 来查看计划和定价。
  
在 **Microsoft 365 管理中心** > **Billing** > **Purchase Services** > **Add Ons 中**，可以选择通信额度以添加信用额度，并且可以随时手动添加信用额度。 You can also enable the **Auto-recharge** option, which will allow automatic account refills when the balance falls below the threshold that you set. 如果选择以一次性金额为通信额度余额提供资金，然后余额降至零，则上述其他呼叫方案将不再起作用，包括免费电话号码。 因此，如果通信额度余额达到 0 (零) ，建议使用 **自动充值** 设置来避免服务中断。 可以通过对 **产品** > 通信额度计 **费** > 来查看当前通信 **额度** 余额。

> [!NOTE]
>你将通过 Lync 服务管理员、公司管理员、计费管理员、公司管理员、用户帐户管理员、支持管理员、服务支持管理员、计费管理员、Lync 服务管理员、设备管理员、应用程序管理员、许可证管理员、云设备管理员、身份验证管理员、特权身份验证管理员、Teams 通信管理员、Teams 通信支持工程师，Teams 通信支持专家，Teams 管理员管理员角色，当充值交易成功时，充值事务失败 (如信用卡) 过期，或者通信信用余额达到 0 (零) 。

每个组织将有不同的通话套餐使用量和费率需要考虑。 需要从当前服务提供商获取此类使用情况数据。 对于已将 Teams 用作其服务提供商的组织，可以通过在 Microsoft Teams 管理中心>分析&报告 **使用** 情况 **报告** > 中查看使用情况数据，然后选择 **PSTN 和 SMS (预览) 使用情况来获取使用情况** 数据。

## <a name="recommended-funding-amounts"></a>推荐的存款金额

我们建议你首先使用少量的资金，对于较小的组织来说，这甚至至少可以是 50 美元。 使用 PSTN 使用情况详细信息报告收集一两个月的 PSTN 使用情况后，你将能够调整金额以满足组织中的正常使用情况。
  
如果已准备好设置这一切，请参阅 [为组织设置通信额度](set-up-communications-credits-for-your-organization.md)。
