# <a name="create-your-champions-program-for-microsoft-teams"></a>为 Microsoft Teams 创建支持者计划

冠军对于提高组织中的认识、采用和教育至关重要。 冠军是一个人，主要通过帮助他人，谁有兴趣的新技术 (特别是 Teams) ，并帮助其他员工使用它符合你的最佳做法。 根据组织的规模，你可能将此角色作为人员角色的正式部分，但员工通常会自己承担此角色，因为他们的核心动机是帮助他人。

冠军将以各种方式适应整体启动计划，如下所示。

![冠军启动计划的插图。](media/teams-adoption-champions.png)

冠军应：

- 接受正式培训，提高知识的深度和广度
- 鼓励和授权指导、教书和训练同龄人
- 有一致和积极的强化，确认他们的努力的影响
- 有一个明确的计划要执行

使用我们的 [冠军计划指南](https://aka.ms/M365Champions) 在组织中生成此计划。 我们对此社区的最佳做法是：

- 加入 [冠军计划](https://aka.ms/O365Champions)。 请确保冠军计划的领导者至少 (，并且可能所有公司冠军) 都是该计划的成员。 这种免费的每月社区呼叫将为运行自己的计划以及充分利用 Teams 和其他 Microsoft 365 或Office 365服务提供有价值的信息。 可以在自己的内部程序中重复使用程序材料。

- 使用 Teamwork 冠军团队：你的冠军应使用你为其设置的团队进行所有程序通信、反馈和查找资源。  这样做，他们将越来越熟悉产品及其好处。

- 定期举行每月会议：定期举行每月会议，提高社区的热情和凝聚力。 这些可以是亲自会议和虚拟会议的组合，但它们的一致性是拥有繁荣的冠军社区的关键。

    > [!TIP]
    > 若要帮助制定议程，请在公众社区呼叫后的一周内安排会议。 

- 明确计划要求：一些冠军计划要求成员参加每月会议，为他们支持的用户组安排办公时间，并为新用户提供培训。 无论你的要求如何，请确保冠军清楚如何成功参与该计划。

- 明显奖励你的冠军：你的冠军将让你对项目的进展和潜在的陷阱有重要见解。 奖励他们经常参与。 小手势大有裨益，员工往往不是在寻找金钱收益，而是感谢对他们贡献的认可。 找到适当的和有趣的方式来感谢你的社区。 

- 尽早经常训练冠军：你的计划开始，每月会议可用于在冠军社区中培养技能。 利用这些机会让他们加入，进行他们最初的培训，鼓励他们参加我们的课程，并确保他们准备好支持他们的员工，因为他们在试验阶段前进。
