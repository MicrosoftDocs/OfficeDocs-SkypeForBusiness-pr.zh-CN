# <a name="create-your-champions-program-for-microsoft-teams"></a>为 Microsoft Teams 创建支持者计划

支持者对于提高组织的认知度、采用率和教育至关重要。 奖杯是一个人，主要通过帮助其他对新技术感兴趣的人， (Teams) 并帮助其他员工根据你的最佳实践使用它。 根据组织的规模，你可能将此角色作为人员角色的正式部分，但员工通常自己承担此角色，因为他们的核心动机是帮助其他人。

支持者会以多种方式参与总体发布规划，如下所示。

![成功推出规划插图](media/teams-adoption-champions.png)

比赛者应：

- 接受正式培训，以提高其深度和知识广度
- 受到鼓励并有权指导、教学和训练其对等方
- 具有一致和积极的强化，可增强他们工作的影响
- 制定明确的计划来执行

使用我们的 [支持计划指南](https://go.microsoft.com/fwlink/?linkid=854665) 在你的组织中构建此计划。 我们针对此社区最佳实践包括：

- 加入 [支持计划](https://aka.ms/O365Champions)。 确保最低要求是"比赛"计划负责人 (可能所有公司) 都是该计划的成员。 此免费的每月社区通话将提供一些有价值的信息，用于运行自己的计划并获取 Teams 和其他 Microsoft 365 或 Office 365 服务。 可以在自己的内部程序中重复使用计划材料。

- 使用 Teamwork 支持团队：你的队友应该使用你为它们设置的团队进行所有计划沟通、反馈和查找资源。  在执行此操作时，他们将逐渐熟悉产品及其优势。

- 定期举行月度会议：通过定期召开每月会议促进社区积极性与内聚性。 这些会议可以是个人会议与虚拟会议的组合，但它们的一致性对于拥有一个成功的支持者社区来说很关键。

    > [!TIP]
    > 为了帮助制定日程，请安排公共社区通话后一周的会议。 

- 明确计划要求：一些金牌计划要求成员参加每月会议，为支持的用户组保留办公时间，并为新用户提供培训。 无论你的要求如何，请确保支持者清楚知道如何成功参与该计划。

- 明显奖励你的战利者：你的得主会提供对项目进度和潜在陷阱的重要见解。 他们经常因为参与而受到奖励。 小手势会大有作为，员工通常不会寻求货币收益，而是感谢认可他们的贡献。 找到适合且有趣的方式来感谢社区。 

- 提前并经常培训得奖人：你的计划开始，每月会议可用于在支持者社区中培养技能。 利用这些机会让他们上车、进行初始培训、鼓励他们参加我们的课程，并确保他们已准备好在试验阶段为员工提供支持。
