# <a name="kaizala-and-microsoft-teams"></a>Kaizala和Microsoft Teams 

2019年4月，我们[宣布了](https://techcommunity.microsoft.com/t5/microsoft-kaizala-blog/update-on-kaizala-features-coming-to-microsoft-teams/ba-p/974525)将Kaizala与Microsoft Teams一起实施的计划。 Kaizala功能补充了Teams强大的通信和协作功能，Microsoft 365是团队合作的中心。

打开与Microsoft Teams做生意的全新方式的大门。 Teams扩展了Kaizala的功能 - 将聊天、会议、通话、协作、应用集成和文件存储合并到单个接口中。 Teams可帮助客户从投资中获得更多价值，并改善一线员工的日常体验。

Teams通过帮助组织从不同的应用数组迁移到单个安全中心，将团队所需的内容（包括聊天、会议和呼叫）与本机集成到Office 365应用，从而改善工作场所协作。 目前，全球组织使用Teams跨位置、时区和语言进行协作。

Microsoft Teams将Kaizala替换为Microsoft 365内部员工和扩展网络中的人员的主要通信客户端。

Kaizala 专业版功能现已在Teams中提供，包括现在作为Teams[应用模板](/microsoftteams/platform/samples/app-templates)（如[轮询](/microsoftteams/platform/samples/app-templates#poll)、[调查](/microsoftteams/platform/samples/app-templates#survey)、[清单](/microsoftteams/platform/samples/app-templates#checklist)、[测验](/microsoftteams/platform/samples/app-templates#quiz--)和[培训](/microsoftteams/platform/samples/app-templates#training--)）提供的Kaizala操作。 我们最近发布了任何Teams用户通过其电话号码与外部用户交互的功能。 查看 [博客](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/microsoft-teams-users-can-now-chat-with-any-teams-user-outside/ba-p/3070832) 和 [文章](https://support.microsoft.com/en-us/office/add-or-invite-people-outside-your-teams-org-to-a-chat-6897ab47-9f60-4db6-8b95-18599714fe57) 以了解详细信息。

使用数字通信和协作工具启用组织网络的每个成员可以改变工作完成方式。 本概述将帮助你开始从Kaizala到Microsoft Teams的过渡。

继续阅读有关Kaizala和Teams的常见问题。

## <a name="faq"></a>常见问题

## <a name="when-should-my-organization-move-from-kaizala-to-teams"></a>我的组织何时应从Kaizala迁移到Teams？

我们鼓励所有Microsoft 365和Office 365客户立即开始使用[Teams](https://www.microsoft.com/microsoft-teams/group-chat-software?ms.officeurl=teams&rtc=1&OCID=AID2388518_SEM_Ks5ySdZ9)，以利用Teams提供的丰富功能。 这些功能包括为 46 种语言和 181 个市场中的移动用户提供简单而强大的体验。 通过使用 Microsoft Flow 和 PowerApps 为一线工作人员提供轮班管理、来宾访问以及对自定义工作流和应用的支持等功能。 对于Teams中尚不可用的特定Kaizala功能，我们将在将来发布更新的时间线。

## <a name="how-can-i-continue-to-use-kaizala-pro-if-i-had-purchased-the-standalone-offering"></a>如果我购买了独立产品/服务，如何继续使用Kaizala 专业版？

截至 2020 年 7 月 1 日，Microsoft 已终止Kaizala 专业版独立服务的销售。 Kaizala和Teams将继续作为[Microsoft 365和Office 365计划的一部分](/Office365/Kaizala/migrate-kaizala-pro)进行访问。

## <a name="how-can-i-get-started-with-teams"></a>如何开始使用Teams？

从应用商店下载[Teams](https://www.microsoft.com/microsoft-teams/group-chat-software)，并访问[帮助中心](https://support.microsoft.com/teams?ui=en-us&rs=en-us&ad=us)中的培训和资源以开始使用。 你还可以加入[Teams技术Community](https://techcommunity.microsoft.com/t5/microsoft-teams/ct-p/MicrosoftTeams)，提供反馈，对功能请求进行投票，并与出色的社区互动。

## <a name="what-can-i-do-to-prepare-for-the-transition-of-kaizala-to-teams"></a>如何为Kaizala过渡到Teams做好准备？

现在开始将Teams用于能够促进组织间通信的解决方案。 Teams不仅仅是Kaizala的升级，它是一个强大的工具，打开了全新的经营方式的大门。 Teams对于各种一线工作人员用户方案具有独特的定位，这要归功于[为中小型企业 (SMB) 而设计的Teams概要](https://www.microsoft.com/microsoft-365/blog/2021/12/01/new-microsoft-teams-essentials-is-built-for-small-businesses/)，以及[通过输入电话号码或完整电子邮件地址以供个人使用Teams来与外部用户协作的能力](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/microsoft-teams-users-can-now-chat-with-any-teams-user-outside/ba-p/3070832)。

## <a name="will-i-be-able-to-use-teams-to-collaborate-with-customers-and-partners-outside-my-organization-using-a-phone-number"></a>我能否使用Teams使用电话号码与组织外部的客户和合作伙伴进行协作？

是的，使用Teams 连接，可以通过输入要访问的完整电子邮件地址或电话号码来邀请任何Teams用户聊天，并与拥有 Microsoft 个人帐户的任何人开始一对一或群聊。 无需切换租户。 IT 管理员和 Microsoft 个人帐户Teams用户提供了一系列控件和工具，以确保安全性。 查看 [支持页](https://support.microsoft.com/en-us/office/add-or-invite-people-outside-your-teams-org-to-a-chat-6897ab47-9f60-4db6-8b95-18599714fe57)。 有关详细信息，请参阅 [博客](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/microsoft-teams-users-can-now-chat-with-any-teams-user-outside/ba-p/3070832)。
