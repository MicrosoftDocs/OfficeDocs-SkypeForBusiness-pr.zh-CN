# <a name="create-your-first-teams-in-microsoft-teams"></a>在 Microsoft Teams 中创建首批团体

推动采用 Teams 的最佳方式是使用该产品。 建议使用 Teams 管理 Microsoft 365 或Office 365部署。 还应创建专注于团队合作和工作效率的员工社区。  (这些初始团队可能会随时间而变化。) 一些建议的团队包括：

| 团队名称 | 建议的频道名称 |
| --------- | ---------------------- |
| 了解 Teams | 常规</br> 打招呼</br> 如何实现？</br>反馈 </br> 将来团队的理念 </br> 支持 |
| Microsoft Teams 实现 | 常规 <br/> 意识和采用 <br/> Business Engagement <br/> 早期采用者计划 <br/> 反馈和见解 <br/> 安全性和符合性 <br/> 策略和规划 <br/> 服务运行状况和事件 <br/> 培训 <br/> 水凝器聊天 |
| 团队合作冠军 | 常规 <br/> 最佳做法、培训和操作方法 <br/> 冠军角 <br/> 反馈和支持 <br/> 程序潜在顾客 <br/> 社交中心 |

在 Microsoft Teams 中，[选项卡库](/microsoftteams/platform/concepts/tabs/tabs-overview)允许你访问管理员已启用的所有 Microsoft 365 或Office 365和第三方应用。 为了从产品中获取最大价值，我们建议将关键资源固定为每个相应通道中的选项卡。 请参阅下面的示例。

![显示默认选项卡和自定义选项卡的屏幕截图。](media/teams-adoption-tab-example.png)

在初始团队中，我们建议固定以下项目，以便在团队频道中轻松参考。 这些资产是 OneNote 笔记本部分的补充。 如果你正在与我们的 FastTrack 团队合作，他们将帮助你配置这些第一个团队。 

|团队/频道名称 | 固定选项卡 |
|----------------- | ---------- |
| **Microsoft Teams 部署** ||
| 常规 | Web 选项卡： [Teams 文档](./index.yml) |
| 意识和采用 | [Teams 采用中心](https://aka.ms/DriveTeamsAdoption)<br/>[Microsoft Teams Customer Success Kit](https://aka.ms/TeamsCustomerSuccess)|
| 业务参与 | Wiki - Engagement 指南<br/>表单 – Teams 咨询请求<br/>列表 – Engagement 请求 |
|早期采用者 | 固定反馈社区 <br/> OneNote for FAQ 和常见主题 |
| 反馈和报告 | 为 Microsoft 365 或Office 365管理员固定 Microsoft 365 或Office 365使用情况报告 (仅)  |
| 安全性和合规性 | Microsoft 信任门户 <br/> [安全性和符合性文档](/office365/securitycompliance/index)<br/> [路线图](/office365/securitycompliance/security-roadmap) |
| 策略和规划 | 协作北星 PowerPoint <br/> 服务采用项目计划 |
| 培训 | 自定义学习培训门户 <br/> [Teams 联机培训](https://aka.ms/TeamsTraining) |
| **团队合作冠军**|  |
| 常规 | 用于程序公告 <br/> 固定冠军计划概述 |
| 最佳做法 | 适用于团队、频道和聊天 PowerPoint 的最佳做法 <br/> 团队生命周期 |
| 冠军角 | 会议笔记和演示文稿 |
| 社交中心 | RSS 连接器：Teams 博客和驱动采用博客 |
| 培训和操作方法 | 培训门户链接 |

## <a name="making-the-switch-to-microsoft-teams"></a>切换到 Microsoft Teams

开始使用 Teams 时，你和项目团队的其他成员必须致力于使用 Teams 聊天、频道和资源来运行项目。 你自己对 Teams 的使用对采用项目的质量至关重要。 建议通过继续通过电子邮件与核心项目团队对话来抵制项目碎片化。 通过切换到 Teams，你将为最终希望员工接受的行为建模。 设置基调，先走，然后开始享受将项目信息放在一个位置！
