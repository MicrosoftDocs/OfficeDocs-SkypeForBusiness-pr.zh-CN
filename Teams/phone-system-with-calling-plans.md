## <a name="dial-plans"></a><span data-ttu-id="97bc5-685">拨号计划</span><span class="sxs-lookup"><span data-stu-id="97bc5-685">Dial plans</span></span>

<span data-ttu-id="97bc5-686">Office 365 的电话系统功能中的[拨号计划](https://support.office.com/article/What-are-dial-plans-2f0cfb59-1ca1-4e31-84ce-09d0b1a7ce1b)是一组规范化规则，用于将拨打的电话号码转换为替代格式（通常为 [E.164](https://go.microsoft.com/fwlink/?linkid=859014) 格式）以便进行呼叫授权和呼叫路由。</span><span class="sxs-lookup"><span data-stu-id="97bc5-686">A [Dial Plan](https://support.office.com/article/What-are-dial-plans-2f0cfb59-1ca1-4e31-84ce-09d0b1a7ce1b), a Phone System feature of Office 365, is a set of normalization rules that translates dialed phone numbers into an alternate format (typically [E.164](https://go.microsoft.com/fwlink/?linkid=859014) format) for call authorization and call routing.</span></span> 

<span data-ttu-id="97bc5-687">拨号计划允许用户按其习惯的方式（例如，对于本地呼叫省略区号，对于国内呼叫省略国家/地区代码，甚至在拨打电话时使用简短数字拨号）拨打电话号码。</span><span class="sxs-lookup"><span data-stu-id="97bc5-687">A dial plan allows users to dial phone numbers the way they are accustomed to, such as omitting area code for local calls, omitting country code for domestic calls, or even using short digit dialing when performing conference dial out.</span></span>

<span data-ttu-id="97bc5-688">在 Office 365 的电话系统功能中，有两种类型的拨号计划：</span><span class="sxs-lookup"><span data-stu-id="97bc5-688">Within the Phone System feature of Office 365, there are two types of dial plans:</span></span>

-   <span data-ttu-id="97bc5-689">**服务拨号计划**。</span><span class="sxs-lookup"><span data-stu-id="97bc5-689">**Service dial plan**.</span></span> <span data-ttu-id="97bc5-690">这是默认的拨号计划，基于 Office 365 使用位置适用于用户，无法修改。</span><span class="sxs-lookup"><span data-stu-id="97bc5-690">This is the default dial plan and applied to users based on Office 365 usage location, and it cannot be modified.</span></span>
-   <span data-ttu-id="97bc5-691">**租户拨号计划**。</span><span class="sxs-lookup"><span data-stu-id="97bc5-691">**Tenant dial plan**.</span></span> <span data-ttu-id="97bc5-692">这是租户中的自定义拨号计划，进一步分成两种类型：</span><span class="sxs-lookup"><span data-stu-id="97bc5-692">This is a customizable dial plan within a tenant, and further divided into two types:</span></span>
    -   <span data-ttu-id="97bc5-693">**租户-全局拨号计划** - 该拨号计划适用于租户中的所有用户。</span><span class="sxs-lookup"><span data-stu-id="97bc5-693">**Tenant-global dial plan**—the dial plan applies to all users within the tenant.</span></span>
    -   <span data-ttu-id="97bc5-694">**租户-用户拨号计划** - 该拨号计划仅适用于特定用户。</span><span class="sxs-lookup"><span data-stu-id="97bc5-694">**Tenant-user dial plan**—the dial plan applies only to specific users.</span></span>

<span data-ttu-id="97bc5-696">分配给用户的有效拨号计划是服务拨号计划（基于用户的 Office 365 使用位置）和租户拨号计划（可以是租户-全局拨号计划或租户-用户拨号计划）的组合。</span><span class="sxs-lookup"><span data-stu-id="97bc5-696">The effective dial plan assigned to users is the combination of service dial plan (based on user’s Office 365 usage location) and tenant dial plan (can be either tenant-global dial plan or tenant-user dial plan).</span></span>
