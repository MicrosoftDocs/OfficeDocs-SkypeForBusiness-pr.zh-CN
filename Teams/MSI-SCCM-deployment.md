<a name="install-microsoft-teams-using-msi-with-sccm"></a>安装 Microsoft 团队使用 MSI （与 SCCM)

Microsoft 已提供批量 admins MSI 文件部署团队选择用户或计算机。 管理员可以使用此文件，以便用户无需手动下载团队应用程序远程部署团队。 System Center Configuration Manager (SCCM) 是用于托管部署的 MSI 文件的主要工具。
