# <a name="manage-telephone-numbers-for-your-organization"></a>管理组织的电话号码

目前，Microsoft 支持两种电话号码类型： 

- [**用户号码**](#user-telephone-numbers)，也称为订阅者号码，可分配给组织的用户。

- [**服务号码**](#service-telephone-numbers)，分配给音频会议、[自动助理](plan-auto-attendant-call-queue.md)或呼叫 [](deploy-audio-conferencing-teams-landing-page.md)[队列等服务](plan-auto-attendant-call-queue.md)。

Microsoft 正在努力简化数字类型，但目前需要确定：

- 哪些用户位置需要 Microsoft 提供的新电话号码？
- 需要哪种类型的 (或) 电话号码？
- 如何将现有电话号码移植到Teams？

你获取和管理电话号码方式因 PSTN 连接选项的不同而不同。

- 有关管理 Microsoft 呼叫计划的电话号码的信息，请参阅 [管理呼叫计划的电话号码](manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization.md)。

- 有关为接线员管理电话号码连接，请参阅使用接线员电话[设置连接。](operator-connect-configure.md#set-up-phone-numbers)

- 有关管理直接路由的电话号码的信息，请参阅配置 [电话号码并启用企业语音和语音邮件](direct-routing-enable-users.md#configure-the-phone-number-and-enable-enterprise-voice-and-voicemail-online)。

如果需要其他号码类型或其他号码类型（Microsoft Teams管理中心中显示的数字类型外，可以将电话号码请求提交到 电话[号码服务中心](https://pstnsd.powerappsportals.com/)。

## <a name="user-telephone-numbers"></a>用户电话号码

有两种类型的用户电话号码，可以分配给贵组织的用户：  
    
- **地理数字** 与地理区域有关系，并且最常见。 例如，在大多数情况下，只能在特定地址、城市、州或国家/地区内使用地理电话号码。
    
- **非地理号码称为** 国家/地区号码，有时称为 VoIP 号码。 这些数字与国家/地区内的地理区域没有关系。 例如，从国家/地区中的任何位置呼叫号码时，非地理号码的成本通常相同。 此外，一些国家/地区（如丹麦）仅提供非地理数字。


## <a name="service-telephone-numbers"></a>服务电话号码  

本部分介绍许可中包含的 Microsoft 可用的服务编号。 有关接线员或直接连接提供的服务号码的信息，请与提供商联系。 

Microsoft 提供两种类型的服务电话号码-收费和免费电话号码，可分配给音频会议、自动助理或呼叫队列等服务。 服务号码的并发呼叫容量高于用户号码。 服务号码的可用性因国家/地区而异，而号码类型 (是收费号码还是免费电话号码) 。 Microsoft 在每个国家/地区的电话许可证决定了该号码可用于哪些方面。
    
 - **收费服务号码** - 有两种类型的收费服务号码，这可能会导致呼叫者产生收费费用：
    
   - **地理数字** 地理数字与地理区域有关系。 例如，在大多数情况下，只能在特定地址、城市、州或国家/地区内使用地理电话号码。
        
   - **非地理数字** 非地理数字是国家/地区号码，与国家/地区内的地理区域没有关系。 例如，从国家/地区中的任何位置呼叫号码时，非地理号码的成本通常相同。
   
- **免费服务号码** - 这些服务号码通常不会导致呼叫者产生收费费用。 Teams 60 多个国家/地区提供国家/地区免费电话号码。
    
    > [!CAUTION]
    > 在某些情况下，一些国家/地区和发起号码类型（例如源自移动电话的呼叫）可能会向呼叫者收取收费费用。
