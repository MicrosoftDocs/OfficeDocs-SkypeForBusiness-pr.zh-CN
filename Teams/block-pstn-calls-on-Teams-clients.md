# <a name="understand-how-users-can-block-pstn-calls-on-microsoft-teams-desktop-and-mobile-clients"></a>了解用户如何阻止 Microsoft 团队桌面和移动客户端上的 PSTN 呼叫

如果用户接收到他们不希望在其 Microsoft 团队桌面或移动版 (iOS 或 Android) 客户端上接收的呼叫, 则他们可以使用 "阻止呼叫" 功能防止今后来自发件人的呼叫。 用户还可以管理其阻止的呼叫者列表。

此功能允许用户减少以下类型的通话:

- 未知呼叫者 (没有呼叫者 ID)
- 垃圾邮件通话
- Robo 通话
- 从 telemarketers 拨打电话
- 与他人通话的电话。

有关阻止呼叫者的详细信息和步骤, 请参阅[管理团队中的呼叫设置](https://support.office.com/article/Manage-your-call-settings-in-Teams-456cb611-3477-496f-b31a-6ab752a7595f)。
