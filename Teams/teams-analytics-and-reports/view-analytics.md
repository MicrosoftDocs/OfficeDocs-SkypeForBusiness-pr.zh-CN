# <a name="view-analytics-in-teams"></a>查看团队中的分析

在 Microsoft 团队中，用户可以查看其所属团队和频道的分析。 此信息使用户可以深入了解其团队的使用模式和活动。 用户可以在三个级别查看活动用户、帖子、回复等的数据。

- **跨团队分析**为用户提供了在单个列表视图中作为其成员或所有者的所有团队的使用数据的大致概述。
- **每团队分析**为用户提供了更详细的视图，显示特定团队的使用数据。
- **每信道分析**为用户提供了更为精细的视图，显示特定频道的使用数据。

用户可以筛选这些视图中的任何一种，以查看特定时间段内的数据。

## <a name="view-cross-team-analytics"></a>查看跨团队分析

1. 在团队中，在 "团队" 列表底部的 "**加入或创建团队**" 旁边，单击 "**管理团队**"。
2. 单击 "**分析**" 选项卡。
3. 选择日期范围以显示您是其成员或所有者的所有团队的使用数据。

    ![跨团队分析视图的屏幕截图](../media/view-analytics-cross-team.png)

    |项目 |说明  |
    |--------|-------------|
    |**名称** - 按 WAN 链路进行筛选（筛选器位于图形右侧）。   |团队的名称。 |
    |**活动用户**   |在指定时间段内团队活动的 "团队" 和 "趋势" 行中的活动用户数。
    |**人员**   |在指定时间段内团队的总人数。 这包括团队所有者、团队成员和来宾。|
    |**来宾**   |指定时间段内团队的来宾人数。 |
    |**自检**   |在指定时间段内，在团队聊天中发布的新消息数。 |
    |**条**   |指定时间段内团队聊天中的答复次数。 |
    |**类型**   |团队是否为私人团队或公共团队。|

## <a name="view-per-team-analytics"></a>查看每个团队的分析

1. 在 "团队" 中，转到所需团队，单击 "**更多选项（...）**"，然后单击 "**管理团队**"。
2. 单击 "**分析**" 选项卡。
4. 选择日期范围以显示团队的使用数据。  

    ![每个团队分析视图的屏幕截图](../media/view-analytics-per-team.png)

    |项目 |说明  |
    |--------|-------------|
    |**摘要**   |团队活动摘要，包括以下内容：<ul><li>**用户**：指定时间段内的用户总数。 这包括团队所有者、团队成员和来宾。</li> <li>**文章**：在指定时间段内在团队聊天中发布的新消息数。</li><li>**回复**：指定时间段内团队聊天中的答复次数。</li> <li>**应用**：添加到团队的应用数。</li><li>**会议**：按团队级别组织的团队会议数。</li> </ul> |
    |**活动用户**   |活动用户和非活动用户数。|
    |**角色**   |按角色（包括团队所有者、工作组成员和来宾）的用户数。|
    |**活动用户**图表  |每日有效次数。 将鼠标悬停在给定日期的点上可查看该日期的活动用户数。|
    |**消息**图表  |按日期在团队聊天中发布的消息总数。 将鼠标悬停在给定日期的圆点上可查看在该日期发布的新文章和答复的数量。|

> [!TIP]
> 你还可以通过单击[跨团队分析视图](#view-cross-team-analytics)中的团队在列表中查看每个团队分析。

## <a name="view-per-channel-analytics"></a>查看每通道分析

1. 在 "团队" 中，转到所需的频道，单击 "**更多选项（...）**"，然后单击 "**管理频道**"。
2. 单击 "**分析**" 选项卡。
3. 选择日期范围以显示频道的使用数据。  

    ![每通道分析视图的屏幕截图](../media/view-analytics-per-channel.png)

    |项目 |说明  |
    |--------|-------------|
    |**摘要**   |频道活动摘要，包括以下内容：<ul><li>**用户**：指定时间段内的用户总数。 这包括团队所有者、团队成员和来宾。</li> <li>**文章**：在指定时间段内发布到频道的新邮件数。</li><li>**回复**：在指定时间段内频道中的答复次数。</li> <li>**应用**：添加到频道的应用数。</li> </ul> |
    |**消息**图表  |按日期发布到频道聊天的邮件总数。 将鼠标悬停在给定日期的圆点上可查看在该日期发布的新文章和答复的数量。|

> [!TIP]
> 你还可以通过在 "[每个团队的分析" 视图](#view-per-team-analytics)的下拉列表框中选择通道来查看每个通道分析。
    
> [!NOTE]
> 我们将活动用户定义为在桌面客户端、移动客户端和 web 客户端中执行有意操作的用户。 有意行动的示例包括开始聊天、呼叫、共享文件、编辑团队中的文档、参与会议等。 我们去除了诸如自动启动、最小化屏幕或关闭应用之类的被动操作。 我们还跨单个用户 ID 消除了所有操作。
