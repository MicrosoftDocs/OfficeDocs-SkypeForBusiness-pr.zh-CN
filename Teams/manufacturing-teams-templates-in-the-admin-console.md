# <a name="use-manufacturing-team-templates-in-the-admin-center"></a>在管理中心使用制造团队模板

通过提供预定义的设置、频道和预安装应用模板，可以使用团队模板快速轻松地创建团队。

团队模板预先定义了围绕制造需求设计的团队结构。 还可以扩展团队模板，以创建根据特定组织需求定制的团队。

在本文中，我们将介绍每个团队模板并推荐如何使用它们。

如果你负责在整个制造组织中规划、部署和管理多个团队，则本文适合你。 你已在贵组织内部署 Teams 服务。 如果尚未部署 Teams，首先请阅读[如何部署 Microsoft Teams](./deploy-overview.md)。

若要了解有关团队模板的一般信息，请参阅 [团队模板入门](get-started-with-teams-templates-in-the-admin-console.md)。

## <a name="quality-and-safety"></a>质量与安全

与制造工厂团队集中通信、访问资源和工厂运营。 包括策略和过程文档、培训视频、安全通知、轮班交接过程。

| 基本模板类型|baseTemplateId| 此基本模板包含的属性 |
| ------------------|-- |----------------------------------------------------- |
|质量与安全|`com.microsoft.teams.template.QualitySafety` |频道： <ul><li>常规<li>公告</li><li>第 1 行</li><li>第 2 行</li><li>第 3 行</li><li>安全</li><li>培训</li><li>维护</li><li>有趣的内容</li></ul> 应用： <ul><li>Wiki</li><li>Planner</li></ul>|
