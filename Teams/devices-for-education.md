# <a name="teams-devices-for-education"></a>Teams 教育版的设备

## <a name="overview"></a>概述

### <a name="hybrid-and-blended-education-is-here-to-stay"></a>混合和混合教育在此保持

Learning混合和混合。 教师和学生都需要在连续的空间中实现流畅的体验，在学习方式和位置方面需要灵活性。 教育不会限制于物理教室或严格虚拟环境;远程参与与个人学习一样动态。 混合和混合学习提供的灵活性（虽然并不是全部都是新内容）对于很多人来说都是全新的，并且将具有广泛的影响。 

新式混合教室将无边界。 学生和教师将受同步和异步学习以及交互式多模式参与支持，旨在满足每种情况下每个学生的需求。 Teams设备旨在支持一系列空格，以便不听到任何语音。 

### <a name="student-and-educators-alike-excel-with-microsoft-teams-rooms"></a>学生和教师都使用 Microsoft Teams 会议室

-   Microsoft Teams 会议室实现安全、安全、以学生为中心的学习，从课堂集成到聊天室，以便进行更集中的协作，甚至支持一对一地解决特定学生的需求和问题。

-   在数字或混合学习环境中使用个人设备和共享设备可帮助学生社交并获取他们将一直使用的关键技能。

-   Teams功能不限于教室。 教师和管理员可以轻松从家庭、办公室或已优化其环境的任何位置与同一学校或整个学区内的其他教职员工联系。 借助Teams，专业开发可从任何位置访问。

### <a name="bring-digital-flexibility-to-physical-spaces"></a>为物理空间带来数字灵活性

为了确保远程工作的学生和教师可以完全参与组协作和会议，教育工作区将需要一些技术来帮助缩小旧的个人模型与新的混合和远程空间之间的分隔。

## <a name="space-considerations"></a>空间注意事项

### <a name="what-is-the-role-of-the-space"></a>空间的角色是什么？

物理空间旨在容纳将在其中发生的各种活动。 不同类型的活动或连接可能会促进不同的会议行为。 例如，教室将看到与会议室行为截然不同的行为。

帮助定义空间的使用方式。 为了有效使用教室或会议空间，必须思考的是教室和会议空间的布置和完成时间。

对于现有空间，该技术应该与会议室的现有布局保持一致。 例如，如果为演示文稿和就地说明设置了表和台式，则技术应该遵循。 麦克风和相机需要以最佳方式在房间中放置，而不只是在它们适合的任何位置添加，而不考虑会议室的使用方式。 远程参与者需要像个人参与者一样清楚地查看演示文稿材料。

对于新空间，会议室的预定使用应该同时驱动用户和技术考虑事项。 例如，在设计用于促进多个用途的聊天室 (如单个工作和组交互) ，为了提高效率，应分隔为两个单独的区域。

*空间中的活动将侧重于协作共同创建吗？*

创造力需要围绕用户协作设计的布局和技术部署。 适当的空间便于用户与技术无缝协作，并消除即使是最现代用户的冲突。 Teams设备设计为足够用户友好，让学生可以很轻松地导航，并且教师可以控制和自定义首次使用。 

*与个人参与者和远程参与者共享内容优先的空格如何？*

## <a name="technology-considerations"></a>技术注意事项

正确的音频和视频技术选择将帮助用户从会议Microsoft Teams最大。

*音频*

音频注意事项对于确保远程参与者和房间内参与者都可以清楚地听到和理解至关重要。 麦克风和扬声器的覆盖必须包含会议室中每一个座位，而不会留空。

### <a name="what-are-the-right-audio-devices-for-each-space"></a>每个空间适合哪些音频设备？

将正确的麦克风和扬声器与每个空间配对将提供最佳效果，使听到声音时不会分散注意力。

-   在可能干扰的空间（如家庭的公共房间或教室的桌面）中，耳机通过仅对会议参与者限制声音提供可能的最佳体验。 耳机麦克风设计为仅选取会议语音，并且包含扬声器，以便只有与会者可以听到所进行内容。

-   在更私有 (例如，学生对设备) ，设备选择可以扩展到包括个人扬声器等内容。 这些个人设备允许与耳机相同的对话质量级别，但无需限制必须佩戴设备。

-   在共享空间中，选项会扩展，但相同的原则仍然存在：会议参与者需要聆听并清楚地听到其他人的声音。

    -   一体式音频设备是小型空间的易于部署、高质量的选项，参与者数量有限，与音频设备相对接近。

    -   在较大的空间中，将麦克风和扬声器分开，以确保空间内所有参与者的最大覆盖范围。 许多大中型会议室解决方案都作为工具包的一部分提供，无需配置，只需物理安装麦克风和扬声器。

    -   在最最大、最复杂的空间中，提供了解决方案，允许将各种麦克风和扬声器类型一同使用，例如从大型大厅捕获观众和观众语音，同时将呼叫带到所有人。 这些解决方案最好由在设计和部署更涉及技术的解决方案方面具有专业知识的 [Microsoft](https://cloudpartners.transform.microsoft.com/teams-mrpp) 合作伙伴部署。

*Video*

视频与音频一样重要。 捕获面部表达式或手势等物理的非语言通信与课堂或会议内容一样重要，以确保最大程度的参与和理解。 会议室需要配备相应的屏幕大小，以便每个人都可以正常体验会议，无论是在会议室中还是远程。 会议室还需要具有正确的相机覆盖范围和 FOV () 以便有效捕获远程加入的所有会议室内会议参与者。

### <a name="what-are-the-right-cameras-for-each-space"></a>每个空间的右相机是什么？

新式视频是新式视频协作Teams要求。 相机技术具有高级技术，现在可用于所有空间的高质量相机。

-   笔记本电脑和平板电脑等设备中包含的相机是供参与视频呼叫的个人使用的一个很好的起点。 对于希望为个人空间带来更丰富的视频体验的用户，外部相机代表从嵌入设备向上扩展，并且可在更多方案和配置中使用，从而确保呼叫参与者共享自己最佳质量的视图。 这些设备通过安装在每台电脑上的简单 USB 连接轻松安装。

-   在共享空间中，选项和功能会扩展。 一键式音频设备（如音栏）中包含的相机可能会覆盖较小的会议室。 这些相机通常具有宽的视区，确保它们一次捕获空间内每个参与者。 其中许多相机还具有智能帧图像 (以便相机只显示会议中的人员，而不是空的) 跟踪和了解空间利用率。

-   在较大的房间中，相机采用从不同距离捕获不同位置的参与者所需的技术。 这些相机可以捕获组或专注于特定个人，同时保持出色的视觉清晰度。 通常，仅跟踪和启用活动扬声器的设备会用于较大的空间，以便许多参与者在参与会议时可以单独进行特别推荐。

### <a name="how-do-you-interact-virtually-beyond-video-streams"></a>如何虚拟地与视频流交互？

Teams会议不仅仅是对参与者的音频和视频呼叫。 有多种方法可以共享内容并与内容动态交互。

常见方案是直接从参与者的设备共享内容。 各个用户可以直接从各个应用程序共享完整的音频和视觉体验，也可以将屏幕的全部内容直接共享到会议，以便所有人查看。 这将扩展到已安装设备的所有共享空间。 Microsoft Teams 会议室还可以选择使用有线监视器连接来允许参与者共享，即使他们缺乏单独加入会议的能力。

对于来自物理世界的内容，会议室系统Microsoft Teams空间可以处理白板或文档照相机，允许对空间中的内容进行协调的审阅和讨论。

对于完全交互式体验，交互式显示 (如 Surface Hub 系列产品和 Microsoft 数字白板) 使以交互方式为就地学生和远程学生创建内容变得容易。
