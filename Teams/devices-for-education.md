# <a name="teams-devices-for-education"></a>Teams教育设备

## <a name="overview"></a>概述

### <a name="hybrid-and-blended-education-is-here-to-stay"></a>混合和混合教育随时可保持

Learning混合混合。 教师和学生都需要一种跨一系列空间实现的流畅体验，这由在学习方式和位置方面的灵活性需求所推动。 教育不限于物理教室或严格虚拟环境;远程参与与个人学习一样动态。 混合学习和混合学习提供的灵活性对许多人来说不是一个新增功能，并且将具有意义广泛的影响。 

现代混合教室将是无限的。 学生和教师将受同步和异步学习以及交互式多模式参与支持，旨在满足每个学生在每种情况下的需求。 Teams设备已设计为支持一系列空间，因此不会听到任何语音。 

### <a name="student-and-educators-alike-excel-with-microsoft-teams-rooms"></a>学生和教师都使用 excel 和 Microsoft Teams 会议室

-   在数字或混合学习环境中使用个人和共享设备可帮助学生社交和获取他们一生将使用的关键技能。

-   Teams功能不限于课堂。 教师和管理员可以轻松地从他们家庭、办公室或已优化其环境的任何位置，与同一学校或整个学区内的其他教职员工联系。 借助Teams，可从任何位置访问专业开发。

## <a name="space-considerations"></a>空间注意事项

### <a name="what-is-the-role-of-the-space"></a>空间的作用是什么？

物理空间旨在容纳将发生的各种活动。 不同类型的活动或连接可能会促进不同的会议行为。 例如，教室的行为与会议室的行为要大为不同。

家具可帮助定义空间的使用方式。 若要使教室或会议空间有效，必须考虑家具及完成时间。

对于现有空间，该技术应该与房间的现有布局保持一致。 例如，如果为演示和个人指导设置了桌子和桌子，则技术应该遵循。 麦克风和摄像头需要以最佳方式在房间中定位，而不只是在它们适合的任何位置添加，而不考虑房间的使用方式。 远程参与者需要像查看参与者一样清楚地查看演示材料。

对于新空间，预定使用房间应推动家具和技术方面的考虑。 例如，设计用于促进多种用途的聊天室时 (例如单个工作和组交互) ，应该将空间分隔成两个单独的区域，以便获得最大效率。

## <a name="technology-considerations"></a>技术注意事项

正确的音频和视频技术选择可帮助用户从会议Microsoft Teams。

*音频*

音频注意事项对于确保远程参与者和室内参与者都可以清楚地听到和理解至关重要。 麦克风和扬声器覆盖范围必须包括房间中的每个席位，且不会留下间隔。

### <a name="what-are-the-right-audio-devices-for-each-space"></a>每个空间适合哪些音频设备？

将每个空间的合适麦克风和扬声器配对，可提供最佳结果，让声音清晰无干扰。

-   在可能嘈杂的空间（例如家庭中的公用会议室或教室的办公桌）中，耳机通过仅向会议参与者限制声音，提供可能的最佳体验。 耳机式麦克风设计为仅拾取会议语音，并且包含扬声器，以便只有与会者可以听到正在进行中的内容。

-   在更私密 (例如，学生的床位) ，设备选项可以扩展到包括个人免提电话等内容。 这些个人设备允许与耳机上相同的对话质量级别，但无需使用设备。

-   在共享空间中，选项会扩展，但相同的原则仍然存在：会议参与者需要被听取并清楚地听到其他人的声音。

    -   一体式音频设备是易于部署的高质量选项，适用于参与者数量有限且相对靠近音频设备的较小空间。

    -   在较大的空间中，麦克风和扬声器分开，以确保空间内所有参与者的最大覆盖范围。 许多大中型会议室的解决方案都作为工具包的一部分提供，无需配置，只需实际安装麦克风和扬声器。

    -   在最大且最复杂的空间中，提供的解决方案允许将各种麦克风和扬声器类型一起使用，例如从大型大厅捕获观众和观众的声音，同时将呼叫带到每个人。 当 Microsoft 合作伙伴在设计和部署涉及更多技术[](https://cloudpartners.transform.microsoft.com/teams-mrpp)的解决方案方面具有专业知识时，最好是实现这些解决方案。

*Video*

视频与音频一样重要。 捕获面部表情或手势等物理非语言通信与教室或会议内容一样重要，以确保最大程度的参与和理解。 会议室需要配备合适的显示大小，以便每个人在会议室或远程都可以正常体验会议。 会议室还需要具有正确的相机覆盖范围和 FOV () ，以便有效捕获远程加入的所有会议室会议参与者。

### <a name="what-are-the-right-cameras-for-each-space"></a>每个空间适合哪些摄像头？

优质视频是新式协作Teams必需。 相机技术已显著提升，现在所有空间都提供高质量的相机。

-   笔记本电脑和平板电脑等设备中包含的摄像头是参与视频呼叫的个人使用的起点。 对于希望为个人空间提供更丰富的视频体验的用户，外部摄像头代表嵌入式设备中的一步，可用于更多的方案和配置，确保呼叫参与者共享自己的最佳质量视图。 这些设备通过每个电脑上找到的简单 USB 连接轻松安装。

-   在共享空间中，选项和功能会扩展。 一键式音频设备（如声栏）中包含的摄像头可能会覆盖较小的房间。 这些相机通常具有宽广的视野，确保它们通过一次拍摄捕获空间中每个参与者。 其中许多相机还具有图像图像 (以便相机只显示参加会议的人，而不是空) 来跟踪和了解空间利用率。

-   在较大的房间中，相机采用在不同距离的不同位置捕获参与者所需的技术。 这些摄像头可以捕获组或关注特定个人，同时保持出色的视觉清晰度。 可以仅跟踪活动发言人并仅提供其功能的设备会定期在较大的空间中使用，以便许多参与者可以在会议过程中单独受到推荐，因为参与者会贡献自己的想法。

### <a name="how-do-you-interact-virtually-beyond-video-streams"></a>如何在视频流之外以虚拟方式交互？

Teams会议不仅仅是参与者的音频和视频呼叫。 有多种方法可以与内容进行动态共享和交互。

一种常见方案是直接从参与者的设备共享内容。 单个用户可以将单个应用程序的完整音频和视觉体验或者其屏幕的整个内容直接共享到会议，供所有人查看。 这扩展到安装了设备的所有共享空间。 Microsoft Teams 会议室还可以选择使用有线监视器连接来允许参与者共享，即使他们无法单独加入会议。

对于来自物理世界的内容，Microsoft Teams会议室系统的空间可以处理白板或文档相机，允许对空间中的内容进行协调的审阅和讨论。

为获得完全交互式的体验， (交互式显示 Surface Hub 系列产品和 Microsoft 数字白板) 便于以交互方式为就地学生和远程学生创建内容。
