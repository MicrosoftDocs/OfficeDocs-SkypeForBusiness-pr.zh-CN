# <a name="meeting-room-uses-and-layouts"></a>[会议室使用和布局](#tab/uses)

## <a name="overview"></a>概述

Microsoft Teams 会议室解决方案提供用途的灵活性，甚至在整个空间中移动的灵活性，而传统会议室不能。 例如，Teams 会议室设备，用户可以使用数字白板和智能人员跟踪摄像头增强协作，确保无论每个本地参与者在房间中在哪里都可以看到。 任何类型的空间和任何类型的会议都可以使用设备或Teams 会议室进行优化Teams 会议室解决方案。

### <a name="presentation-focused-meetings"></a>以演示文稿为中心的会议 

会议和会议是第一个存储桶，此会议通常更正式。 音频视觉是一项优先任务，并且侧重于呈现内容。 设备通常已修复，允许用户在从表格中心控制台与设备交互时保持坐在座位上。

会面和演示空间优先呈现和讨论，这意味着必须针对内容共享和远程共同创建优化布局和技术。 每个人都需要参与讨论，并且必须能够查看共享内容并与之交互。

**开会和展示****空间布局：** 表格应固定并位于房间前显示器中。 在较小的房间中，表格可以附加到墙壁上，但在较大的空间中，必须分隔它们以适应相机注意事项。

### <a name="co-creation-focused-meetings"></a>以共同创建为中心的会议

会议和共同创建会议通常不太正式，优先处理共同创建、墨迹书写和白板。 访问个人文件也是此存储桶的重点。 设备可以是移动设备，可鼓励用户起立、收集和与设备交互。

适当的空间可以无缝地连接用户和技术，并消除即使是最新手用户的摩擦。 Teams设备设计为足够用户友好，使青少年学生或新用户能够轻松导航，并且让教师或管理员能够控制和自定义首次使用。

**满足并共同创建空间布局：** 表和共同创建设备可以具有灵活的位置，并且应该容纳在表或设备中工作的人组。

### <a name="presentation-and-co-creation-focused-meetings"></a>以演示文稿和共同创建为中心的会议

请记住，这两种类型的聊天室都可以根据需要处理这两种类型的体验。 房间还可以设计，通过针对每种用途使用技术轻松适应这两者。 需要记住的重要一点就是，无论你正在配备什么空间，从专用办公室到讲课室，都需要启用Teams。

**组合演示文稿和共同创建空间布局：** 可以将其他会议场景中的用例组合在一起，以形成多用途会议室，其中以演示文稿为中心的会议和共同创建工作通常可以同时以同等的效率实现。

## <a name="technology-considerations"></a>技术注意事项

正确的音频和视频技术选择可帮助用户从会议Microsoft Teams。

### <a name="audio"></a>音频

音频注意事项对于确保远程参与者和室内参与者都可以清楚地听到和理解至关重要。 麦克风和扬声器覆盖范围必须包括房间中的每个席位，且不会留下间隔。

*每个空间的合适音频设备*

需要一个空间的音频源，以确保会议室中不存在音频问题。

将每个空间的合适麦克风和扬声器配对，可提供最佳结果，让声音清晰无干扰。

-   在共享空间中，选项会扩展，但相同的原则仍然存在：会议参与者需要被听取并清楚地听到其他人的声音。

    -   一体式音频设备是易于部署的高质量选项，适用于参与者数量有限的较小空间，它们相对靠近音频设备。

    -   在 **较大的空间中**，麦克风和扬声器分开，以确保空间内所有参与者的最大覆盖范围。 许多大中型会议室的解决方案都作为工具包的一部分提供，无需配置，只需实际安装麦克风和扬声器。

    -   在最大且最复杂的空间中，提供的解决方案允许将各种麦克风和扬声器类型一起使用，例如从大型大厅捕获观众和观众的声音，同时将呼叫带到每个人。 与在设计和部署更涉及技术的解决方案方面具有专业知识的 Microsoft 合作伙伴一起部署时，最好是实现这些解决方案。

### <a name="video"></a>视频

视频与音频同等重要。 捕获面部表情或手势等物理非语言通信与教室或会议内容一样重要，以确保最大程度的参与和理解。 会议室需要配备合适的显示大小，以便每个人在会议室或远程都可以正常体验会议。 正因如此，会议室还需要具有正确的相机覆盖范围和视区 (FOV) 才能有效捕获远程加入的所有会议室会议参与者。

*每个空间的合适摄像头*

与良好的音频一样重要，高质量的视频对于现代网络协作Teams必需。 相机技术已显著提升，现在所有空间都提供高质量的相机。

-   所有会议都应使用笔记本电脑和平板电脑等设备中包含的摄像头，以确保每个人都能看到和听到。 对于希望为个人空间提供更丰富的视频体验的用户，外部摄像头代表嵌入式设备中的一步，可用于更多的方案和配置，确保呼叫参与者共享自己的最佳质量视图。 这些设备通过每个电脑上的简单 USB 连接轻松安装。

-   在共享空间中，选项和功能会扩展。 **一键** 式音频设备（如声栏）中包含的摄像头可能会覆盖较小的房间。 这些相机通常具有宽广的视野，确保它们通过一次拍摄捕获空间中每个参与者。 其中许多相机还具有图像智能帧等功能，因此相机只显示参加会议的人，而不是空椅子或计数人员，以跟踪和了解空间利用率。

-   在 **较大的房间** 中，相机采用在不同距离的不同位置捕获参与者所需的技术。 这些摄像头可以捕获组或关注特定个人，同时保持出色的视觉清晰度。 可以仅跟踪活动发言人并仅提供其功能的设备会定期在较大的空间中使用，以便许多参与者可以在会议过程中单独受到推荐，因为参与者会贡献自己的想法。

## <a name="microsofts-vision-for-the-future-of-meeting-rooms"></a>Microsoft 对会议室未来的愿景

工作的未来将是流畅、动态的，并且由云提供动力。 Microsoft 支持会议体验，允许用户随时随地参加会议。 这些会议是混合的，这意味着你可以无缝加入它们。 使用 Microsoft 设备增强的会议是包容性的;每个语音都听到，每个人都清楚地表示。

Microsoft 将看到一个未来，其中会议是沉浸式会议，空间是围绕人们设计的。 我们的动态会议视图让每个人都能够相互连接到内容：正式演示文稿与协作聊天配对。 数字画布释放了组创造力。 无论身在何处，每个人都可以面对面连接。

对会议未来的愿景基于一个简单的信念：会议应提供的不仅仅是一次事务。 会议将为每个人设计，无论他们是在会议室还是全球。

## <a name="what-is-an-enhanced-microsoft-teams-room"></a>什么是增强型Microsoft Teams聊天室？

在 Microsoft 的混合工作区中，远程参与者和房间内参与者的需求会告知下一代房间标准。 增强聊天室是一Microsoft Teams会议室，旨在提供提升的混合会议体验，促进更自然和沉浸式的连接，同时让会议中的每个人都感觉包括、代表和高效。 聊天室遵循一组通过结合使用软件和硬件功能与Teams 会议室实现的原则。

增强型Microsoft Teams聊天室的概念围绕一组体验原则支持的两个关键目标设计。

**目标 1：参加会议的每个人都感觉包括、代表和高效。**

设计原则：

-   远程参与者可以在会议室中建立真正的状态

-   会议室参与者可以在会议中保留其个人身份

-   所有会议参与者都可以进行无边界协作。

**目标 2：连接感觉自然且沉浸式。**

设计原则：

-   体验针对人机连接进行缩放，从而启用感觉每个人都在同一房间的交互。

增强的会议室体验是所有这些组件和创意放在一起，没有定义体验的单一功能。 此外，还可以自行部署增强型Microsoft Teams 会议室中的功能和概念。

特定功能部署的示例：

-   在所有类型的会议空间中利用我们新的前行前会议室显示体验，其中Microsoft Teams会议室解决方案支持它。

-   使用传统的会议室体验实现曲线表和Microsoft Teams布局，以改进会议室中的人工连接，同时保持所有会议空间的体验连续性。

## <a name="when-should-you-deploy-this-room-type"></a>何时应部署此聊天室类型？

增强Microsoft Teams 会议室不是要取代组织内部的所有其他会议空间，而是充当对现有使用会议空间Microsoft Teams。 应将其在组织中以战略方式部署到最重视与远程会议参与者进行面对面互动的空间和位置。

Microsoft Teams 会议室通过软件更新提供以人为中心的混合会议体验。 但是，某些硬件技术和房间配置指南在优化一些增强的体验方面发挥着关键的作用。 它是这些组件的组合，构成增强型Microsoft Teams聊天室。 评估想要Microsoft Teams 会议室改进为增强型聊天室Microsoft Teams，请考虑以下事项：

-   需要进行哪些硬件升级和/或投资。

-   大多数会议室旨在最大程度地利用会议室维度和会议室容量之间的紧密关系来最大化有价值的方形资源。 增强型Microsoft Teams会议室将会议体验的优先级高于最大化正方形。 并非每个组织都可以按这种方式确定每个会议空间的优先级，因为它可能会减少他们可以使用的会议室数。

### <a name="what-is-front-row"></a>什么是前行？

为了让会议室中的人员在会议前后与远程参与者建立更好的连接并支持协作，Microsoft 引入了新的内容布局，Microsoft Teams 会议室前行。 在视觉上，此新布局将视频库移动到屏幕底部，并固定为特定比率，以保持人脸大小生动。 借助这种布局，聊天室内参与者可以在水平平面上面对面地看到远程同事，这类似于他们在同一房间中的情况。

会议内容由上下文流畅的组件（如议程、任务和笔记）环绕，这些组件可以实时更新，帮助参与者保持参与和高效。 此外，会议聊天清晰可见，以便从 Teams 会议室Teams加入 Teams 会议时，可以轻松实时查看并参与对话。 在增强Microsoft Teams聊天室中，此内容显示在宽纵横比屏幕上，以最大化会议内容的大小以及可同时查看的参与者数。

单显示器和双显示器配置都支持此布局。 虽然前行布局在增强型房间中影响特别大，但前行可用于 Teams 上运行的任何 Windows 会议室，而不考虑家具与硬件配置。

### <a name="how-do-you-co-create-content"></a>如何共同创建内容？

在Microsoft Teams 会议室，用户可以使用智能捕获功能利用传统的模拟白板。 对于所有参与者都可以一起绘制和墨迹的现代化体验，Microsoft Whiteboard可以使用触摸屏房间前显示器、配套设备（如电脑、手机或平板电脑）和 Microsoft Surface Hub。

尽管许多参与者可能会选择通过个人设备将 Teams 会议加入 Teams 呼叫来共享此类会议室中的内容，但会议室中的本地视频演示文稿也应以易于访问的方式在表格中可用。

## <a name="infrastructure"></a>基础结构 

### <a name="infrastructure-power"></a>基础结构电源

至少应在表和表上为最终用户 (和设备安装) 电源连接。 如果共同创建设备在空间中使用，则可能还需要在它们位于其 (电源) 。

### <a name="infrastructure-data"></a>基础结构数据

应同时在表 (为最终用户提供网络连接) 在 Teams 会议室 解决方案主网络的设备装载位置 (提供) 。 如果在空间中使用共同创建设备，则还需要网络连接才能连接到 Teams 会议室 解决方案。

### <a name="acoustic-wall-treatments"></a>声学墙壁处理

应对此类型的房间中至少两面墙壁进行声学处理，以防止会议期间出现回声和环境噪音问题。

### <a name="window-treatments"></a>窗口处理

来自此类房间的光线可能会对照明造成负面影响，给相机图像和房间前显示引入刺眼和阴影。 采用窗口的任何空间也应采用窗口处理，以允许在会议期间阻止外部光线。

### <a name="floor-treatments"></a>楼层处理

此类房间的楼层应经过装饰或以声学方式处理，以防止会议期间出现回声和环境噪音问题。

### <a name="lighting"></a>照明 

应提供顶灯以均匀地亮起空间，同时会议室参与者不会出现冲刷或坐在暗空间中。 应尽量减少房间前显示器、共同创建的设备或空间内的其他反射表面的直接照明，以避免引入刺眼。

房间前扬声器阵列和顶置扬声器适用于此类空间。

需要定位说话人，为所使用设备或设备的认证范围内的所有会议室参与者提供覆盖范围。

### <a name="front-of-room-display"></a>房间前显示

#### <a name="display-type"></a>显示类型

此类房间需要使用超宽纵横比、高分辨率房间前显示器，该屏幕大小适合空间的宽度和深度，以确保最大可见性和清晰度。

显示类型示例包括：

-   激光投影仪

-   LED 视频墙壁

-   特殊型的"LED"或"OLED"平面显示

#### <a name="display-functions"></a>显示函数

与Microsoft Teams 会议室解决方案一样，增强型Microsoft Teams 会议室需要使用可适应以下功能的显示器才能正常运行：

-   基于 HDMI 同步的睡眠和快速从睡眠状态唤醒。

    -   对于投影仪，必须支持自动预热和冷却期，无需用户干预。

-   显示器必须支持设计所需的相应 EDID 和Microsoft Teams 会议室分辨率。

#### <a name="display-placement"></a>显示位置

增强聊天室要求将显示屏放置在所有会议室会议参与者的前面，与表格正对。

显示器的水平中心线必须与表格位于同一中心线上。

显示器的垂直中心线不能超过 +/- 15 度偏移量，这一点与典型的会议室会议参与者在座位上座位上的眼线偏移量不能超过 15 度。

#### <a name="display-count"></a>显示计数

增强型会议室要求显示是单个宽纵横比显示，或者是显示器的混合组合，当会议室内会议参与者看到时，可以实现单一显示的效果。
