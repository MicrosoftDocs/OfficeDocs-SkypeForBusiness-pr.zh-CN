<a name="edit-guest-user-information"></a>编辑来宾用户信息
===========================

要编辑来宾帐户（例如，显示名称或个人资料照片），请访问 Azure Active Directory 门户。 有关详细信息，请参阅[了解身份和 Azure Active Directory](https://support.office.com/article/Understanding-Office-365-Identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9)。

目前，您无法从 Microsoft 团队管理中心、Microsoft 365 管理中心或 Exchange 管理中心编辑来宾信息。
