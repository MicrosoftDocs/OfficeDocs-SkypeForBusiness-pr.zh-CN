<a name="best-practices-for-organizing-teams-in-microsoft-teams"></a>在 Microsoft Teams 中组织团队的最佳做法
======================================================

团队是围绕共同的目标共同收集的人员集合。 此组人员可能位于一个部门或整个组织内。 将它们放在一起的结果就是它们的发展结果。 团队成员可以在不同的步调工作，也可以以不同的方式创建资产，但在我们的体验中，他们通常会快速协作，我们称之为 "高 velocity 团队团队"。  

在创建团队之前，请考虑目标、项目或工作项以及你的组织中的哪些人员可以通过协作来帮助交付。 一旦你确定它们，请将这些人员或组添加到团队以开始协作。 由于成员可以随着时间的推移而更改，因此最好为每个团队指定多个所有者。 有关详细信息，请参阅[管理团队](https://support.office.com/article/Teams-and-Channels-df38ae23-8f85-46d3-b071-cb11b9de5499)。

查看此简短视频，了解如何组织跨组织或单个用途的团队的一些示例：

> [!VIDEO https://youtu.be/XAowXcmQ-kA]

## <a name="add-teams-gradually"></a>逐渐添加团队

第一次推出团队时，我们建议从少量团队和团队成员开始。 开始时添加新的人员或组。 团队的强大功能是，当您添加新人员或组时，他们可以快速掌握已讨论的内容，因为无论何时加入，用户都可以使用对话和文件。 避免创建一组具有相同成员的不同团队的诱惑;而是在单个团队中创建信道。

## <a name="create-channels-to-focus-discussions"></a>创建用于重点讨论的频道

创建团队后，最好开始考虑您需要支持的各种不同项目和对话的类型。 创建初始频道，以便用户了解在何处参与和查找现有对话。 使用描述性频道名称，让人们可以轻松了解每个对话的位置。 将工具（如 OneNote、Power BI 或 Planner）作为选项卡添加到频道，以便成员拥有他们所需的一切，直接在频道中使用。 您还可以将常用网页作为选项卡添加到频道。 

了解有关最终用户的这些快速提示的详细信息，请查看有关[团队工作](https://support.office.com/article/teams-and-channels-df38ae23-8f85-46d3-b071-cb11b9de5499#ID0EAABAAA=Work_in_teams)的详细信息。 

最佳做法：使用较大成员集和更多频道创建团队。 最大限度地减少需要人员参与的团队数量。 团队中的频道应被视为主题或工作流，以帮助团队组织其工作以实现其联合目标。 不应创建特定数量的频道。 每个团队都应根据其工作、优先级和样式来手工创建频道。 

将标准频道用于团队中的每个人都可参与的对话。 当你需要具有选择成员组的重点协作空间时，请利用[团队中的专用频道](private-channels.md)。

较大的组织可能希望创建团队作为 "模板"，以便对其捕获的有关特定工作类型的信息进行标准化。 这对于战略客户管理、课堂管理、卫生保健方案、理赔管理、事件管理和适用于特定行业的其他方案非常有用。 若要了解详细信息，请查看[中小型企业的](smb-templates.md)[团队模板](get-started-with-teams-templates.md)和团队模板入门。

## <a name="use-the-general-channel"></a>使用常规频道

默认情况下，创建团队时，将为你创建**常规频道**。 此频道有许多有用的用途：

- 使用它可共享团队要实现的功能的概述，如项目章程或团队中的人员。
- 将其用于新团队成员的加入以及新团队成员将发现有用的其他高级信息。
- 将其用于通知，或配置 SharePoint 新闻连接器以将您的新式状态报告发布到此频道。  
- 对于新的或单个用途的团队，你可以在决定团队如何最大程度地支持你的目标时，在开始时是唯一的频道。

您不能删除、重命名或取消收藏 "常规频道"。 频道按字母顺序显示（"常规" 通道位于顶部）。 在具有多个频道的团队中，使用 "**隐藏**" 或 "**显示**" 显示最常使用的频道。

若要了解详细信息，请在 "[团队和频道](https://support.office.com/article/teams-and-channels-df38ae23-8f85-46d3-b071-cb11b9de5499#ID0EAABAAA=Work_in_channels)" 页面上的 "频道" 选项卡**中**peruse。



## <a name="consider-setting-up-moderation-in-your-channels"></a>考虑在频道中设置裁决

团队所有者可以为频道启用裁决，以控制哪些人可以开始新文章以及答复该频道中的帖子。 设置裁决后，您可以选择一个或多个团队成员进行审阅。 （默认情况下，团队所有者是 "审阅者"。）有关详细信息，请参阅[在 Microsoft 团队中设置和管理通道裁决](manage-channel-moderation-in-teams.md)。
