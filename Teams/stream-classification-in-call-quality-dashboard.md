## <a name="unclassified-streams"></a><span data-ttu-id="3b97b-238">未经分类的流</span><span class="sxs-lookup"><span data-stu-id="3b97b-238">Unclassified Streams</span></span>

<span data-ttu-id="3b97b-239">在 CQD 中，当交互式连接建立 (ICE) 连接失败或未报告计算流分类所需的所有指标时，流被标记为未 _分类_ 。</span><span class="sxs-lookup"><span data-stu-id="3b97b-239">In CQD, a stream is marked _Unclassified_ when Interactive Connectivity Establishment (ICE) connectivity fails or when all the metrics required to compute the stream classification are not reported.</span></span>

<span data-ttu-id="3b97b-240">要检查 ICE 连接故障，请检查 "First Connectivity Ice" 和 "Second Connection Ice" 维度是否为 "FAILED" 值。</span><span class="sxs-lookup"><span data-stu-id="3b97b-240">To check for ICE connectivity failures, examine the dimensions "First Connectivity Ice" and "Second Connectivity Ice" for a "FAILED" value.</span></span> <span data-ttu-id="3b97b-241">如果任一值指示失败，则流将标记为未 _分类_。</span><span class="sxs-lookup"><span data-stu-id="3b97b-241">If either value indicates a failure, the stream is marked as _Unclassified_.</span></span>

<span data-ttu-id="3b97b-242">如果未 _分类_ 流的 ICE 连接成功，则该流可能被视为未 _分类_ ，因为未报告密钥流度量值。</span><span class="sxs-lookup"><span data-stu-id="3b97b-242">If ICE connectivity succeeded for an _Unclassified_ stream, the stream is likely considered _Unclassified_ because key stream metrics were not reported.</span></span> <span data-ttu-id="3b97b-243">这些指标可能不会被报告的原因有几个：</span><span class="sxs-lookup"><span data-stu-id="3b97b-243">There are a few reasons these metrics may not be reported:</span></span>
